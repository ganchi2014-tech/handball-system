## プロジェクト概要

**目的：** 高校ハンドボール部（全国レベル）のバックプレーヤーが試合後に自分で振り返り、次の行動を自分で決める「自己問答システム」。

**コーチ（東岸さん）の哲学：**
- 「この世にある全てのハンドボールを私は認めている。相性や向き不向きがあるだけ」
- じゃんけん構造：絶対最強の戦術は存在しない
- OF↔DFは相互依存：「DFに合うOF、それに合うDF……」という連鎖
- バイアスなき辞書：コーチが言うとバイアスがかかるので、AIが全羅列する

**対象選手：** バックプレーヤー（ウィングではない）

1. **ハンドボール辞書**：全戦術・技術・状況を羅列→パターン分け→じゃんけん関係

## コーチ補足知識（必ずコードに反映すること）

### GK
- **3方向の法則**：左/中/右（常に2/3が閉じていて1/3が開いている）
- **波型GK**：コース予測で先に動く → 早打ちが有効（GKが動く前に打つ）
- **スポット型GK**：動かずにコースを絞る → 引きつけてから最後に角度で打つ
- GK×DF連携：DFがコースを絞ってGKの仕事を減らす

### ポストの4プレー
1. **回転プレー**：DFに背中を向けた状態から体を回転させてシュート
2. **フェイクプレー**：受けた瞬間に逆方向フェイクでDFを外す
3. **フラッシュプレー**：6mライン付近に瞬間的に飛び込んでパスを受ける
4. **スクリーンプレー**：ポストがDFに壁を作り、バックプレーヤーの突破を助ける

### コンタクト技術（山側/谷側）
- DFが斜めに構えたとき「山側（高い方）」を越えさせないのが原則
- 「谷側（低い方）」は前後ステップ（横ではなく前後）で押し出す
- コンタクト中も常に次のポジション（勝ち位置）を意識

### ポストDF V字位置取り
- ポストOFとゴールを結ぶ線上（V字の頂点）に立つ
- ポストOFより先にボールを受ける位置を取る（先読み）

## 【じゃんけんマップ — 全カテゴリ横断統合】

### 核心原則
- 絶対最強の戦術は存在しない／全てにカウンターがある／「先に読んだ方が選択肢を持てる」

### OF↔DFの基本サイクル
```
クローズDF（内側に立つ）→ OFに外パス・外シュートの選択肢 → ワイドOF優位
→ DFがワイドに対応 → ラインが広がる → 内側にスペース発生 → クローズOF（内突破）優位
→ クローズDFに戻る
```

### 数的状況サイクル
```
マンツーマンDF → スクリーンでマーク剥がし → フリー確保
→ ゾーンDF（スペース消し） → 外シュートで引き出す → ゾーン間にスペース → 1on1突破
→ マンツーマンに戻る
```

### GKサイクル
```
波型GK（コース予測で動く） → 早打ちでGKが動く前に打つ
→ スポット型GK（動かずコース絞り） → 引きつけて最後に角度で打つ → 波型GKに戻る
※常に「左・中・右」の2/3は閉じている。開いている1/3を読む勝負
```

### 身体特性サイクル
```
高身長ブロックDF → 低弾道早打ち（踏み切らない）で股下/低コーナー
→ GKが下に合わせる → 高跳躍ジャンプシュートで上部コーナー → 高身長DFのブロックに戻る
```

### ゲームクローズサイクル
```
大量リード → ポゼッション戦術 → パッシブプレー警告（強制攻撃）
→ 攻撃ミス → カウンター機会 → 点差縮まる → 負けチームが7on6
→ 空ゴールリスク → DFが引く → 7on6でピボットがフリー
→ ゾーン守備でコンパクト化 → 得点率低下 → 時間切れ
```

### DF陣形サイクル
```
0-6（外シュート多発）→ 1-5（前衛プレッシャー）→ 前衛の裏をピボットで使う → 0-6に戻る
マンツーマン → スクリーンで剥がし → スイッチ → ミスマッチ → ゾーン → 外シュート → マンツーマンに戻る
```

### ポストプレーサイクル
```
ポストV字位置 → ポストOFが動く → DFが追う → バックプレーヤー突破
→ DFがバックに集中 → ポストへのパスコース開く → ポストのV字位置に戻る
「そこにいるだけでDFが迷う」= 存在自体が脅威
```

### スクリーンサイクル
```
スクリーン利用（カットイン）→ DFがスイッチ → ミスマッチ → スクリーナーをポップアウト
→ 外シュート機会 → DFがポップを追う → カットインのフリー復活 → スクリーン利用に戻る
```

## 【基礎技術：パスキャッチ・シュート・コンタクト】

### パスキャッチ
- ボールを迎えに行く（待つのではなく動いて受ける）
- DFとの間に体を入れてインターセプトを防ぐ
- 受けながら次の選択肢を決める（視野確保）
- 受ける前に：DFの位置・動き・スペースを確認
- じゃんけん：DFがインターセプト狙い→フェイクパスで動かしてから本パス→DFがフェイクに慣れる→タイミングが遅れてインターセプト可能に戻る

### シュートの種類
| シュート | 特徴 |
|---|---|
| ジャンプシュート | 踏み切って最高点で打つ。滞空時間でGKを惑わす |
| 早打ち（ノージャンプ） | 踏み切らずに打つ。GKの動作前に放てる |
| ダイブシュート | 体を投げ出して6mライン内側から打つ |
| フォールシュート | 接触を受けながら体を傾けて打つ |
| ループシュート | GKの上を越えて放物線でゴールへ |
| リバウンドシュート | GKやDFに当たって跳ね返りを詰める |
| レグシュート | 自分の股の間からバックハンドで打つ（奇襲） |

### GK対策
| GKタイプ | 特徴 | 有効な打ち方 |
|---|---|---|
| 波型 | コースを読んで先に動く | 早打ち（GKが動く前に放つ） |
| スポット型 | 動かずコースを絞る | 引きつけてから最後に角度で変える |

**波型GKへの早打ちの原理（コーチ直伝）**：GKが「シューターの最高到達点に先取りしにいく」イメージで動く場合、早打ち（最高点に達する前に打つ）はGKの意識外のシュートになる。GKは最高点を想定して動いているため、それより低いタイミングへの対応が遅れる。

GKの法則：左/中/右の常に2/3が閉、1/3が開。その「開いている1/3」を見つけて打つ。

### コンタクト技術（山側/谷側）— コーチ直伝
- **山側**：DFの体の高い方。越えられると完全に背後を取られる
- **谷側**：DFの体の低い方。前後ステップで対応できる
- OFの戦略：山側越えのフェイクでDFが傾いたところで谷側へ切り返す
- DFの原則：山側を越えさせないことが最優先。谷側は前後ステップで押し出す
- **勝ち位置**：ボールを受ける前に「相手DFより先に有利なポジション」を取ること

## 【個人対決：1:1・2:2・スクリーン・勝ち位置】

### 1on1の選択肢
| DFの状況 | OFの推奨行動 |
|---|---|
| クローズDF（内側） | 外へのフェイントから外シュート、または内突破 |
| ワイドDF（外側） | 内突破またはピボットへのパス |
| DFが前傾姿勢 | バックステップしてシュート |
| DFが後傾姿勢 | 前への加速でドリブル突破 |

じゃんけん：ドリブル突破→DFが縦を切る→フェイクから外シュート→DFが外を警戒して距離を取る→突破スペース発生→ドリブル突破に戻る

### 2on2のパターン
| パターン | 説明 |
|---|---|
| ピック＆ロール | スクリーンしてからゴール方向に転身してパスを受ける |
| ピック＆ポップ | スクリーンしてから外に開いてシュートチャンス |
| クロス | 2人が交差することでDFのマーク交換を強制 |

### 勝ち位置
ボールを受ける前に「相手DFより先に有利なポジション」を取ること。勝ち位置にいれば受けた瞬間にすでに選択肢がある。

## 【数的状況・速攻・バックチェック】

### 数的有利（OF：多い側）
- 2on1：DFがどちらかに寄った瞬間に逆へパス。DFが中間なら近い方からシュート
- 3on2：DF2人の外側を使い、引き伸ばして中央スペースを作る
- 数的有利は即座に活かす（時間をかけるとDFが戻る）

### 5on6守備（数的不利）
- DFラインを引いてコンパクトゾーン
- 外シュートはGKに任せる
- 早い攻撃は禁物（カウンターリスク）
- 2分間を0点で凌ぐのが目標

### 速攻
| 種類 | 人数 | 説明 |
|---|---|---|
| 1stウェーブ | 1〜2人 | GKスロー直後の最速カウンター |
| 2ndウェーブ | 3〜4人 | 1stが止められた後に合流 |
| セカンドチャンス速攻 | 全員 | DFが整い切る前の全員参加 |

### バックチェック
- シュートを打った瞬間から始まる（止まらない）
- 最低2名が即時帰陣
- よくあるミス：シュートの結果を見てから動く（→遅い）

## 【認知・判断・行動・声掛け】

### 認知→判断→行動の3段構造
①認知：DFの位置/重心/目線、GKの空きコース、スペース、味方の位置
②判断：オートマティズム（約束事）で速くする。優先順位：ピボットフリー→走り込む味方→1on1→キープ
③行動：技術の自動化（考えなくてもできる）

### エコロジカルアプローチ
- 練習は試合に近い状況で行う
- DFがいない練習は試合に直結しない
- クローズDFを見たら自然に外が見える→認識したら外へ（アフォーダンス）

### 声掛けの優先順位（バックプレーヤー）
1. DFの位置報告（「クローズ！」「ワイド！」）
2. フリーの選手への呼びかけ（「来い！」）
3. DFのマーク確認（「俺が見る！」）
4. GKへの指示（「右！」「低い！」）

## 【GK・DF陣形】

### GK（コーチ補足統合）
- 3方向の法則：左/中/右（常に2/3が閉、1/3が開）
- 波型GK：コース予測→早打ちが有効
- スポット型GK：コース絞り→引きつけてから角度変化が有効
- GK×DFの連携：DFがコースを絞る→GKの仕事が減る

### DF陣形

| 陣形 | 長所 | 弱点 | 崩し方 |
|---|---|---|---|
| 0-6 | ゴール前密集、体力消耗少 | 外シュートに弱い、前線プレッシャー薄い | 遠距離外シュートで引き出す |
| 1-5 | 外シュートに前衛プレッシャー | 前衛の裏にスペース、前衛の体力消耗 | 前衛を引き付けてピボットパス |
| 2-4 | 前線2人でプレッシャー | 後衛4人でゴール前が薄い | 横幅を使う |
| 3-2-1 | 縦のプレッシャーが強い | 横の連係が難しい | 横幅展開 |
| 3-3 | 前後バランス | 前後分断しやすい | 後衛3人の横幅が薄い |

DF選択フレーム：外シュート多い→1-5 or 2-4 / ピボットが強い→0-6 / 速攻出したい→3-2-1 / エースを止めたい→マンツーマン指定

## 【ポストプレー（OF・DF）】

### ポストOFの核心
「そこにいるだけでDFが迷う」= 存在自体が脅威

### ポストの4プレー（コーチ直伝）
1. **回転プレー**：DFに背中を向けた状態からパスを受け、体を回転してシュート
2. **フェイクプレー**：パスを受けた瞬間に逆方向フェイク→DFが反応したところで本来の方向へシュート
3. **フラッシュプレー**：6mライン付近に瞬間的に飛び込んでパスを受ける。DFが油断した瞬間に
4. **スクリーンプレー**：ポストがDFに壁を作り、バックプレーヤーの突破を助ける。ロールアウトでフリーになる可能性も

### ポストOFのじゃんけん
```
ポストが6mラインに構える → DFがポストを警戒 → バックプレーヤーのスペース発生
→ DFがバックに集中 → ポストへのパスコース開く → ポスト経由シュート
→ DFがポスト警戒に戻る
```

### ポストDFのV字位置取り（コーチ直伝）
- ポストOFとゴールを結ぶ線上（V字の頂点）に立つ
- ポストOFより先にボールを受ける位置を取る（先読み）
- ポストOFが動いたらDFが追う→別スペース発生→声かけで補う

## 【ゲームマネジメント・ファウル戦略】

### パッシブプレー回避
- 審判が警告シグナル（手上げ）後、6回以内にシュートを打つ
- 常にシュートの脅威を見せながらボールを回す
- DFを動かしていること（静止したボール回しはNG）

### ゲームクローズ（タイミングマトリクス）
| リード差 | 残り時間 | 戦術 |
|---|---|---|
| 5点以上 | 5分以内 | 徹底したボールキープ・タイム使い |
| 3-4点 | 5分以内 | バランス維持・確実なシュート |
| 1-2点 | 2分以内 | セーフシュートのみ・速攻厳禁 |

### ファウル大原則（コーチ直伝）

> **「退場と天秤にかけて上回るものは基本的にない。いついかなるときも退場はしてはいけない。」**

「ファウルしてでも止める」「速攻を退場覚悟で止める」は誤った判断。

### ファウル損得計算
| 状況 | ファウルする | ファウルしない |
|---|---|---|
| 外シュートのみの場面 | ✗（GKに任せる） | ○ |
| **退場リスクがある場面** | **✗（絶対しない）** | **○** |
| フリースローを与える位置・退場リスクなし | △（状況次第） | ○ |
| 外シュートのみの場面 | ✗（GKに任せる） | ○ |
| 終盤・リード中 | ✗ | ○ |
| 終盤・ビハインド | ○ | ✗ |

### タイムアウト判断
- 使う場面：相手に3連続失点、相手のモメンタムが明確、重要セットプレー前
- 使わない場面：自チームのモメンタムが来ているとき、相手DFが崩れているとき

### 7on6（GK外し）タイミング
| ビハインド | 残り時間 | 推奨度 |
|---|---|---|
| 2点差 | 残り2分 | ◎ 定番 |
| 3点差 | 残り4分 | ○ 状況次第 |
| 4点差 | 残り7分 | △ 早めの博打 |

| 高スピード | カウンター・エクスプロージョン突破 | スタミナ消耗 |
| 高パワー | 6m付近シールド・押し込み・ファウル誘発 | フェイント反応遅延 |
| 左利き | 非対称プレッシャー・逆スピンパス | 左サイドからの外角が窮屈 |
| 高ジャンプ | 最高点シュート・空中フェイント | スタミナ消耗 |

**最難対応コンビ**：高ジャンプ×左利き（高さと角度が両方非標準）

**左右混在の非対称プレッシャー**：左利きと右利きが交互に攻撃→DFが毎回ブロック方向を変えなければならない

### 選手タイプ別マトリクス
| 特性組み合わせ | 推奨ポジション | 推奨戦術 |
|---|---|---|
| 高身長×高ジャンプ×右利き | バックライト/センター | ジャンプシュート・上部コーナー |
| 低身長×高スピード×左利き | バックセンター | 内切り突破・股下シュート |
| 高パワー×低スピード | ピボット | ブロック・ファウル誘発 |
| 高判断速度×両利き | センタープレーヤー | ゲームメイク |

### スカウティング項目
**攻撃面**：シュート成功率（ポジション別）、エースのシュートコースのクセ、ピボットへのパス頻度、速攻の発動条件、セットプレーのサインパターン
**守備面**：ゾーンかマンツーかハイブリッドか、プレスのタイミング、GKの弱点、DF選手のファウル傾向

## 【試合文脈による戦術変化】

### スコア差別戦術
| 状況 | 方針 |
|---|---|
| 大量リード（5点以上） | ポゼッション・主力温存・ファウル管理 |
| 小差リード | バランス維持・確実なシュート |
| 同点 | 先取点で心理的優位・ハイテンポ |
| 小差負け | リスクを取る・速攻徹底・7on6準備 |
| 大量負け | ギャンブルモード・モメンタム連続得点 |

### 時間帯別戦術
- **前半序盤（0-10分）**：偵察フェーズ。相手DFシステムを確認
- **前半終盤（25-30分）**：ハーフタイム前の1点は心理的影響大→積極攻撃
- **後半中盤（40-50分）**：疲労差の活用。足が止まったDF選手へ集中
- **残り5分**：1点の価値最大化。タイムアウト使い時が最重要
- **ラスト2分**：負け→7on6最終手段 / 勝ち→速攻せず時間を使う

### 退場者が出た場合
| 状況 | 戦術 |
|---|---|
| 自チームが2分退場（5on6） | コンパクトゾーン・早い攻撃禁物 |
| 相手が2分退場（6on5） | 積極攻撃・退場選手が戻る前に |
| 複数名退場 | 完全守備シフト・7on6は絶対しない |

## 【セットプレー・サインプレー・スクリーン技術】

### フリースロー時の攻撃パターン
**クイックスロー系**：笛の直後に素早くスロー。スローワー突破、ピボットへのダイレクトパス
**セットオフェンス系**：クロスランニング、ダブルスクリーン、バックプレーヤーミドルシュート

### サインの種類
| 媒体 | 具体例 |
|---|---|
| 言語 | 「A！」「2！」等 |
| 手のサイン | オープン/クローズハンド・指の数 |
| 立ち位置 | センター寄りかサイド寄りか |
| 動作 | ドリブル開始でパターンA |

### スクリーンの種類と合法要件
**スクリーンの本質（コーチ直伝）**：「自分の体を壁にして相手の進行を妨げること」がメイン。相手の押すパワーを0にする押し返しはOK、押し返しすぎはNG。足を大きく開く・腕や肘で押すのは基本NG。審判によって激しいやり合いを認める人とすぐファウルを吹く人がいる→試合序盤に傾向を把握する。

合法要件：スクリーナーは静止・DFの視野内から入る・手足は密着・自ら接触を引き起こさない

| 種類 | 説明 |
|---|---|
| ステーショナリースクリーン | 完全静止でDFの進路を塞ぐ |
| クロススクリーン | 2名が交差しながら互いにスクリーン |
| ダブルスクリーン | 2名が並んでエースのために壁を作る |
| スクリーン＆ロール | スクリーン後にゴール方向に転身してパスを受ける |

### DFのスクリーン対策
| 対策 | 説明 | 有効場面 |
|---|---|---|
| ファイトオーバー | スクリーナーの前を通ってマーク維持 | エースが外シュートを打てる位置 |
| ゴーアンダー | スクリーナーの後ろを通ってマークに戻る | エースが突破優先の場合 |
| スイッチ | DFが互いにマーク交換して無効化 | 弱点：ミスマッチが生じる |
| ヘルプアンドリカバー | 3人目DFが一時カバー、元のDFが戻る | 連携が難しい上級技 |
