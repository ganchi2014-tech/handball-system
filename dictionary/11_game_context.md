## じゃんけん（試合文脈）

```
大量リードのポゼッション戦術
  → パッシブプレー警告 → 攻撃強制
  → ミスのリスク上昇 → カウンター機会増加
  → 点差縮まる（元に戻る）

7on6攻撃（GK外し）
  → 得点機会+1
  → ボールロスト → 空ゴールへのロングシュート失点
  → リスク回避のセーフプレー → 得点力不足
  → 7on6の発動（元に戻る）

モメンタム維持（選手交代しない）
  → 疲労蓄積 → 後半失速
  → 相手修正完了+自チーム疲弊 → 逆転
  → 早めの選手交代 → 流れが切れ相手に修正時間
  → モメンタム維持（元に戻る）
```
