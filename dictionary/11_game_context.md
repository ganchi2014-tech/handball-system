# 試合文脈による戦術変化

---

## スコア差による戦術変化

| スコア状況 | 基本方針 | 具体的戦術 |
|---|---|---|
| **大量リード（5点以上）** | ポゼッション・温存・リスク回避 | ボールキープ・主力休ませる・ファウル管理 |
| **小差リード（1〜4点）** | バランス維持 | 確実なシュートのみ・DFラインを保つ |
| **同点** | 先取点で心理的優位 | ハイテンポ・GKビッグセーブ→即カウンター |
| **小差負け（1〜4点）** | リスクを取る | 速攻徹底・7on6準備・タイムアウト活用 |
| **大量負け（5点以上）** | ギャンブルモード | 7on6早期発動・モメンタム連続得点で相手を揺さぶる |

---

## 時間帯による戦術変化

### 前半序盤（0〜10分）
- 相手のDF強度・システムを確認する偵察フェーズ
- エースを温存しつつ攻めてDFの弱点を探る
- 確実な選択肢を優先（スロースタートを避ける）

### 前半中盤（10〜20分）
- 最初の10分で得た情報をもとに攻め方を確立
- ピボットの活用を増やす（DFが疲れ始める）

### 前半終盤（25〜30分）
- **ハーフタイム前の1点は心理的影響が大きい** → 積極攻撃
- 相手のタイムアウト残数を確認

### 後半序盤（30〜40分）
- ハーフタイム修正の実行テスト
- 相手の修正を読んでカウンター修正

### 後半中盤（40〜50分）
- 疲労差の活用：足が止まってきたDF選手への集中攻撃
- 選手交代で自チームの疲労をリセット

### 後半終盤（残り5分）
- 1点の価値が最大化
- タイムアウトの使い時判断が最重要
- 7on6発動タイミングの判断

### ラスト2分
- 負けているチーム：7on6最終手段
- パッシブプレー警告回避：ボールを早く回す
- 勝っているチーム：速攻しない・時間を使う

### ラスト30秒
- 負けているチーム：無理でもシュートを打ち、リバウンドを狙う
- 勝っているチーム：ファウルさせながらボールキープ

## 退場者が出た場合

| 状況 | 戦術 |
|---|---|
| 自チームが2分間退場（5on6） | DFラインを引いてコンパクトゾーン・早い攻撃禁止 |
| 相手が2分間退場（6on5） | 積極攻撃・素早い仕掛け（退場選手が戻る前に）|
| 自チーム主力が退場 | 代替選手投入・残選手でシステム再構築 |
| 複数名退場 | 完全守備シフト・7on6は絶対しない |

---

## モメンタムの読み方

### 自チームのモメンタムが来ているサイン
- 3連続得点以上
- GKのビッグセーブ連続
- 相手のミス増加
- スタンドの盛り上がり

### 相手チームのモメンタムが来ているサイン
- 3連続失点
- 自チームのミス増加
- 相手の攻撃テンポが上がっている

## じゃんけん（試合文脈）

```
大量リードのポゼッション戦術
  → パッシブプレー警告 → 攻撃強制
  → ミスのリスク上昇 → カウンター機会増加
  → 点差縮まる（元に戻る）

7on6攻撃（GK外し）
  → 得点機会+1
  → ボールロスト → 空ゴールへのロングシュート失点
  → リスク回避のセーフプレー → 得点力不足
  → 7on6の発動（元に戻る）

モメンタム維持（選手交代しない）
  → 疲労蓄積 → 後半失速
  → 相手修正完了+自チーム疲弊 → 逆転
  → 早めの選手交代 → 流れが切れ相手に修正時間
  → モメンタム維持（元に戻る）
```
