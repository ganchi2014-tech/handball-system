# クローズ/ワイド・セットOF陣形・ウィングプレー

---

## クローズ/ワイド フレームワーク（核心）

バックプレーヤーの全判断はまずここから始まる。

### 定義
- **クローズ**：DFが内側（ゴール寄り）に立っている状態
- **ワイド**：DFが外側（サイドライン寄り）に立っている状態

### OFの選択肢

| DFの状態 | OFの有利な選択 | 理由 |
|---|---|---|
| クローズ（内側） | 外パス・外シュート・サイド展開 | 内側が塞がれているので外が空く |
| ワイド（外側） | 内突破・ドリブル突破・ピボットパス | 外が塞がれているので内が空く |

### じゃんけん
```
クローズDF
  → OFが外に展開 → 外シュート・サイドパス
  → DFがワイドに対応
  → 内側にスペース → クローズOF（内突破）
  → DFがクローズに戻る
```

## セットOF陣形

### 基本陣形

**3-3陣形**（センター3＋ウィング2＋ピボット1）
- バックプレーヤー3人が横に並ぶ
- ピボット1人が6mライン付近
- ウィング2人がサイド
- じゃんけん：0-6DFには外シュートで対抗 / マンツーマンにはスクリーンで対抗

**2-1-3陣形**（ピボットを高い位置に置く）
- ピボットを9mライン付近に引き出す
- DFの陣形を崩してスペースを作る
- ピボットが引き出したスペースにバックプレーヤーが切り込む

**ポストオフェンス中心陣形**
- ピボットを6mラインに固定
- バックプレーヤー全員がピボットを経由したプレーを第一選択
- DFがピボットに集中 → バックの突破スペース確保

### セットOFのサインプレー
- Aサイン：右展開
- Bサイン：左展開  
- Cサイン：ピボット経由
- ポジション別の動き出しを事前に決めておく

## ウィングプレー（バックプレーヤー視点）

### クロスアタック（バックプレーヤーによるウィング攻略）
- バックプレーヤーがサイドに流れてウィングと交差する
- DFのマーク交換を強制する
- 交差後にどちらがシュートを打つかをサインで決める

### ウィングへの供給パターン
- **外パス**：クローズDFの外からウィングへ
- **バックドア**：ウィングがゴール裏に回り込んでパスを受ける
- **ロブパス**：DFの頭上を越えてウィングへ（高身長OFに有効）

### コーナーシュートのじゃんけん
```
ウィングがコーナーからシュート
  → GKがニアポストを締める
  → ファーコーナーへのシュート（角度シュート）
  → GKがファーをケア
  → ニアへのシュートまたはロールインシュート
  → GKがニアをケアに戻る
```

## オフボール（ボールを持っていないときの動き）

### バックプレーヤーのオフボール原則
1. **常に動く**：止まっているとDFがマークしやすい
2. **スペースを作る動き**：自分が動くことで別の選手のスペースを作る
3. **受ける準備**：いつでもパスを受けられる位置に入り続ける
4. **DFを引き付ける**：ボールと関係ない方向のDFを動かす

### オフボールの種類
| 動き | 目的 |
|---|---|
| カットイン | ゴール方向へ切り込んでパスを受ける |
| ポップアウト | 外に開いてシュートチャンスを作る |
| バックドアカット | DFの背後を抜けてパスを受ける |
| スクリーンムーブ | スクリーナーのために動く |
| ダミーラン | 囮として動いて別のスペースを作る |
