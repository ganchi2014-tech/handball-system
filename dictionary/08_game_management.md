# ゲームマネジメント・ファウル戦略

---

## パッシブプレー

### ルールの基本
- 審判が「警告シグナル（手を上げる）」を出したら6回以内にシュートを打たなければならない
- 違反したら相手ボールのフリースロー
- 警告シグナルは「明らかに攻撃意図がない」と審判が判断したときに出る

### 合法的な時間の使い方
- ボール回しは合法（DFを動かしながら回す）
- ドリブル突破の試みを見せながら回す
- ピボットへの当てパスを繰り返す（毎回シュートの脅威を見せる）
- シュートフェイクを入れてGKを動かす

### パッシブプレー回避チェックリスト
1. 常にシュートの脅威を見せているか
2. DFを動かしているか（静止したボール回しはNG）
3. ピボットへの当てパスを使っているか
4. タイムアウトを取るタイミングはいつか（警告後は使えない）

## ゲームクローズ（リードを守る）

### タイミングマトリクス

| リード差 | 残り時間 | 戦術 |
|---|---|---|
| 5点以上 | 10分以上 | 通常のポゼッション |
| 5点以上 | 5分以内 | 徹底したボールキープ・タイム使い |
| 3-4点 | 5分以内 | バランス維持・確実なシュート選択 |
| 1-2点 | 2分以内 | セーフシュートのみ・速攻厳禁 |
| 同点 | 終盤 | 先取点で心理的優位 |

### ボールキープ技術
- ピボットでボールを保持させる（ファウルを誘う）
- コーナー付近でのキープ
- GKのスローイン前に時間を使う
- ファウルされたらすぐに立つ（プレーを続けるふり）

## ファウル戦略

### 損得計算マトリクス

| 状況 | ファウルする | ファウルしない |
|---|---|---|
| 相手が確実に1点を取る場面、退場リスクなし | ○（ファウルで止める） | ✗ |
| 相手が確実に1点を取る場面、退場リスクあり（累積） | △（状況次第） | ○ |
| 速攻で2on1の状況 | ○（前の選手をファウル） | ✗ |
| 外シュートのみの場面 | ✗（GKに任せる） | ○ |
| 終盤・リード中 | ✗（時間を使わせる） | ○ |
| 終盤・ビハインド | ○（止めて時間を作る） | ✗ |

### 退場累積管理
- 試合前半でのファウル累積は後半のリスクになる
- 主力選手の退場は戦術的に大きなマイナス
- 「ファウルしてでも止める」判断は退場リスクとのバランスで決める
- 2分間退場中は5on6→コンパクトゾーンで凌ぐ

### 戦略的ファウル
- **速攻をファウルで止める**：確実な失点を2分退場リスクと天秤にかける
- **フリースローを与える位置の選択**：ゴールから遠い位置でファウルする
- **ファウルのタイミング**：GK位置が良ければあえてシュートを打たせる選択も

## タイムアウト判断

### 使うべき場面
- 相手に3連続得点されたとき（モメンタムを切る）
- 相手のモメンタムが明確に出ているとき
- 重要なセットプレーの前（作戦確認）
- 選手が混乱しているとき

### 使ってはいけない場面
- 自チームのモメンタムが来ているとき（流れを切ってしまう）
- 試合序盤（後半に温存する）
- 相手DFが崩れているとき

### 「5秒ルール」的な原則
- タイムアウト要求から5秒以内に出なければならない
- タイムアウト後の最初のプレーは準備したセットプレーを即実行

## 7on6（GK外し）

### 発動タイミング
| ビハインド | 残り時間 | 推奨度 |
|---|---|---|
| 2点差 | 残り2分 | ◎ 定番 |
| 3点差 | 残り4分 | ○ 状況次第 |
| 4点差 | 残り7分 | △ 早めの博打 |
| 1点差 | 残り30秒 | ○ 最終手段 |

### 7on6の攻撃原則
- 数的有利を活かして素早くシュートまで持ち込む
- ピボットへの早いパスが最有効
- ボールロストは空ゴール失点 → 慎重な判断が必要

### 相手の7on6対策（DFとして）
- 空ゴールへのロングシュートを狙う
- ボールロストを誘うプレッシャー
- コンパクトゾーンで5on6を守り切る

## モメンタム管理

### モメンタムが来ているサイン
- 連続得点（3点以上）
- GKのビッグセーブ連続
- 相手のミスが増えている
- スタンドの雰囲気

### モメンタムを維持する
- 選手交代を最小限にして流れを切らない
- タイムアウトを取らない（相手に修正の機会を与えない）
- 高いプレッシャーを継続する

### モメンタムを取り戻す
- 即タイムアウト（最優先）
- 選手交代（空気を変える）
- DF陣形変更（パターンを崩す）
- GKへの声かけ（精神的サポート）
