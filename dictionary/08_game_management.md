# ゲームマネジメント・ファウル戦略

---

## パッシブプレー

### ルールの基本
- 審判が「警告シグナル（手を上げる）」を出したら6回以内にシュートを打たなければならない
- 違反したら相手ボールのフリースロー
- 警告シグナルは「明らかに攻撃意図がない」と審判が判断したときに出る

### 合法的な時間の使い方
- ボール回しは合法（DFを動かしながら回す）
- ドリブル突破の試みを見せながら回す
- ピボットへの当てパスを繰り返す（毎回シュートの脅威を見せる）
- シュートフェイクを入れてGKを動かす

### パッシブプレー回避チェックリスト
1. 常にシュートの脅威を見せているか
2. DFを動かしているか（静止したボール回しはNG）
3. ピボットへの当てパスを使っているか
4. タイムアウトを取るタイミングはいつか（警告後は使えない）

## ゲームクローズ（リードを守る）

### タイミングマトリクス

| リード差 | 残り時間 | 戦術 |
|---|---|---|
| 5点以上 | 10分以上 | 通常のポゼッション |
| 5点以上 | 5分以内 | 徹底したボールキープ・タイム使い |
| 3-4点 | 5分以内 | バランス維持・確実なシュート選択 |
| 1-2点 | 2分以内 | セーフシュートのみ・速攻厳禁 |
| 同点 | 終盤 | 先取点で心理的優位 |

### ボールキープ技術
- ピボットでボールを保持させる（ファウルを誘う）
- コーナー付近でのキープ
- GKのスローイン前に時間を使う
- ファウルされたらすぐに立つ（プレーを続けるふり）

## ファウル戦略

### ファウル（罰則）の種類と重大度（2025年競技規則 第16条 + コーチ直伝）

ファウルは一括りにしてはいけない。「何が与えられるか」で意味が全く違う。

---

#### 【ボールの扱いによる結果】

**① フリースロー（第13条） ◎ ナイス**
- 相手に9m付近以遠からの攻撃権を与える
- GKとDFが整う時間が生まれる → 守りやすい状況
- 戦術的に与えてよいファウルの結果
- 与える位置が重要：ゴールから遠い位置ほど良い
- 主な発生原因：軽微な接触・パッシブプレー・段階的違反

**② ペナルティスロー（7mスロー）（第14条） ✗ ダメ**
- ゴール正面7mからのフリーシュート。GKと1対1
- 世界トップレベルで成功率80〜85%。ほぼ失点確定
- 主な発生原因：
  - ゴールエリア内（6mライン内）でのDFの反則
  - 明確な得点機会（シュート動作中・フリーでゴールへ向かっている）を妨害
  - 第8条9項・第8条11項に該当する重大な反則
  - コート外の人物がボールに触れる等
- DFとして最も避けるべき状況

---

#### 【選手へのペナルティ】

**③ 警告（イエローカード）（第16条3項） △ 受け取ったら修正をかける**
- 各チームに最大3回まで。選手個人には1回のみ
- 意味：「**この審判はこの基準で吹く**」というアナウンス
- 受け取ったら：次のプレーから基準に合わせて修正をかける
- 戦略的読み方：どのプレーで警告が出たかを記録して審判の基準を把握する
- 警告自体はプレー継続できるが、同じ行為の繰り返しは②退場に直結する

**④ 退場（2分間除外）（第16条4〜5条） ✗✗ いかなる場合もダメ**
- 2分間フィールドから除外。チームは5on6の数的不利
- 退場中は交替選手を入れられない（2分後に戻れる）
- 同一選手が3回退場 → 失格
- コーチ原則：**「退場と天秤にかけて上回るものは基本的にない」**
- 1点を止めるためでも退場は割に合わない（2分間の5on6は複数失点リスク）
- 主な発生原因：
  - 繰り返される反則
  - 明確な得点機会の妨害（7mにならない状況で）
  - 抗議行為・非スポーツマン的行為

**⑤ 失格（第16条6〜9条） ✗✗✗ 絶対にダメ**
- 試合から永久除外。退場と異なり2分後も戻れない
- ただし2分経過後はチームに交替選手が戻ることは可（16条9条）
- 失格＋報告書提出（最重度）：暴力行為等。交替も不可の場合あり
- 主な発生原因：
  - 特に危険な反則・暴力行為
  - 非スポーツマン的行為の繰り返し
  - 競技役員・コーチへの侮辱行為

---

### ファウル判断の優先順位

```
①フリースローを与えることになる → ◎ 積極的に選べる手段
②7mスロー（ペナルティ）になる → ✗ 絶対に避ける
③警告が出る                    → △ 修正のシグナルとして受け取る
④退場（2分間）になる          → ✗✗ いかなる場合も避ける
⑤失格になる                   → ✗✗✗ 絶対にダメ
```

### 具体的な判断場面

| 場面 | 最善の判断 |
|---|---|
| 速攻を止めたい（ゴールから遠い） | ①フリースローになる位置でファウル |
| GKと1対1の状況（相手がフリー） | ファウルしない。どうせ②7mか④退場。GKに任せる |
| 6mライン付近での接触 | ②7mを絶対に与えない。ファウルするなら外側で |
| 外シュートのみの場面 | GKに任せてファウルしない |
| 審判が③警告を出した | 「この審判のライン」を把握。同じプレーを繰り返さない |
| ④退場になりそう | 1点を失っても退場より良い。必ずファウルしない判断 |

### 第8条（反則行為）の概要

**段階的な反則（第8条3〜6条）**：最初は①フリースロー。繰り返すと③警告→④退場→⑤失格へ昇格

**重大な個人ファウル（第8条7条）**：
- a) 相手選手への危険な行為
- b) 後ろからの接触（ボールではなく人を狙う）
- c) 3m違反（フリースローを妨害）
- 結果：③警告〜④退場

**特に重大な個人ファウル（第8条8条）**：
- 明確な得点機会（チャンスを直接潰す行為）
- 相手の顔・首への行為
- 激しい体当たり・押し込み
- 結果：④退場〜⑤失格 ＋ ②7mスロー

**暴力行為（第8条11条）**：
- 試合中・ハーフタイム・試合後の暴力
- 結果：⑤失格（＋報告書）＋ ②7mスロー

## タイムアウト判断

### 使うべき場面
- 相手に3連続得点されたとき（モメンタムを切る）
- 相手のモメンタムが明確に出ているとき
- 重要なセットプレーの前（作戦確認）
- 選手が混乱しているとき

### 使ってはいけない場面
- 自チームのモメンタムが来ているとき（流れを切ってしまう）
- 試合序盤（後半に温存する）
- 相手DFが崩れているとき

### 「5秒ルール」的な原則
- タイムアウト要求から5秒以内に出なければならない
- タイムアウト後の最初のプレーは準備したセットプレーを即実行

## 7on6（GK外し）

### 発動タイミング
| ビハインド | 残り時間 | 推奨度 |
|---|---|---|
| 2点差 | 残り2分 | ◎ 定番 |
| 3点差 | 残り4分 | ○ 状況次第 |
| 4点差 | 残り7分 | △ 早めの博打 |
| 1点差 | 残り30秒 | ○ 最終手段 |

### GKを外すタイミングとサイン
- **フリースロー時**：最も安全。DFが整っている間にGKが走れる
- **ゴールスロー後**：GK自身がコートに残って外から入れ替わる
- **タイムアウト明け**：7人が最初から揃った状態でスタートできる
- **サイン**：コーチが手を叫ぶ / 特定のコールワード / GKへのジェスチャー

---

### 7on6 フォーメーション

**パターンA：3-3フォーメーション（標準）**
```
　　　　LW　　　　　　RW
　バックライト　　バックレフト
　　　バックセンター
　　　　　PV
（PV = ピボット。6m上に構える）
```
- 外野3人・内側3人（PVは6mライン上）
- ボール回しで相手DF7人を動かしてから崩す
- ピボットへの素早いパスが第一選択

**パターンB：2-4フォーメーション（ピボット2枚）**
```
　LW　　　　　　　　　RW
　　バックライト　バックレフト
　　　PV左　　　PV右
```
- ピボット2人を6m左右に配置
- どちらかにパスが通れば至近距離からのシュート
- DFが2人を同時にケアしにくい

**パターンC：4-2フォーメーション（外野4人）**
```
LW　バックライト　バックレフト　RW
　　　　PV　バックセンター
```
- 外野4人でパスを回してDFを横に引き伸ばす
- 引き伸ばした後にピボットへ通す
- 遠距離シュート力が高いチームに有効

---

### 7on6 攻撃の第一選択フロー

```
7on6開始直後：
│
├─ ピボットへの即パスコースはあるか？
│   YES → 即パス → ピボットがシュート
│   NO  ↓
│
├─ 外野4人でのボール回しでDFを横に動かす
│   → 1人が「クローズ」を確認したら逆へ展開
│   → 逆端でシュートチャンスがあれば打つ
│   → なければ再びピボットへ当てパスを試みる
│
└─ フリースロー or タイムアウト取得 → リセット
```

**7on6でやってはいけないこと（失敗パターン）**：

| 失敗 | 結果 |
|---|---|
| パスミスでボールロスト | 空ゴール失点ほぼ確定 |
| 急ぎすぎて判断が雑になる | DFがインターセプト → カウンター |
| 外シュートを急いで打つ | GKなしでも外れたら終わり |
| ピボットへのパスにこだわりすぎる | 時間切れ / パッシブ警告 |
| 全員がゴール方向を向く | ボールロスト時に誰も帰れない |

**リスク管理の原則**：
```
7on6 = 「1点以上を取りに行く」ではなく「1点差を縮める可能性を作る」
→ 無理なシュートより確率の高いシュートを選ぶ
→ ボールロストしたら空ゴール → 逆に点差が広がる最悪のシナリオ
```

---

### 相手の7on6対策（DFとして）
- 空ゴールへのロングシュートを常に狙う（インターセプト = 空ゴール直行）
- ボールロストを誘うプレッシャー（ただし7mを与えないこと）
- コンパクトゾーンで5on6を守り切る（全員でゴール前を固める）

## モメンタム管理

### モメンタムが来ているサイン
- 連続得点（3点以上）
- GKのビッグセーブ連続
- 相手のミスが増えている
- スタンドの雰囲気

### モメンタムを維持する
- 選手交代を最小限にして流れを切らない
- タイムアウトを取らない（相手に修正の機会を与えない）
- 高いプレッシャーを継続する

### モメンタムを取り戻す
- 即タイムアウト（最優先）
- 選手交代（空気を変える）
- DF陣形変更（パターンを崩す）
- GKへの声かけ（精神的サポート）
