# 相手チームの視点・スカウティング・アジャストメント

---

## スカウティング項目

### 攻撃面
- シュート成功率のポジション別集計（バックライト・バックレフト・センター・サイド・ピボット別）
- エース選手のシュートコース偏り（利き手・跳躍方向のクセ）
- ピボットへのパス頻度と成功率
- 速攻の発動条件（どのシチュエーションで仕掛けるか）
- セットプレーのサインと実行パターン
- センタープレーヤーの左右展開のクセ
- ドリブル突破の方向（利き手方向か逆か）
- アシスト源（どの選手からエースへのパスが多いか）

### 守備面
- ゾーンDFか、マンツーマンか、ハイブリッドか
- ゾーンDFの場合：6-0・5-1・3-2-1などの種別
- プレスのタイミング（センターラインか6m手前か）
- GKの弱点（上部左コーナーが弱い、下段（足元）のシュートへの反応が遅い 等）
- DF選手のファウルを取られやすい傾向
- カウンター発動の条件（シュート後の戻り速度）
- スイッチングの速度と頻度

### コンディション・メンタル面
- 前半/後半でパフォーマンスがどう変わるか
- 点差が開いた場合の崩れ方
- タイムアウトのタイミング（どのスコア状況で取るか）
- 退場者が出た後の戦術変更パターン

## ハーフタイム修正

### 攻撃修正
| 問題 | 対策 |
|---|---|
| エースへの二枚掛け | 逆サイドのフリー選手へ切り替え |
| ピボットパスが潰される | フラッシュのタイミング変更 |
| 速攻が出ない | GKからのロングスローを積極活用 |
| シュート成功率が低い | コーナーポジションからのシュートを増やし角度変更 |
| DFに読まれている | セットオフェンスのパターンを第二プランに変更 |
| パス回しが遅い | ワンタッチパスの徹底 |

### 守備修正
| 問題 | 対策 |
|---|---|
| 外シュートを決められる | 0-6から5-1に切り替え、前衛でプレッシャー |
| エース1人に決められ続ける | マンツーマン担当を指定 |
| ピボットに決められる | ピボットのマークを常時2枚（DF2人）に |
| 速攻を繰り返し食らう | シュート後の最低2名の即時帰陣を徹底 |
| GKの弱点が露呈 | GK交代（控えGKの特性と合わせる） |

## リアルタイムアジャストメント（試合中）

| 相手の変化 | こちらの対応 |
|---|---|
| 6-0→5-1に変更 | 5-1の前衛を引き付けてからバックプレーヤーが外シュート |
| マンツーマンに変更 | スクリーンプレーを多用してマークを剥がす |
| 前線プレスを強化 | ロングパスでプレスを越える（GK→速攻選手） |
| エースへの二枚掛け | 逆サイドのフリー選手を使う |
| ゾーンを狭めた | 外からの遠距離シュートで引き出す |
| DFラインが高い | ピボットの背後（相手DFラインとGKの間の空間）へ通すスルーパス（DFの間を抜くパス） |
| DFラインが低い | 外シュートのコースが開く→積極的に打つ |

## アジャストメントのじゃんけん

```
相手の6-0ゾーンDF
  → 外からの遠距離シュート（ゾーン外で打つ）
  → 相手が5-1に変更（前衛がプレッシャー）
  → 5-1前衛を引き付けてからピボットパス（前衛の裏＝前衛と後ろのDFの間にできる空間）
  → 相手がマンツーマンに変更
  → スクリーンでマーク剥がし → 1on1突破
  → 相手が6-0に戻る

相手エース二枚掛けマーク
  → 数的有利サイドへの展開
  → 相手がゾーン再構成（サイドも警戒）
  → エース解放（マーク手薄になる）
  → エースへの集中攻撃
  → 相手エース二枚掛けに戻る
```
