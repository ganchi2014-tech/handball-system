## 幹：GKが直面する物理的制約（全技術・戦術の起点）

```
ハンドボールGKの全技術・戦術は、一つの物理的事実から派生する：

  6mシュートのフライトタイム ：約240ms
  7mスローのフライトタイム  ：約350ms
  GKの反応＋動作に必要な時間：500〜1000ms

→「見てから動く」だけでは、どのGKも物理的に間に合わない
→ この制約は技術・体格・経験では解消できない。全GKに共通する絶対条件。
```

この制約への3つの解決策が、GK技術・戦術の「枝の大分類」を作る（詳細出典 → 科学的根拠セクション）：

| 解決策 | 方法 | 具体的な現れ |
|---|---|---|
| **A：ポジション** | 動かなくていいよう、最初から体でコースを塞ぐ | コース絞り型・射角理論・面積カバーの幾何学 |
| **B：先読み** | ボールが手を離れる前に行先を予測して先に動く | 先読み型・キュー読み・193ms前の動作開始 |
| **C：前進（角度縮小）** | シューターに近づきゴールを小さく見せる | 縦方向ポジショニング・6m対応30〜40%縮小 |

> エリートGKはA＋B＋Cを状況に応じて組み合わせる。「先読み型/コース絞り型」の2分類は、AとBの優先度を外から観察するときの**実用ラベル**。研究上の科学的タイプ分類ではない（スポーツ科学ではアンティシペーション能力を連続量として扱う）。

**シューター（バックプレーヤー）視点の逆手**：

| GKの解決策 | シューター側の対策 |
|---|---|
| **A：ポジション** | GKを**動かす**（フェイク・引きつけ）→ 動いた逆が空く |
| **B：先読み** | **早打ち**でGKが動く前に放つ / **逆を突く**（GKの動き始めの逆へ） |
| **C：前進** | **ループシュート（ヘーバー）**でGKの頭上を越す（打ち方 → `02_individual.md`） |

> GKがA・B・Cを組み合わせるほど、シューター側も複合対策が必要になる（じゃんけんの多重化）。

## 核心原則（コーチ直伝）

### 3方向の法則

> **構造的位置づけ**：3方向の法則は「解決策A（ポジション）」の幾何学的帰結。GKが正しいポジションを取ると、体の面積カバーによって自然と2/3が閉じる。解決策B（先読み）や解決策C（前進）では自動的には成立しない原則。

- ゴールは常に「左・中・右」の3エリアに分割される
- GKは体のサイズと位置から、常に**2/3を閉じていて、1/3が開いている**
- シューターはその「開いている1/3」を見つけて打つのが基本戦略
- 縦も同様：上/中/下の3エリア。常に2/3が閉、1/3が開

### GKタイプ2分類（解決策A・Bの観察ラベル）と「読み取りプロセス」

| 旧名称 | 新名称 | 英語/独語 | 本質 |
|---|---|---|---|
| 波型 | **先読み型（アクティブ型）** | アンティシペーション（先読み） / アクティヴGK（積極的GK） | 予測して先に動く |
| スポット型 | **コース絞り型（ポジション型）** | Position / Passives GK | コースを塞いで待つ |

> ⚠ **分類の性質**：この2分類は「解決策AとBのどちらを優先しているか」を外から観察する実用ツール。実際のGKは試合内でダイナミックに切り替える。DHB/EHF/IHFの公式文書に明示的定義はなく、スポーツ科学論文もこの分類を採用していない（研究はアンティシペーション能力を連続量として扱う）。

**試合序盤の読み取りプロセス（最重要）**：

```
序盤3〜5本：まず「普通に打つ」→ GKの反応パターンを観察

観察ポイント：
  ① GKはリリース前に動き始めたか？（先読み型の兆候）
  ② GKは動かずコースを塞いでいたか？（コース絞り型の兆候）
  ③ どのコースを空けているか（空き2エリアはどこか）
  ④ シュートフェイントに反応したか・しなかったか

判定：
  → 3本以上でリリース前に動く → 先読み型と判断
  → ほぼ動かない・精度を要求される → コース絞り型と判断
  → 状況で変わる → 混合型 → 場面ごとに使い分け
  ★ 観察期間中（判定前）の共通最善手：低コース（腰〜膝） ← タイプに関係なく有効
```

**GK自身のタイプ選択基準（GK選手・GKコーチ向け）**：

> 相手GKを分析するだけでなく、**自分がどちらのタイプで臨むかをGK自身が試合前・試合中に決める**。

| 場面 | 推奨タイプ | 理由 |
|---|---|---|
| 相手シューターの精度が高い・コース変化が鋭い | **コース絞り型** | 先読みが外れると必ず失点。体で最大面積を守る |
| 相手シューターが同じパターンを繰り返している | **先読み型** | パターンが読める = 先に動ける。読みが当たれば確実に防げる |
| 試合終盤・疲労が蓄積している | **コース絞り型** | 先読みの集中力が落ちると読みがズレやすい。体を張って守る |
| 7mスロー（ペナルティ） | **コース絞り型が基本** | 先読みで飛ぶとシューターに逆を突かれる。ルーティンと確率ゲームで対応 |

**低コース（腰〜膝）は判定前から有効**：どちらのタイプにも一定の効果がある。序盤の観察期間中はここを狙うのが最も安全。

## 基本スタンス（グルントシュテルング）― エリート観測データ

*出典: Journal of Applied Biomechanics 32(5) 2016 / ヴァニャ・ラディッチ コーチング*

> ⚠ **これは「エリートGKの観測平均値」であり、処方箋ではない。** 体格・身長・スタイルによって最適値は変わる。「自分はなぜここが違うのか」を考える参照点として使う。

| 身体部位 | エリート観測値 | 備考 |
|---|---|---|
| 足幅 | 肩幅の約**2.5倍** | 統計的平均。小柄なGKは比率が変わる場合がある |
| 膝の角度 | 軽度屈曲 | 大腿四頭筋の即時活性化のため。過度な屈曲は疲労を招く |
| 重心の前後 | 足首軸より**前方**に投影 | 踵に乗ると反応が遅れる |
| 体重支持 | 足の前3分の2（前足部） | つま先だけは×、踵だけは× |
| 手の位置 | 肘を約90°屈曲、腕は体幹より前方・腰の高さ | 両腕が独立して動けることが必須 |
| ゴールラインからの距離 | 約**1.10m**前方（エリート実測平均） | バイオメカニクス最適値は**1.44m**（→縦方向ポジショニングセクション参照） |
| 腕の独立性 | 両腕がそれぞれ独立して動けること | ノルウェー式："armene beveger seg uavhengig" |

```
スタンスのじゃんけん：
  足幅広く（安定重視） → 大きな横移動ができない → シャッフル Stepが届かない
  足幅狭く（機動性重視） → 股下シュートに弱い → ボールが股下を抜ける
  解決：肩幅2.5倍 + 前足部重心 + フットワークで補完
```

## フットワーク4種類

*出典: ハンドボール エボリューション（進化） / ヴァニャ・ラディッチ コーチング / DHB Torwartschule*

| 名称 | 説明 | 使用場面 |
|---|---|---|
| **シャッフル Step**（サイドステップ） | 足を交差させずに横移動。両足の幅を常に保ち重心が安定する。最も頻用 | 短距離の横移動（基本） |
| **T-Step**（Tステップ） | 動きたい方向と逆の足を、もう一方の足のかかと横にT字型に引き寄せ重心を移動させる。次のシャッフルへの切り替え準備動作。シャッフルStepと違い、方向軸そのものを変えるときに使う | 方向転換・軸を切り替えるとき |
| **Cross Step**（クロスステップ） | 一方の足をもう一方の足の前に交差させながら移動。速度が上がるが安定性はやや低下 | 長距離の素早い横移動 |
| **Forward-Backward Step** | 前進でシューティングアングルを狭める（プレッシャーアウト）／後退でリカバリー | シューターへの前進・ループ対応後退 |

**重要原則**：シャッフル Step移動では足を交差させない。Cross Stepは大きな距離を素早くカバーする場面限定。

## ポジショニング角度理論

### 射角の二等分線原則

```
         S（シューター）
        /  \
       / 射角\  ← ∠LSR（両ポストへの角度）
      /↙二等分\線
     /    GK   \
    L─────────────R
  左ポスト       右ポスト

シューターが右に移動すると：
  S──────S'
             \
              \射角' （より狭くなる）
               GK' （右に移動）
         L─────────────R
```

- GKは常に「**射角（シューターを頂点とし、両ゴールポストを両辺とする角度∠LSR）の二等分線**」上に立つ
- 「ゴール中心とシューターを結ぶ線」ではない（シューターが正面にいるときのみ一致する）
- シューターが左右に動くと、GKはゴールの弧に沿って追従する（半円軌道）
- **つま先は常にシューターの投球手の方向に向ける**

**実用的な理解**：GKはシューターの方向に向かって「角を詰める（shoot the angle）」ように動く。シューターが右に移動すれば GK も右へ。左なら左へ。

### 縦方向ポジショニング（前後距離）― シューター距離別の最適立ち位置

*出典: Journal of Applied Biomechanics 2016 (PubMed 27341496) / ヴァニャ・ラディッチ コーチング (vanjaradic.fi)*

> GKの前後位置はシューターまでの距離・角度によって変わる。左右の二等分線原則と組み合わせて使う。

| シューター距離・状況 | 初期待機位置 | 前進後の推奨位置 | 根拠 |
|---|---|---|---|
| **9mバックコート** | ゴールライン前方**1.10m**（エリート実測平均） | バイオメカニクス最適値 **1.44m**（半歩前） | 2016年韓国代表実測研究。従来推奨0.75mは短すぎ |
| **6mライン（ピボット等）** | **0.5〜1.0m**（初期待機） | 前進後 **1.5〜2.5m**（1歩40〜80cm） | 240msのフライトタイム → 純粋反応不可。ポジションのみ有効 |
| **ウィング（通常角度）** | ポスト寄りに**タイミング良く1歩前進** | 前進方向：コーナーではなく**シューターの手の方向へ** | コーナー方向へ前進するとバックポストが開く |
| **ウィング（極小角度）** | **前進しない**。ポジション保持のみ | — | 前進で逆にゴールが開く。ポストに近い場合は後退も |
| **ウィング（大きな角度）** | 積極的に前進してアングル縮小 | シューターに近づく | 広いスペースを持つウィング対策 |

**前進の原理（Verkleinerung des Winkels / 角度縮小）**：
```
GKが前進 → シューターから見たゴール開口角が縮小
  （タンジェント関数：GKが近づくほどGKの体が射角の大部分を占める）
  → 6mシュートへの前進30〜40%アングル縮小（ヴァニャ・ラディッチ実証データ）
```

**前進の最適タイミング**：「シューターがシュートを決意した瞬間（もはや方向を変えられない段階）」に前進を合わせる。早すぎ → ループシュートのリスク。遅すぎ → アングル縮小効果ゼロ。

### 前進（Step Out）の効果と危険性

| 効果 | 危険性 |
|---|---|
| シューティングアングルを30〜40%縮小 | タイミングが合わないとコーナーを大きく開ける |
| シューターに心理的プレッシャー | ループシュート（ヘーバー）で頭上を抜かれる |
| 近距離では防御面積が増加 | 早打ち（最高点前）に対応できなくなる |
| — | ウィング極小角度では逆効果（ゴールが開く） |

**6mライン対応の反応時間計算**：
```
6m先から90km/hシュートのフライトタイム：約240ms
  視覚処理（75ms）+ 判断（50〜75ms）+ 運動開始（100〜125ms）= 合計225〜275ms
  → 純粋な反応では間に合わない
  → ポジショニング前進 + アンティシペーションのみが有効な対応手段
```

## 先読み型GK（アクティブ型）― 詳細メカニズム

> **前提**：このセクションは「序盤の観察でGKが先読み型と判定した後」の攻略・理解に使う。

**何をしているか**：
```
シューターが助走・踏み切り
 ↓
GKが「このシューターはあそこに打つ」と予測
（予測根拠：助走の角度・利き手・過去のパターン・体の向き）
 ↓
ボールが手から離れる前にGKが先にそのコースへ動く
 ↓
シューターが打った瞬間にはGKはもうそこにいる
```

**なぜ有効か**：科学的に確認済みの事実として、GKの反応+動作には500〜1000msが必要だが、シュートのフライトタイムは約350ms（7mスロー）〜240ms（6mシュート）しかない。つまり純粋な反応では間に合わない。エリートGKはリリース**193ms前**から動き始めている（→ 科学的根拠セクション参照）。

**弱点（シューター視点）**：
- GKが動くタイミングより**早く**打つ（早打ち）→ GKはまだ動いていない or 動き始めだけ
- GKが「右に動く」予測をした瞬間に**左**に打つ（逆を突く）
- バックハンドフェイク → GKのコース予測を完全に狂わせる

**早打ちの原理（コーチ直伝）**：
- GKが「シューターの最高到達点に先取りしにいく」イメージで動く場合
- 早打ち（最高点に達する前に打つ）はGKの意識の外のシュートになる
- GKは最高点を想定して動いているため、それより低いタイミングへの対応が遅れる

**じゃんけん（先読み型GKとの戦い）**：
```
最高点シュート → GKが先読みで塞ぐ → 早打ちに変える → GKが早打ちに慣れる
 → さらに早いタイミングに変える or 最高点に戻す（GKが混乱）
 → バックハンドフェイクを加える → GKの予測が不安定になる
```

**GKが先読みを多用して失点が増えた場合の自己修正**：上部の「GK自身のタイプ選択基準」テーブルを再確認 → 「相手の精度が高い」「試合終盤」に該当すればコース絞り型への切り替えを検討。

## コース絞り型GK（ポジション型）― 詳細メカニズム

> **前提**：このセクションは「序盤の観察でGKがコース絞り型と判定した後」の攻略・理解に使う。

**何をしているか**：
```
GKは動かない。ただし「最初から体でコースを塞いでいる」

左右×上下 = 6エリア
 ┌───────────────┐
 │ 左上 │ 中上 │ 右上 │
 │──────────────│
 │ 左下 │ 中下 │ 右下 │
 └───────────────┘

GKの体が「4エリアを塞ぎ、2エリアを意図的に空けている」
シューターはその「空き2エリア」を見つけて正確に打つ必要がある
```

**なぜ有効か**：動かないことで「先読みを外れるリスク」がない。体のサイズと立ち位置で常に2/3を守れる。

**幾何学的根拠（面積カバー）**：
```
ゴール幅：300cm
GKの肩幅（体幅）：約60〜65cm
両腕を展開したスパン：身長 × 約1.03（例：190cmのGKで約196cm）

→ 体 + 腕スパンでゴール幅の約65〜70%を物理的にカバーできる
→ シューターは残り約30〜35%（1エリア）に「正確に」打つ必要がある
→ コース絞り型はこの「確率の壁」を最大化する戦略
```

先読み型の科学的根拠が「193ms前から動く」という時間軸なら、コース絞り型の根拠は「体積と面積の幾何学」。どちらが有効かはシューターの精度・スピード・フェイント能力による（じゃんけん構造）。

**弱点（シューター視点）**：
- GKを**動かす**（引きつける・フェイクを入れる）→ GKが動いた方向の逆が空く
- 「空き2エリア」を正確に特定して打つ（精度が問われる）
- **低いコース**：コース絞り型GKは高さを変えにくい。低めが空いていることが多い

**角度変化の原理**：
- ジャンプしながら体の向きを変えて「GKが読んでいたコースの逆」に打つ
- GKが「中央」を塞いでいれば → 端の低いコースを狙う
- GKが「右下」を強調して塞いでいれば → 左上（GKの反対上）を狙う

**じゃんけん（コース絞り型GKとの戦い）**：
```
「空き2エリア」に正確に打つ → GKが次第にそのコースを先読みで塞ぐ（先読み型に変化）
 → 引きつけてから角度変化で逆を突く → GKが動くようになる（先読み型に変化）
 → 早打ちも組み合わせる → GKが混乱してどちらか一方のタイプに戻る
```

## セービング技術（ポジション別）

*出典: ハンドボール エボリューション（進化） / DHB Torwartschule / FFHB オメイエ分析*

### ウイングシュート対応

```
体重はポスト側の脚に → 反対脚を解放して反応用に使用
ポスト側の手：肘屈曲・高めに構え → 上角保護
反対側の手：約90°屈曲・中間高さ → バックコーナー保護

腕-脚同期セービング（パラーデ（セーブ動作） bras-jambe / ブラジャンブ）：
  逆脚の膝を上げながら肩のレベルを上げる（ホップ動作と連動）
  → 防御面積を縦方向に拡大
```

**じゃんけん（ウイングvs GK）**：
```
ウイングが角度つきシュート → GKがポスト寄りにポジション → 遠コーナー狙い
  → GKがポスト離れる → ポスト際（近コーナー）シュート
  → GKがポスト寄りに戻る → 遠コーナーサイクルに戻る
```

### バックコートシュートへのダイビング

*出典: PMC7739716 (角速度/Moment/パワー of Hip-Knee-Ankle) / ハンドボール エボリューション（進化） / ヴァニャ・ラディッチ コーチング*

**踏み切りの原則（コントララテラル〈動きたい方向と逆側の脚〉主推進）**：

> GKのダイビングは**動きたい方向と逆側（コントララテラル）の脚が主推進脚**。右方向にダイブするなら左脚で蹴る。

```
右方向ダイブの6フェーズシーケンス（左右対称で左ダイブも同様）：

  ①準備     : 基本姿勢（肩幅2.5倍・前足部重心）
  ②左脚踏み切り: 左脚（コントララテラル）の股関節伸展筋で水平推進力を生成（最大ピークモーメント）
  ③右脚推進  : 右脚（イプシラテラル）が伸展しながら体幹を右方向へ送り出す
  ④飛行フェーズ: ヒップが動作方向を先行 → 肩・腕がフォローする順序で回転
               左脚は屈曲させ上方への推進を継続
               右脚は伸展位を保つ
  ⑤腕伸展   : 右腕をボール方向へ最大伸展（低いボールは手背・前腕で、高いは手掌でキャッチ or リダイレクト）
  ⑥着地     : 足首側面→膝→大腿→ヒップ→側胸部の順に「順次着地」
               （着地時の最大荷重：3〜8kN〈キロニュートン＝力の単位〉＝体重の約4.2〜8.6倍の衝撃）
               手首・肘で受けない（骨折リスク）
```

**ダイビングの2タイプ（ドイツ語）**：

| タイプ | 名称 | 特性 | 適用 |
|---|---|---|---|
| 水平跳躍型 | ザイトヴェルツシュプルング（サイドウェイスプリング） | 横方向の水平運動量を最大化 | 低いボール・中間高さ |
| 複合跳躍型 | トアヴァルトシュプルング（ゴールキーパージャンプ） | 垂直+水平の複合跳躍 | 上角・ミドルのボール |

**コーチング原則（ヴァニャ・ラディッチ）**：  
「足がスタート位置を決め、ヒップが動作方向を決め、腕と手はフィニッシュの要素にすぎない」

| ボールの高さ | 優先技術 |
|---|---|
| 低いボール | 崩れ落ちながら前腕・手背でブロック |
| 中間・高い | 手でキャッチ or リダイレクト（偏向）|

**GK選手の実践ポイント**：
- 右ダイブの訓練では「左脚で蹴る」感覚を意識的に練習する
- ダイビング中にヒップが回転を先行させ、腕が後から追う順序を体に覚えさせる
- 着地後に即座に立ち上がれる「転がり着地」の習得（ローリング動作）

### ループシュート（ヘーバー）対応

- 相手がGKの前進を利用してGKの頭上に「置く」ように放るシュート
- **シュリットシュプルングテクニック**（ドイツ式ステップジャンプ技術）で対応：
  - **ボール方向の脚（同側脚）を踏み出してジャンプ**（右側のループなら右脚を踏み出す）
  - ジャンプしながら同側の腕を上方に伸ばす
  - 踏み出し脚のつま先を外向きにして防御面積を横方向に広げる
  - 着地は安全に両脚で（片脚着地は足首・膝の捻挫リスク）
- **D-ユース（10歳前後）から教育開始が推奨**（DHB指針）

### ポスト（ピボット）連携場面のGK対応

> GK・CP・GKコーチ全員に関わる重要場面。ポストが絡む攻撃はGKの「視線の分割」を要求する。

**ポストがいる場面でGKが直面する問題**：
```
ボールをバックプレーヤーが持っている
  ↓
GKはボールを見たい（シュートを読むため）
  ↓
しかし6m付近のポストも意識が必要（パス→即シュートの危険）
  ↓
「ボールかポストか」の注意配分が問われる
```

**DHB原則（ドイツ語）**：*"Das Auge folgt dem ボール, das Bewusstsein folgt dem Kreis"*  
（目はボールを追うが、意識はポストを追う）

| ポスト連携の場面 | GKの対応 | CPへの示唆 |
|---|---|---|
| ポストが6mラインで静止 | 視線はボール保持者。ポスト位置を「周辺視野」で把握 | ポストの存在がGKの重心を中央に固定させる → 端へのシュートが有利になる |
| ポストがボールを受けて即ターン | ターン完了・腕を引いた瞬間に前進ステップ（ヘラウスラウフェン） | 「受け→即ターン→シュート」パターン1が最速。スピードが核心 |
| バックがポストスクリーンを使いシュート | スクリーンを無視してボール保持者（バック）を追う | スクリーンでGKの前進タイミングが乱れる → 前進プレッシャーを奪える |
| ポストへのフェイクパス後にバックがシュート | GKがポスト方向に反応した瞬間に逆コースへシュート | フェイクパスはGKの意識をポストに向けさせる最も効果的な動作 |

**ピボットの4シュートパターン（GKが事前に把握すべき）**：
1. **受け→即ターン→シュート**（最も速い。タイミングで対応）
2. **受け→フェイク→シュート**（フェイクに反応するとシュートが決まる）
3. **ボディフェイント→逆方向シュート**（先読みが全て外れる）
4. **フォールアウェイショット**（後ろに倒れながら打つ。GKの前進プレッシャーを外す）

**ピボット（シューター）側の選択優先度**：
1. まず**パターン1**（速度勝負）→ DFがつく前・GKが反応する前に即シュート
2. GKが前進してきた場合 → **パターン4**（フォールアウェイ）→ 解決策Cへの対抗
3. GKが先読みで動いた場合 → **パターン3**（ボディフェイント）→ ヒップの向きを偽る
4. GKが動かず待っている場合 → **パターン2**（フェイク後シュート）→ 動かして空きを作る

**アンティシペーションのキュー信頼性ランク**（Huesmann et al. PMC10591308 / ヴァニャ・ラディッチ）：

| ランク | キュー | 信頼性 | 説明 |
|---|---|---|---|
| ①最高 | **ヒップの回転方向** | 最信頼 | シューターは必ずターゲット方向にヒップを向ける（物理的制約） |
| ② | **肩の向き** | 高 | ヒップに連動する。ただしフェイクで外せる場合がある |
| ③ | **腕の角度・肘の位置** | 中 | 高さ（上角/下角）・パワーの推定に有効 |
| ④最低 | **フットワーク・アプローチ** | 低 | 文脈情報としてのみ有効 |

**GKコーチの指導ポイント**：
- 「目はボール（主視野）、意識はポスト（周辺視野）」の二重注意を意識的に訓練する
- ポストが6m以内にいる場合は基本スタンスで「若干中央寄り」にポジション
- Huesmann (2023)：エキスパートGKは**チーム全体の攻撃タスク配分**（誰が主に打つか）を試合前情報として活用
- DF連携：「ディフェンダーはGKがボールを常に見られるよう立ち位置に配慮する」（ヴァニャ・ラディッチ）
- ポストのシュートパターン4種をスカウティングで事前把握 → どのパターンが多いかを試合前にGKに伝える

### バックプレーヤーのポジション別：GKが守りにくいコース（幾何学から導く）

GKは射角∠LSRの二等分線に立つため、**シューターのポジションによってGKの立ち位置が変わり、守りにくいコースが決まる**。

```
LB（バックレフト）からシュート：
  GKは左ポスト寄りに立つ（ニアポストを閉じる）
  → 右コーナー（ファーコーナー）が相対的に空く
  → 右上・右下が有効コース。初手は右下（低コース）
  GKへの示唆：ファーコーナー（遠い側のすみ）へ腕を素早く突き出して弾くパンチ動作が最重要

LW（レフトウィング）からシュート：
  角度が極端に小さい → GKは左ポスト際にほぼ密着して立つ
  → 使えるコースは事実上2択に絞られる
     ①ファーコーナー（右側）への強打 ← 主選択肢
     ②GKが前に出すぎた場合：ニアポスト際の隙間
  → 縦方向詳細 → 縦方向ポジショニングセクション「ウィング（極小角度）」参照
  GKへの示唆：極小角度では前進するとゴールが逆に開く。ポジション保持が基本

CB（センターバック）からシュート：
  GKはほぼ中央（射角が最大・左右均等）
  → 全コースが均等に守られる
  → 初手：低コース（腰〜膝）← GKタイプ不明・観察中の共通最安全選択
  → タイプ判明後：高さの選択（上コーナー vs 低コース）か「コース変化フェイント」
  GKへの示唆：左右の読み合い。フェイントへの対応力が核心

RB（バックライト）からシュート：
  GKは右ポスト寄りに立つ（ニアポストを閉じる）
  → 左コーナー（ファーコーナー）が相対的に空く
  → 左上・左下が有効コース。初手は左下（低コース）
  利き手別補足：
    右利き → ファーコーナー（左）は体を交差させる動作が必要（技術難度高め）
    左利き → ファーコーナー（左）が自然な方向 → 威力・精度の両立が容易（左利き優位ポジション）
  GKへの示唆：LBの逆。左腕・左脚の対応訓練が特に重要

RW（ライトウィング）からシュート：
  LWの左右対称。角度が極端に小さい → GKは右ポスト際にほぼ密着して立つ
  → 使えるコースは事実上2択に絞られる
     ①ファーコーナー（左側）への強打 ← 主選択肢
     ②GKが前に出すぎた場合：ニアポスト際の隙間
  → 縦方向詳細 → 縦方向ポジショニングセクション「ウィング（極小角度）」参照
  利き手別補足：
    右利き → ファーコーナー（左）が自然な方向 → 有利
    左利き → ファーコーナー（左）は体を交差させる動作（技術難度高め）
  GKへの示唆：LW対応の左右対称
```

**GKコーチへの含意**：ポジション別のセービングリーチ（腕の届く範囲）をドリルで確認し、ファーコーナーへの反応を弱点として重点的に強化する。

## 科学的根拠（数値データ）

### 反応時間とフライトタイム

*出典: PMC3590836 / Biomechanical Analysis of アンティシペーション（先読み）*

| 指標 | 数値 |
|---|---|
| 7mスローのボール飛行時間 | 約**350ms**（0.35秒）@ 20 m/s |
| 6mシュートのフライトタイム | 約**240〜270ms** @ 80〜90 km/h |
| GKの反応+動作に必要な時間 | **500〜1000ms** |
| エリートGKの平均反応開始 | ボールリリースの**193ms前** |
| エリートGKの成功率 | **67%** |
| 未経験GKの平均反応開始 | リリースの209ms前（ただしタイミングのバラつきが大きく±127ms＝毎回大きく早すぎたり遅すぎたりして安定しない） |
| 未経験GKの成功率 | **24%** |

**結論：GKは純粋な反応だけではセービング不可能。アンティシペーション（先読み）は必須。**

### 視線研究（Gaze Behavior / クワイエット Eye）

*出典: PMC4664728（フロンティアズ（学術誌） in 心理学 2015）/ PMC11940369（MDPI Brain Sciences 2025）*

**男女GKの視線戦略の違い**：

| 性別 | 視線の主対象 | 戦略タイプ |
|---|---|---|
| 男性GK | 投球腕・前腕 | 運動軌跡の動的追跡 |
| 女性GK | 頭部・体幹 | 記憶ベースのパターン認識 |

**クワイエット Eye（QE）効果**：
- セービング成功と「最終固視の持続時間（QE）の長さ」に正の相関
- 男性GK：成功/失敗の差 = 平均**122ms**（p<0.001）
- 女性GK：成功/失敗の差 = 平均**92ms**（p=0.005）
- → **見続ける（QE延長）がセーブ率向上に直結**

**左利き対策の問題**（PMC10591308 / German Sport University Cologne 2023）：
- 左利きシューターへのアンティシペーション精度は右利きより**低い**
- 慣れの非対称性：右利きに比べて学習経験が少ない
- 対策：左利き専用の練習機会を意識的に増やす

### アンティシペーションが利用するキュー2段階

```
【試合前情報】
  対戦相手のパターン（スカウティング）
  個々のシューターの習慣（利き手・よく使うコース）

【スロー中情報】
  ジャンプの軌跡・腕の振り
  DFのブロック位置（どこをGKがカバーすべきか確認）
  シューターの体の向き・重心移動
```

### デンマーク / Game リーディング（読み）

### スペイン / 大型GK型

**理念**：身体能力（長身・高跳躍）と高い技術完成度の融合

### Mats オルソン式（独自流派・北欧ベース）

> **注記**：オルソンはスウェーデン人だが、ノルウェー女子代表GKコーチとして20年以上活動。「スウェーデン式」ではなく彼個人が体系化した独自流派として扱う。

**理念**：「最も重要なのは身体ではなく、脳だ（The most important thing to save the ball is the brain, not the body）」

**経歴**：スウェーデン代表294キャップ（1979-1997）・1990年世界選手権金・1997年世界最優秀GK・2024年EHFハンドボール殿堂入り。GKコーチとして：ノルウェー女子代表（2005-現在）・SG Flensburg-Handewitt（2010-2017 / 2024/25-）・スウェーデン男子代表（2013-2021）・IHF CCM（コーチング・メソッド委員会）メンバー（2018-）

**4段階育成モデル**（IHF教育センター「ゴールキーパー coach education: Basic concepts in a different way」）：

*出典: IHF ゴールキーパー Coach Education Online Symposium 2020（youtu.be/Ekb_2JmgoFM）/ EHF ハンドボール マスターマインド記事*

| 段階 | 名称 | 環境 | 具体的ドリル形式 | コーチングの重点 |
|---|---|---|---|---|
| **1** | Simple テクニック トレーニング（基礎技術） | **脅威なし**（シューターなし or 静的） | ①コーチがボールを投げる→GKが止める（高さ・左右・バウンド・足元の各方向） ②両手ドリブル移動 ③フットワーク4種の反復 | **自信の構築を最優先**。正確なフォームの自動化。失敗を批判しない |
| **2** | Complex テクニック トレーニング（複合技術） | **実シューターあり**（戦術的プレッシャーは限定） | ①複数シューター交互シュート ②6ポジション→2秒以内に正しいアングル移動ドリル ③フィジカル負荷後のセービング（疲労下での技術維持） | **チャレンジと自信のバランス**。セーブ率の測定を開始 |
| **3** | リーディング the Shot（シュートの読み） | **キュー訓練**（シューターの身体言語を読む） | ①シューターがヒップ→肩→腕を意図的に見せる→GKが先読みして動く ②映像で自分の動作開始タイミングを確認（「早すぎ/遅すぎ」分析） ③認知テスト：静止画・動画からキューを読んで方向を予測 | 「早すぎ vs 遅すぎ」のフィードバック。**キューの意識化** |
| **4** | リーディング the Game（ゲーム全体の読み） | **試合文脈**（相手チーム全体の傾向を読む） | ①スクリメージ形式でDFコミュニケーション統合 ②試合映像分析→「このチームは誰が主に打つか」を語言化 ③ハーフタイム戦略討議 | **プロセスを問う**：「なぜそう判断したか」。正解/不正解より意図を評価 |

```
オルソンの核心：
  多くのコーチは ステージ 1 と 2 で止まる
  → GKが「身体で止める」ことを覚えても「脳で止める」には至らない
  → ステージ 3 と 4 こそ世界トップとその他の差
  
  「ジュニアでも ステージ 3 の訓練は早期から始められる」（オルソン）
  技術が未熟でもゲームを読む習慣は早期に形成できる
  
  ステージ 3 のキュー信頼性ランク（研究根拠: PMC10591308 / ヴァニャ・ラディッチ）：
    ①ヒップ回転（最信頼）→ ②肩の向き → ③腕/肘の位置 → ④フットワーク（参考程度）
```

「従来のGKコーチは技術・フィジカルから積み上げる。オルソンは リーディング（読み） を独立した育成段階として最上位に置く」

**オルソン式4段階と「3つの解決策」の対応**：

| オルソン段階 | 主に育成する解決策 |
|---|---|
| ステージ 1-2（技術訓練） | 解決策A（ポジション）・解決策C（前進）の身体化 |
| ステージ 3（シュートの読み） | 解決策B（先読み・キュー読み）の開発 |
| ステージ 4（ゲームの読み） | A＋B＋Cの状況判断による統合運用 |

> 「多くのコーチはステージ 1-2で止まる」というオルソンの批判の構造的意味：解決策AとCしか教えない、解決策Bが育たない。

**アンティシペーション（先読み）の方法論**：
- シューターの**身体言語（ボディ Language）の読み取り**が核心：肩・腰の位置、腕の動き、接近速度からコースを予測
- 「ヴィジョン」概念：どこにボールが来るかを「見て理解する」能力（先天的素質 + 後天的訓練）
- **個人別戦略の設計**：特定シューターのファボリットショット（得意コース）を「あえて与えるか、不得意コースに誘導するか」を事前に設計（映像分析と統計を組み合わせる）
- Katrine Lundeとの20年の師弟関係で具体化：「マッツに初めて会ったとき、自分がどう考えているかを問われた。すぐに自分の考え方が変わった」（Lunde証言）

**信頼（Trust）の三角形**：
「ゴールキーピングは信頼についてだ——GK自身・ディフェンダー・ゲーム全体との間の信頼」
→ DF連携をテクニックではなくメンタル的基盤として位置づける

**メンタル理論（7要素）**：

| 要素 | 内容 |
|---|---|
| テンション・コントロール | プレッシャー下での集中管理 |
| カリスマ | コート上の支配力・存在感 |
| フロー | ゾーン状態への入り方と維持 |
| ゲーム読みの心理戦略 | Strategies for reading the game |
| 相手への心理的対抗 | 心理学 against opponents |
| 冷静さと集中の維持 | Staying cool and concentrated |
| フォーカス・リフォーカス | ミス後の精神的リセット |

**段階別コーチング哲学**：

| GKの段階 | 重点 |
|---|---|
| 若手・未経験 | **自信の構築**を最優先。課題より楽しさと成功体験 |
| エリート・経験者 | **チャレンジと自信のバランス**。限界への挑戦と心理的支持の両立 |

「世界クラスのGKの最大の違いは精神的な部分。テクニックも戦術も誰でも学べる。しかし最高のGKはより優れたゲーム読み能力を持ち、プレッシャーの中でのプレーを楽しめる」

**現代GK論**：GKは「速攻の起点（Starting Point of a Fast Break）」として戦略的重要性が増した。FP/GK交代ルール変更以降、GKの戦術範囲が「ショットストッパー」を超えて攻撃構築者へと拡張。

## GK×DF連携原則（アプヴェア-GK・フェアブント）

**基本の連携**：
- DFがコースを絞る → GKの仕事が減る（残り1/3だけカバーすればよい）
- DFが絞れていない → GKは2/3以上カバーが必要 → 無理が生じる
- GKへの声かけ：「右！」「押し出せ！」でGKの意識をコースに向ける

**5-1との連動（ドイツ式 / コーチ直伝）**：
- トップDFが前進（ヘラウスリュッケン）する → GKは前衛の背後スペースをカバーする意識
- トップDFが引く → GKは外シュートラインを意識してポジション調整
- **ポストの位置**もGKの判断材料（ポストがどこにいるかでカバーの優先順位が変わる）

**6-0との連動**：
- フェアシーベン（シフト）でDFがボールサイドに寄る → GKは逆サイドのスペースを意識
- ヘラウスリュッケン（飛び出し）後 → GKは前衛が戻るまでの一瞬、前衛の裏スペースをカバー

```
DFのブロック位置をGKにリアルタイム共有
  → GKがカバーすべき「残りスペース」を瞬時に把握
  → GKとDFで守備面積を分業（完全補完）
  → 上位レベルでは「動的連動」に発展（シフトに連動してGKも動く）
```

## GKのコーチング機能（声・指示・コート指揮官）

GKはコート全体を正面から見渡せる唯一のポジション。防御だけでなく、コート指揮官として機能する。

**情報共有の3層システム**：

| レベル | タイミング | 内容 |
|---|---|---|
| 試合前情報 | ウォームアップ中 | 相手シューターの特徴・よく使うコース（スカウティング共有） |
| インゲーム声かけ | プレー中・セット前 | DF陣形・マーク・スペースの警告 |
| 非言語信号 | 瞬間的 | 手のジェスチャー・アイコンタクト（ノイズの多い試合中） |

**具体的な声かけ**：

| 声かけ | 使用場面 |
|---|---|
| 「左、出ろ！」 | 左サイドのシューターへの前進指示 |
| 「右、出ろ！」 | 右サイドへの前進指示 |
| 「ポスト、マーク！」 | 6mプレーヤーが危険位置に入ったとき |
| 「マーク交替！」 | スクリーン・クロス後のマーク交替 |
| 「内側に絞れ！」 | DFが外に引き出された後の修正 |
| 「出ろ！プレッシャー！」 | バックプレーヤーへの前進プレッシャー指示 |
| 「後ろ！注意！」 | 背後スペースへの警告 |
| 「前3枚、出ろ！」 | 速攻時の前線人数確認 |
| 「左ウィング空いてる！」 | スローアウト先の指示 |
| 「速く！」 | 即スローアウト（クイックスローオフ）の発動合図 |

**コーナーシュート対応の役割分担**：
- DFがロングコーナー（遠い側）を担当
- GKがショートコーナー（ポスト際）を担当
- → 役割分担を事前に明示することで守備漏れを防ぐ

**各国の声かけ文化**：

| 国 | スタイル | 特徴 |

## 状況別GK戦略

### 5on6守備（自チームが退場1人の状況）

> ⚠ **用語注意**：「5on6守備」＝**自チームが退場1人**でDFが5人しかいない状態（相手OF6人に対してDF5人で守る）。「相手チームがGKをFPに交代した7on6攻撃」とは別状況 → 後者は次の「7on6守備」セクションで扱う。

```
自チームが退場1人 → DFが5人（通常6人）に減少
  → 守備ラインに構造的な穴が生まれる
  → GKの役割変化：
      ① やや後退してポジション（ワイドシュートへの角度カバーを優先）
      ② 「次の1本を止める」mindsetに切り替え（連続失点を防ぐ）
      ③ 集中を一つのボールに絞る（全体を追いすぎない）
      ④ DFへの声かけを増やす（5人DFは情報が届きにくくなる）
```

**5on6でのメンタルリセット**：
- 失点後：次のプレーへの行動ベースリセット（「構える」「前に出る」の身体的動作で気持ちを切り替える）
- 感情的なリアクションをコントロール → DF陣への落ち着いた声かけを継続

### 7on6守備（相手がGK抜き6人OF）

> **視点注意**：このセクションは**自チームGKが相手の7on6攻撃を守る**場面。自チームが7on6で攻撃する（こちらがGKをFPに交代する）場合のOF戦術は `08_game_management.md` 参照。

```
相手がGKポジションにFPを投入 → 自チームは5人DF+GKで対応
  → GKの最重要準備：「ボールを奪った瞬間の即ロングスロー」
  → 相手GKポジションは空き → ロングスローで直接得点チャンス

EHF研究：7on6の守備でボールを奪えた場面のうち約70%が、空いた相手ゴールへ直接スローを狙えるチャンス（速攻得点機会）につながる
→ GKは常に「奪ったら即ロング」を意識して構える
```

声かけ：ボールを奪った瞬間 "キープ!" と叫び、FPに即ロングスローを促す

### スコア差別GK戦略

| スコア状況 | GK戦略 | メンタル対応 |
|---|---|---|
| 大差リード | ルーティンの維持。実験的判断は避ける | 集中切れへの自己管理 |
| 大差ビハインド | 「次のボール（next ball）」思考。一本ずつに集中 | 失点リセット → 即行動（呼吸・身体キュー） |
| 接戦・終盤 | 呼吸ルーティン + 身体的キュー（スタンス確認）+ ポジティブ内対話 + フィールドスキャン | メンタルスタンスの安定を優先 |
| 同点・最終局面 | ボールをリリースするまでシューターの手から視線を外さない（QE効果を引き出す）。先読みより「見て判断」 | Trust三角形（GK-DF-ゲーム）で精神基盤を保つ |

### 失点直後の立て直し動作フロー（GK選手共通）

> 詳細メンタルマネジメント → `18_mental.md`（フォーカス/Refocus・3秒ルール）

```
失点した瞬間：
  Step 1（0〜2秒）: ボールがゴールに入った後、即座にグランドスタンスに戻る
                     → 視線をボール（相手ゴール前）から前方に戻す
  Step 2（2〜5秒）: 短い自己対話（「次」「構える」「OK」）
                     → 感情を外に出さない（相手に「揺らいだ」を見せない）
  Step 3（5秒以内）: DFへの声かけを再開
                     → 「切り替え！」「次！」などで自分とDFを同時にリセット
  Step 4（次のプレー開始前）: スタンス確認・前足部重心の確認
                     → 身体的キューで頭を「次のプレー」に向ける
```

**オルソン式**：「フォーカス・リフォーカス（フォーカス/Refocus）」— ミス後3秒で完全に前を向く。「過去は変えられない。次のボールだけを考える」  
**ノルウェー式**：メストリング（達成感）の維持 — 失点しても「自分はできる」という感覚を身体レベルで保つことがパフォーマンスの安定に直結

## GKスローアウト（速攻起点）の意思決定

> ノルウェー式 イニシエリングスファーゼン（速攻発動フェーズ）の詳細 → `15_norway_gk_df_school.md`

**意思決定の優先順位**：

```
① フリーの選手はいるか（スペースへのパス）
    ↓ いなければ
② バックチェックの状況（相手が戻っているか）
    ↓ 十分戻っていれば
③ スペース（ティーフェンラウムへの長距離スロー）でリスクをとる
```

**スローの距離別判断**：

| 距離 | 判断基準 | ターゲット |
|---|---|---|
| ロングスロー | 相手GKポジションが空き（7on6後）/ バックチェックに大きなギャップ | 空ゴール直接 / ウィングへのロングパス |
| ミドルスロー | 相手が半分戻り始めた段階 | センター付近のフリー選手 |
| ショートスロー | 相手が全員戻っている | 近くの味方で安全に組み立て |

**タイミングの原則**：
- 理想はセービング後1秒以内にスローアウト開始（クイックスローオフへの橋渡し）
- ただし「速く」より「正確に」。誤ったスローアウトはターンオーバーに直結

**クイックスローオフ vs テンポゲーゲンシュトース**：

| 用語 | 意味 | 使用条件 |
|---|---|---|
| クイックスローオフ（速いスタート） | 相手が得点セレブレーション中にGKが即スロー | 相手が気づく前・全員が戻る前の隙を突く |
| テンポゲーゲンシュトース（速攻） | 標準的な速攻（GKスローから2〜3人で仕掛ける） | バックチェックに数的ギャップがある |

**ノルウェー式（イニシエリングスファーゼン）の原則**：
- ウィングへのパスを第一選択肢とする（ウィングが最も速くスペースを作れる）
- GKのスロー後は「次のセットポジションに戻る」ではなく、スロー自体が攻撃の第一手

```
じゃんけん（スローアウトの読み合い）
GKが即ロング（クイックスローオフ）を多用
  → 相手が失点後すぐにバックチェックを開始（スローアウト警戒）
  → GKがショートスローで組み立てに切り替える
  → 相手がバックチェックをゆるめる → 即ロングが再び有効
  → 「どちらをするか」の読み合いループ
```

| **ポジショニング精度** | 射角∠LSR二等分線上に立てているか | 映像で各シーン確認 |
| **アンティシペーション品質** | 先読みのタイミングが早すぎ/遅すぎでないか | 失点シーンの分析 |
| **コーチング（声かけ）量と精度** | 1試合あたりの有効な声かけ数・内容の正確性 | 映像＋音声で確認 |
| **スローアウト判断** | 速攻起点となったGKスローの成功率 | ターンオーバー vs 速攻比率 |
| **失点後リセット速度** | 失点後にGrunststellungに戻るまでの時間 | 3秒以内が目標 |
| **7mスロー vs セット時のパフォーマンス差** | 7mと通常シュートでセーブ率に差がないか | 別集計で比較 |

**コーチングの評価原則**（オルソン式より）：
- 若手GKには「できた」フィードバックを優先
- 課題指摘は具体的な動作で（「見続けろ」→「ボールが手を離れるまで目を外すな」）
- ゲームリーディング（リーディング the Game）は計測しにくいが、失点パターン分析で推定できる

## じゃんけん（GK総合統合）

```
【2タイプのサイクル】
先読み型GK（コミット）
  → 早打ち（フライングシュート）でGKの動作前に放つ
  → GKがコース絞り型に変化（待つ）
  → 引きつけてから角度変化 or 空き2エリアを精度で打つ
  → GKが次第にそのコースを先読みで塞ぐ（先読み型に戻る）
  → 早打ちに戻る

【GK vs シューターの認知戦】
GKがアンティシペーションでリリース193ms前に動く
  → シューターがそのタイミングを読んで逆コースへ
  → GKが中立重心を保ち読みを遅らせる（QE延長）
  → シューターが「GKはどちらに動くか」読もうとする → 相互読み合いのエスカレーション

【左利きの非対称じゃんけん】
左利きシューター登場
  → GKのアンティシペーション精度が低下（経験不足）
  → 左利きシューターが優位になる
  → GK・チームが左利き専用の対策練習 → 精度が上がる
  → 左利きが希少性を失うと右利き並みの対策が完成

【大型GK vs 高跳躍シューター】
200cm大型GK（スペイン型）が面積でコースを消す
  → 高跳躍ジャンプシュートで上コーナーを狙う
  → 大型GKがシュリットシュプルングテクニックで上を塞ぐ
  → 低コース（股下・腰）を狙う
  → 大型GKが低姿勢に構える → 上コーナーが空く → 高跳躍に戻る

【オルソン式 リーディング the Game サイクル】
GKが「技術 → シュートの読み → ゲーム全体の読み」を習得
  → 相手チームがGKのパターンをスカウティング
  → GKが個人別戦略を試合ごとに変える（映像分析 × 統計で設計）
  → 相手も対策を変える → 読み合いの軍拡競争（「脳の勝負」）
  「脳の勝負」はフィジカルの衰えとは無関係 → キャリアが延びる

【オルソン式 メンタルのじゃんけん】
GKがTrust（信頼）を持ってプレー → シューターが「読まれた」と怖気づく
  → シューターが意識しすぎてミスシュート
  → GKがミス → フォーカス/Refocus（7要素）で即リセット → Trust回復
  → 相手が「GKは揺らいだ」と見抜いてプレッシャーを与える
  → Trust の三角形（GK-DF-ゲーム全体）でチーム全体が補完 → 循環

【GK声かけ（コート指揮官）のじゃんけん】
GKが「出ろ！」→ DFが前進してプレッシャー → バックがジャンプシュートを選ぶ
  → DFが出すぎると背後スペースが開く → ポストへのパスが通る
  → GKが「絞れ！」→ DFが引く → バックがシュートスペースを確保
  → 声かけ1つでDFの陣形が変わる → 最適な声かけのタイミングと内容が問われる
  「GKはコートを最もよく見えている」→ 声かけの精度がそのままDFの質になる

【5on6退場時のじゃんけん】
GKが後退してワイドカバー（角度消し優先）→ 中距離コースが空く → バックがミドルシュート
  → GKが前進してミドルを閉じる → サイドへのパスが通る
  → 5人DFの「詰める/引く」が声かけなしに混乱 → GKが統制の要になる
  退場明け（相手が数的均等に戻る）→ 元の6-0連動に戻る準備を声かけで促す

【スローアウトのじゃんけん（再掲：総合統合版）】
GKが即ロング（クイックスローオフ）を多用
  → 相手が失点後すぐにバックチェック開始 → ロングが封じられる
  → GKがショートスローで近くから組み立てる → 相手がバックチェックをゆるめる
  → 即ロングが再び有効に → 読み合いのループ
  「失点した瞬間こそ、GKの目はすでに次の攻撃先を探している」
```

## トピックM：GKが物理的に守れない面積の定量計算

> **このセクションの位置づけ**：「幹：GKが直面する物理的制約」の数値的補強。GKが「なぜポジションと先読みが必要か」を面積・距離・時間の3軸で数字化する。

### ゴール面積とGKカバー可能面積の比較

```
ゴール全体面積：
  幅 3.00m × 高さ 2.00m = 6.00㎡

GKの体（静止状態）がカバーできる面積の推計：
  ・体幅（肩幅）：約 0.60〜0.65m
  ・両腕展開スパン：身長 × 約1.03
      190cm GK → スパン約 1.96m
      180cm GK → スパン約 1.85m
  ・立位でのカバー高さ（頭頂〜膝）：約 1.30〜1.40m

  体 + 腕スパンでの横幅カバー率：
      190cm GK（スパン1.96m）→ ゴール幅300cmの約 65%
      180cm GK（スパン1.85m）→ ゴール幅300cmの約 62%

  → 残り約 35〜38%（横幅換算で約 105〜115cm）は
    GKが「静止状態で体に届かないエリア」
```

### 反応時間 × 動作速度 = 到達可能距離の計算

```
6m先からの 90km/h シュートのフライトタイム：約 240ms

GKが「見てから動く」場合の所要時間：
  視覚処理：75ms
  判断    ：50〜75ms
  運動開始 ：100〜125ms
  ─────────────────
  合計    ：225〜275ms → ボールはすでにゴール内

GKが反応（動作開始）してから240ms以内に移動できる距離：
  ・腕のみの振り出し速度（肩から手先まで）：
      動作開始から 200ms で腕を 0.5〜0.7m 伸展可能（⚠確認推奨）
  ・重心移動（横ステップ）：
      動作開始から 240ms で横に 0.3〜0.5m 移動可能（⚠確認推奨）

→「純粋な反応」で届くゴール拡張距離：約 0.3〜0.7m（方向・高さによる）
→ 6mシュートに対して「静止状態＋反応移動」で実際にカバーできるゾーン：
   ゴール幅の約 75〜80%（横幅換算で 225〜240cm）
→ 「物理的に反応不可能なゾーン」：残り 20〜25%（60〜75cm）

⚠確認推奨：腕伸展・重心移動の到達距離は研究によって数値が異なる。
　上記はヴァニャ・ラディッチの指導データ + 一般的バイオメカニクス推計値の組み合わせ。
　PMC3590836（2012）は時間軸データを提供するが到達距離の直接数値は明示していない。
```

### 「物理的に反応不可能なゾーン」の位置

```
GKが正面向き・基本スタンスで立っている場合、
下記エリアが最も「反応到達が困難」なゾーンとなる：

  ┌─────────────────────────────────────┐
  │ ●左上角  │         │  ●右上角 │ ← 腕を最大伸展しても届きにくい高さ
  │（最難）  │  カバー可 │（最難）  │
  │──────────────────────────────────────│
  │         │         │         │
  │  カバー可 │ GK中心   │  カバー可 │ ← 体・腕スパンの範囲
  │         │         │         │
  │──────────────────────────────────────│
  │ ●左下角  │  カバー可 │  ●右下角 │ ← 低い外角コース（重心移動が必要）
  └─────────────────────────────────────┘

  最も守りにくい4ゾーン：4コーナー（特に上コーナー2箇所）
  上コーナー：ゴール最上部（高さ1.7〜2.0m）× 両サイド端
  → ここへの正確なシュートは「物理的反応限界」に最も近い

  ただし6mシュートでは、至近距離から上のすみ（上コーナー）へ正確に通すこと自体がシューターにも難しい（じゃんけん構造）
```

### 前進戦術の数値的効果

```
GKのゴールライン前方距離と見かけのゴール幅の関係（タンジェント関数による推計）：

  シューター位置：6m正面（センターバック）

  GKがゴールライン前方 1.10m（エリート実測平均）にいる場合：
    シューターからGKまでの距離 = 6.00 - 1.10 = 4.90m
    GKのゴール端までのスパンが射角に占める比率 → 基準値

  GKが 1.44m 前進した場合（バイオメカニクス最適値）：
    シューターからGKまでの距離 = 6.00 - 1.44 = 4.56m
    → 距離が 4.56/4.90 ≒ 0.93 倍に短縮
    → 射角内でGK体幅が占める角度は距離に反比例して増大
    → 概算で射角の約 7〜10% 追加縮小効果（⚠確認推奨）

  GKが 2.50m まで前進した場合（6mシュート対応の積極前進）：
    距離 = 3.50m
    → 1.10m時と比較して射角内の有効体幅カバーが約 40% 増加
    → ヴァニャ・ラディッチの「30〜40%アングル縮小」と対応

  要約：
    前進 1m あたりの射角縮小効果 ≒ +5〜10%（距離・角度・体格による）（⚠確認推奨）
    積極前進（+1.4m以上）で射角縮小 30〜40%（ヴァニャ・ラディッチ実証）
```

### GKが取るべき戦略的示唆（数値から導く）

物理計算が示す4つの含意：

| 含意 | 根拠 | 実践への反映 |
|---|---|---|
| **ポジションが最優先** | 静止状態での体カバー率（62〜70%）は動いてのカバーより確実 | まずポジション → 次に前進 → 最後に先読み |
| **前進は面積ではなく角度を変える** | 前進でGK体積が変わらなくても射角が縮小する | 前進タイミング精度が面積計算より重要 |
| **4コーナーは「死角」として受け入れる** | 物理的反応限界ゾーンを消そうとすると他が空く | 4コーナーは「打てたら相手を褒める」覚悟で対応 |
| **先読みなしでは数字上セーブ不可** | 反応時間（225〜275ms）＞フライトタイム（240ms） | アンティシペーションはオプションではなく必須インフラ |

```
シューター側への示唆：
  「物理的反応不可能なゾーン」は存在するが、
  そこへの正確なシュートはシューターにも高難度。
  → GKが「難しいコースを狙わせる」戦略として
    あえてコーナーを少し空けてシューターを誘う手もある
  → 「難しいコース = GKに届かない = 決まる」ではない
    コースが狭く高難度 = シューターもミスしやすい（じゃんけん）
```

*出典・参照：PMC3590836（2012）反応時間データ / Journal of Applied Biomechanics 32(5) 2016 ポジショニング実測値 / ヴァニャ・ラディッチ コーチング（vanjaradic.fi）アングル縮小データ*

## トピックN：GKが持つ「DFに見えない情報」の体系化

> **このセクションの位置づけ**：「GKのコーチング機能（声・指示）」の認知的基盤。なぜGKの声かけが価値を持つかを、GKが持つ情報優位性の観点から構造化する。

### GKがゴールライン後方から見えて、DFには見えない情報の分類

```
GKの立ち位置の特性：
  ・ゴールライン付近（エリート実測 1.10〜1.44m 前方）から全コートを正面視
  ・自分はOF選手に向き合っているため、OFの背面・サイドが全て見える
  ・DFはOFに向き合うため、「自分の背後のOF配置」が見えにくい
```

**分類1：OFの背番号・ポジション認識（DFの背後情報）**

```
DFがOFと向き合っている場合：
  → DFの視界に「自分が守っているOF」は見える
  → しかし「自分の後ろにいる別のOF」が見えない
     （特に：フォローアップのバック選手 / 遅れて走り込む選手）

GKには見える情報：
  → どの番号のOFがどのポジションにいるか
  → 複数OF選手が重なった場合の奥にいる選手
  → DF陣形の外側にいるウィング選手の位置

典型的な場面：
  DF2人が前出たとき → 背後に空いたスペースにOFが走り込む
  → GKが最初に気づく → 「後ろ！スペース！」の声
```

**分類2：ピボットへのパスコース（DFがピボットをマーク中の死角）**

```
DFがピボット（ポスト）をマークしている場合：
  → DFはピボットを背中方向に押さえるため
    「ボール保持者（バック）からGK方向」を向きにくい
  → ピボット-GK間のパスコースがDF視界から外れやすい

GKには見える情報：
  → バック選手の助走方向・ボール保持位置
  → DF-ピボット間にパスコースが「空いているか・塞がれているか」
  → バックがパスを出すか・シュートを選ぶかの直前の身体の向き

典型的な場面：
  バックが助走してきてDF手前でパスを選択
  → GKがそのコースを事前に「空いてる！パス注意！」で警告可能
  → DF陣が声で再配置（ヘルプDF投入）可能
```

**分類3：バックプレーヤーの助走速度・ボール保持位置（DF一歩早い読み）**

```
DFがバックプレーヤーを止めようとする場合：
  → DFはバックを追いながら接触タイミングを測る
  → 視線がバックの身体中心に集中 → 「ボールを持った腕の位置」を
    細かく追跡する余裕が小さい（⚠確認推奨）

GKには見える情報：
  → バックの助走速度（速いほど早打ち傾向）
  → ボールを持った腕の高さ・引き方（シュートの種類の予測根拠）
  → ジャンプの踏み切り方向（シュートコースの先読み最優先キュー）

GKのアドバンテージ：
  → DFが「接触を維持する」ことに意識が向く場面で
    GKは「シュートを読む」ことに専念できる
  → Huesmann et al. (2023, PMC10591308)：
    エキスパートGKはチーム全体の攻撃タスク配分を事前情報として活用する
    → 「今日は誰が主に打っているか」を試合序盤に把握
    → DFが個々のマッチアップに集中している間、GKが全体のシュート役割を追跡
```

**分類4：DFフォーメーション全体の穴（俯瞰的位置から見える構造的ギャップ）**

```
DFが陣形を組んでいるとき：
  → 各DFは自分の担当エリア・担当OFを視認
  → 「陣形全体として今どこが薄いか」を把握するには
    全DFを同時に視界に入れる位置が必要

GKには見える情報：
  → DFがボールサイドに引き寄せられてファーサイドが空く瞬間
  → DF陣形の横幅が詰まりすぎてウィングへのパスコースが空く瞬間
  → DFの1人がヘラウスリュッケン（飛び出し）した後の背後スペース
  → ピボット-DFの数的バランス（誰がフリーか）

典型的な場面：
  5-1でトップDFが前出た直後 → 背後の穴を「GKのみが俯瞰で即認識」
  → 「後退！戻れ！」または「背後注意！」で穴を即指示

研究根拠：
  「whether defense players are present in a goalkeeper's field of vision will 
  influence which kinematic and contextual information are available」
  （PMC10591308 / Huesmann et al. 2023）
  → DFの配置がGKの知覚に影響を与える = GKはDFの位置から情報を追加で得ている
```

### 「GK = チームの情報ハブ」という概念の各国での扱い

### GKからの情報が最も価値を持つ場面3つ

**場面①：DFがヘラウスリュッケン（飛び出し）した直後の背後スペース**

```
状況：トップDFまたはサイドDFが前に出てプレッシャー
問題：DFが前出た瞬間、自分の背後に空きスペースが生まれる
     → そのDFは背後を見えない（前向きにOFに向かっているため）

GKのアドバンテージ：
  → 背後スペース + 走り込むOF選手を「唯一リアルタイムで認識できる」立場
  → 声かけ「後ろ！スペース！」が0.5〜1秒の先手を生む
  → 声がなければ失点。声があれば隣のDFがカバーできる可能性
```

**場面②：ピボットへの浮きパスが通る直前**

```
状況：バックプレーヤーがDF手前から低いパスではなく「浮かせたパス」を選択
問題：DFはバックプレーヤーを追っていてピボット方向の視線が乏しい
     → ピボットがDFの背後・脇でフリーになっていても気づきにくい

GKのアドバンテージ：
  → ピボットのポジション + バックの腕の動き（パス軌道）を同時視認
  → 「ポスト！背後！」で0.3〜0.5秒前に警告
  → DFが微修正できれば浮きパスを潰せる
```

**場面③：相手が速攻を仕掛けてくる切り替えの瞬間**

```
状況：GKがセービングまたはキャッチした直後、相手が即速攻を開始
問題：DF選手はシュートシーンに集中していたため
     「今、自分の背後に相手がいるか」を即時判断できない

GKのアドバンテージ：
  → セービング直後にコート全体を即視認（視線を変えるだけでよい）
  → 「戻れ！左2人空いてる！」で速攻を即時判断・コントロール
  → GKスローアウトの方向決定（シュネレ・ミッテ vs ショートスロー）の
    情報も同時に処理
  → ノルウェー式の「セービングで終わらず即速攻起点になる」思想の
    情報的根拠がここにある
```

**じゃんけん（情報ハブとしてのGKの限界）**：
```
GKが情報をDFに伝える → DFが前出る → GKの死角（DF背後）がさらに広がる
  → GKが「出すぎ」を警告 → DFが躊躇する → OFが前進スペースを得る
  → GKがまた「出ろ！」の声かけ → 声かけの精度と信頼性が問われる

GKの声が多すぎる場合：
  → DFが「また出るか」と慣れる → 重要な声かけが埋もれる
  → 情報の「質 vs 量」のバランスが問われる（⚠確認推奨：最適声かけ頻度の研究は限定的）
```

> **コーチへの示唆**：GKの「コーチング（声かけ）量と精度」をパフォーマンス評価指標として計測する（→「エリートGKパフォーマンス基準」セクション参照）。声の数だけでなく「DFが動いたか・動かなかったか」で精度を評価することが重要（⚠確認推奨：声かけ精度の標準的評価手法は未確立）。
