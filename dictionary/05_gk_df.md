# GK・DF陣形

---

## GK（ゴールキーパー）

### コーチ補足知識（直伝）

**3方向の法則**
- ゴールは常に「左・中・右」の3エリアに分割される
- GKは体のサイズと位置から、常に2/3を閉じていて、1/3が開いている
- シューターはその「開いている1/3」を見つけて打つのが基本戦略
- 縦も同様：上/中/下の3エリア。常に2/3が閉、1/3が開

**GKタイプ（名称改定版）**

| 旧名称 | 新名称 | 英語/独語 | 本質 |
|---|---|---|---|
| 波型 | **先読み型（アクティブ型）** | Anticipation / Aktives GK | 予測して先に動く |
| スポット型 | **コース絞り型（ポジション型）** | Position / Passives GK | コースを塞いで待つ |

---

### 先読み型GK（旧：波型）― 詳細メカニズム

**何をしているか**：
```
シューターが助走・踏み切り
 ↓
GKが「このシューターはあそこに打つ」と予測
（予測根拠：助走の角度・利き手・過去のパターン・体の向き）
 ↓
ボールが手から離れる前にGKが先にそのコースへ動く
 ↓
シューターが打った瞬間にはGKはもうそこにいる
```

**なぜ有効か**：人間の反応速度（0.15〜0.25秒）では、近距離の速いシュートには「見てから動く」では間に合わない。だから「予測して先に動く」戦略を取る。

**弱点（シューター視点）**：
- GKが動くタイミングより**早く**打つ（早打ち）→ GKはまだ動いていない or 動き始めだけ
- GKが「右に動く」予測をした瞬間に**左**に打つ（逆を突く）
- バックハンドフェイク → GKのコース予測を完全に狂わせる

**早打ちの原理（コーチ直伝）**：
- GKが「シューターの最高到達点に先取りしにいく」イメージで動く場合
- 早打ち（最高点に達する前に打つ）はGKの意識の外のシュートになる
- GKは最高点を想定して動いているため、それより低いタイミングへの対応が遅れる

**じゃんけん構造（先読み型GKとの戦い）**：
```
最高点シュート → GKが先読みで塞ぐ → 早打ちに変える → GKが早打ちに慣れる
 → さらに早いタイミングに変える or 最高点に戻す（GKが混乱）
 → バックハンドフェイクを加える → GKの予測が不安定になる
```

---

### コース絞り型GK（旧：スポット型）― 詳細メカニズム

**何をしているか**：
```
GKは動かない。ただし「最初から体でコースを塞いでいる」

左右×上下 = 6エリア
 ┌───────────────┐
 │ 左上 │ 中上 │ 右上 │
 │──────────────│
 │ 左下 │ 中下 │ 右下 │
 └───────────────┘

GKの体が「4エリアを塞ぎ、2エリアを意図的に空けている」
シューターはその「空き2エリア」を見つけて正確に打つ必要がある
```

**なぜ有効か**：動かないことで「先読みを外れるリスク」がない。体のサイズと立ち位置で常に2/3を守れる。

**弱点（シューター視点）**：
- GKを**動かす**（引きつける・フェイクを入れる）→ GKが動いた方向の逆が空く
- 「空き2エリア」を正確に特定して打つ（精度が問われる）
- **低いコース**：コース絞り型GKは高さを変えにくい。低めが空いていることが多い

**角度変化の原理**：
- ジャンプしながら体の向きを変えて「GKが読んでいたコースの逆」に打つ
- GKが「中央」を塞いでいれば → 端の低いコースを狙う
- GKが「右下」を強調して塞いでいれば → 左上（GKの反対上）を狙う

**じゃんけん構造（コース絞り型GKとの戦い）**：
```
「空き2エリア」に正確に打つ → GKが次第にそのコースを先読みで塞ぐ（先読み型に変化）
 → 引きつけてから角度変化で逆を突く → GKが動くようになる（先読み型に変化）
 → 早打ちも組み合わせる → GKが混乱してどちらか一方のタイプに戻る
```

---

**GKタイプ別 OFの対策まとめ**

| GKタイプ | 基本対策 | 応用対策 | 有効フェイント |
|---|---|---|---|
| **先読み型** | 早打ち（GKが動く前に放つ） | GKの動き始めを見てその逆 | ①シュートフェイク / ⑦バックハンドフェイク |
| **コース絞り型** | 引きつけてから角度変化 | 空き2エリアを特定して精度で打つ | ①シュートフェイク（GKを動かす）/ ③ボディフェイント |
| **不明（試合序盤）** | 3〜5本観察してタイプを特定 | 低コース（腰〜膝）は両タイプに有効 | まず普通に打って動きを確認する |

> ※フェイント8種類の詳細は `dictionary/02_individual.md` 参照

**GK×DFの連携原則**
- DFがコースを絞る → GKの仕事が減る（残り1/3だけカバー）
- DFが絞れていない → GKは2/3以上カバーが必要 → 無理が生じる
- GKへの声かけ：「右！」「押し出せ！」でGKの意識をコースに向ける

## DF陣形

### 0-6（シックスゼロ）

**構造**：6人全員がゴールライン付近に一列に並ぶ

| 長所 | 弱点 |
|---|---|
| ゴール前の密集度が高い | 外からの遠距離シュートに弱い |
| ピボットへのパスを防ぎやすい | 前線プレッシャーが薄い |
| 体力消耗が少ない | 速攻発動が遅れる |

**0-6の3基本動作（ドイツ語専門用語）**：

| 動作 | ドイツ語 | 内容 |
|---|---|---|
| シフト | **Verschieben**（フェルシーベン） | ボール側へラインが一体となって横移動。ボールサイドに密集を作る |
| 飛び出し | **Herausrücken**（ヘラウスリューケン） | インナーDFがボールを持つバックに前進プレッシャー |
| 受け渡し | **Übergeben**（ユーバーゲーベン） | 飛び出した後、隣のDFにマークを渡してラインを維持 |

**Tiefenraum**（ティーフェンラウム）= ゴール前の危険なスペース。0-6の目的はここを最小化すること。

**崩し方（OF視点）**：
- Verschieben（シフト）の直後 → 逆サイドへ素早い展開でスペースを使う
- Herausrücken（飛び出し）の背後 → ピボットを走り込ませる
- Übergeben（受け渡し）の一瞬 → クイックパスで瞬間的なマーク外れを突く
- 遠距離から積極的に外シュート → DFラインを引き出す

**じゃんけん**：
```
0-6ゾーン → 外シュートを打たれる → 1-5に変更 → 前衛の裏をピボットで使う → 0-6に戻る
Herausrückenで前に出る → 背後にスペース → ピボット走り込み → Herausrücken禁止 → 外シュートが打ち放題に戻る
```

### 1-5（ワンファイブ）

**構造**：前衛1人＋後衛5人

| 長所 | 弱点 |
|---|---|
| 前衛でシューターへプレッシャー | 前衛の裏にスペースができる |
| 外シュートを封じやすい | 1人の前衛が抜かれると数的不利 |
| 速攻カウンターを誘いやすい | 前衛選手の体力消耗が激しい |

**前衛1人の役割**：
- シューターのシュートを遅らせる（ブロック）
- パスコースを切る
- ゾーンカバーに戻る責任も持つ（←重要：前衛が戻れないと6-5の穴になる）

**トップDFの連動（コーチ直伝）**：
- トップDFは単独で動くのではなく、**裏のフルバックDFとの位置関係**を常に把握した上で動く
- **ポストの位置**も連動判断の要素（ポストがどこにいるかでカバーの優先順位が変わる）
- カバー（穴を埋める動き）と牽制（相手を動かす動き）を状況に応じて使い分ける
- 試合の流れによって判断するもの。数式的に「この場面はカバー」ではなく流れを読む

**崩し方（OF視点）**：
- 前衛を引き付けてからピボットへパス（前衛の裏のスペース）
- 前衛を左右に動かして疲弊させる

---

### 2-4（ツーフォー）

**構造**：前衛2人＋後衛4人

| 長所 | 弱点 |
|---|---|
| 前線で2人のプレッシャー | 後衛4人だとゴール前が薄い |
| センタープレーヤーを挟める | 横幅をカバーしきれない |

---

### 3-2-1（スリーツーワン）

**構造**：前3人・中2人・後1人（縦に積み上げる）

| 長所 | 弱点 |
|---|---|
| 縦方向のプレッシャーが強い | 横の連係が難しい |
| 速攻からそのまま守備移行しやすい | 全員の連動が必要で習得が難しい |

---

### 3-3（スリースリー）

**構造**：前衛3人・後衛3人

| 長所 | 弱点 |
|---|---|
| 前後のバランスが取れる | 前後の分断が生じやすい |
| 前線での数的有利を作りやすい | 後衛3人では横幅のカバーが難しい |

---

## DF陣形の選択フレーム

```
相手に外シュートが多い → 1-5 or 2-4（前線プレッシャー）
相手にピボットが強い → 0-6（ゴール前密集）
速攻を出したい → 3-2-1（縦の流れを作る）
エースを止めたい → マンツーマン担当を指定
相手に合わせた柔軟変更 → ハーフタイム修正
```

## DF陣形じゃんけん（全体）

```
0-6 → 外シュートに弱い → 1-5 → 前衛の裏が弱い → 0-6に戻る
マンツーマン → スクリーンに弱い → ゾーン → 外シュートに弱い → マンツーマンに戻る
3-2-1 → 縦スペースに弱い → 0-6 → 横スペースに弱い → 3-2-1に戻る
```
