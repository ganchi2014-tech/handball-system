# 個人対決：1:1・2:2・スクリーン・勝ち位置

---

## フェイント8種類（OF個人技術の核心）

> **フェイントの本質**：相手（DFまたはGK）を動かして、動いた方向の逆を使うこと。動かさなければフェイントではない。

---

### フェイント①　シュートフェイク（Wurfbewegungstäuschung）

**動作**：シュートするモーション（腕を引く・体を回す・肩を入れる）を見せて実際には打たない

| 項目 | 内容 |
|---|---|
| **長所** | GKとDFの両方を同時に騙せる。使用機会が最多（常にシュートの意図を見せられる）。実際のシュートと組み合わせることで相乗効果が最大 |
| **効くDF/GK** | 前に詰めてくるDF（プレッシャーDF・Herausrücken）/ **先読み型GK**（先に動くGKはシュートモーションで釣れる） |
| **じゃんけんの逆** | 「シュートが来るまで動かない」静的待機型DFには効かない → そこにはドライブで突く |
| **カウンターDF** | 腕の動きではなく**重心（骨盤の向き）**を見て反応する。腕だけのフェイクは重心には現れない |
| **カウンターGK** | コース絞り型GKは動かないので、フェイクには引っかからない。フェイク後に本当に打たれるまで待てる |

**じゃんけん構造**：
```
シュートフェイク → DFが前に出る → 背後にスペース → ドライブ突破
ドライブ突破が続く → DFが引く → シュートフェイクが不要になりそのまま打てる
```

---

### フェイント②　パスフェイク（Passtäuschung）

**動作**：パスを出す方向・モーションを見せて実際には出さない（または全く逆方向に出す）

| 項目 | 内容 |
|---|---|
| **長所** | 隣のDFを横に動かしてスペースを作れる。DFがインターセプト狙いで出てきた瞬間に逆を使える |
| **効くDF/GK** | インターセプトを積極的に狙うDF（前に出てくるDF）。連動・カバーが速いDF（動く前に別の選択肢を取れる） |
| **じゃんけんの逆** | 「手の動きには動かない」慎重なDFには効かない → そこにはドライブかシュートフェイクで対応 |
| **カウンターDF** | **手の動きではなく腰（重心）の向き**を見る。パスフェイクは手だけで偽れるが腰は偽りにくい。位置を保つ（動かない判断） |
| **GKへの影響** | パスフェイクはGKより「隣のDF」を動かすためのフェイント。GKには直接効かないことが多い |

---

### フェイント③　ボディフェイント（Körpertäuschung / 重心フェイント）

**動作**：体の重心・肩・上半身を一方向に傾けてDFをその方向に動かし、逆方向に突破する

| 項目 | 内容 |
|---|---|
| **長所** | 小さい動作でも高い効果。ドライブの前段階として汎用性が最高。「目線フェイント」と組み合わせると二重効果 |
| **効くDF** | 重心の傾きに反応するDF（目線・上半身を見ているDF）。反応速度が速いDFほど引っかかりやすい |
| **じゃんけんの逆** | 重心フェイントを使い続けると → DFが「腰を見る」ようになる → 次はステップフェイントに切り替える |
| **カウンターDF** | **腰（骨盤の向き）**を見る。上半身は偽れるが骨盤の向きは偽りにくい。上半身だけが傾いて骨盤が動いていない → フェイク確定 |

---

### フェイント④　ステップフェイント（Schritttäuschung / 踏み込みフェイント）

**動作**：踏み込む方向・タイミングを偽る（内側に踏み込む素振りを見せて外に出る、またはその逆）

| 項目 | 内容 |
|---|---|
| **長所** | DFの足を釘付けにしてからドライブできる。踏み込みは「本番の予告」なのでDFは反応せざるを得ない |
| **効くDF** | 足の向き・踏み込みを見て反応するDF。体勢が崩れやすい（重心が移動しやすい）DF |
| **じゃんけんの逆** | DFが足を見なくなる（骨盤を見るようになる）→ 今度はボディフェイントが効く |
| **カウンターDF** | 足ではなく**骨盤・腰の向き**を基準にする。踏み込みに対して「一拍遅らせて反応する」（フェイクに食いつかない） |
| **注意** | ステップフェイントは踏み込みの「深さ」が重要。浅い踏み込みはフェイクと見抜かれやすい |

---

### フェイント⑤　スピードフェイント（Tempowechsel / テンポ変化）

**動作**：急加速→急停止→再加速（またはその逆）でDFの反応タイミングをずらす

| 項目 | 内容 |
|---|---|
| **長所** | 繰り返し使える（タイミングが毎回変わる）。特定の「見る場所」に依存しない。体力があれば試合終盤でも有効 |
| **効くDF** | スピードで対応するDF（相手の速さに合わせて動くDF）。先読みが得意なDF（タイミングを狂わせられる） |
| **じゃんけんの逆** | DFが「距離を保って追いかけない」戦略に切り替えると効果が落ちる → 別のフェイントと組み合わせる |
| **カウンターDF** | 相手のスピード変化より先にポジションを取る。「追いかけない」選択（相手のペースに乗らない）。距離を一定に保つ |

---

### フェイント⑥　ターンフェイント（Drehtäuschung / ターン方向偽り）

**動作**：内側に入る・またはターンする動きを見せて実際には逆方向に動く。または体を回して背を向けて死角を作る

| 項目 | 内容 |
|---|---|
| **長所** | DF背後のスペースに入り込める。ポストプレーヤーとの相性が特に良い（ゴール前での展開）。密着されるほど効果が大きい |
| **効くDF** | 密着マーク型（マンツーマン）。内側を切っているDF（クローズポジション）。ターン方向を読んで先回りしてくるDF |
| **じゃんけんの逆** | DFが「ターンを防ぐために先に回り込む」と → ターンフェイントからの逆方向突破が生きる |
| **カウンターDF** | ターン方向を先に塞ぐポジショニング（先に体を入れる）。マークを手放して下がる（ターンを使えない距離を作る）。Übergeben（受け渡し）と連動させる |

---

### フェイント⑦　バックハンドフェイク（Rückhandtäuschung）

**動作**：バックハンドシュート（または逆手パス）のモーションを見せて通常のシュート/パスに切り替える（またはその逆）

| 項目 | 内容 |
|---|---|
| **長所** | GKが予測するコースを完全に裏切れる。非利き手を「脅威」として認識させることで利き手の選択肢も増える |
| **効くDF/GK** | **先読み型GK**（コース予測が基本戦略 → バックハンドモーションで予測を完全に狂わせられる）。パスコースを読んでいるDF |
| **前提条件** | バックハンドが「実際に使える技術」でないと脅威にならない。練習量が必要 |
| **じゃんけんの逆** | コース絞り型GK（コースを塞いで待つ）には限定的。GKが動かないならバックハンドの方向を変えても塞がれるコースは変わらない |
| **カウンターGK** | コース絞り型GKはフォームで判断せず体の向きで守るため、バックハンドフェイクに影響されにくい |

---

### フェイント⑧　ダブルフェイント（Doppeltäuschung / 2連フェイント）

**動作**：フェイントを2回連続で使う（例：パスフェイク→シュートフェイク→本シュート、内→外→内 の2回切り）

| 項目 | 内容 |
|---|---|
| **長所** | 「1回のフェイントには耐えられる」全国レベルDFを超えられる。フェイント後の「再読み直し」が遅いDFに有効 |
| **効くDF** | 1回のフェイントは耐えられるが2回目への対応が遅いDF。「フェイントを読んで動いた」DFが次の反応に戻れる前に仕掛ける |
| **前提条件** | 2回フェイントをかけても体の軸が崩れない体幹が必要。1回目のフェイントに自分も乗りすぎない |
| **カウンターDF** | ダブルを予測して「2回目で動く」（1回目で動かない練習）。距離を保って両方のフェイントを関係なくポジションを維持する |

---

### フェイント全体のじゃんけんマップ

```
DFが重心・腕を見ている → シュートフェイク・パスフェイクが効く
DFが重心だけを見る → ステップフェイント・ターンフェイントが効く
DFが足を見る → ボディフェイント・スピードフェイントが効く
DFが1回のフェイントに耐える → ダブルフェイントが効く
DFが先読みで動く → スピードフェイントが効く（タイミングをずらす）

先読み型GK → シュートフェイク・バックハンドフェイクが効く（予測を裏切る）
コース絞り型GK → 引きつけてから打つ（GKを動かしてから逆を打つ）
```

### 実戦でのフェイント選択フレーム

```
相手DFが「何を見ているか」を確認する
→ 腕・上半身を見ている   → ①シュートフェイク / ②パスフェイク
→ 重心（腰）を見ている   → ④ステップフェイント / ⑥ターンフェイント
→ 足の踏み込みを見ている → ③ボディフェイント / ⑤スピードフェイント
→ 1回耐えてくる         → ⑧ダブルフェイント
→ 先読みで飛んでくる     → ⑤スピードフェイント（タイミング変化）
→ GK対策が必要         → ①シュートフェイク / ⑦バックハンドフェイク（先読み型GK向け）
```

## 1:1（一対一）

### OFの原則
1. **勝ち位置を先に取る**：ボールを受ける前に有利な位置を確保
2. **DFの重心を動かす**：フェイントで重心を崩してから仕掛ける
3. **コンタクト（山側/谷側）を活用**：→ `01_basics.md` 参照

### 1on1の選択肢
| 状況 | 推奨行動 |
|---|---|
| クローズDF（内側） | 外へのフェイントから外シュート、または逆に内突破 |
| ワイドDF（外側） | 内突破またはピボットへのパス |
| DFが前傾姿勢 | バックステップしてシュート |
| DFが後傾姿勢 | 前への加速でドリブル突破 |
| DFが横に広い | 縦（前方）への突破 |

### 1on1のじゃんけん
```
ドリブル突破（縦）
  → DFが縦を切る（体を張る）
  → フェイクから外シュート
  → DFが外シュートを警戒（距離を取る）
  → ドリブル突破のスペース発生
  → ドリブル突破（縦）に戻る
```

## 2:2（二対二）

### OFの原則
- 2人の連係でDFのどちらかを崩す
- スクリーンを使って1on1の局面を作る
- ピック＆ロール（スクリーン＆ロール）が基本

### 2on2のパターン
| パターン | 説明 |
|---|---|
| ピック＆ロール | スクリーンしてからゴール方向に転身してパスを受ける |
| ピック＆ポップ | スクリーンしてから外に開いてシュートチャンス |
| クロス | 2人が交差することでDFのマーク交換を強制 |

### DFの2on2対応
- スイッチ（マーク交換）：即座に対応できるが、ミスマッチのリスク
- ファイトオーバー：マークを外さない（体力消耗大）
- ヘルプ：3人目のDFがカバー（チームDF必須）

## 勝ち位置（Winning Position）

> ボールを受ける前に「相手DFより先に有利なポジション」を取ること

### 重要性
- 勝ち位置にいれば、ボールが来た瞬間にすでに選択肢がある
- 勝ち位置を取れていないと、受けてから考えることになり判断が遅れる
- 「先に読んだ方が選択肢を持てる」の具体的な体現

### 勝ち位置の取り方
1. DFの動きを先読みして、DFが移動する前に目標地点に入る
2. オフボール（ボールを持っていないとき）から準備する
3. パサーとのアイコンタクトを活用する

### 勝ち位置とDFの攻防
```
OFが勝ち位置を先取する
  → DFが後追いで対応
  → OFがボールを受けた瞬間にすでに選択肢がある
  → DFが先読みして勝ち位置に先回り
  → OFが別の勝ち位置を探す（動き直し）
  → 「先に読んだ方が勝つ」の無限ループ
```
