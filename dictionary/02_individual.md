## フェイント8種類（OF個人技術の核心）

> **フェイントの本質**：相手（DFまたはGK）を「動かす」こと。動かさなければフェイントではない。動いた方向の逆が使える。

**8種の全体像**

| # | 名前 | 何をする動き（一言） | 主な用途 | クロアチア/ユーゴ語 |
|---|---|---|---|---|
| ① | 廻旋フェイント | DFの手を腕の回転で外し差し込む | 接触DF（ホールディング）への対処 | Lažnjak rotacijom |
| ② | ターンフェイント | 体をくるっと回転させる | DF背後への侵入 | Okret / Rotacija |
| ③ | ダブルフェイント | フェイントを2連続でかける | 1回耐えるDFへの対処 | Kombinacija lažnjaka |
| ④ | シュートフェイント | シュートのフォームを全部作って打たない | GK・ブロックDFを動かす | Lažnjak šutom |
| ⑤ | パスフェイント | パスを出す方向に腕を振って出さない | 隣のDFを動かす | Lažnjak dodavanjem |
| ⑥ | ステップフェイント | 踏み込む足の方向を偽る | DFの重心を崩す | Lažnjak smjerom |
| ⑦ | ボディフェイント | 肩・上半身だけを一方向に傾ける | DFの視線を釣る | Lažnjak tijelom |
| ⑧ | スピードフェイント | 全力疾走→急停止→また加速 | DFのリズムをずらす | Lažnjak promjenom tempa |

> **注意**：③ダブルフェイントは「フェイントの種類」ではなく「①②④⑤⑥⑦⑧を2連続でかける使い方の原則」。分類上は7種＋1ルール。
> **補足**：GKセクション（05_gk_df.md）で「先読み型GKを最も狂わせるフェイント」として挙げられている**バックハンドフェイク**は、④シュートフェイントの特殊形として下記に記述。

### クロアチア/ユーゴ学派フェイント体系との対照

クロアチア語では **Fintiranje（フィンティラニェ）** がフェイント全体の総称。
日本の8種は Fintiranje の主要カテゴリとほぼ一対一で対応する。

| Fintiranje 分類（クロアチア語） | 対応する日本の分類 | 補足 |
|---|---|---|
| Lažnjak tijelom | ⑦ボディフェイント | 上半身・体幹で偽る |
| Lažnjak šutom | ④シュートフェイント | GK・ブロッカーDF向け |
| Lažnjak dodavanjem | ⑤パスフェイント | 隣のDF向け |
| Lažnjak smjerom | ⑥ステップフェイント | 方向・足の踏み込みで偽る |
| Lažnjak promjenom tempa | ⑧スピードフェイント | テンポ変化 |
| Lažnjak frekvencijom | （⑥⑧の中間） | ステップのリズム変化で崩す |
| Lažnjak rotacijom | ①廻旋フェイント | DFの腕を外す |
| Okret / Rotacija | ②ターンフェイント | 体全体を軸回転 |
| Kombinacija lažnjaka | ③ダブルフェイント | 2種以上を連続 |

**「200種」の構造（クロアチア学派の考え方）**：
```
基本フェイント動作（約8〜10種）
  × タイミング変奏（早い・標準・遅い）
  × 体勢変奏（立ち・ジャンプ中・落下中・ひねり）
  × コンタクト対応（DF圧力あり・なし・強い）
  ＝ 数百の「名前のついた技術」

例：Lažnjak šutom（シュートフェイント）だけでも
  → タイミング早め × ジャンプ頂点 × DFあり = 特定の名称
  → タイミング遅め × 落下中 × DFなし = 別の名称
  ……というように細分化されている
```

> **実用的な見方**：8種を「骨格」、クロアチア学派の細分化を「バリエーション」として理解する。
> 8種を習得したうえで、実戦で「どのタイミング・どの体勢・どの圧力下で使うか」を経験で積むのが「200種への道」。

### フェイント①　廻旋フェイント（腕回し・接触解放）

**【前回の記述は根本的に間違っていた。正しい内容に修正する】**

**体の動き**：
```
状況：DFがホールディング（腕を掴む）またはプッシング（腕を押す）をしてきた

① 掴まれた / 押されている腕を、円を描くように回す
  （外側→下→内側→上 または その逆方向）

② 円運動でDFの手が腕から外れる（または接触が切れる）

③ 外れた瞬間に即座に「差し込む」（ドライブ・カットイン）

結果：DFに腕をコントロールされた状態から、自分のスペースに突入できる
```

**なぜ機能するか**：
- DFが腕を掴んでいる → 円運動で「腕の表面」がDFの手の内側から外側へ移動する → 掴みが外れる
- 掴みが外れた瞬間 = DFのコントロールが失われた瞬間 → その0.5秒が突破の窓

| 項目 | 内容 |
|---|---|
| **長所** | 接触を積極的に使ってくるDFを逆用できる。掴まれるほど廻旋が決まりやすい |
| **主な対象** | **ホールディング・プッシングをしてくるDF**（接触型・マンツーマン密着型） |
| **GKへの効果** | 廻旋自体はGK対策ではない。差し込んだ後のシュートでGK対策が始まる |
| **カウンターDF** | 腕を掴まない（ホールディングをやめる）→ ただしそうすると今度は腕が自由になり別のフェイントが使いやすくなる |
| **反則との境界** | 廻旋でDFの腕を「払う」動作がOFから積極的に接触すると反則になる場合がある。あくまで「外す」動作で |

**じゃんけん構造**：
```
DFがホールディングで腕を掴む → 廻旋で外す → 差し込む
DFが廻旋を嫌がって掴まなくなる → 腕が自由 → 他のフェイント（④⑥⑦）が使いやすくなる
廻旋→差し込みを繰り返す → DFがファウルでも止めようとする → フリースロー獲得
```

**好奇心担当が気づいた補足（修正版）**：
廻旋フェイントがハンドボール特有なのは「ホールディングが一定許容される」競技特性から。バスケットボールは腕を掴むと即反則→廻旋が必要な場面が生まれない。ハンドボールの接触文化が廻旋という技術を産んだ。

**批判的思考担当の修正（前回「廻旋はGK向け」は誤り）**：
前回の指摘「廻旋はGKメイン技術でDFには効きにくい」は完全に逆だった。廻旋の主対象はDF（接触型）であり、GKへの直接効果はない。この修正で廻旋フェイントの「効く対象」表が根本から変わる。

### フェイント②　ターンフェイント（体の回転・反転）

**体の動き**：
```
パターンA（入り込みフェイント）：
ゴールに背を向けてDF背後に入る動き → DFが追いかける → 逆に反転して逆方向へ

パターンB（ターン方向偽り）：
右にターンする動き（右足を軸に右回転）を始める → DFが右ターンに対応 → 左に切り返す

パターンC（背向けキープ）：
DFに背を向けてボールをキープ → DFが前に出た瞬間に振り向いてシュートまたはパス
```

| 項目 | 内容 |
|---|---|
| **長所** | DF背後のスペースへ入り込める。密着マークされるほど効果が大きい（密着DF = 背後のスペースが大きい） |
| **効くDF** | 密着マンツーマン。内側を切っているDF（クローズポジション）。ターン方向を先読みで塞ごうとするDF |
| **弱点** | ターン中は一瞬ゴールから背を向ける。視野が狭くなる瞬間がある |
| **カウンターDF** | ターン方向を先に塞ぐ（先に体を入れる）。または逆に距離を取ってターンを使えないようにする |
| **ポストとの相性** | ポストプレーヤーはゴール前でこのフェイントを多用。V字位置取りからのターンフェイントが定番 |

**じゃんけん構造**：
```
右ターンフェイントを繰り返す → DFが右をふさぎに先回り → 左ターンが空く
DFが距離を取ってターンを防ぐ → 今度はシュートやパスの距離が生まれる
```

### フェイント③　ダブルフェイント（2連フェイント）

**体の動き**：
```
例①：パスフェイント → シュートフェイント → 本シュート
  右にパスの動き（DFが右を見る）
  → 戻して今度はシュートモーション（DFが前に出る）
  → 実際には左へドライブ

例②：左ステップ → 右ステップ → 左突破
  左に踏み込む（DFが左に重心移動）
  → 戻して右に踏み込む（DFが右に戻ろうとする）
  → 左に一気に加速（DFが間に合わない）

例③：ターンフェイント → 廻旋フェイント → シュート
  2種類を組み合わせる
```

| 項目 | 内容 |
|---|---|
| **長所** | 「1回のフェイントには耐えられる」全国レベルDFを超えられる。1回目で動いたDFが元の体勢に戻れる前に2回目が来る |
| **前提条件** | 1回目のフェイントで自分も体が崩れないこと。体幹が弱いと1回目で自分が転倒リスク |
| **効くDF** | 1回は耐えられるが2回目への切り替えが遅いDF。「フェイントを読んだ」と思って動いたDF |
| **カウンターDF** | 1回目で動かない訓練（「2回目で動く」ルール）。距離を保って1回目も2回目も関係なくポジション維持 |

### フェイント④　シュートフェイント（シュートモーション・シュートフォーム）

**体の動き**：
```
腕を引く（ボールを耳の横へ）
→ 体を回す（腰の回転を始める）
→ 体重を前脚に乗せる
→ ここまでシュートと全く同じ動き → でも放さない

GKは「打ってくる」と判断して動く
↓
実際には：ドライブ / パス / 廻旋フェイントへ移行
```

| 項目 | 内容 |
|---|---|
| **長所** | GKとDFを同時に騙せる唯一のフェイント。使用機会が全フェイント中最多 |
| **効くGK** | **先読み型GK**（シュートモーションでコースを予測して先に動く） |
| **効くDF** | 前に詰めてシュートブロックを狙うDF（Herausrücken中のDF） |
| **カウンターGK** | コース絞り型GK（シュートフォームに反応せず待てる）。ただしその場合はドライブで使う |
| **カウンターDF** | 骨盤の向きを見る。シュートフェイントは腕・肩が動いても骨盤は必ずしも動かない |

**じゃんけん構造**：
```
シュートフェイント → GKが先に動く → 空いたコースへ本シュート
本シュートが続く → GKが待つようになる（コース絞り型に変化）→ ドライブで突く
```

### フェイント⑤　パスフェイント（パスモーション・視線誘導）

**体の動き**：
```
パスを出す相手の方向に体と腕を向ける
→ 腕を振り始める（パスの動作）
→ DFがそのパスコースを切りにいく / インターセプトに来る
→ 実際には出さない → 別方向へパス または ドライブ
```

| 項目 | 内容 |
|---|---|
| **長所** | 隣のDFを横に動かしてスペースを作れる。DFがインターセプトに来た瞬間にその背後が空く |
| **効くDF** | インターセプトを狙うDF（前に出てくる積極的なDF）。カバーリングが速いDF（動く前に別の選択肢を作れる） |
| **GKへの影響** | GKへの直接効果は薄い。あくまで「隣のDF」を動かすためのフェイント |
| **カウンターDF** | 腕の動きより腰・体の向きを見る。インターセプトに飛ばずポジションを維持する |

### フェイント⑥　ステップフェイント（踏み込みフェイント・フットワーク）

**体の動き**：
```
パターンA（インカットフェイント）：
右足を内側（ゴール方向）に強く踏み込む
→ DFが内を切りにくる（または体重が右に移動）
→ 左足で外に踏み出してそのまま外突破

パターンB（アウトカットフェイント）：
右足を外側に踏み込む
→ DFが外を切りにくる
→ 素早く内側にカットイン

ポイント：踏み込みの「強さ」と「深さ」が重要。
浅い踏み込みは騙されない。体重が乗った踏み込みほど効果大。
```

| 項目 | 内容 |
|---|---|
| **長所** | 踏み込みは「本番の宣言」なのでDFは反応せざるを得ない。特に足の向きを見ているDFに有効 |
| **効くDF** | 足の向き・踏み込みを見て反応するDF。重心が移動しやすいDF（体のバランスが崩れやすい選手） |
| **カウンターDF** | 足より骨盤を見る。骨盤が動いていない踏み込みはフェイク。「一拍待って反応」の訓練 |

**じゃんけん構造**：
```
ステップフェイント多用 → DFが足を見なくなる（骨盤を見るようになる）
→ ボディフェイント（肩だけ傾ける）が効くようになる
```

### フェイント⑦　ボディフェイント（肩・上半身の傾き）

**体の動き**：
```
右肩を右に落とす（上半身だけが右に傾く）
→ DFが「右に来る」と反応して右に体重を移す
→ 実際には左に踏み込んでドライブ

ポイント：
- 腰（骨盤）は動かさない → 上半身だけで偽る
- 骨盤まで動かすと本番と区別がつかなくなる（それはステップフェイントになる）
- 動作が小さくても効く。大きく動く必要はない
```

| 項目 | 内容 |
|---|---|
| **長所** | 動作が小さく素早い。エネルギー消費が少ないので繰り返し使える |
| **効くDF** | 上半身・肩の動きを見ているDF。反応速度が速いDFほど引っかかりやすい（速く反応できるから釣れる） |
| **カウンターDF** | 骨盤の向きを見る。肩が右に傾いても骨盤が左を向いていたらフェイク |

**ステップフェイントとの違い**：
- ステップフェイント：**足が動く**（踏み込む）
- ボディフェイント：**上半身が傾く**（足はまだ動いていない）

### フェイント④-B　バックハンドフェイク（GKコース予測撹乱）

**GKセクション（05_gk_df.md）で「先読み型GKを最も狂わせる技術」として言及されているため、ここに詳細を記述。**

**体の動き**：
```
通常のシュートフォームで構える（ここまで④シュートフェイントと同じ）
↓
ボールを手から離す直前に、手首を「通常とは逆方向」に返す
↓
ボールが手から離れた後の軌道が、フォームから予測されるコースと全く異なる
```

**なぜ機能するか**：
- 先読み型GKは「腕の振り・体の回転・重心の方向」からコースを予測して先に動く
- バックハンドフェイクは**腕の軌道を最後の瞬間に変える**
- GKがすでに「予測したコース」に動いている → 体が止まれない → 反対コースが空く

| 項目 | 内容 |
|---|---|
| **難度** | 高い。手首の強さと柔軟性が必要 |
| **有効な対象** | 先読み型GK専用（コース絞り型GKには通じにくい）|
| **有効なシューター** | バックプレーヤー・ウィング両方。特に右利きが左コーナーへ打つ場面 |
| **カウンターGK** | コース絞り型に変える（動かなければ騙されない）|

**じゃんけん構造（GKとの戦い）**：
```
バックハンドフェイク → GKのコース予測が外れる → 得点
GKがバックハンドフェイクを警戒 → 動きを遅らせる（待つ方向に変化）
→ コース絞り型GKに変化 → 今度はドライブで突破
→ GKがドライブを警戒して前に出る → バックハンドフェイクが復活
```

### フェイント⑧　スピードフェイント（テンポ変化・緩急）

**【重要な再定義：スピードフェイントは「決めつけ行為＝1/2のギャンブル」】**

スピードフェイントは「DFが今のテンポに合わせる」という前提に賭ける行為。
- 賭けが当たる → DFが急停止に対応できず置き去り
- 賭けが外れる → DFは元の距離を保ったまま → 自分だけ急停止してチャンスを潰す

つまり**読み合いではなく、確率1/2の決め打ち**。「最強のフェイント」ではなく「読まれた瞬間に最弱になるフェイント」。

**体の動き**：
```
パターンA（急停止→再加速）：
全力で走る → 急ブレーキ（DFも止まろうとする と仮定） → 即座に逆方向に加速

パターンB（スローダウン→急加速）：
ゆっくり近づく → DFがリラックスする瞬間 と仮定 → 一気に加速してドライブ

パターンC（リズム変化）：
一定のリズムで動いてDFをその拍に乗せる と仮定 → 急に拍のタイミングを変える
```

| 項目 | 内容 |
|---|---|
| **長所** | 当たれば一気に決まる。「どこを見れば防げる」基準がないので情報処理の遅いDFには有効 |
| **効くDF** | 相手の速さに本当に追従するDF。リズムに無意識に乗ってしまうDF |
| **当たる確率を上げる方法** | ①事前スカウティングで「相手DFが追従型か独立型か」を見ておく ②試合中、最初の数回で相手のリアクションを確認してから本番で使う |
| **カウンターDF** | 「相手のペースに乗らない」訓練済みDF。距離を一定に保ち、自分のリズムで動く。ゴール方向への移動コースだけを塞ぐ |
| **致命的な弱点** | DFが追従しなかった場合、自分だけ動きを止めて時間を消費する。シュートチャンスをむしろ失う |

**じゃんけん構造（1/2ギャンブルの本質）**：
```
スピードフェイント発動 → DFが乗る ✓ → 抜ける
スピードフェイント発動 → DFが乗らない ✗ → 自滅（時間ロス・距離ロス）
 → 次は別のフェイント（読み合いで決まるフェイント）に切り替える
```

**好奇心担当の「最強説」への反駁（批判的思考担当による訂正）**：
前回「⑧スピードフェイントは見る場所がない唯一のフェイント」として最強候補に挙げたが、これは誤り。
「見る場所がない」= **読み合いが成立しない** = **賭けに頼るしかない**ということ。
本当に最強のフェイントは「DFの反応を見てから選べる」フェイントであり、スピードフェイントはむしろ**最も賭博性が高いフェイント**。

### フェイント8種のじゃんけんマップ（修正版）

**【批判的思考担当の指摘②③を反映：カウンターは「骨盤一択」ではなく状況で分かれる】**

```
■ DFの守り方別：どのフェイントが効くか

DFが腕を掴む・押す（ホールディング・プッシング）
  → ①廻旋フェイント（腕を外して差し込む）

DFが腕・手の動きだけを見ている
  → ④シュートフェイント / ⑤パスフェイント（腕の動きで釣る）

DFが肩・上半身を見ている
  → ⑦ボディフェイント（肩だけ傾ける）

DFが足の踏み込みを見ている
  → ⑥ステップフェイント（踏み込む足の方向を偽る）

DFが骨盤まで見ている（上級DF）
  → ②ターンフェイント（体全体を動かす）or ⑧スピードフェイント（速さで見る余裕を奪う）

DFが1回のフェイントを耐えてくる
  → ③ダブルフェイント（2連続で1回目の反応を使う）

■ GK別：どのフェイントが効くか

先読み型GK（先に動く）
  → ④シュートフェイント（打つ前に動かして逆を打つ）が有効
  → ①廻旋は「差し込んだ後のシュート」段階なのでGK対策は別途必要

コース絞り型GK（動かない）
  → フェイントより「どこが空いているか」の精度 or「引きつけてGKを動かす」
  → ④シュートフェイントでGKを動かしてから逆コースを狙う
```

**カウンターDFの原則（批判的思考担当の修正：骨盤一択は不正確）**：

| フェイントの種類 | 正しいカウンター |
|---|---|
| 腕・肩だけが動く静的フェイント（④⑤⑦） | 骨盤の向きを見る |
| 足が動く動的フェイント（⑥） | 一拍待って骨盤の向きを見る |
| 体全体が動く（②⑧） | 骨盤より「ゴールへの移動コース」を塞ぐ |
| 接触型（①） | 腕を掴まない（ホールディングをしない） |
| 2連続（③） | 1回目で動かない訓練 / 距離を保ってポジション維持 |

### 実戦でのフェイント選択フレーム

```
ボールを持った瞬間に確認する2つのこと：

【確認①】DFは接触してきているか？
  YES → ①廻旋フェイント（腕を外して差し込む）が最優先
  NO  → 確認②へ

【確認②】DFは何を見ているか？
  腕・手を見ている  → ④シュートフェイント / ⑤パスフェイント
  肩を見ている      → ⑦ボディフェイント
  足を見ている      → ⑥ステップフェイント
  骨盤を見ている    → ②ターンフェイント / ⑧スピードフェイント
  1回耐えてくる     → ③ダブルフェイント
  わからない        → ⑧スピードフェイント（タイミング変化はどのDFにも普遍的に効く）
```

**好奇心担当が追加したい問い（未解決）**：
- フェイントの「癖（予備動作）」を消せるか？→ 消せる選手が本物のフェインター
- 最も読みにくいフェイントは⑧スピードフェイント説（「見る場所」がない唯一のフェイント）
- 廻旋は「接触処理技術」か「フェイント」か → 両方。接触処理の動作が同時に突破の起点になる

## 1:1（一対一）

### OFの原則
1. **勝ち位置を先に取る**：ボールを受ける前に有利な位置を確保
2. **DFの重心を動かす**：フェイントで重心を崩してから仕掛ける
3. **コンタクト（山側/谷側）を活用**：→ `01_basics.md` 参照

### 1on1の選択肢
| 状況 | 推奨行動 |
|---|---|
| クローズDF（内側） | 外へのフェイントから外シュート、または逆に内突破 |
| ワイドDF（外側） | 内突破またはピボットへのパス |
| DFが前傾姿勢 | バックステップしてシュート |
| DFが後傾姿勢 | 前への加速でドリブル突破 |
| DFが横に広い | 縦（前方）への突破 |

### 1on1のじゃんけん
```
ドリブル突破（縦）
  → DFが縦を切る（体を張る）
  → フェイクから外シュート
  → DFが外シュートを警戒（距離を取る）
  → ドリブル突破のスペース発生
  → ドリブル突破（縦）に戻る
```

---

### 突破体系：プロドール（Prodiranje）― クロアチア/ユーゴ学派

クロアチア語で **Prodiranje（プロジラニェ）** = 「突破・侵入」全般の総称。
1on1の「どう突破するか」の分類が日本語よりはるかに細かい。

| クロアチア語 | 日本語 | 内容 | 対応するDF状態 |
|---|---|---|---|
| Direktni prodor | 直接突破 | DFに対してまっすぐ切り込む | ワイドDF（外に立っている） |
| Prodor s lažnjakom | フェイントつき突破 | フェイントを先に入れてから突破 | クローズDF・距離を詰めたDF |
| Prodor s promjenom smjera | 方向転換突破 | 逆方向フェイントから急転換 | 重心を片側に乗せているDF |
| Prodor pod kotom | 角度突破 | 外→内の角度でDFの内側に入る | 正面に立っているDF |
| Prodor uz blokadu | スクリーン利用突破 | スクリーナーを使って突破コースを作る | マンツーマンDF |
| Prodor u dubinu | 縦突破 | ゴール方向へのダイレクトドライブ | DFが横に広がっているとき |
| Prodor širicom | 横展開突破 | サイドへの展開から突破 | DFがゴール正面を固めているとき |

**プロドールとフェイントの関係**：
```
フェイント（Fintiranje）= 相手を「動かす」行為
突破（Prodiranje）      = 動かした後に「入る」行為

フェイント → 突破 の順番が必須。
「フェイントを入れたが突破しなかった」= チャンスを消費しただけ。
「突破した = フェイントが機能した」の証明。
```

### シュート技術分類（クロアチア語対照）

| クロアチア語 | 日本語 | 特徴・使用条件 |
|---|---|---|
| Suručni šut | ジャンプシュート（標準） | 最高到達点で放つ。最も多用 |
| Šut iz pada | フォールシュート | 空中でバランスを崩しながら打つ。至近距離で有効 |
| Leti šut | フライングシュート | ウィングが全速で飛び込んで打つ（デュエルシュート） |
| Zakucavajući šut | スマッシュシュート | 上から叩きつける。GKの頭上を越す |
| Odskočni šut | バウンスシュート | バウンドさせてGKの手前を通す |
| Rotacijski šut | ターンシュート（廻旋シュート） | 体をひねりながら逆方向へ打つ |
| Šut bez doskoka | ノーランディングシュート | 着地せず落下中に打つ。GKのタイミングが外れる |
| Šut iz skoka s rotacijom | ジャンプ＋回転シュート | 最高難度。GKの予測を完全に外す |

> バックプレーヤーが主に使うのは Suručni šut / Zakucavajući šut / Rotacijski šut。
> Leti šut はウィング専用に近い。

## 2:2（二対二）

### OFの原則
- 2人の連係でDFのどちらかを崩す
- スクリーンを使って1on1の局面を作る
- ピック＆ロール（スクリーン＆ロール）が基本

### 2on2のパターン
| パターン | 説明 |
|---|---|
| ピック＆ロール | スクリーンしてからゴール方向に転身してパスを受ける |
| ピック＆ポップ | スクリーンしてから外に開いてシュートチャンス |
| クロス | 2人が交差することでDFのマーク交換を強制 |

### DFの2on2対応
- スイッチ（マーク交換）：即座に対応できるが、ミスマッチのリスク
- ファイトオーバー：マークを外さない（体力消耗大）
- ヘルプ：3人目のDFがカバー（チームDF必須）

## 勝ち位置（Winning Position）

> ボールを受ける前に「相手DFより先に有利なポジション」を取ること

### 重要性
- 勝ち位置にいれば、ボールが来た瞬間にすでに選択肢がある
- 勝ち位置を取れていないと、受けてから考えることになり判断が遅れる
- 「先に読んだ方が選択肢を持てる」の具体的な体現

### 勝ち位置の取り方
1. DFの動きを先読みして、DFが移動する前に目標地点に入る
2. オフボール（ボールを持っていないとき）から準備する
3. パサーとのアイコンタクトを活用する

### 勝ち位置とDFの攻防
```
OFが勝ち位置を先取する
  → DFが後追いで対応
  → OFがボールを受けた瞬間にすでに選択肢がある
  → DFが先読みして勝ち位置に先回り
  → OFが別の勝ち位置を探す（動き直し）
  → 「先に読んだ方が勝つ」の無限ループ
```
