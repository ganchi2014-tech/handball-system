# 基礎技術：パスキャッチ・シュート・コンタクト

---

## パスを受ける前のルーティン（勝ち位置取り）

**「受けてから考える」は遅い。受ける前に全部決まっている状態を作る。**

### ステップ1：スペース確認（オフボール中）
```
隣の選手がボールを持った瞬間：
→ 自分のDFの位置を確認（クローズかワイドか）
→ ピボットのポジションを確認（6mにいるか）
→ GKの位置を確認（左/中/右どこが空いているか）
```

### ステップ2：勝ち位置を取る
```
DFより先に有利なポジションに入る

クローズDF（内側）に立っている場合：
→ 外側に「受けられるポジション」を取る
→ 受けたらすぐ外シュートまたはサイド展開が打てる状態

ワイドDF（外側）に立っている場合：
→ 内側への突破コースを確保してから受ける
→ 受けたらすぐドライブが打てる状態
```

### ステップ3：受け方の選択
| 状況 | 受け方 |
|---|---|
| フリーで受けられる | 両手キャッチ → 次の動作へ |
| DFが近い | ワンハンドキャッチ → そのままドリブルまたはシュート |
| DFが前から来ている | 体でシールドしながら受ける（インターセプト防止） |
| 高いボールが来る | ジャンプして最高点でキャッチ（DFより先に確保） |
| パスが逸れそう | 迎えに行く（待たない） |

### ステップ4：受けると同時に次を決定済み
```
受ける → 考える → 動く　（× 遅い）
考える → 受ける → 動く　（○ 速い）
```

## パスキャッチ

### 羅列
- ボールを迎えに行く（待つのではなく動いて受ける）
- 両手でしっかり受けてから次の動作
- インターセプトを防ぐ受け方：DFとの間に体を入れる
- 受けながら次の選択肢を決める（視野確保）
- ワンハンドキャッチ（素早い展開のため）
- バックハンドキャッチ（逆手で受けて素早く逆転）
- フォームとして：肘を体の外に出さない、手首を柔らかく
- 受ける前に：相手DFの位置・動き・スペースを確認

### じゃんけん
```
DFがインターセプトを狙う
  → OFが受ける前にDFに気付く → 角度を変えてパス
  → DFが角度を変えてカバー
  → OFがフェイクパスでDFを動かしてから本パス
  → DFがフェイクに反応しなくなる
  → 本パスのタイミングが遅れ → インターセプト可能に戻る
```

## シュート

### 種類の羅列
- **ジャンプシュート**：踏み切って最高点で打つ。滞空時間でGKを惑わす
- **早打ち（ノージャンプ）**：踏み切らずに打つ。GKの動作前に放てる
- **ダイブシュート**：体を投げ出して6mライン内側から打つ
- **フォールシュート（倒れ込み）**：接触を受けながら体を傾けて打つ
- **ループシュート（山なりシュート）**：GKの上を越えて放物線でゴールへ
- **リバウンドシュート**：GKやDFに当たって跳ね返りを詰める
- **フリーシュート（助走なし）**：立ち止まった状態からのシュート（外角から）
- **サイドシュート**：極端な角度（サイドポジション）からのシュート
- **レグシュート（股抜き）**：自分の股の間からバックハンドで打つ（奇襲）
- **ドリブルシュート**：ドリブルで加速しながら近距離で打つ

### GK対策（コーチ補足統合）
| GKタイプ | 特徴 | 有効な打ち方 |
|---|---|---|
| 先読み型 | コースを読んで先に動く | 早打ち（GKが動く前に放つ） |
| コース絞り型 | 動かずコースを絞る | 引きつけてから最後に角度で変える |

**GKの法則**：左/中/右の常に2/3が閉じていて1/3が開いている。その「開いている1/3」を見つけて打つのが基本。

### シュートコース選択のじゃんけん
```
ニアコーナー（手前の角）を狙う
  → GKが寄ってくる（ニアをケア）
  → ファーコーナー（遠い角）が空く
  → GKがファーをケア
  → ニアが空く
  → ニアコーナーに戻る

上を狙う
  → GKが上をケア（手を上げる）
  → 下が空く（股下・低コーナー）
  → GKが下をケア
  → 上が空く
  → 上を狙うに戻る
```

## ドリブル技術

### ドリブルの種類と使い分け

| 種類 | 動き | 使う場面 |
|---|---|---|
| **スピードドリブル（プッシュ）** | ボールを斜め前に強く弾きながら全力で走る | 速攻・前方にスペースがある突破 |
| **コントロールドリブル（保護）** | ボールを体の横・後ろ寄りで低く叩く | DFのプレッシャー下・時間を作るとき |
| **低位置ドリブル（バウンス）** | 腰〜膝の高さで非常に低くつく | DFが腕を出してくるとき・密集地帯 |
| **クロスオーバードリブル** | 体の前を横断させて左右を切り替える | DFが一方向を完全に切っているとき |
| **バックビハインドドリブル** | 体の後ろで左右を切り替える | DFが前から腕を出してくるとき |
| **ロールターンドリブル** | ボールを保持したまま体を回転してDFを背に突破 | DFが正面から密着してきたとき |

---

### ドリブルフェイント（ドリブル中の仕掛け）

| フェイント | 動き | 狙い |
|---|---|---|
| **ジャブステップ** | ドリブル中に片足を一方向に強く踏み込む | DFの重心を崩してから逆へ突破 |
| **ヘジテーション（緩急）** | ドリブルのリズムを突然変える（遅→速または速→遅） | DFがリズムに合わせた瞬間に裏をつく |
| **イン＆アウト** | 内側に踏み込みそうな動きから外に切り返す | DFが内を切りに来たときに外が空く |
| **スピンムーブ** | ドリブル中にその場で1回転して逆方向へ | DFがボールに手を出してきた瞬間に有効 |

### ドリブルのじゃんけん
```
スピードドリブルで縦突破
  → DFが体を張って縦を切る
  → 止まってコントロールドリブルでシュートフェイント
  → DFが前に出る（ブロックを狙う）
  → DFの脇を抜けてドリブル突破またはパス

DFがドリブルのボールを狙って腕を出してくる
  → 低位置ドリブルでボールを守る
  → DFが腕を下げる（低いボールに対応）
  → 通常の高さに戻して速いドリブル
```

## 踏み切り・シュート打点

### 踏み切りの種類

| 種類 | 動き | 特徴 |
|---|---|---|
| **2ステップ（標準）** | 右足 → 左足（または逆）で踏み切る | ハンドボールの基本。安定・パワーが出る |
| **ユーロステップ** | 1歩目を一方向・2歩目を逆方向に大きく踏む | DFがついてくる → 2歩目でかわす |
| **1ステップ（早打ち用）** | 最後の1歩だけで打つ | 最速。GKが動く前に放てる。パワーは犠牲になる |
| **ノーステップ（立ち打ち）** | ジャンプせずに立ったまま打つ | フリーの外角シュートで使用 |

---

### シュート打点（空中のどこで打つか）

```
上昇中（最高点より前）
→ 早打ち効果あり。GKが「最高点で打つ」と予測して先読みしている場合に有効
→ 弱点：腕の力が足りないとシュートが甘くなる

最高点
→ 最もパワーと精度が出る。GKへのコースが最も広い（見下ろす角度になる）
→ GK視点：最も読まれやすいタイミング

下降中（最高点より後）
→ GKが「最高点で打つ」と予測していると動作が微妙にズレる
→ フォールシュート（シュト・イズ・パダ（フォールシュート））の延長
→ 弱点：体勢が不安定。ファウルのリスクも上がる
```

**GKタイプ別の打点選択**：
| GKタイプ | 最適な打点 | 理由 |
|---|---|---|
| 先読み型（動く前提） | 上昇中（早打ち） | GKが最高点を予測して先に動いている |
| コース絞り型（待つ） | 最高点 + コース変化 | GKは動かない → 打点より「コースの精度」が勝負 |
| 混合型（試合中に変化） | 最初の数本で観察してから判断 | 序盤は普通に打ってGKの反応パターンを確認 |

---

### 早打ち詳細（コーチ直伝を展開）

GKの「最高点予測」を外す技術。先読み型GKが多い現代ハンドボールで最重要。

```
状況：GKが「シューターの最高到達点で先読みして動く」と仮定

早打ち（上昇中に打つ）
→ GKはまだ「先読みのモーション前」か「動き始めだけ」
→ GKの体が完全に動く前にボールが届く

さらに：
GKが早打ちに慣れてくる
→ 早打ちを予測して「早めに動く」ようになる
→ 最高点に戻す（GKが過剰に早く動いてコースが空く）
→ GKが混乱して判断基準を失う
```

**早打ちに必要な体の準備**：
- 助走しながらすでに腕を引いておく（ジャンプ前に準備完了）
- ジャンプと同時または上昇中に腕の振りを終える
- 最高点を「待たない」意識が必要

## コンタクト技術（山側/谷側）

コーチ直伝の核心技術。

### 定義
- DFが斜めに体を向けて構えたとき、体の傾きに「高い側（山側）」と「低い側（谷側）」が生まれる
- **山側**：DFの体の高い方。越えられると完全に背後を取られる
- **谷側**：DFの体の低い方。ここを通ろうとすると前後ステップで対応できる

### OFの視点
- DFが斜め構えをしているとき、**山側を越えようとするフェイク**で、DFが山側をケアするために体を傾ける
- そこで**谷側（実は守りやすい側）**へ切り返すと、DFは体勢を戻しきれない

### DFの視点
- 山側を越えさせないことが最優先
- 谷側に来た場合は**前後ステップ（横ではなく前後）**で押し出す
- コンタクト中も常に次の**勝ち位置**（相手より先に有利なポジション）を意識して動く

### 勝ち位置の概念
- ボールを受ける前に「相手DFより先に有利なポジション」を取ること
- 勝ち位置にいれば、ボールが来た瞬間にすでに選択肢がある
- 勝ち位置を取れていないと、受けてから考えることになり遅れる

---

### 山側/谷側の応用（じゃんけん展開）

**基本セットアップ**：
```
DFが斜め構え（例：右肩を前に出して右側を切っている）
→ 山側 = 右（越えると完全に背後）
→ 谷側 = 左（守りやすいが、体が傾くと対応が遅れる）
```

**OFが使うじゃんけん**：
```
山側フェイク（右）を繰り返す
  → DFが山側を警戒して右重心になる
  → 谷側（左）への切り返し → DFが体勢を戻しきれない
  
谷側（左）への切り返しを繰り返す
  → DFが谷側をケアするために構えを変える
  → 山側（右）への突破が開く

DFが斜め構えをやめて正面構えに変える
  → 山側/谷側の非対称がなくなる
  → 代わりにステップフェイント・クロスオーバードリブルが有効になる
```

**コンタクト時の応用**：
- DFが密着（腕を当ててくる）→ 山側方向に廻旋フェイントを使いながら谷側へ差し込む
- DFが腕を伸ばして前から押してくる（プッシング）→ 廻旋で外してから山側へ突破
- DFが横から押してくる → 押されている方向（谷側）と逆の山側へのドライブが最速の抜け方

## ボールのグリップ・滑り対策（JHA 2018規定準拠）

### JHA 国内規定の正確な事実
- **靴への松ヤニは「可」**（JHA公式規定）— 靴に塗っておいて、そこから指へ補充する方式は認められる
- **他部位（手のひら・ボール直接など）への松ヤニは「不可」**
- ただし **会場使用上の条件によっては、靴の松ヤニも認めない場合がある** → 会場・大会要項を要確認
- 詳細は `31_equipment.md` 【L】アクセサリー ⑨ 靴への松ヤニ 参照

### 状況別の現実的対策
| 状況 | 対策 |
|---|---|
| 手が乾燥して滑る | 水で軽く湿らせて拭く／吸汗リストバンド／靴の松ヤニ（会場可なら）から指へ補充 |
| 手汗が多すぎて滑る | タオルでこまめに拭く／DRYHANDS（バスケ用）で代用 |
| ボール表面が古くて滑る | 新品ボールへ／表面を軽く湿らせる |
| 緊張で手汗が止まらない | 滑り止めスプレー（バスケ用で代用可） |
| グリップが必要 | **大会要項で松ヤニ可否を確認** → 可なら靴限定で塗布 |

### 注意点
- 手のひらに松ヤニを直接塗ったり、ボールに塗ったりするのは規定違反
- 「靴に少量塗っておいて、必要時に指で軽く触れて補充」が正しい使い方
- 床への影響を考慮し、試合後はチームで床掃除する責任あり
