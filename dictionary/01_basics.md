# 基礎技術：パスキャッチ・シュート・コンタクト

---

## パスキャッチ

### 羅列
- ボールを迎えに行く（待つのではなく動いて受ける）
- 両手でしっかり受けてから次の動作
- インターセプトを防ぐ受け方：DFとの間に体を入れる
- 受けながら次の選択肢を決める（視野確保）
- ワンハンドキャッチ（素早い展開のため）
- バックハンドキャッチ（逆手で受けて素早く逆転）
- フォームとして：肘を体の外に出さない、手首を柔らかく
- 受ける前に：相手DFの位置・動き・スペースを確認

### じゃんけん
```
DFがインターセプトを狙う
  → OFが受ける前にDFに気付く → 角度を変えてパス
  → DFが角度を変えてカバー
  → OFがフェイクパスでDFを動かしてから本パス
  → DFがフェイクに反応しなくなる
  → 本パスのタイミングが遅れ → インターセプト可能に戻る
```

## シュート

### 種類の羅列
- **ジャンプシュート**：踏み切って最高点で打つ。滞空時間でGKを惑わす
- **早打ち（ノージャンプ）**：踏み切らずに打つ。GKの動作前に放てる
- **ダイブシュート**：体を投げ出して6mライン内側から打つ
- **フォールシュート（倒れ込み）**：接触を受けながら体を傾けて打つ
- **ループシュート（山なりシュート）**：GKの上を越えて放物線でゴールへ
- **リバウンドシュート**：GKやDFに当たって跳ね返りを詰める
- **フリーシュート（助走なし）**：立ち止まった状態からのシュート（外角から）
- **サイドシュート**：極端な角度（サイドポジション）からのシュート
- **レグシュート（股抜き）**：自分の股の間からバックハンドで打つ（奇襲）
- **ドリブルシュート**：ドリブルで加速しながら近距離で打つ

### GK対策（コーチ補足統合）
| GKタイプ | 特徴 | 有効な打ち方 |
|---|---|---|
| 波型 | コースを読んで先に動く | 早打ち（GKが動く前に放つ） |
| スポット型 | 動かずコースを絞る | 引きつけてから最後に角度で変える |

**GKの法則**：左/中/右の常に2/3が閉じていて1/3が開いている。その「開いている1/3」を見つけて打つのが基本。

### シュートコース選択のじゃんけん
```
ニアコーナー（手前の角）を狙う
  → GKが寄ってくる（ニアをケア）
  → ファーコーナー（遠い角）が空く
  → GKがファーをケア
  → ニアが空く
  → ニアコーナーに戻る

上を狙う
  → GKが上をケア（手を上げる）
  → 下が空く（股下・低コーナー）
  → GKが下をケア
  → 上が空く
  → 上を狙うに戻る
```

## コンタクト技術（山側/谷側）

コーチ直伝の核心技術。

### 定義
- DFが斜めに体を向けて構えたとき、体の傾きに「高い側（山側）」と「低い側（谷側）」が生まれる
- **山側**：DFの体の高い方。越えられると完全に背後を取られる
- **谷側**：DFの体の低い方。ここを通ろうとすると前後ステップで対応できる

### OFの視点
- DFが斜め構えをしているとき、**山側を越えようとするフェイク**で、DFが山側をケアするために体を傾ける
- そこで**谷側（実は守りやすい側）**へ切り返すと、DFは体勢を戻しきれない

### DFの視点
- 山側を越えさせないことが最優先
- 谷側に来た場合は**前後ステップ（横ではなく前後）**で押し出す
- コンタクト中も常に次の**勝ち位置**（相手より先に有利なポジション）を意識して動く

### 勝ち位置の概念
- ボールを受ける前に「相手DFより先に有利なポジション」を取ること
- 勝ち位置にいれば、ボールが来た瞬間にすでに選択肢がある
- 勝ち位置を取れていないと、受けてから考えることになり遅れる
