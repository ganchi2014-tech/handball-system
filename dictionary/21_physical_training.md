## 【なぜフィジカルが必要か】バックプレーヤーの試合負荷データ

```
研究（PMC 2023, Physical Demands during Official Competitions in Elite Handball）

バックプレーヤーの1試合平均走行距離：約4.96 km
  ※ウィング（4.23 km）・ピボット（3.91 km）より多く、全ポジション中最長

高速走行（High-speed running）距離：センターバック平均 243.2 m/試合
スプリント距離：センターバック平均 62.0 m/試合
試合中の心拍数：HRmax の85〜90%で推移（ほぼ全力に近い強度）

→ バックプレーヤーは全ポジション中、最も体力資源を消費するポジション。
→ 投擲能力（シュート速度）と跳躍能力の両立が最重要要件。
```

## 【A】主要筋群と役割（なぜその筋肉か）

### A-1. 上半身

| 筋群（日本語） | 原語 | ハンドボールとの関連 |
|---|---|---|
| 回旋筋腱板（棘上筋・棘下筋・小円筋・肩甲下筋） | Rotator Cuff | シュート動作の減速制御。1試合100本以上の投擲に耐える肩の土台 |
| 三角筋 | Deltoid | オーバーハンドスローの推進力 |
| 前鋸筋 | Serratus Anterior | スロー時の肩甲骨安定化 |
| 僧帽筋下部 | Trapezius Lower | 投球肩（Thrower's Shoulder）予防に不可欠 |
| 広背筋 | Latissimus Dorsi | 腕の加速・減速の両方に関与 |
| 腹斜筋 | Obliques | 体幹回旋→サイドアームシュートの動力源 |
| 体幹（インナーマッスル） | Core / Rumpfstabilität | 下半身の爆発力を上半身へ伝達する連結部 |

### A-2. 下半身

| 筋群（日本語） | 原語 | ハンドボールとの関連 |
|---|---|---|
| 大臀筋・中臀筋 | Gluteus Maximus/Medius | 踏み切り・方向転換・ランジの主動力 |
| 大腿四頭筋 | Quadriceps | ジャンプシュートの離地力 |
| ハムストリングス | Hamstrings | スプリント加速・着地衝撃吸収。Nordicで損傷率50%減 |
| 腸腰筋 | Hip Flexors | 跳躍後の着地コントロール・走フォーム |
| 内転筋群 | Adductors | 横方向の急停止・サイドステップ安定 |
| 下腿三頭筋 | Gastrocnemius/Soleus | 踏み切り・スタートダッシュ |

## 【B】筋力トレーニング（Schnellkrafttraining / Maximalkrafttraining）

### 基本方針

```
研究エビデンス：
ベンチプレス1RM（最大1回挙上重量）とシュート速度の相関係数 r = .637（有意差あり）
→ 上半身筋力の向上はシュート速度の向上に直結する

推奨強度：
  最大筋力期（Maximalkraft） → 85〜95% 1RM
  爆発力期（Schnellkraft）   → 60〜80% 1RM（最大速度で挙げる）
```

### オフシーズン12週間プログラム（週3回）

**フェーズ1：筋肥大期（Hypertrophy / Muskelaufbau）— 週1〜4**

| 種目 | セット×レップ | 負荷 | 目的 |
|---|---|---|---|
| スクワット（Squat） | 4×8〜10 | 70〜75% 1RM | 大腿四頭筋・臀筋・内転筋 |
| ルーマニアンデッドリフト（RDL） | 3×10 | 65〜70% 1RM | ハムストリングスの偏心訓練 |
| ベンチプレス（Bench Press） | 4×8〜10 | 70〜75% 1RM | シュート速度と直接相関 |
| シーテッドケーブルロウ | 3×10 | 中重量 | 肩甲骨後退筋・広背筋 |
| 懸垂 / ラットプルダウン | 3×8〜10 | 自重/補助 | 肩安定化・投球拮抗筋 |
| オーバーヘッドプレス | 3×8 | 65% 1RM | 三角筋・肩甲骨安定 |
| パロフプレス（Pallof Press） | 3×12 | バンド | 体幹回旋剛性（Anti-rotation） |

**フェーズ2：最大筋力期（Maximalkraft）— 週5〜8**

| 種目 | セット×レップ | 負荷 |
|---|---|---|
| スクワット | 5×3〜5 | 85〜90% 1RM |
| デッドリフト | 4×4 | 85% 1RM |
| ベンチプレス | 5×3〜5 | 85〜90% 1RM |
| 加重懸垂 | 4×4〜5 | 自重+荷重 |
| ブルガリアンスプリットスクワット | 3×6/脚 | 中重量 |

**フェーズ3：爆発力期（Schnellkraft / Power）— 週9〜12**

| 種目 | セット×レップ | 負荷 | ポイント |
|---|---|---|---|
| パワークリーン（Power Clean） | 4×3 | 75〜80% 1RM | 全身爆発力 |
| ハングスナッチ（Hang Snatch） | 3×3 | 70% 1RM | 上方爆発力 |
| スクワットジャンプ（Squat Jump） | 4×5 | 30〜40% 1RM | 下肢パワー |
| メディシンボールスラム | 4×8 | 3〜5 kg | 体幹爆発力・シュート動作模倣 |
| 爆発的ベンチプレス | 3×5 | 60〜70% 1RM（最大速度） | 投球速度向上 |

### ハンドボール特化種目（Handball-specific exercises）

```
ケーブルウッドチョップ（Cable Woodchop）
  → 斜め上から斜め下へ。腹斜筋・体幹回旋。シュート動作と直結。

サイドランジ（Side Lunge）
  → 内転筋・大臀筋。横方向の急停止動作に直結。

ゴブレットスクワット（Goblet Squat）
  → 体幹と下肢の協調。方向転換前の準備姿勢。

ラテラルバンドウォーク（Lateral Band Walk）
  → 中臀筋強化。膝・股関節外側安定化。足首捻挫予防にも。

ワンアームランドマインプレス（One-arm Landmine Press）
  → 片手プレス・回旋安定性。実際の投擲動作に近い。
```

## 【C】ジャンプトレーニング（Plyometrics / Sprungkrafttraining）

### 研究エビデンス

```
PMC 2023（ユース女子ハンドボール・10週間・週2回）：
  SJ（反動なし跳び）・CMJ（反動あり跳び）・立ち幅跳びが有意改善

メタアナリシス（エリートハンドボール選手）：
  SJ 改善率：4.5〜13.9%
  CMJ 改善率：5.7〜11.4%

複合プログラム（筋力＋プライオメトリクス 12週間）：
  SJ +6.1%、CMJ +3.8%、連続40回ジャンプ +6.8%
```

### テスト種目

| テスト名 | 略称 | 内容 |
|---|---|---|
| Squat Jump | SJ | 反動なし垂直跳び（爆発的同心収縮） |
| Countermovement Jump | CMJ | 反動あり垂直跳び（SSC使用） |
| CMJ with Arms | CMJA | 腕振り反動ありCMJ |
| Standing Long Jump | SLJ | 立ち幅跳び |
| Drop Jump | DJ | 台上からの落下→反動跳び（衝撃吸収＋爆発力） |

### プライオメトリクス種目（レベル別）

```
基礎レベル（Grundstufe）：
  ボックスジャンプ / スクワットジャンプ / 幅跳び / スキッピング

中級レベル（Aufbaustufe）：
  ドロップジャンプ / ラテラルバウンド（横跳び） / 片脚CMJ / ディプスジャンプ

上級レベル（Hochleistungsstufe）：
  ボール操作を含む反応跳躍 / ジャンプ中のメディシンボール投げ / ハードルホップ
```

**注：** CMJ（遅いSSC）はSJ（同心収縮のみ）より高いトレーニング効果を得やすい

## 【D】スピード・アジリティ（Schnelligkeit / Agility）

### ハンドボール特化型ドリル

**Tテスト（T-Test Agility）**
```
設定：コーンをT字型に配置（縦10m・横各5m）
実施：前方10mダッシュ → 左5mサイドシャッフル → 右10mシャッフル → 左5mシャッフル → バックペダルでスタートへ
基準：男性 9.5秒未満 = 優秀
```

**プロアジリティ（Pro Agility / 5-10-5 Shuttle）**
```
設定：3点コーン（各4.5m）
実施：中央スタート → 右4.5m → 左9m → 中央4.5m
目的：短距離での最大加速・急停止・切り返し
```

**コート特化型スプリント**
```
シャトルスプリント：6mライン〜センター往復（2×15m）
バックプレーヤー特化：25m往復ダッシュ（長い距離より短距離が試合に即している）
バック＆フォース：ゴール〜センターの前後ダッシュ繰り返し
```

**ラテラルバウンド（Lateral Bounds / Skate Jumps）**
```
横方向への単脚跳躍繰り返し
→ サイドステップの爆発的パワー。コートカバーに直結。
```

## 【E】持久力トレーニング（Ausdauer / Endurance）

### ハンドボールの持久力特性

```
試合時間：60分（前後半各30分）
試合中平均心拍数：HRmax の85〜90%
バックプレーヤー：高強度活動の割合が全ポジション中最多

有酸素能力の目標値（男子エリート）：VO₂max 50〜60 ml/kg/min
```

### HIIT プロトコル（推奨3種類）

**Type A：4×4分（ハンドボール最多使用）**
```
内容：4分間 HRmax の90〜95%強度 → 4分アクティブリカバリー（60〜65%）
セット：4セット
頻度：週2回 × 8〜12週間
根拠：ハンドボール選手のVO₂max改善に最も高いエビデンス（系統的レビュー 2024）
```

**Type B：短インターバル（EHF推奨・試合特異性が高い）**
```
内容：10秒間 最大速度の95%で走行 → 20秒リカバリー
セット：2セット × 22本
特徴：ハンドボールの動作パターン（短い爆発＋短い回復）に最も近い
```

**Type C：長インターバル（有酸素容量構築）**
```
内容：3分間 最大速度の85% → 3分リカバリー
セット：5セット
特徴：有酸素ベースの構築に有効
```

**スモールサイドゲーム（Small-Sided Games / SSG）**
```
少人数・狭コートでのゲーム形式
効果：持久力・技術・戦術を同時に鍛えられる
研究：SSGグループはTテスト・シュート速度・Yo-Yoテストで有意改善
実践：スペイン・スカンジナビアのクラブで有酸素トレーニングの主要代替手段として使用
```

**Yo-Yo インターミッテントリカバリーテスト（YYIRT1）**
```
正式名：Yo-Yo Intermittent Recovery Test Level 1
内容：20mシャトルランを漸増速度で実施。各ラン後10秒アクティブリカバリー
用途：ハンドボール選手の持久力評価の標準テスト
```

## 【F】年齢別指針（DHB Athletikkonzept 2024 / Rahmentrainingskonzeption）

### DHBの長期育成5段階（Langfristiger Leistungsaufbau）

| 段階 | ドイツ語名 | 対象年齢 | 主な内容 |
|---|---|---|---|
| 1 | Grundlagenschulung（基礎教育） | ～12歳 | 多様な動作・楽しさ優先 |
| 2 | Grundlagentraining（基礎トレーニング） | 12〜14歳 | 協調性・自体重・体幹安定 |
| 3 | Aufbautraining I（発展I） | 14〜16歳 | 外部負荷導入（砂袋・ケトルベル） |
| 4 | Aufbautraining II（発展II） | 16〜18歳 | バーベル筋力・爆発力の系統的発達 |
| 5 | Anschlusstraining（接続） | 19歳〜 | 競技特化・個人特化 |

### 高校生（16〜18歳 / Aufbautraining II）の具体的指針

```
週2〜3回、セッション当たり30〜45分
毎練習前にアスレチックウォームアップ（15分）
バーベル筋力トレーニングの本格導入
Schnellkraft（爆発力）とMaximalkraft（最大筋力）の両方を系統的に訓練
固有受容感覚トレーニング（バランスボード等）の導入

ノルウェー研究（PMC 2021・15〜18歳・205名）による警告：
  週6回以上のセッションは過負荷リスクあり
  週間傷害発生率 53%、重篤な怪我の割合 38%
  → RPEスケールを使用し、週sRPE 1,000〜1,400 AU以内に抑える
```

### シーズン区分と重点

| 期間 | ドイツ語 | 主な目標 | 内容 |
|---|---|---|---|
| プレシーズン（6〜8週） | Vorbereitungsphase | 体力基盤構築 | 高volume筋力・有酸素基盤 |
| インシーズン（競技期） | Wettkampfphase | 試合パフォーマンス維持 | 週2回の維持筋力・SSG型有酸素 |
| オフシーズン（移行期） | Übergangsphase | 回復と再構築 | アクティブリカバリー・有酸素ベース |

| 曜日 | 内容 | 負荷 |
|---|---|---|

| MD+2 | 筋力トレーニング（高負荷） | 高 |
| MD+3 | テクニカル・タクティカル＋SSG | 中 |
| MD-3 | 筋力（維持目的・低volume） | 中 |
| MD-2 | スピード・アジリティ | 中 |
| MD-1 | 軽い技術練習・活性化 | 低 |
| 試合当日 | 試合 | 最高 |

## 【H】ケガ予防プログラム（Verletzungsprävention）

### ハンドボール主要傷害部位

```
全傷害の約50%が膝・足首に集中
バックプレーヤーは肩の過負荷傷害（オーバーユース）が最多

Thrower's Shoulder（投球肩）：
  内旋制限GIRD（Glenohumeral Internal Rotation Deficit）
  ＋外旋過剰の組み合わせで発症
```

### 予防エクササイズ

**肩（Schulter / Shoulder）：**
```
外旋運動（External Rotation in 90° Abduction）
  → 90°外転位で内旋・外旋。回旋筋腱板強化。

Yレイズ（Y Raise）
  → 下部僧帽筋・回旋筋腱板・肩甲骨安定筋群。

バンド外旋（Band External Rotation）
  → 低負荷・高反復。毎日実施可能。

研究（Frontiers 2024）：
  6週間の等速性偏心外旋訓練で肩の柔軟性・筋構造が有意改善
  （男子ジュニアハンドボール選手対象）
```

**膝・ハムストリングス：**
```
ノルディックハムストリング（Nordic Hamstring Exercise / NHE）
  → 週1〜2回・20週間継続で大腿二頭筋損傷率50%減（研究エビデンス）
  → 高校生に特に推奨

片脚スクワット（Single-leg Squat）
  → 膝関節アライメント強化
```

**足首（Sprunggelenk / Ankle）：**
```
バランスボードトレーニング
  → 固有受容感覚（Propriozeption）強化

ラテラルバンドウォーク
  → 中臀筋強化→下肢アライメント安定→捻挫予防
```
