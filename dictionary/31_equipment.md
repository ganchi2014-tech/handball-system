## 【B】ボールサイズ規定

### IHF/JHA 公式ボールサイズ
| サイズ | 周囲 | 重量 | 対象 |
|---|---|---|---|
| **3号** | 58〜60cm | 425〜475g | 男子高校・大学・社会人 |
| **2号** | 54〜56cm | 325〜375g | 女子高校・大学・社会人／男子中学 |
| **1号** | 50〜52cm | 290〜330g | 女子中学・小学高学年 |
| **0号** | 46〜48cm | 240〜290g | 小学低学年 |

### 高校での実用
- **男子**：3号球
- **女子**：2号球
- 部活で使う練習ボールも同サイズ（試合と感覚を合わせる）
- 自主練用ボール購入時は **必ず同サイズ** を買う

### ボールの選び方
| 観点 | チェックポイント |
|---|---|
| **メーカー** | モルテン（公式試合球）、ミカサ、セレクト、ヒュンメル |
| **公認球マーク** | IHF公認（試合で使う）、練習球（やや安価） |
| **表面素材** | 合成皮革（屋内用）、ゴム製（屋外・初心者用） |
| **グリップ感** | 凹凸が深い = 滑りにくい（モルテンH3X5001など） |
| **新品か中古か** | 試合前1ヶ月で新品に交換（表面が劣化） |

### ボールの保管
- 直射日光NG（変形・劣化）
- 湿気の少ない場所
- 空気圧チェック（指で押して **少し凹む程度**）
- 試合前に必ず空気圧確認

## 【C】シューズ選び（最も悩むカテゴリ）

### ハンドボールシューズの基本要件
1. **インドア専用** — 屋外シューズ（ランニングシューズ）は体育館で滑る／床を傷める
2. **ノンマーキング底** — 黒い跡が床に付かないゴム底
3. **横方向のサポート** — サイドステップが多いのでアッパーが横ズレを支える
4. **クッション性** — ジャンプ着地の衝撃吸収

### 推奨メーカー・モデル
| メーカー | 特徴 | 価格帯 |
|---|---|---|
| **ミズノ ウェーブメダル** | 国内定番、初心者〜上級者 | 8,000〜15,000円 |
| **アシックス ゲルブラスト** | クッション重視、長時間プレー向き | 12,000〜18,000円 |
| **ヒュンメル** | デンマークブランド、欧州プロも使用 | 10,000〜20,000円 |
| **ミズノ ライトニング** | バレーボール兼用可、軽量 | 10,000〜16,000円 |
| **アディダス** | ハンドボール専用が少ない、卓球シューズで代用も | 8,000〜15,000円 |

### 自分の足型に合わせた選び方
| 足型 | 推奨 |
|---|---|
| **甲が高い** | アシックス系（甲の余裕あり） |
| **足幅が広い** | ミズノ4Eモデル |
| **足幅が狭い** | ヒュンメル、アディダス |
| **疲れやすい** | クッション厚めのモデル（ゲルブラストなど） |
| **足首が弱い（捻挫癖）** | ハイカット or テーピング併用 |

### サイズの選び方
- つま先に **0.5〜1cm の余裕** を残す
- ジャンプ着地で **足が前にズレない** か確認
- 靴ひもをしっかり締めた状態で歩いて違和感ないか
- 必ず **両足試着**（左右で微妙にサイズ違う場合あり）

### 買い替えタイミング
- 底が **平らに磨り減ったら** 即交換（グリップ消失=怪我リスク）
- 通常使用で **6ヶ月〜1年**
- 試合用と練習用を分けると **長持ち**
- 大会前1ヶ月は新品に履き慣らす

### シューズの手入れ
- 試合後 **必ず汚れを拭く**
- 中敷きを外して乾燥
- 月1回 **靴ひもを外して洗濯**（手洗い）
- シューズ用消臭剤 or 重曹で臭い対策

## 【D】松ヤニ（ハーツ／レジン）の使用ルール

### 松ヤニとは
- 樹脂を加工した粘着物質、ボールに塗ってグリップ向上
- ヨーロッパでは標準使用、日本では **体育館によって使用可否が違う**

### 使用可否の確認
| 体育館タイプ | 松ヤニ可否 |
|---|---|
| **専用ハンドボール体育館** | 可（少数） |
| **市民体育館・公共施設** | 多くは **禁止**（床を傷めるため） |
| **学校体育館** | 学校の判断（管理者に確認） |
| **大会会場** | 大会要項に明記される |

### 使用する場合の手順
1. **試合前**：手のひらに少量を塗る（指の腹中心）
2. **試合中**：パスキャッチ時に必要なら追加
3. **試合後**：必ず **完全に拭き取る**（床に残すと違反）

### 松ヤニの種類
| タイプ | 特徴 |
|---|---|
| **液体タイプ（Selectなど）** | 塗りやすい、ハイレベル向け |
| **クリームタイプ（Trimonaなど）** | 中庸、初心者向け |
| **チョークタイプ** | 軽い粘着、女子・中学生向け |

### 床への影響と対策
- 松ヤニ残留 = 床が滑る → 怪我の原因
- 試合後の床掃除（モップ）は **チームの責任**
- 専用クリーナー（Trimona Cleanerなど）を持参

### 高校での現実
- **高校生公式戦の多くは松ヤニ禁止**
- 練習でも体育館使用ルールに従う
- 「グリップが足りない時」は **手汗で代用**（水で軽く湿らせて拭く）

## 【E】ウェア・ユニフォーム規定

### 公式戦ユニフォームの基本
| 項目 | 規定 |
|---|---|
| **チーム統一** | フィールドプレーヤー6名は同色 |
| **GK** | フィールドプレーヤーと **異なる色** |
| **背番号** | 1〜99（4桁不可） |
| **GK背番号** | 通常 1, 12, 16 |
| **キャプテンマーク** | 上腕に色違いのバンド |
| **広告・スポンサーロゴ** | 大会規定範囲内のみ |

### 高校生の練習ウェア
- 動きやすい **吸汗速乾Tシャツ**（綿NG = 汗で重くなる）
- **ハーフパンツ or 短パン**（膝までの長さ）
- **着圧スパッツ**（任意）— 太もも・ふくらはぎの疲労軽減

### 試合用追加装備
| 装備 | 目的 |
|---|---|
| **着圧スパッツ（CW-X, スキンズ等）** | 筋肉サポート、疲労軽減 |
| **アームスリーブ** | 肘の保温・接触保護 |
| **指テーピング** | 突き指予防（特にGK・ピボット） |
| **足首サポーター** | 捻挫癖がある選手向け |
| **膝サポーター** | ジャンパー膝（PV/SB向け） |
| **マウスピース** | 顔面接触対策（推奨） |

### ユニフォーム規定の落とし穴
- **背番号印刷の剥がれ** → 試合直前にチェック
- **異性チーム混合練習** → 配色被り注意（試合前にビブス確認）
- **新ユニフォーム導入** → IHF/JHA規定に沿っているか確認

## 【F】アクセサリー禁止・着用規定

### 試合中に **禁止** のもの
| アイテム | 理由 |
|---|---|
| **指輪** | 接触時の怪我リスク |
| **ピアス・イヤリング** | 同上、テーピングで隠してもNG |
| **ネックレス** | 引っ掛かり・首絞めリスク |
| **腕時計** | 接触時の怪我 |
| **ヘアピン（金属）** | 怪我リスク |
| **長い爪** | 相手選手への傷つけ |

### 試合中に **OK** のもの
| アイテム | 条件 |
|---|---|
| **ヘアゴム（布製）** | 髪が顔にかからないように |
| **メガネ** | スポーツメガネ（ストラップ付き）推奨 |
| **コンタクトレンズ** | 推奨（メガネより安全） |
| **テーピング** | 自分の怪我予防のため |
| **サポーター** | 同上 |
| **マウスピース** | むしろ推奨 |

### 髪型の規定
- 長髪は **必ず結ぶ**（プレー中の視界・接触対策）
- **三つ編み** は審判から「金属ピン使用」と疑われやすい → 通常のポニーテール推奨
- 試合直前に **コーチ or キャプテンが視認チェック**

### 爪・身だしなみ
- **爪は短く** — 試合前日にカット
- 爪先が指を超えないこと（審判チェック対象）
- **テープ** で爪を覆う方法もあるが、剥がれやすい

## 【G】試合に持って行くものリスト

### 必須（忘れたら試合に出られない可能性）
- [ ] **ユニフォーム上下**（背番号確認）
- [ ] **インドアシューズ**（紐の確認）
- [ ] **靴下**（予備1足含めて2足）
- [ ] **タオル**（顔用・床用2枚）
- [ ] **スポーツドリンク or 水**（500ml × 2本以上）
- [ ] **テーピング・サポーター**（自分の必需品）
- [ ] **携帯電話**（連絡用、試合中はベンチへ）

### 推奨
- [ ] **着圧スパッツ**
- [ ] **アームスリーブ・マウスピース**
- [ ] **エナジーバー/おにぎり/バナナ**（ハーフタイム補食）
- [ ] **保冷剤 + アイスバッグ**（試合後の冷却用）
- [ ] **ジャージ/ウィンドブレーカー**（体温管理）
- [ ] **シャワー道具**（試合後）
- [ ] **着替え**（試合後の通学用）
- [ ] **ルールブック or アプリ**（試合中の疑問解決）
- [ ] **筆記用具・ノート**（試合振り返り用）

### あると便利
- [ ] **モバイルバッテリー**
- [ ] **薬（鎮痛剤・湿布）**
- [ ] **ビニール袋**（汚れたウェア入れ）
- [ ] **コインケース**（自販機・移動用）
- [ ] **保険証コピー**（怪我対応）

### チェック方法
- **前夜に並べる** → 朝バタバタしない
- リストを **スマホメモ** に保存（毎試合確認）
- **試合バッグ専用** に統一（普段使いと分ける）

## 【H】装備の予算とコスパ

### 高校3年間の標準予算
| 項目 | 1年目 | 2年目 | 3年目 | 合計 |
|---|---|---|---|---|
| シューズ | 10,000 | 10,000 | 10,000 | 30,000円 |
| ユニフォーム（部費） | 15,000 | - | - | 15,000円 |
| 練習ウェア | 5,000 | 3,000 | 3,000 | 11,000円 |
| ボール（自主練用） | 5,000 | - | 5,000 | 10,000円 |
| サポーター類 | 3,000 | 2,000 | 2,000 | 7,000円 |
| **合計目安** | **38,000** | **15,000** | **20,000** | **73,000円** |

### コスト削減のコツ
- **シューズ**：型落ちモデルを公式ショップで（30%安）
- **ボール**：チーム共同購入で割引
- **ウェア**：練習用はユニクロ・ワークマンの吸汗速乾シャツで代用可
- **サポーター**：医療用テーピングは100円ショップでも一定品質

## 【I】よくある失敗と対策

| 失敗 | 対策 |
|---|---|
| 「シューズ大きすぎて足が前にズレる」 | 試着時にジャンプ着地を試す |
| 「ボールが滑る」 | 松ヤニ可否確認、または手の汗で湿らせる |
| 「ユニフォーム背番号が剥がれた」 | 試合前日にチェック、両面テープで応急処置 |
| 「指輪を外し忘れた」 | 試合直前にキャプテンが全員チェック |
| 「タオル忘れた」 | 部室の予備 or チームメイトに借りる |
| 「靴ひもが切れた」 | 替えひもを常備（100円） |
| 「水分忘れた」 | 会場の自販機 or 共同水筒 |

## 【J】じゃんけん：装備による戦術選択

| 状況 | 装備の判断 |
|---|---|
| 雨天移動の試合 | シューズは **濡れない袋** に入れて持参 |
| 暑い会場 | 着圧スパッツより **冷感ウェア** 優先 |
| 寒い会場 | アームスリーブ・レッグウォーマー必須 |
| 連戦（2試合目） | 試合間に **新しい靴下** に交換（足の蒸れ対策） |
| 怪我明け復帰 | 該当部位の **サポーター必須**（再発予防） |
