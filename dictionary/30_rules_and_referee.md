# 30. ルール・反則・審判辞典（日本高校生向け）

> 「今のファウル？」「パッシブって何？」を試合中に自分で判断できるようになるための辞典。
> **設計哲学**：コーチに聞かずに、自分で読んで自分で判断する。
> **対象**：日本の高校ハンドボール選手（インターハイ・春高・国体・地区大会レベル）。

---

## 【A】このファイルの使い方

「**試合中に何が反則だったか分からない**」が最大の困りごと。本ファイルは：

1. **場面別反則早見表** — 「今の動作」がどのルール条文に該当するか
2. **接触の線引き（第8条）** — どこまでが合法／フリースロー／警告／2分／失格か
3. **判定エスカレーション** — 警告→2分退場→失格→赤+青 の境界
4. **パッシブプレー詳細** — 高校生が一番混乱する判定
5. **主審・副審の見分け方** — どっちに聞けばいい？

## 【B】IHF/JHA ルールの基本構造

### ルールの優先順位
1. **IHFルール（International Handball Federation）** — 世界共通
2. **JHA（日本ハンドボール協会）規程** — 日本国内追加
3. **大会要項** — 各大会の細則（試合時間・延長・等）

### 試合の基本フォーマット（高校）
| 項目 | 規定 |
|---|---|
| 試合時間 | 前半30分 + 後半30分（高校公式戦） |
| ハーフタイム | **15分** が標準（大会要項で10分の例あり） |
| タイムアウト | 各チーム前半1回・後半2回（合計3回／チームタイムアウト1回1分） |
| 延長戦 | 5分休憩後、10分間（前半5分+後半5分、ハーフタイム1分）。第二延長も同形式 |
| 7m勝負（PK戦） | IHF: 5本ずつ → サドンデス／**日本高校大会の多くは3本ずつ → 3本ずつ追加 → サドンデス**（大会要項を必ず確認） |
| 選手登録 | 16名以内（IHF）／大会要項で増減あり |
| 同時出場 | コートに7名（GK含む） |

## 【C】反則早見表

### 攻撃側の反則（OF）
| 反則名 | 動作 | 罰則 |
|---|---|---|
| **ダブルドリブル** | ドリブル止めて再開 | フリースロー（相手） |
| **オーバーステップ（4歩以上）** | ボール受領後 **3歩までOK**、4歩目から反則。ジャンプして空中でボールをキャッチし、両足を同時に地面につけた場合、その着地は歩数（ステップ）に数えない（IHFルール第7条3項。2025年7月改正） | フリースロー |
| **3秒オーバー** | ボール保持したまま **3秒以上** 動かない（ドリブルもしない） | フリースロー |
| **ライン踏み** | OF選手が攻撃時の6mラインを踏む | ターンオーバー |
| **ピボット6m侵入** | ピボットが6m内に立ったままパス受ける | フリースロー |
| **チャージング（攻撃ファウル）** | 攻撃側（OF）の選手が守備側（DF）の選手を押し倒す | フリースロー＋警告対象 |
| **空中6m侵入** | ジャンプシュート後の着地が6m内（シュート行為で離れた場合は可） | 場合により無効 |
| **ファイトオフ** | OFが腕で押し返す | フリースロー |

### 守備側の反則（DF reglas）
※「DF」は守備（ディフェンス）の略。守る側の選手がやってしまう反則をまとめた表です。

| 反則名 | 動作 | 罰則 |
|---|---|---|
| **ホールディング** | OFを抱える・腕で抑える | フリースロー + 警告対象 |
| **ハッキング（叩く）** | OFの腕・手を叩く | フリースロー + 2分退場対象 |
| **プッシング** | 体で押し倒す | フリースロー + 2分退場対象 |
| **トリッピング（足払い）** | 足を引っ掛ける | 2分退場 + 危険なら赤 |
| **顔面・首への接触** | 上半身への危険プレー | 即時2分退場 or 赤 |
| **背後からの接触** | 見えない位置からの接触 | 2分退場 |
| **GKの6m外への出過ぎ** | GKがゴールエリア外でボール扱う | フリースロー |
| **GKラインクロス（7m時）** | 7mシュート時GKがラインを越える | 再投げまたは無効 |

### 共通の反則
| 反則名 | 動作 | 罰則 |
|---|---|---|
| **パッシブプレー** | 攻撃意図なくボール回し（→詳細はD章） | フリースロー（相手） |
| **タイムアウト不正取得** | 規定外でタイム要求 | チームに警告 |
| **抗議・暴言** | 審判への異議 | 警告→2分→失格 |
| **ベンチ外コーチング** | コーチエリア外からの指示 | チーム警告 |
| **不適切な交代** | 交代ゾーン外からの出入り | 2分退場（出場選手） |

## 【C-2】接触の線引き：合法 → フリースロー → 警告 → 2分 → 失格（IHF競技規則 第8条）

> 「どこまでの接触がOKで、どこからが反則・ペナルティか」を競技規則そのもので理解する章。
> ハンドボールは接触のあるスポーツ。**接触＝即反則ではない**。「何を・どこに・どれだけの強さで」当てたかで罰が変わる。
> （出典：IHF Rules of the Game / Rule 8「Fouls and Unsportsmanlike Conduct」2025年3月版）

### ① まず「やってよい接触」＝笛は鳴らない（第8条1項）
守備（DF）が攻撃（OF）に対して、次の接触は**認められている**：
- **曲げた腕（ベントアーム）を相手の体に当てて、ついていく・止める**（腕を伸ばして押さえ込むのはNG）
- **胴体（トランク）で相手の進路をブロックする**（位置取りの競り合い。腕でなく体で）
- **開いた手で、相手の手からボールをはたく**（狙うのはボール。手をつかむのはNG）
※ ブロック＝「相手が空きスペースへ動くのを止める」こと。基本は受け身（パッシブ）に行う。

### ② やってはいけない接触＝基本はフリースロー（第8条2項）
- 相手の手からボールを**つかむ・叩き落とす**
- **腕・手・脚で相手をブロックして押しのける／押し出す**（危険な肘の使用も含む）
- 相手を**ホールド（体やユニフォームをつかむ）**（相手がプレー続行できても反則）
- 相手に**走り込む・跳び込む**
→ これらは原則 **フリースロー**（明確な得点機会の阻止なら **7mスロー**）。

### ③ 線引きの核心：体に向かった反則は「段階的罰則」（第8条3項）
**ボールでなく相手の体に向かった反則**は、フリースロー／7mに**加えて**、人へのペナルティが段階的に上がる：
**警告（イエロー）→ 2分退場 → 失格**

どの段階かは、審判が次の**4基準**を組み合わせて判断する（＝これが「線引き」の正体）：
| 基準 | 内容（重くなる方向） |
|---|---|
| **A 位置** | 正面 ＜ 横 ＜ **背後**（見えない背後ほど重い） |
| **B 当てた部位** | 胴体 ＜ **シュート腕・脚** ＜ **頭・首・のど**（頭頸部が最も重い） |
| **C 強さ・状況** | 軽い ＜ 強い／相手が**全力で動いている**最中ほど重い |
| **D 結果** | 体やボール扱いへの影響・動きを止めた・プレーを中断させた |

（シュート動作中／スペースへ走り込み中／高速移動中、などの場面性も考慮される）

### ④ いきなり2分退場（警告を飛ばす・第8条4項）
危険を顧みない反則は、警告なしで**直接2分退場**：
- **高強度**、または**全力疾走中の相手**への反則
- **長く抑える・引きずり倒す**ホールド
- **頭・首・のど**への反則
- **胴体やシュート腕を強く叩く**
- **跳んでいる相手の脚・足をつかむ**等、体勢を崩させる
- **高速で走り込む・跳び込む**

### ⑤ 失格（レッド・第8条5項）＝相手の健康に危険
- 高強度、または**相手が無防備で身を守れない**状況での反則
- 走行・跳躍・シュート中に**体勢を完全に崩させる**
- **顔・のど・首**への特に攻撃的な接触／無謀な態度での反則
※ 接触が小さくても、**空中や全力疾走で身を守れない瞬間**なら危険＝失格になりうる（強さでなく「危険度」で判断）。

### ⑥ 失格＋報告書（ブルー・第8条6項）
- 特に無謀・危険・**故意・悪質**で、プレーと無関係な行為 → 試合後に書面報告し、協会が追加処分を審査。

### 高校生向け早わかり：同じ「止める」でも段階が変わる
| 場面 | 判定の目安 |
|---|---|
| 曲げた腕を相手の体に当てて並走 | **合法**（笛なし） |
| 相手の腕をつかんで止めた | **フリースロー**（繰り返せば警告） |
| 走り込むバックを正面から強く突いた | **2分退場** |
| 跳んだシューターのシュート腕を強く叩いた | **2分退場〜失格** |
| 背後から、無防備な相手の首・頭へ接触 | **失格** |
| 故意の危険な暴力的行為 | **失格＋報告書（青）** |

> 覚え方：**「ボールを狙えば軽い／体を狙うほど重い」「正面より背後、胴より頭が重い」「相手が無防備なほど重い」**。
> 迷ったら「①の合法接触（曲げた腕・胴体ブロック）の範囲に収まっているか？」を基準にする。
> ※ 上の【C】反則早見表の罰則は"代表的な段階"。実際は本章④〜⑥の強さ・危険度で1段階・2段階上がる。

## 【D】パッシブプレー（高校生が最も混乱する判定）

### パッシブとは
**攻撃側が得点意図のないボール回しを続けている**と審判が判定したとき。

### 判定プロセス（IHF 2022年7月以降の現行ルール）
1. **予告サイン**：副審が黄色い旗を頭上で振る（「攻撃意図を示せ」）
2. **4回のパス制限**：予告後、攻撃側に**4回までのパス**が許される（旧6回・旧旧8回から短縮）
3. **4回以内にシュートに至らなければ** → パッシブ判定 → 相手フリースロー
4. ※ 4回目のパス後にフリースロー／スローイン（ボールがサイドラインの外に出たときに外から投げ入れて再開するプレー）が与えられた場合、攻撃を完了させるため**追加1パス**まで認められる

### パッシブを取られないコツ
- 予告サインが出たら**2〜3回目のパスで仕掛けを始める**
- 予告サイン中は**シュートフェイント・ドリブル突破を見せる**
- 仕掛けに失敗してもシュートまで持ち込めば**サインリセット**

### 場面別判定基準
| 状況 | パッシブ取られる？ |
|---|---|
| リード守りで時間消化したい | 取られる（戦術的に取られる覚悟） |
| シュート2本連続で外して攻撃停滞 | 取られやすい |
| ピボットへのパスを5本連続 | 「攻撃意図あり」と判定される可能性高い |
| ボール失わずキープ重視 | 取られる |

## 【E】退場のエスカレーション

### 段階別ペナルティ
| 段階 | 罰則 | 出場可否 |
|---|---|---|
| ① 警告（イエローカード） | 注意のみ | そのまま継続 |
| ② 2分退場 | 2分間、人数不利 | 2分後復帰 |
| ③ 失格（レッドカード） | 試合終了まで退場 | 復帰不可 |
| ④ 失格＋報告書（ブルーカード） | 試合終了+次戦以降出場停止 | 復帰不可・要協会審査 |

### イエロー（警告）の閾値（IHF公式）
- **同一選手は1試合に1枚のみ**：それ以降の本来イエロー相当の反則は直接 2分退場
- **1チーム合計3枚まで**：4枚目以降の本来イエロー相当の反則も2分退場
- **ベンチで誰でも貰える**：選手・コーチ・帯同医（ベンチオフィシャルは1試合1枚で打ち止め）
- イエロー対象動作：軽度のホールディング、押し合い、抗議、遅延行為など

### 2分退場（イエロー2分相当）
- ハッキング、プッシング、危険でない接触ファウル
- 同一選手が **3回目の2分退場** → 自動的に失格（赤）
- 2分中は **自チームが人数不利（コートプレーヤー5人＋GK1人の合計6人。通常より1人少ない状態）** → 攻撃選択肢が減る

### 失格（レッド）
- 危険なプレー（顔面接触、足払い、背後からの強い接触）
- 暴言・暴力行為
- 失格選手が出ても **2分後にチームは7人に戻れる**（選手交代可能）
- ただし **失格選手本人** はその試合終了まで出場不可

### 失格＋報告書（青）
- 特に悪質と判断された場合
- 次戦以降の出場停止審査対象
- ベンチオフィシャルも対象

## 【F】主審・副審の役割分担

### 2人制レフリー（IHF標準）
| 審判 | 配置 | 主な責任 |
|---|---|---|
| **主審（Court Referee）** | コート両サイドライン外 | 全体の判定・カード提示・最終決定 |
| **副審（Goal Referee）** | ゴールライン後方 | ゴール判定・6mライン侵入・パッシブサイン提示 |

### 判定権限の優先順位
1. **ゴール判定** → 副審が一次判定（ゴールライン後方から見えるため）
2. **ファウル判定** → 主審が優先（角度が広い）
3. **両者の判定が割れた場合** → **主審が最終決定**
4. **得点・退場の最終決定** → 主審

### 質問・確認のタイミング
- **試合中の異議**：基本NG（警告対象）。「教えてください」の姿勢で
- **タイムアウト中**：キャプテンのみ主審に質問可
- **ハーフタイム/試合後**：審判控え室への直接抗議は禁止、書面で大会本部経由

## 【G】ベンチオフィシャル（コーチ・帯同医）

### コーチエリアの規則
- **指定エリア内のみ立ち入り可** — 通常はベンチ前 ±2m
- エリア外からの指示 → チーム警告
- 試合中の選手交代は **交代ゾーン**（自陣ハーフライン側）からのみ
- ベンチ役員数：原則4名以内（コーチ・アシスタント・帯同医・マネージャー）

### グリーンカード（タイムアウトカード）
- 各チーム3枚（前半1、後半2）
- ベンチオフィシャルがタイムキーパーに渡す
- 渡した時点で次の自チームボール保持時にタイムアウト発動
- カード提示は **1分以上のタイムアウトを取る場合のみ**

## 【H】試合進行の細部

### キックオフ（試合開始）
- コートセンターから1名がパス
- 残り選手は **自陣ハーフ** に位置
- 笛と同時にプレー開始（笛前の動きは反則）

### 失点後の試合再開
- スコア後、相手チームが **センターからスローオフ**
- **クイックスローオフ（即時センター投擲）**：スローオフ直後の速攻も合法
- 自陣ハーフラインを越えて待ち構えるのは反則

### ゴールキーパーの特殊規則
- ゴールエリア内では足を使ってよい
- ゴールエリア内ではボール保持時間制限なし
- ゴールエリアを出るとフィールドプレーヤー扱い
- GK着用ユニフォームは他選手と異なる色

### 7mスロー判定
| 状況 | 7m獲得？ |
|---|---|
| 明らかな得点機会をDFが反則で阻止 | ✓ 必ず7m |
| 6m付近で軽度ホールディング → シュート阻止 | ✓ 7m |
| ハーフライン付近で軽度反則 | ✗ 通常フリースロー |
| ピボットへの背後ホールディング | △（ケースによる）シュート機会の有無で判断 |

## 【I】自己組織化のための判定スキル

### 試合中に自分で判定を読み解く3ステップ
1. **笛が鳴った瞬間に位置を見る** — 主審の指方向＝反則発生位置
2. **主審の手のサインを見る** — フリースロー方向・退場・タイムアウト等
3. **副審の旗を見る** — パッシブ予告・ゴール判定

### 主審サイン（高校生が覚えるべきTOP10）
1. **両手で押す動作** → プッシング（押した側に反則）
2. **片手で腕を叩く動作** → ハッキング
3. **両手で抱える動作** → ホールディング
4. **指を立てる + 黄色提示** → 警告
5. **指2本立てる** → 2分退場
6. **指を回す動作** → ステップオーバー
7. **手のひらを下に何度も** → ダブルドリブル
8. **両手を頭上で交差** → 失格
9. **片手で「T」型** → タイムアウト
10. **黄色い旗を頭上で振る（副審）** → パッシブ予告

## 【J】よくある誤解の修正

| 誤解 | 正しい理解 |
|---|---|
| 「3歩以上歩いてはいけない」 | 正解は **3歩まで歩いてOK**（4歩目から反則） |
| 「ボールを2秒以上持ってはいけない」 | 正解は **3秒以上**（IHFは「3秒」と明記） |
| 「6mラインを踏んだら反則」 | OF着地時のみ。**ジャンプ中はOK**、空中シュートはルール内 |
| 「赤カードは即試合終了」 | 失格選手のみ。**チームは2分後に7人に戻る** |
| 「タイムアウトは何度でも取れる」 | 各チーム前半1回・後半2回・合計3回まで |
| 「審判の判定は絶対」 | 確かに最終判定だが、**主審に「教えてください」と聞くのは可** |

## 【K】ルール改正の追いかけ方

### 公式情報源
- **IHF公式サイト**：International Handball Federation Rules of the Game（年次改訂）
- **JHA公式サイト**：日本ハンドボール協会 ルールブック日本語訳
- **高体連**：高校選手権大会要項（毎年更新）

### 最近の主要改正
- **2022年7月**：パッシブ予告後のパス回数を **4回** に短縮（旧6回、それ以前は無制限）
  ※4回目のパス後にフリースロー／スローイン獲得時は追加1パスのみ認められる
- **GK外し（7on6）**：GKがフィールドプレーヤーとして交代可能（2016〜）
- **ブルーカード（青）**：失格＋報告書の区別が明確化
- **2025年7月以降**：審判付加報告（重大反則）の運用が改定（IHF最新）

## 【K-2】アクセサリー・身だしなみ規定（試合前必須チェック）

### 試合中に **禁止** のもの
| アイテム | 理由 |
|---|---|
| **指輪** | 接触時の怪我リスク |
| **ピアス・イヤリング** | 同上、テーピングで隠してもNG |
| **ネックレス** | 引っ掛かり・首絞めリスク |
| **腕時計** | 接触時の怪我 |
| **金属ヘアピン** | 怪我リスク |
| **長い爪** | 相手選手への傷つけ |

### 試合中に **OK** のもの
| アイテム | 条件 |
|---|---|
| **布製ヘアゴム** | 髪が顔にかからないように |
| **スポーツメガネ** | ストラップ付き推奨 |
| **コンタクトレンズ** | メガネより安全、推奨 |
| **テーピング・サポーター** | 自分の怪我予防のため |
| **マウスピース** | 顔面接触対策、むしろ推奨 |

### 髪型の規定
- 長髪は **必ず結ぶ**（プレー中の視界・接触対策）
- **三つ編み** は審判から金属ピン使用と疑われやすい → ポニーテール推奨
- 試合直前に **キャプテン or コーチが視認チェック**

### 爪・身だしなみ
- **爪は短く** — 試合前日にカット
- 爪先が指を超えないこと（審判チェック対象）
- テープで爪を覆う方法もあるが剥がれやすい

### 試合前 30秒チェックリスト
- [ ] 指輪を外した
- [ ] ピアス・イヤリングを外した
- [ ] ネックレスを外した
- [ ] 腕時計を外した
- [ ] 髪を結んだ（長髪の場合）
- [ ] 爪を切った（指先を超えていない）
- [ ] 金属ピンを使っていない

→ キャプテンが全員に「外した？」と確認する習慣を作る。

## 【M】じゃんけん：ルールを使った戦術

| 状況 | じゃんけんの一手 |
|---|---|
| リードして時間消化したい | パッシブ取られる覚悟でボール回し（戦術的選択） |
| 相手にイエロー2枚（次は2分） | 接触多めに仕掛ける（3枚目=2分退場狙い） |
| 自分が2分退場 | チーム5+GKで守る → カウンター速攻狙い |
| 相手キーマンが2分退場 | その時間内に4点差以上つける目標 |
| パッシブ予告された | ピボット中心の3-4本パスで突破 or 個人技で抜く |
