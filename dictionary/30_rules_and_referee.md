# 30. ルール・反則・審判辞典（日本高校生向け）

> 「今のファウル？」「パッシブって何？」を試合中に自分で判断できるようになるための辞典。
> **設計哲学**：コーチに聞かずに、自分で読んで自分で判断する。
> **対象**：日本の高校ハンドボール選手（インターハイ・春高・国体・地区大会レベル）。
> **出典**：（公財）日本ハンドボール協会『ハンドボール競技規則（2025）』＝IHF "Rules of the Game for Indoor Handball"（**2025年7月1日施行**）。本文中の「第○条○項」「8：3」等の表記は公式条文番号に対応。条文の数値・判定基準はこの公式規則書で検証済み。
> ※ ただし**試合時間・延長・7m勝負の本数・装具の細部などは大会要項（高体連・各連盟）が優先**する場面がある。最終確認は必ず大会要項と試合前の主審確認で。

## 【A】このファイルの使い方

「**試合中に何が反則だったか分からない**」が最大の困りごと。本ファイルは：

1. **場面別反則早見表** — 「今の動作」がどのルール条文に該当するか
2. **接触の線引き（第8条）** — どこまでが合法／フリースロー／警告／2分／失格か
3. **判定エスカレーション** — 警告→2分退場→失格→赤+青 の境界
4. **パッシブプレー詳細** — 高校生が一番混乱する判定
5. **主審・副審の見分け方** — どっちに聞けばいい？

## 【B】IHF/JHA ルールの基本構造

### ルールの優先順位
1. **IHFルール（International Handball Federation）** — 世界共通
2. **JHA（日本ハンドボール協会）規程** — 日本国内追加
3. **大会要項** — 各大会の細則（試合時間・延長・等）

### 試合の基本フォーマット（成年・高校）※カッコ内は公式条文
| 項目 | 規定 |
|---|---|
| 試合時間 | **前半30分 + 後半30分**（成年・高校の標準。第2条1）。中学生は前後半各25分、小学生はJクイック方式 |
| 休憩（ハーフタイム） | **10分が標準**。主催者権限で延ばせるが**最長15分**まで（第2条1）。※旧版辞典の「15分標準」は誤り |
| チームタイムアウト | **1分間**。公式の基本は**前半・後半に各1回ずつ（延長戦は不可）**（第2条10）。主催者権限で各チーム最大3回まで増やせるが、**前半・後半それぞれ最大2回・後半残り5分は1回のみ**（競技規則解釈3）。請求は自チームのボール保持中に「グリーンカード（T表示）」で |
| 延長戦 | 同点で勝敗を決める場合、5分休憩後に**前後半各5分**（間に1分休憩）。第2延長も同形式（第2条2） |
| 7m勝負（7mスローコンテスト） | 公式（IHF/JHA）は**各チーム5名ずつ・交互に1投**、コイントスで先後攻。5本ずつで同点なら先後攻を入れ替え、改めて5名選出（重複可）して決着まで（第2条2注）。**ただし日本の高校大会は大会要項で「3本ずつ」など本数が異なることが多い → 要項を必ず確認** |
| 選手登録 | **16名まで**（2025改正で14→16名。第4条1）。大会要項で上限が下がる場合あり |
| チーム役員（ベンチ） | **5名まで**（2025改正で4→5名。第4条2）。うち1名を「チーム責任者」に指名 |
| 同時出場 | コートに**7名**（GK1名＋コートプレーヤー6名）。GKを置かない作戦（7on6）なら**コートプレーヤー最大7名**（第4条1） |
| 競技開始 | 各チーム**5名以上**いれば試合開始可。試合中いつでも16名まで補充可（第4条1） |

## 【B-3】歩数（ステップ）の正確な数え方（第7条3項＋2025★★ガイドライン）

> 高校生が「トラベリング取られた／取られなかった」で一番もめる部分。**「3歩までOK・4歩目から反則」は正しい**が、"いつ0歩スタートになるか"が肝心。

### 0歩（ノーステップ）になる＝得する場面
- **空中で味方からのパスを受け**、片足または両足同時で着地 → その着地は**0歩**。そこから**さらに3歩**動ける（実質ジャンプキャッチで1歩得する）

### 1歩と数える基本（第7条3項a〜d）
- 両足で立ってキャッチ → 片足を上げて下ろす／動かすと1歩
- 片足立ちでキャッチ → もう一方の足を着くと1歩
- 空中でキャッチして片足着地 → 同じ足で跳んで着地、または他方の足を着くと1歩

### ⚠️ 引っかかりやすい例外：**ドリブルの後**は0歩にならない
- **自分のドリブルをキャッチ**した場合は「パスを受ける」に当たらない → **ジャンプ中にキャッチして着地しても、その着地は第1歩**（0歩ではない）。2025年★★ガイドラインで明確化
- つまり「ジャンプキャッチで1歩得」は**味方からのパス限定**。ドリブル→ジャンプキャッチ→着地→3歩、はオーバーステップ

### その他
- 転倒して滑った後・ボールに飛びついて取った後に立ち上がってプレー再開＝**反則ではない**
- ドリブル（はずませる）→つかむと、その瞬間から**3歩以内かつ3秒以内**に手放す（第7条4項）
- ダブルドリブル＝一度つかんだ後に再びドリブル開始は不可（第7条7項）。ただし「ファンブル（キャッチし損ない）」は複数回触れてOK

## 【C】反則早見表

### 攻撃側の反則（OF）
| 反則名 | 動作 | 罰則 |
|---|---|---|
| **ダブルドリブル** | ドリブル止めて再開 | フリースロー（相手） |
| **オーバーステップ（4歩以上）** | ボール保持後 **3歩までOK**、4歩目から反則（第7条3項）。**空中でパスを受けて着地した場合、その着地は0歩**（＝そこから3歩動ける）。⚠️ただし**自分のドリブル後**に空中でキャッチして着地した場合は0歩にならず**着地が第1歩**（2025年★★ガイドライン）。詳細は【B-3】 | フリースロー |
| **3秒オーバー** | ボール保持したまま **3秒を超えて**持つ（床にボールをつけて押さえる状態も保持に含む。第7条2項） | フリースロー |
| **ライン踏み** | OF選手が攻撃時の6mラインを踏む | ターンオーバー |
| **ピボット6m侵入** | ピボットが6m内に立ったままパス受ける | フリースロー |
| **チャージング（攻撃ファウル）** | 攻撃側（OF）の選手が守備側（DF）の選手を押し倒す | フリースロー＋警告対象 |
| **空中6m侵入** | ジャンプシュート後の着地が6m内（シュート行為で離れた場合は可） | 場合により無効 |
| **ファイトオフ** | OFが腕で押し返す | フリースロー |

### 守備側の反則（DF）
※「DF」は守備（ディフェンス）の略。守る側の選手がやってしまう反則をまとめた表です。

| 反則名 | 動作 | 罰則 |
|---|---|---|
| **ホールディング** | OFを抱える・腕で抑える | フリースロー + 警告対象 |
| **ハッキング（叩く）** | OFの腕・手を叩く | フリースロー + 2分退場対象 |
| **プッシング** | 体で押し倒す | フリースロー + 2分退場対象 |
| **トリッピング（足払い）** | 足を引っ掛ける | 2分退場 + 危険なら赤 |
| **顔面・首への接触** | 上半身への危険プレー | 即時2分退場 or 赤 |
| **背後からの接触** | 見えない位置からの接触 | 2分退場 |
| **GKの6m外への出過ぎ** | GKがゴールエリア外でボール扱う | フリースロー |
| **GKラインクロス（7m時）** | 7mシュート時GKがラインを越える | 再投げまたは無効 |

### 共通の反則
| 反則名 | 動作 | 罰則 |
|---|---|---|
| **パッシブプレー** | 攻撃意図なくボール回し（→詳細はD章） | フリースロー（相手） |
| **タイムアウト不正取得** | 規定外でタイム要求 | チームに警告 |

| **ベンチ外コーチング** | コーチエリア外からの指示 | チーム警告 |
| **不適切な交代** | 交代ゾーン外からの出入り | 2分退場（出場選手） |

## 【C-2】接触の線引き：合法 → フリースロー → 警告 → 2分 → 失格（IHF競技規則 第8条）

> 「どこまでの接触がOKで、どこからが反則・ペナルティか」を競技規則そのもので理解する章。
> ハンドボールは接触のあるスポーツ。**接触＝即反則ではない**。「何を・どこに・どれだけの強さで」当てたかで罰が変わる。
> （出典：JHA競技規則2025 第8条＝IHF Rules of the Game Rule 8「Fouls and Unsportsmanlike Conduct」2025年7月1日施行）

### ① まず「やってよい接触」＝笛は鳴らない（第8条1項）
守備（DF）が攻撃（OF）に対して、次の接触は**認められている**：
- **曲げた腕（ベントアーム）を相手の体に当てて、ついていく・止める**（腕を伸ばして押さえ込むのはNG）
- **胴体（トランク）で相手の進路をブロックする**（位置取りの競り合い。腕でなく体で）
- **開いた手で、相手の手からボールをはたく**（狙うのはボール。手をつかむのはNG）
※ ブロック＝「相手が空きスペースへ動くのを止める」こと。基本は受け身（パッシブ）に行う。

### ② やってはいけない接触＝基本はフリースロー（第8条2項）
- 相手の手からボールを**つかむ・叩き落とす**
- **腕・手・脚で相手をブロックして押しのける／押し出す**（危険な肘の使用も含む）
- 相手を**ホールド（体やユニフォームをつかむ）**（相手がプレー続行できても反則）
- 相手に**走り込む・跳び込む**
→ これらは原則 **フリースロー**（明確な得点機会の阻止なら **7mスロー**）。

### ③ 線引きの核心：体に向かった反則は「段階的罰則」（第8条3項）
**ボールでなく相手の体に向かった反則**は、フリースロー／7mに**加えて**、人へのペナルティが段階的に上がる：
**警告（イエロー）→ 2分退場 → 失格**

どの段階かは、審判が次の**4基準**を組み合わせて判断する（＝これが「線引き」の正体）：
| 基準 | 内容（重くなる方向） |
|---|---|
| **A 位置** | 正面 ＜ 横 ＜ **背後**（見えない背後ほど重い） |
| **B 当てた部位** | 胴体 ＜ **シュート腕・脚** ＜ **頭・首・のど**（頭頸部が最も重い） |
| **C 強さ・状況** | 軽い ＜ 強い／相手が**全力で動いている**最中ほど重い |
| **D 結果** | 体やボール扱いへの影響・動きを止めた・プレーを中断させた |

（シュート動作中／スペースへ走り込み中／高速移動中、などの場面性も考慮される）

### ④ いきなり2分退場（警告を飛ばす・第8条4項）
危険を顧みない反則は、警告なしで**直接2分退場**：
- **高強度**、または**全力疾走中の相手**への反則
- **長く抑える・引きずり倒す**ホールド
- **頭・首・のど**への反則
- **胴体やシュート腕を強く叩く**
- **跳んでいる相手の脚・足をつかむ**等、体勢を崩させる
- **高速で走り込む・跳び込む**

### ⑤ 失格（レッド・第8条5項）＝相手の健康に危険
- 高強度、または**相手が無防備で身を守れない**状況での反則
- 走行・跳躍・シュート中に**体勢を完全に崩させる**
- **顔・のど・首**への特に攻撃的な接触／無謀な態度での反則
※ 接触が小さくても、**空中や全力疾走で身を守れない瞬間**なら危険＝失格になりうる（強さでなく「危険度」で判断）。

### ⑥ 失格＋報告書（ブルー・第8条6項）
- 特に無謀・危険・**故意・悪質**で、プレーと無関係な行為 → 試合後に書面報告し、協会が追加処分を審査。

### 高校生向け早わかり：同じ「止める」でも段階が変わる
| 場面 | 判定の目安 |
|---|---|
| 曲げた腕を相手の体に当てて並走 | **合法**（笛なし） |
| 相手の腕をつかんで止めた | **フリースロー**（繰り返せば警告） |
| 走り込むバックを正面から強く突いた | **2分退場** |
| 跳んだシューターのシュート腕を強く叩いた | **2分退場〜失格** |
| 背後から、無防備な相手の首・頭へ接触 | **失格** |
| 故意の危険な暴力的行為 | **失格＋報告書（青）** |

> 覚え方：**「ボールを狙えば軽い／体を狙うほど重い」「正面より背後、胴より頭が重い」「相手が無防備なほど重い」**。
> 迷ったら「①の合法接触（曲げた腕・胴体ブロック）の範囲に収まっているか？」を基準にする。
> ※ 上の【C】反則早見表の罰則は"代表的な段階"。実際は本章④〜⑥の強さ・危険度で1段階・2段階上がる。

## 【D】パッシブプレー（高校生が最も混乱する判定）

### パッシブとは
**攻撃側が得点意図のないボール回しを続けている**と審判が判定したとき。

### 判定プロセス（第7条11〜12項／競技規則解釈4。2022年7月改正の現行ルール）
1. **予告合図**：レフェリーが**片腕を頭上に上げる**（ジェスチャー17）。「攻撃の意図を示せ」の合図。⚠️**黄色い旗ではない**（ハンドボールに旗は無い／旧版辞典の「黄色い旗」は誤り）
2. **最大4回のパス**：予告後、攻撃側はシュートを打つまで**最大4回のパス**でフィニッシュしなければならない（旧6回・旧旧無制限から短縮）。※**国内は主催者権限で最大6回まで**緩和可（解釈4D注。年齢・種別による）
3. **5回目のパスを味方が受けた時点**でシュートに至っていなければ → パッシブ判定 → 相手フリースロー（中断時のボール位置から）
4. ※**4回目のパス後**にフリースロー／スローイン／GKスローを得た場合、攻撃を完了させるため**追加1パス**まで認められる

### パスの数え方（ここが細かい・解釈4D）
**カウントが継続される（リセットされない）：**
- 予告後に攻撃側がフリースロー／スローインを得ても、パス回数のカウントは**継続**
- パスやシュートがDFにブロックされ、再び攻撃側に戻っても（GKスローになっても）**継続**

**そもそもパスに数えないもの：**
- DFの反則で十分にコントロールできなかったパス（結果、攻撃側のフリースロー）
- パスをDFが触ってコート外に出た（結果、スローイン）／シュートがブロックされ外に出た

### ⚠️ 予告ナシでも"直接"パッシブを取られる3つの特別状況（解釈4E）
予告合図を出していなくても、レフェリーは即パッシブ判定できる：
1. **明らかな得点チャンスの放棄**：前にオープンスペースがありDFもいないのに、シュートも良い位置の味方へのパスもしない（速攻の中断、1対1で抜けるのに打たない等）
2. **ウィングからシュートを打たずに戻す**：ウィングに十分なスペースがあり接触もないのにゴールエリアへ飛び込みながらバックへパスを戻す
3. **相手陣内マンツーマンに対し攻めない**：プレッシャーを受けていないのにドリブル・横パスだけでゴールへ向かわない

### パッシブを取られないコツ
- 予告が出たら**2〜3本目のパスで仕掛けを始める**（4本フルに使わない）
- 予告中は**シュートフェイント・ドリブル突破・ピボットへの当て**で「攻める意図」を見せ続ける
- 仕掛けに失敗してもシュートまで持ち込めば**カウント・予告はリセット**

### 場面別判定基準
| 状況 | パッシブ取られる？ |
|---|---|
| リード守りで時間消化したい | 取られる（戦術的に取られる覚悟で。最後の数分は計算ずく） |
| 速攻でDF不在なのに打たず作り直す | **直接パッシブ**（解釈4E①）に取られうる |
| 攻める意図を見せずキープ重視のパス回し | 取られる |
| シュート脅威ゼロで外野だけ回す | 取られやすい |
| 当てパス＋シュートフェイクでDFを動かし続ける | 取られにくい（積極的な攻撃意図と判定） |

## 【E】退場のエスカレーション

### 段階別ペナルティ
| 段階 | 罰則 | 出場可否 |
|---|---|---|
| ① 警告（イエローカード） | 注意のみ | そのまま継続 |
| ② 2分退場 | 2分間、人数不利 | 2分後復帰 |
| ③ 失格（レッドカード） | 試合終了まで退場 | 復帰不可 |
| ④ 失格＋報告書（ブルーカード） | 試合終了+次戦以降出場停止 | 復帰不可・要協会審査 |

### イエロー（警告）の閾値（第16条1注）
- **同一選手は1試合に1枚のみ**：それ以降の本来イエロー相当の反則は直接 2分退場
- **プレーヤーはチーム合計3枚まで**：4枚目以降の本来イエロー相当の反則も2分退場
- **チーム役員（ベンチ）はチームで合計1枚まで**：それ以降は2分退場（チーム役員の2分退場もチーム合計1回まで）
- **すでに2分退場した選手には、その後イエローを出せない**（直接2分以上）
- イエロー対象動作：軽度のホールディング、押し合い、抗議、遅延行為など

### 2分退場（イエロー2分相当）
- ハッキング、プッシング、危険でない接触ファウル
- 同一選手が **3回目の2分退場** → 自動的に失格（赤）
- 2分中は **自チームが人数不利（コートプレーヤー5人＋GK1人の合計6人。通常より1人少ない状態）** → 攻撃選択肢が減る

### 失格（レッド）
- 危険なプレー（顔面接触、足払い、背後からの強い接触）
- 暴言・暴力行為
- 失格選手が出ても **2分後にチームは7人に戻れる**（選手交代可能）
- ただし **失格選手本人** はその試合終了まで出場不可

### 失格＋報告書（青）
- 特に悪質と判断された場合
- 次戦以降の出場停止審査対象
- ベンチオフィシャルも対象

## 【F】主審・副審の役割分担

### 2人制レフリー（IHF標準・第17条1）
> 重要：IHFでは**2人のレフェリーは"同等の権限"**。「主審が上で副審が下」ではない。位置で役割が呼び分けられ、攻守が替わると2人は入れ替わる。
| 呼称 | 配置 | 主な担当 |
|---|---|---|
| **コートレフェリー** | 攻撃の後方（サイドライン寄り） | 後方からの反則・カード提示 |
| **ゴールレフェリー** | ゴールライン後方 | 得点判定（笛2回）・6mライン侵入・スローイン判定 |

### 判定が割れたときの公式ルール（第17条6〜7）
1. **得点判定** → ゴールレフェリーが見やすい（笛2回＋ジェスチャー12）
2. **罰則の重さで2人の見解が割れた** → 協議して決定。**合意できなければ"重い方"の罰則**を適用（第17条6）
3. **どちらのボールかで割れた** → 協議。**合意できなければコートレフェリーの見解を優先**（第17条7。この時は必ずタイムアウト）
4. **両レフェリー・TOの事実判定は最終**。競技規則に従っていない判定のみ異議可。試合中に話しかけられるのは各「チーム責任者」だけ（第17条11）

### 質問・確認のタイミング
- **試合中の異議**：基本NG（警告対象）。「教えてください」の姿勢で
- **タイムアウト中**：キャプテンのみ主審に質問可
- **ハーフタイム/試合後**：審判控え室への直接抗議は禁止、書面で大会本部経由

## 【G】ベンチオフィシャル（コーチ・帯同医）

### コーチングゾーンの規則
- **コーチングゾーン内のみ立ち入り可**（第1条11で公式に新設＝センターラインから3.5m〜アウターゴールラインから8m）。場所に余裕があればベンチ真後ろも可
- ゾーン外からの指示 → スポーツマンシップに反する行為として段階的罰則
- 試合中の選手交代は **自チームの交代ライン**（センターラインから左右4.5mまで）を通ってのみ
- ベンチ役員数：**5名まで**（2025改正。うち1名がチーム責任者＝レフェリー等と話せる唯一の人）

### グリーンカード（チームタイムアウト請求カード）
- 約**15×20cm**、両面に大きく「**T**」表示（競技規則解釈3）
- **チーム役員がジャッジーズテーブルに置いて請求**（自チームのボール保持中のみ）
- タイムキーパーが笛を吹くまでにボールを失わなければ、そのチームに即タイムアウト付与
- 枚数は基本「前半・後半各1回」。主催者が最大3回を認める場合は番号付き（「1」「2」「3」）カードを配布し前後半各最大2回（解釈3）
- 50秒経過で「あと10秒」の音合図 → 計1分

## 【H】試合進行の細部

### スローオフ＝試合開始・失点後の再開（第10条）
- センターライン中央（中央から左右1.5m以内）から、笛の合図の**3秒以内**にどの方向へでも投げる。投げる選手は片足をセンターライン上に置き続ける
- **前後半の開始時**は、両チームとも**全員が自陣サイド**にいなければならない
- **得点後の再開**は、得点されたチームがスローオフ。このとき**スローを行う相手チームはコートのどちらサイドにいてもよい**（ただしスロアーから**3m以上**離れる）。＝開始時のような「自陣に戻れ」の制限はない
- **クイックスローオフ（即・センターから投げて速攻）は合法**。笛が鳴った瞬間、味方はセンターラインを越えて走ってよい
- ※スローオフエリア（直径4mの円）を採用する大会では、円の中から投げ、相手は円の外

### ゴールキーパーの特殊規則
- ゴールエリア内では足を使ってよい
- ゴールエリア内ではボール保持時間制限なし
- ゴールエリアを出るとフィールドプレーヤー扱い
- GK着用ユニフォームは他選手と異なる色

### 7mスロー判定
| 状況 | 7m獲得？ |
|---|---|
| 明らかな得点機会をDFが反則で阻止 | ✓ 必ず7m |
| 6m付近で軽度ホールディング → シュート阻止 | ✓ 7m |
| ハーフライン付近で軽度反則 | ✗ 通常フリースロー |
| ピボットへの背後ホールディング | △（ケースによる）シュート機会の有無で判断 |

#### 「明らかな得点チャンス」の公式定義（競技規則解釈6）— ここに当てはまる妨害は7m
> コート上のどこで起きても、下の状況をDFやチーム役員が反則で妨害したら**7mスロー**（第14条1）。
- **(a)** 6mライン付近で、ボールと体をコントロールした攻撃側が、**正当な方法では誰も止められない状態でシュートできる**とき（まだボールを持っていなくても、すぐ受けられて正当には防げない状況を含む）
- **(b)** ボールと体をコントロールして**GKに向かって独走（ブレイク）**していて、正面から止められる相手がいないとき
- **(c)** GKがゴールエリアを離れていて、**無人のゴールへ誰にも止められず投げられる**とき

**判定の核心（第14条2）**：反則があっても、攻撃側がボール・体をコントロールできていれば即7mにはしない。レフェリーは「本当に得点チャンスが消えたか」を見極めるまで笛を待つ＝**得点できたらそのまま、チャンスが完全に消えたら7m**（アドバンテージ）。

### 出血・負傷時のルール（第4条10〜11）— 自分が怪我したらどう動く
- **出血した／ユニフォームに血がついた**ら、**自分からすぐコート外へ**（通常の交代として）。止血・処置が終わるまで戻れない（第4条10）。指示に従わないとスポーツマンシップに反する行為で罰則
- **コート内で倒れて治療を受けた**場合：原則、その選手は**自チームが3回の攻撃を終えた後**でないとコートに戻れない（第4条11第2段落）。これは「痛いふり」での時間稼ぎ防止
  - ⚠️**日本では高校・中学・小学のユースでこの"3回攻撃"ルールの導入を保留**。実際の運用は**各連盟の通達・大会要項による**ので、自分の大会の扱いを確認（第4条11注）
  - ※ハーフタイムを挟めば、攻撃回数に関係なく戻れる
- 救護でコートに入れるのは**チーム役員かプレーヤー2名まで**（複数負傷時は1人につき最大2名。第4条11ガイドライン）

## 【I】自己組織化のための判定スキル

### 試合中に自分で判定を読み解く3ステップ
1. **笛が鳴った瞬間に方向を見る** — レフェリーが示すスローの方向（ジェスチャー7／9）＝どちらのボールか

## 【J】よくある誤解の修正

| 誤解 | 正しい理解 |
|---|---|
| 「3歩以上歩いてはいけない」 | 正解は **3歩まで歩いてOK**（4歩目から反則。第7条3項） |
| 「ボールを2秒以上持ってはいけない」 | 正解は **3秒まで保持OK・3秒を超えたら反則**（第7条2項） |
| 「ジャンプキャッチの着地はいつでも0歩」 | **味方のパスを空中で受けたとき限定**。自分のドリブル後のジャンプキャッチは着地が第1歩（★★ガイドライン） |
| 「6mラインを踏んだら反則」 | OF着地時のみ。**ジャンプ中はOK**、シュート動作で離れた着地はルール内 |
| 「赤カードは即試合終了」 | 失格選手のみ。**チームは2分後に7人に戻る**（第16条8） |
| 「ハーフタイムは15分が標準」 | 成年・高校は**休憩10分が標準・最長15分**（第2条1） |
| 「タイムアウトは何度でも取れる」 | 基本は前後半各1回・延長なし。主催者権限でも最大3回（前後半各最大2回。第2条10／解釈3） |
| 「ピアスはテープで隠してもダメ」 | 公式は**小さいピアス・イヤリングは完全に覆えば可**（第4条9）。ただし大会要項でより厳しい場合あり |
| 「パッシブは副審が黄色い旗を振る」 | ハンドボールに旗は無い。**レフェリーが片腕を上げる**予告合図（ジェスチャー17） |
| 「審判の判定は絶対」 | 事実判定は最終。ただし**規則に従っていない判定には異議可、チーム責任者がレフェリーに確認できる**（第17条11） |

## 【K-2】アクセサリー・身だしなみ規定（試合前必須チェック）

> ⚠️**この章は公式ルール（第4条9＋「服装や保護を目的とした装具に関する規定」）に合わせて訂正済み**。旧版の「ピアスはテーピングで隠してもNG」は誤り。装具の細部は `31_equipment.md` が完全版。
> ※ただし**大会要項（高体連等）が公式より厳しく「全て外す」と定める場合がある** → 要項と試合前の主審確認が最優先。

### 完全に **禁止**（覆ってもダメ。第4条9）
| アイテム | 理由 |
|---|---|
| **腕時計・ブレスレット** | 接触時の怪我 |
| **ネックレス・チェーン** | 引っ掛かり・首絞めリスク |
| **固定バンドの無い眼鏡／硬いフレームの眼鏡** | 割れ・刺さりリスク |
| **フェイスマスク・ヘルメット・頭部プロテクター** | IHFは形・大きさによらず不可（GKも。顔の一部マスクも不可） |
| **手袋・グローブ（コート上）** | GKも不可。交代地域での防寒は可 |
| **フィンガーバンド** | 不可 |
| **露出した硬い装具** | 肘・膝・足首の硬い部分がむき出し |

### **原則は外すが、危険のない方法で"覆えば"OK**（第4条9）
| アイテム | 条件 |
|---|---|
| **小さいイヤリング・外から見えるピアス** | テープ等で完全に覆い、相手に危険がなければ着用可（旧版「隠してもNG」は誤り） |
| **突起のない指輪** | 同上、覆えば可 |
| **金属・プラスチックのヘアピン** | 完全にテープで覆えば可（柔らかい素材ならそのまま可） |

### 試合中に **OK** のもの
| アイテム | 条件 |
|---|---|
| **柔らかいヘアバンド／ヘッドスカーフ／キャプテンマーク** | 第4条9で明記。単色のスポーツ用 |
| **スポーツメガネ** | 固定バンド付き＋平らなプラレンズ＋フレーム上部が柔らかい材質 |
| **コンタクトレンズ** | メガネより安全、推奨 |
| **テーピング・サポーター** | 肘＝柔らか素材のみ／膝＝硬い部分は完全被覆／足首＝硬い部分も完全被覆なら可 |
| **マウスピース** | 透明か単色なら可（不透明・多色は不可） |
| **松やに** | 使用可。**靴に塗りためるのは可、手・手首にためるのは不可** |

### 髪型の規定
- 長髪は **必ず結ぶ**（プレー中の視界・接触対策）
- **三つ編み** は審判から金属ピン使用と疑われやすい → ポニーテール推奨
- 試合直前に **キャプテン or コーチが視認チェック**

### 爪・身だしなみ
- **爪は短く** — 試合前日にカット
- 爪先が指を超えないこと（審判チェック対象）
- テープで爪を覆う方法もあるが剥がれやすい

### 試合前 30秒チェックリスト
- [ ] 指輪を外した（突起なしなら覆えば可だが、外すのが確実）
- [ ] ピアス・イヤリングを外した（小さい物は完全にテープで覆えば可。大会要項が厳しければ外す）
- [ ] ネックレス・腕時計・ブレスレットを外した（これは覆ってもNG）
- [ ] 髪を結んだ（長髪の場合）
- [ ] 爪を切った（指先を超えていない）
- [ ] 金属ピンは使っていない（使うなら完全にテープで覆う）

→ キャプテンが全員に「外した？」と確認する習慣を作る。

## 【L】場面別 自己判定チェックリスト

### 「今の動作、反則だった？」を試合後に振り返る
- [ ] ボールを何歩持って歩いた？（3歩までOK・パス受けの空中着地は0歩）
- [ ] ボール保持時間は何秒？（3秒までOK・超えたら反則）
- [ ] ドリブル中断後、再ドリブルしていない？
- [ ] DFを抱えた・押した・叩いた動作はあった？
- [ ] 6mラインを着地時に踏んだ？
- [ ] パッシブ予告中、4回以内にシュートに至った？

### 「相手のあのプレー、反則じゃない？」を判断する
- [ ] 接触は明らかにあった？
- [ ] DFの腕・手・体のどこが当たった？
- [ ] 主審・副審の位置から見えていた？
- [ ] 笛が鳴らなかった = 「見えていなかった」or「軽度と判定」

→ 試合後にチームで「あの場面どうだった？」を共有することで、自分のルール理解が深まる。

## 【M】じゃんけん：ルールを使った戦術

| 状況 | じゃんけんの一手 |
|---|---|
| リードして時間消化したい | パッシブ取られる覚悟でボール回し（戦術的選択） |
| 相手にイエロー2枚（次は2分） | 接触多めに仕掛ける（3枚目=2分退場狙い） |
| 自分が2分退場 | チーム5+GKで守る → カウンター速攻狙い |
| 相手キーマンが2分退場 | その時間内に4点差以上つける目標 |
| パッシブ予告された | ピボット中心の3-4本パスで突破 or 個人技で抜く |
