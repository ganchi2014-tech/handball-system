# セットプレー・サインプレー・スクリーン技術

---

## フリースロー時の攻撃パターン

### クイックスロー系（DFが整う前）
- **即時速攻**：笛の直後に素早くスロー。DFが位置についていない瞬間を狙う
- **スローワー自身の突破**：パスフェイクして自らドリブル突破
- **ピボットへのダイレクトパス**：ピボットが6mライン内に飛び込む

### セットオフェンス系（DFが整った後）
- **クロスランニング**：2名がクロスして走り込み、DFのマーク交換を誘う
- **ダブルスクリーン**：2名が並んでエースのためにフリーを作る
- **バックプレーヤーミドルシュート**：良い角度のフリースローならそのまま打つ

### サインの種類
| 媒体 | 具体例 |
|---|---|
| 言語 | 「A！」「2！」等の数字・アルファベット |
| 手のサイン | オープンハンド・クローズハンド・指の数 |
| 立ち位置 | センター寄りかサイド寄りか |
| 動作 | ドリブル開始でパターンA、なしでパターンB |

## スローオフ時の約束事

| パターン | 説明 | 使用条件 |
|---|---|---|
| 即時速攻型 | センターが後方に下げ、サイドへ展開 | 相手DFが整う前 |
| ポゼッション確立型 | 確実にボールを保持してセットプレーへ | リードしている・確実性優先 |
| クイックスローオフ | 得点直後、相手守備が整う前に素早く | 相手のスキを狙う |
| 遅延スローオフ | 時間を使いながらポゼッション | リードしている・残り時間が少ない |

## スクリーン技術

### スクリーンの本質（コーチ直伝）

> **「自分の体を壁にして相手の進行を妨げること」がスクリーンのメイン。**

- 相手が押してきたときに押し返してパワーを0にすることは認められる
- ただし押し返しすぎ（過度な押し返し）はファウル
- 足を大きく開いてスペースを取るのは基本的にNG
- 腕や肘で相手を押すのはNG
- 自分の体そのものを壁にすることが合法の基本

**審判による基準の違い（重要）**：
- 激しいやり合いを認める審判もいる
- 少しでも押し合いがあるとすぐファウルを吹く審判もいる
- 審判の傾向を試合序盤に把握して判断を変える必要がある

### 合法スクリーンの要件
1. スクリーナーは静止している（または止まっている）
2. DFの視野内から入ってから静止（背後からの奇襲はNG）
3. 手足は身体に密着させる（広げると危険行為）
4. スクリーナーが自ら接触を引き起こさない
5. 相手の押すパワーを0にする押し返しはOK。押し返しすぎはNG

### スクリーンの種類

| 種類 | 説明 | 使用場面 |
|---|---|---|
| ステーショナリースクリーン | 完全静止でDFの進路を塞ぐ | バックプレーヤーの突破 |
| ダウンスクリーン | ゴール方向に向かって設定 | ピボットがスペースを作る |
| バックスクリーン | DFの背後に回り込む（グレー） | DFがボールを見ている瞬間 |
| クロススクリーン | 2名が交差しながら互いにスクリーン | フリースロー後のセットプレー |
| ダブルスクリーン | 2名が並んでエースのために壁を作る | エースのフリー確保・最後の手段 |
| スクリーン＆ロール | スクリーン後にゴール方向に転身してパスを受ける | DF2名を同時に動かす |

### DFのスクリーン対策
| 対策 | 説明 | 有効場面 |
|---|---|---|
| ファイトオーバー | スクリーナーの前を素早く通ってマークを維持 | エースが外シュートを打てる位置 |
| ゴーアンダー | スクリーナーの後ろを通ってマークに戻る | エースが突破優先の場合 |
| スイッチ | DFが互いにマークを交換して無効化 | 弱点：ミスマッチが生じる |
| ヘルプアンドリカバー | 3人目DFが一時カバー、元のDFが戻る | 連携が難しい上級技 |

### じゃんけん（スクリーン）
```
スクリーン利用（カットイン）
  → DFがスイッチ対応
  → ミスマッチ発生（スクリーナーが有利）
  → スクリーナーをポップアウト
  → 外シュート機会発生
  → DFがポップを追う → カットインのフリー復活
  → スクリーン利用に戻る

ファイトオーバー（DFがスクリーン前を通る）
  → 外シュートを打つ間を与えない
  → カットインへの対応が遅れる
  → ゴーアンダー（後ろを通る）
  → カットインに対応できる
  → 外シュートが打てる間が生まれる
  → ファイトオーバーに戻る
```
