## 【モールヴァクト（GK）skolen：ノルウェーGKスクール】

> 「ノルウェー代表の優れたGKたちからインスパイアされて作られたGK育成システム」
> GKのトレーニングを「物理（フィジカル）→技術（テクニック）→戦術（タクティクス）」の順に構造化

### 5章構成

| 章 | ノルウェー語 | 内容 |
|---|---|---|
| 1 | **Terminologi og struktur** | 用語と構造：優れた発展を支える考え方の基礎 |
| 2 | **Fysisk grunnlag** | フィジカル基礎：良い技術開発のための身体的土台 |
| 3 | **Tekniske og taktiske aspekter** | 技術・戦術的側面：ゲーム状況の分析とポジション別プレー |
| 4 | **Posisjonsbestemt målvaktstrening** | ポジション別GKトレーニング：各コートポジションからのシュート対応練習 |
| 5 | **Sammensatte øvelser** | 複合練習：複数ポジションからのシュート・ゲームリアルシナリオ |

### GKプレーの4フェーズ（Fasene）

```
① Forflytningsfasen（移動フェーズ）
   → 軽快に動き続ける。ボールだけでなく選手全体を観察する
   → 次のフェーズへの準備を常に行う

② Posisjoneringssfasen（ポジショニングフェーズ）
   → ゴールと攻撃者の位置関係から最適なポジションを取る
   → 読みと先読みでポジションを先取りする

③ Redningsfasen（セービングフェーズ）
   → 実際のシュートを止めるフェーズ
   → 体の重心・反応・脚の使い方が結果を決める

④ Initieringsfasen（イニシエーションフェーズ）
   → セーブ後の速攻発動フェーズ
   → GKからの速いスローアウトがチームの速攻の起点となる
```

### GKの基礎ポジション（Grunnposisjon）

- **ボクサースタンス**に似た構え（重心低く、両足は肩幅以上）
- **Bred beinstilling**（ワイドスタンス）= 安定性の基盤
- **小さいステップで素早い移動**を維持しながらバランスを保つ
- 腹部・背中・腰の**コアスタビリティ**が全ての技術の土台
- 移動中は腕がそれぞれ独立して動けること（「**armene beveger seg uavhengig**」）

### フィジカル要件

| 要素 | 内容 |
|---|---|
| 筋力・爆発力 | ショートスプリント、ダイビング、ジャンプのパワー源 |
| フットワーク | 細かいステップ・方向転換・クロスステップ |
| モータースキル・協調性 | 複雑な動作を自動化する運動神経 |
| コアスタビリティ | 腹・背・腰の安定性 → 全技術の土台 |
| 柔軟性 | 股下・横方向へのリーチを最大化 |

### じゃんけん（GK視点）

```
GKが早く動く（先読み型） → 早打ちでGKの動作前に放たれる
→ GKが動かずコースを絞る（コース絞り型）→ 引きつけてから角度変化で打つ
→ GKが動かなければフリーで打てる → GKが再び早く動く

Initieringsfasen（速攻起点）重視 → GKのスローアウトで速攻が増える
→ 相手チームが速攻対策でバックチェックを強化する → セット攻撃が増える
→ セット攻撃でGKのセービングシーンが増える
```

## 【Forsvarsskolen：ノルウェーDFスクール】

> 「守備は燃えるような意欲（brennende ønske om å få tak i ballen ＝ ボールを奪いたいという強い願望）から始まる」
> 5章構成：方法論 → 前提条件 → 技術・戦術 → 基礎戦術 → 練習

### DFの10コア技術（核心スキル）

| # | ノルウェー語 | 読み | 内容 |
|---|---|---|---|
| 1 | **ベレドスカプスティリング（準備姿勢）** | ベレッドスコープスティリング | 準備姿勢：重心低く、Bred beinstilling（ワイドスタンス） |
| 2 | **フォルフリットニング（移動）** | フォルフリトニング | 移動：0.5〜5mの多方向移動（加速・急停止・方向転換） |
| 3 | **スナップ（速いボール捌き）** | スナップ | インターセプト：パスを切る、スローイン奪取、バウンドボール奪取 |
| 4 | **Blokkere** | ブロッケレ | ブロック：地上・空中からのシュートを止める |
| 5 | **ステンゲ（ブロック動作）** | ステンゲ | スペース閉鎖：相手の移動・パスコースを制限する |
| 6 | **Sikre** | シクレ | 確保・安全確保：主マークを失わずにリスクを予防するポジション |
| 7 | **フィンテ（フェイント）** | フィンテ | DFフェイント：欺く防御動作（攻撃者を誤誘導する） |
| 8 | **Presse** | プレッセ | プレッシャー：相手のリズムと判断を乱す |
| 9 | **Takle** | タックレ | タックル：危険なプレーを防ぐ身体的接触 |
| 10 | **ヒエルペ（ヘルプ動作）** | ヘルペ | ヘルプ：困難な状況にある味方をサポート |

### DFのメンタル前提条件

```
brennende ønske om å få tak i ballen（ボールを奪いたい燃えるような欲求）
→ 集中力（Konsentrasjon）
→ 情報処理能力（情報を素早く取り込み判断する）
→ 自信（セルヴティリット（自信））
→ メンタルルーティン：triggere（トリガー）= 精神的な合図で集中を高める
```

### DFのフィジカル要件

| 要素 | 具体的要求 |
|---|---|
| 加速 | 0.5〜2mの短距離での爆発的加速 |
| 方向転換 | 急停止・素早い方向変化 |
| 脚力 | 長時間の守備姿勢維持 |
| コアスタビリティ | 体幹の安定 |
| 上半身の力 | 接触・押し返しのパワー |
| 作業能力 | 1シーケンス（1回の連続した守備局面）あたり10〜30秒の高強度を維持 |

### 数的状況別戦術

```
同数（1v1, 2v2）→ 個人DFスキルが決定的
数的不利（1v2, 2v3）→ 遅らせる・コンパクトに守る・カバーを待つ
数的有利（2v1, 3v2）→ 積極的な前へのプレッシャー
```

### 個人DFの判断基準（Forsvarsskolen）

```
相手のどちら手利きか → シュートコースの予測
相手のボール保持位置 → プレッシャーの強度
相手の脅威レベル → どこに向かわせるか
能動的 vs 反応的 → 先に動くか待つか
```

### じゃんけん（10技術の連鎖）

```
ステンゲ（ブロック動作）（スペース閉鎖） → 相手がPresse（プレッシャー）で崩す試みをする
→ ベレドスカプスティリング（準備姿勢）でプレッシャーに対応
→ フィンテ（フェイント）（DFフェイント）で攻撃者の判断を遅らせる
→ 攻撃者がスナップ（速いボール捌き）（インターセプト）を避けてパスコースを変える
→ ヒエルペ（ヘルプ動作）で隣が補完 → スナップ（速いボール捌き）チャンス発生
```

## 【アングレープステクニック（攻撃技術）er：ノルウェー攻撃技術メソッド】

> 「個人スキルへの要求は年々高まっている（Det stilles større og større krav til individuelle ferdigheter）」

### 5つのコア攻撃技術

#### 1. Kast og Mottak（パスとキャッチ）

> 「おそらく最も重要な技術の一つ（kanskje en av de viktigste teknikkene）」

| ノルウェー語 | 内容 |
|---|---|
| **ピスケカスト（むち打ちパス）** | ピスクカスト（むち打ちスロー）：鞭を振るような腕のしなりで放つ最速のスロー |
| **シルケルカスト（円運動パス）** | スィルケルカスト（サークルスロー）：腕を円を描くように振るスロー |
| **ステーカスト（押しパス）** | ストゥトカスト（プッシュスロー）：押し出すような短距離速射スロー |
| **シュフルト・カスト（隠しパス）** | シュルトカスト（隠しスロー）：相手に見えにくい角度・位置からのスロー |
| **Mottak med en hånd** | 片手でのキャッチ |
| **Mottak med to hender** | 両手でのキャッチ |

#### 2. フィンテ（フェイント）r（フェイント）

> 「少なくとも左右それぞれ1つの信頼できるフェイントを持つこと（minst én finte hver vei）」

**3つの結果カテゴリ：**
1. 自分で得点（Skåre）
2. 仲間へのパス（Viderespill = 継続パス）
3. ピボットorウィングへのパス（Innspill/Utspill）

**フェイントの核心要素：**
- リズムとテンポの変化
- DFとの正しい距離感（選手によって異なる）
- スペースへのアタック（DFを避けるのではなくスペースを攻める）
- ボールプロテクション
- **Knær over tær**（膝をつま先の上に）＝ 怪我防止と爆発力のための正しい体勢

#### 3. スクート（シュート）d（シュート）

> 「技術は個人によって異なる（Teknikk er individuelt）」
> 基本原則を守りながら各選手が独自のレパートリーを開発する

| ノルウェー語 | 内容 |
|---|---|
| **Piskeskudd** | むち打ちシュート（最も一般的な外シュート） |
| **Sirkelskudd** | サークルシュート（腕を回しながら打つ） |

#### 4. シュペレ（妨害）（スクリーン）

> 「技術・戦術的行動（teknisk-taktisk aksjon）」

- 体でDFの進路を塞いでスペースを作る
- 両足でバランスを保って静止
- 腕を使った保持はNG
- タイミングが重要

#### 5. フォルフリットニング（移動）

- プルプレー状況でのポジション取り
- 新しいスペースへのアクセス
- ゴールへの勢いをつける動き
- 数的優位またはシュートチャンスの創出

### 1対1で勝つ原則（Å vinne duellen）

```
基本：体の向きをゴール方向に向け続ける（Hold kroppen vendt mot målet）
→ バランスと床への接触を維持しながら移動
→ DFではなくスペースをアタックする
→ ボールを受ける前に味方とDFの両方を確認する（スキャニング）
```

**2つのアプローチ：**
```
フォルフリットニング（移動）：スペースを取ることで優位を作る
  → 大きな移動（ゾーン変更・ポジション交換）
  → 小さな調整（プレー読みの前の位置修正）

フィンテ（フェイント）：欺くことで優位を作る
  → リズム変化：加速・減速・一時停止
  → 正しい距離でDFの反応を引き出してから逆をつく
```

## 【Utviklingstrappa：育成段階ラダー】

### 4段階の発達レベル

| レベル | 年齢 | フェーズ | 週あたり練習回数 |
|---|---|---|---|
| Level 1 | 6〜9歳 | 遊びを通じた土台作り | 1〜2回 |
| Level 2 | 10〜12歳 | 基礎スキルの習得 | 2〜3回 |
| Level 3 | 13〜15歳 | スキルの自動化・専門化開始 | 3〜5回 |
| Level 4 | 16〜20歳 | 高度な専門化・シニアへの橋渡し | 7〜12回 |

### 5つのスキルドメイン

```
① 技術スキル（Tekniske ferdigheter）= bevegelsesløsning（動作の解決策。場面ごとに体をどう動かして対応するか、という運動面での答え）
② 戦術スキル（Taktiske ferdigheter）= 状況判断・チーム連携
③ GKスキル（Målvaktferdigheter）= 専門ポジション開発
④ フィジカルスキル（Fysiske ferdigheter）= 協調性・筋力・速度・敏捷性
⑤ メンタルスキル（Mentale ferdigheter）= 自信・集中・モチベーション
```

### メンタルスキルの鍵となるノルウェー語概念

| ノルウェー語 | 意味 | 実践での使い方 |
|---|---|---|
| **メストリング（達成感）** | 習得・達成感 | 「できた」という実感がモチベーションの源泉 |
| **セルヴティリット（自信）** | 自信 | DFに仕掛ける・GKが飛び込む判断力の基盤 |
| **ヴィスアリセーリング（視覚化）** | イメージトレーニング | プレー前に成功シーンを頭の中で描く |
| **インドレ・ディアログ（内的対話）** | 内的対話（セルフトーク） | 「できる」「次はこうする」の自己言語化 |
| **トリッゲレ（誘発する）** | メンタルトリガー | 集中を高める特定の合図・ルーティン |

### ノルウェーの育成哲学（Forsvarsskolen補足）

```
Flerposisjonsspillere（マルチポジション選手）を育てる
→ 若い年代では特定のポジションに固定しない
→ 様々なポジションを経験することで判断力と理解が深まる
→ 成熟したら専門化する（Level 3〜4）

「脱落 = 終わり」ではない設計：
→ 身体・精神の成熟に個人差を認める
→ 一度外れても戻れる仕組み
→ 長期的に多くの選手が開花できる
```

## じゃんけん（全体統合）

```
【GK vs シューター】
GKがコースを読んで先に動く → 早打ち（GKが構えて動き出す前に、素早くシュートを放つこと）
→ GKが動かずコースを絞る → 引きつけてから角度変化
→ GKが絞れていない → GKへの声かけでコース意識

【DF 10技術のサイクル】
ステンゲ（ブロック動作）（閉鎖）→ 相手がフィンテ（フェイント）で崩そうとする
→ ベレドスカプスティリング（準備姿勢）で待つ → スナップ（速いボール捌き）（インターセプト）チャンス
→ 相手がViderespill（継続パス）で展開 → ヒエルペ（ヘルプ動作）で全体が連動
→ Presse（プレッシャー）で次の展開を遅らせる → ステンゲ（ブロック動作）pに戻る

【育成段階のじゃんけん】
早期専門化 → 特定スキルは伸びるが応用力が低下
→ Flerposisjonsspillere（マルチポジション）育成 → 判断力と多様性が高まる
→ 専門化のタイミングが遅れる → 短期的な競技成績は下がるかもしれない
→ しかし長期的に開花する選手が増える（ノルウェーモデルの証明）
```
