## コートの見方（全図共通）

```
          相手ゴール方向
  ┌─────────────────────────────┐
  │  LW    LL    C    RL    RW  │ ← バック/ウィング層（9m付近）
  │              PV             │ ← ポスト（6m付近）
  │  ─────────── 6m線 ──────── │
  │       D₁ D₂ D₃ D₄ D₅ D₆   │ ← 守備陣
  │            [ GK ]           │
  │         ┌────────┐          │
  │         │  GOAL  │          │
  └─────────────────────────────┘

ポジション略称
  LW = 左ウィング      RW = 右ウィング
  LL = 左バック        RL = 右バック
  C  = センターバック  PV = ポスト（ピボット）
  GK = ゴールキーパー  D  = ディフェンス
```

## Part 1：サインシステム（どう伝えるか）

### 1-1. サインを出すタイミング

```
笛が鳴る → 3秒以内に再開しなければならない
            ↑この「準備の3秒間」がサインプレーの設定ウィンドウ

・センター（C）が背を向けてからサインを出す
  → DFから手元が見えない
  → 「誰に何を渡すか」を悟られにくい

・笛直後にすぐサインを出せるほど速攻できる
```

---

### 1-2. コードワード方式（Codewort / 言葉でサイン）

**ドイツ流・コードワード命名文化：**

ハンドボールでは「プレーの動き」ではなく「名前（コードワード）」で呼ぶ。名前自体に意味はなく、チームが記憶しやすいものを選ぶ。

| コードワード例 | 命名の由来 |
|---|---|
| **Polen（ポーランド）** | ポーランド遠征中の試合で使ったプレー（Heiner Brand考案） |
| **Jugo / Jugo Links** | ユーゴスラビア発祥のクロスプレー（最古の定番の1つ） |
| **Russe / Däne** | 各国名をランダムに命名 |
| **Döner / Kölsch / Baguette** | 食べ物シリーズ（日常語で覚えやすい） |
| **Max / Werner / Oma / Mama** | 人の名前シリーズ |
| **Hawaii / Jamaica / Kuba** | 島・地名シリーズ |

> **Heiner Brand（元ドイツ代表監督）の言葉：**  
> 「全プレーの目的はDFを動かすことだ。バックプレーヤーが勢いのある良い位置でボールを受けられれば、それで十分だ」 *(spox.com)*

---

### 1-3. 数字方式（シンプル番号制）

```
「Eins Links！」 → プレー1番、左方向
「Zwei Rechts！」 → プレー2番、右方向
「Neun！」       → プレー9番（固定で覚えておく）
「2 Jamaica！」  → プレー2番、島名が方向を示す（右）
```
- 数字＋島名 or 方向の組み合わせで意味が確定するシステム
- ユースや高校チームに向いたシンプル方式

---

### 1-4. 手のサイン（ベンチ→コート）

| サイン | 意味 |
|---|---|
| 指を1〜4本立てる | プレー番号を示す |
| 手のひらを上に向ける | 攻撃プレー実行 |
| 手のひらを下に向ける | DF変更指示 |
| 腕をクロスさせる | クロスプレー系（Kreuzen） |
| 左右を指差す | そのサイドへ展開 |
| 交代選手が伝達 | コーチが控え選手に伝え→交代で入りながら伝達 |

> **交代でサインを運ぶ（Substitution as Play Carrier）：**  
> 重要なフリースローの前にコーチが意図的に交代を使い、入る選手がプレーを口頭で仲間に伝える。合法かつ効果的。

---

### 1-5. ポジション位置がサインになる（Positional Trigger）

選手の**立ち位置そのもの**がサインになるシステム：

```
例）ポスト（PV）が6m線の左側に立つ → 「左展開」
例）LLが極端に開いた位置に立つ → 「ウィングプレー」
例）CBが9mより前に出る → 「速攻型フリースロー」
```
- DFに言葉を聞かれるリスクなし
- 事前に「この立ち位置＝このプレー」とチームで合意が必要

---

### 1-6. コート内キャプテンのサイン

| サイン | 意味 |
|---|---|
| 拳を握り上げる | プレーAを実行 |
| 手のひらを開く | プレーBを実行 |
| 特定の選手を指差す | その選手が起点（トリガー） |
| 頭を叩く | フリーフロー（即興・自由判断） |
| 口頭でコードワード | 最多使用、センターが叫ぶ |

## Part 2：フリースロー（9m）からのサインプレー

### ─ 基礎プレー群 ─

### P01：シンプルクロス（Einfaches Kreuzen / 「ノイン」）

```
実行前：
  LW  [LL]  C  [RL]  RW
             ↑
             C がボールを持つ

実行中（矢印は走る方向）：
  LW    ←LL       RL→    RW
            ↘   ↙
           （交差点）
             C がどちらか良い方へパス
```

| 項目 | 内容 |
|---|---|
| 独語名 | Einfaches Kreuzen |
| プレーヤー数 | 2名（LL+RL） |
| ポイント | LLとRLが同時にお互いに向かって走り交差。DFの「誰が誰を守るか」を混乱させる |
| DFの反応 | ①スイッチ（交代して守る）②ついていく（衝突リスク）③追跡不能でフリー |
| パス先 | 交差後に有利な位置にいる方を選ぶ |

---

### P02：クロス＋スクリーン（Kreuz mit Block / 「クロイツァー」）

```
実行前：
  LW  LL  C  RL  RW
              PV（6m中央付近に立つ）

実行中：
  PV が D₃ のマーク前に立って「スクリーン設置」
       →→→→RL が PV の体を使って走り込む
  C がRLへパス（スクリーンを抜けた瞬間に）
```

| 項目 | 内容 |
|---|---|
| 独語名 | Kreuz mit Block |
| 別名 | Deckungssuche（守備のスクリーン探し） |
| ポイント | ポストがDFにスクリーンをかけてRLを解放する |
| 決定的タイミング | RLがスクリーンを抜けた**瞬間**にパス → 勢いのあるシュートへ |
| 引用 | ノルウェー代表コーチ Patrick Braarud：「プロハンドボールで最も使われる」 |

---

### P03：ピック＆ロール（Pick & Roll / Deckungssuche）

```
①  RL → C にパス（RL がCに寄る）
②  RL は C のDF（D₄）にスクリーンを設置
③  C がスクリーンを利用してドライブ
④  RL は「ロール」→ 6m方向に反転しポスト位置へ

リードアフター：
  C がシュート    ← 第1選択
  C → PV へパス  ← 第2選択
  C → ロールしたRL（第2ポスト）← 第3選択
```

| 項目 | 内容 |
|---|---|
| バスケットボールとの比較 | ほぼ同じ概念。ハンドボールでは「Deckungssuche」（DFを探す動き）と呼ぶ |
| 選手要件 | Cに1on1突破力、RLはロール後のポジション取りが必要 |

---

### P04：ユーゴ（Jugo / Jugo Links）

```
Jugo Links（左版）：
  C → LL へパス
  LL が左方向に1on1ドライブ
  C が LL の元ポジションへ移動（空いた中央を埋める）

Jugo Links Kreuz（左クロス版）：
  C → LL へパス
  RL が C の後ろをクロス（空のクロス）
  LL：1on1 or クロスしたRLへパス
  → RL が中央でフリーの状態を作る
```

| 項目 | 内容 |
|---|---|
| 独語分類 | Auslösehandlung（起動プレー） |
| 最大の強み | DFが「クロスに反応」するだけで中央にスペースが生まれる |
| バリエーション | Jugo Rechts / Jugo Rechts Kreuz（右版も同様に使用） |

---

### P05：ポーレン（Polen）

```
タイムアウト明け等に使用された「中央混乱」プレー

  LW  LL  [C]  RL  RW
           ↓（Cがドライブ起点）
  LLとRLが逆方向にクロス（空のクロス）
  Cが勢いのある良いポジションへ移動する
  DFが「混乱（Unruhe）している中央」でパスを受けてシュート
```

| 項目 | 内容 |
|---|---|
| 命名 | ポーランド遠征中にHeiners Brandが考案。「ポーランドで使ったプレー」 |
| 目的 | 「バックプレーヤーが勢いを持って良い位置でボールを受けること」 |
| 採用チーム | ドイツ代表（Brand体制）、その後Bundesligaに広まる |

### ─ クロス応用群 ─

### P06：ラング・リンクス / ラング・レヒツ（Lang Links / Lang Rechts）

```
「遅れたクロス」— 同時ではなくパスのタイミングで交差を起こす

① C → LL へ加速パス
② LL は一瞬ためる（遅延）
③ RL が LL 前を横切る（クロス）
④ LL → クロスしたRL へパス
   RLが右サイドへ走りこみシュート体勢
⑤ RWは幅を取り続けてウィングDFを釘付け
```

---

### P07：レア・リンクス「1」（Leer Links "Eins" / 空のクロス）

```
「Leer」= 空（ボールなしのクロス）

① C がボールを持つ
② LL と RL が互いの方向へ走る（クロス）
③ LL は「空でクロス」— ボールを受けない
④ C → RL（中央へ切り込む RL へパス）
⑤ 中央が空いた場所でRLが攻める
```
- DFが「ボールのないクロス」に釣られて動いた隙を突く

---

### P08：ドライアー・クロイツ（Dreier Kreuz / トリプルクロス）

```
3人が連続クロス

① C と LL がクロス（C左・LL中）
② LL（中央に来た）と RL がクロス
③ C が中間ポジションへ移動
④ PV が前衛DFにスクリーン
⑤ 3つのクロスでDF3名が同時に担当変更→混乱
```

| 利点 | DFに3回連続のスイッチ判断を強いる |
|---|---|
| 難点 | 選手間の完璧なタイミング共有が必要 |

---

### P09：ドッペル（Doppel / ダブルエクスチェンジ）

```
① C が LL 前を通り左へ（1回目クロス）
② LL が中央位置に移動
③ LW → LL（中央）へパス
④ PV がギャップに入る
⑤ C は2回目のロール → 有利なシュートポジションへ
```

---

### P10：ヴィボルグ（Viborg / ビボー）

```
（デンマーク・Viborg HKにちなんだ命名）

① C → LL（右中間）へ加速パス
② C が弧を描いて元LLのポジションへ（バランシング）
③ LL → LW へパス
④ LW → C へパス（C が前へ出てくる）
⑤ 最終アクション：シュート / 突破 / クロス継続
```

---

### P11：ゾルベルグ（Solberg / ソルベルグ）

```
（北欧風命名。4人ローテーション）

① C と LL がクロス（1回目）
② LL と RL がクロス（2回目）
③ C がバランシング（3回目のトリガー）
④ RL と C がクロス（4回目 → 完全な4人循環）
⑤ PV が前衛DFに対してスクリーン
```

---

### P12：マクデブルク（Magdeburg / SC Magdeburg スタイル）

```
（SCマクデブルクにちなんだ命名）

① LW-LL-C の三重交換（ダブルエクスチェンジ）
② RL がクロス
③ PV がDFのギャップへポジション
④ C が左バックポジションへ移動
⑤ 最終パス先：最もオープンなプレーヤー
```

---

### P13：ロール（Rolle / "Fickschnitzel"）

```
「Rolle」= ロール（転がり）

通常版：
① C と PV がクロス
② PV → RL へパス
③ PV が再び6m位置へ落ちる（ロール）
④ バックプレーヤーは外側から圧力を維持

「Raus（外）」バリエーション：
① 同じ起点だがCが左へ落ちる
② LL が RL からパスを受ける
③ Cが第2ポストオプションに
```

---

### P14：ユーバーガング（Übergang / バック→ポスト移行プレー）

```
Übergang Halb（サイド移行）：
① LL → C へパス
② LL が対角線上に6mへダッシュ（第2ポストになる）
③ C → RL へパス
④ RL が LL の走りが生んだギャップに入る
⑤ PV が中央DFにスクリーン

Übergang Mitte（中央移行）：
① C → LL（中央方向）へパス
② LL → RL（開いた方向）へパス
③ PV が中央DFをブロック
④ RL → C へ戻す（C が中央へ帰還）
```

### ─ 特技・個人技系 ─

### P15：ケンパトリック（Kempa-Trick）

```
         ┌────────┐
         │  GOAL  │ ← 相手ゴール
         └────────┘
              ↑
         B（ジャンプ中）← 空中でキャッチ → シュート
              ↑
           A（パス） ← 6m内またはその付近で待機

・A がボールを持ってスタンバイ
・B が弧を描いてダッシュ → ジャンプ
・B のジャンプの頂点で A がパス
・B は空中でキャッチ → 着地前にシュート
・Bが6mに入る前にシュートを放てば適法
```

| 項目 | 内容 |
|---|---|
| 命名 | Bernhard Kempa（独、1920-2017）。1954年3月24日に初使用 |
| 有名な使用例 | 2007年世界選手権準決勝・独vs仏（延長）でドイツが炸裂させた |
| ビーチハンドボール | ダブルポイント（2点）になるトリックゴール扱い |
| バックラインから | 走り込みながらジャンプするシューターへのパスでGKをリセットさせない |

### P16：ダイビングポスト（ダイブシュート）

```
・PV が6m前に立つ
・C またはバックがフリースローを受ける
・C が鋭いパスを PV の走りに合わせて出す
・PV が1〜2歩踏み込みながらダイブ（前傾倒）
・体が6m内に入りながら空中でキャッチ → シュート
・着地前にボールを放せば合法
```

| 難易度 | ★★★★☆ — PVの勇気と体の柔軟性が必要 |
|---|---|
| なぜ有効 | 低い角度・予測不能の軌道。GKがセービングポジションを取る前にシュートが来る |

---

### P17：CSSR変形（ウィングへのスレッドパス）

```
（1970年代チェコスロバキア発祥）

① C がフリースロー or オープンパス → RL へ
② C（またはPV）が D₁と D₂の間のギャップに走り込む（ボールなし）
③ RL が C のギャップ通過前にパス
④ C がパスを受けてスレッドパス → カットしてくる LW へ
⑤ LW が角度のあるシュート

最適条件：
  LW が左利きであれば角度が有利
  ウィングDFが開いたまま → C が直接シュート
```

---

### P18：ダブルスクリーン（Double Screen）

```
① C → RL → C → LL と素早くパス回し
② パスが回る間に PV と C（再配置）が
   D₂と D₃に対してダブルスクリーンを設置
③ LL がパスを受けた瞬間
   D₂と D₃がスクリーンに阻まれてクローズできない
④ LL が無人でシュート

配置図：
       D₂         D₃
       |           |
      PV₁        PV₂   ← スクリーン
  LL（ボール保持） → シュート
```

---

### P19：ダイアゴナル（Diagonal / 対角線カット）

```
① C がボールを持ちスタンバイ
② LL が左バックから右センターギャップへ対角線ダッシュ
③ LL が D₃ を横切った瞬間に C がパス
④ LL が勢いに乗ってシュート体勢
```

---

### P20：ウェーブ（Welle / 波状攻撃）

```
  LW → LL → C → RL → RW（左から右へ素早いパス回し）
  ↑パス後の各プレーヤーはゴール方向へダッシュ

・ボールが右に動く間、各選手が内側へ波のように押し寄せる
・右端（RW or RL）がシュート or PV へ縦パス
・折り返し波：右→左でも使用可
```

| 目的 | DFを横方向に引き伸ばしてバック線のスペースを作る |
|---|---|

---

### P21：ワンクロス・ツーストレート（日本代表の定番）

```
① RL が斜め前方にカット → C の前でボールを受ける
② RL が縦方向にドライブ（DFを引きつける）
③ LL と C が RL のドライブに合わせてパラレルダッシュ
④ LL が RL からクロスパスを受け → 縦突進
⑤ C がLL の走りと並走 → LL → C へパス
⑥ C がシュート or 空いた PV or ウィングへ

もし第1段が封じられたら：
  RL/LL/C が反対サイドへポジション移動
  LW がスペース確保
  LL と RL が新サイドのギャップを攻撃
```

| 採用チーム | 日本代表「彗星JAPAN」の最頻出セットプレー |
|---|---|
| 強み | DFに「クロスを追うか縦を追うか」の二択を強いる |

### ─ フリースロー速攻系 ─

### P22：クイックフリースロー（Schneller Freiwurf / 早打ち）

```
笛が鳴った瞬間、DFが整列していないことを利用

① 笛直後に C がすぐボールを持つ
② 後退中のDFが3m離れていないことを確認
③ ウィングが既に前を走っている
④ 即座にロングパス or 直接シュート（3秒以内）

※3m離れていないDFがいてもすぐ笛を吹くわけではないのでリスク計算が必要
```

## Part 3：GKスローからのプレー

### P23：シュネルヴルフ（Schnellwurf / GK→ウィングロングパス）

```
  GK がボールをセーブ or デッドボールを受け取る
    ↓（0.5秒以内に判断）
  RW or LW はすでにダッシュ済みでDFの前に出ている
    ↓
  GK → ウィングへロングスロー（頭上を越えて放る）
  ウィングは走りながら受けてそのままシュート
```

| 成立条件 | ウィングが相手DFより前にいること |
|---|---|
| 失敗原因 | GKの判断が遅い、ウィングが戻っていた、パスが短い |

---

### P24：シュネレ・ミッテ（Schnelle Mitte / 失点直後の速攻スタート）

```
相手がゴールを決めた → 自チームがスローオフ権

通常：選手が揃ってからスローオフ（遅い）

Schnelle Mitte：
① GK が即座にボールを拾い上げる
② センタースローオフ位置の C に素早くパス
③ C は中央ライン（1.5m範囲内）でタッチしてすぐスローオフ
④ 相手DFが守備に戻り切れていない状態で攻撃開始

歴史：TBV Lemgo（2002-03シーズン）・THW Kiel（2001-02）が多用し大会を席巻
```

| IHFルール | スローオフゾーンは中央から1.5m以内。選手はラインを踏めばOK（乗り越え不要） |
|---|---|
| 効果 | 相手が「まだ守備を整えていない」数秒の隙を利用して数的有利を作る |
| 対抗策 | 失点後すぐにDFポジションへ戻る訓練（Schnelle Mitte対策トレーニング） |

---

### P25：GK→ポスト中距離パス（GKビルドアップ）

```
速攻が間に合わない場合：
① GK がポジション取りの良いバックへ短〜中距離パス
② バックがセットオフェンス開始ポジションへ配給
③ 通常のセットオフェンスに移行

GKが7人目として参加する変形：
① GKをベンチに下げてフィールドプレーヤー追加
② →「7on6」フォーメーションへ移行
```

## Part 4：7on6（GK外し攻撃）

### P26：7on6 基本フォーメーション

```
         相手ゴール方向
  ┌─────────────────────────────┐
  │  LW  LL  CB  RL  RW  PV   │ ← 通常6名+元GK（7人目）
  │           PV₂              │ ← ダブルポストにする場合
  │  ─────── 6m線 ─────────── │
  │  D₁ D₂ D₃ D₄ D₅ D₆       │ ← 守備6名（GKなし）
  │         [ 空ゴール ]         │
  └─────────────────────────────┘

元GKの配置パターン：
  A：CBポジションに入る（追加バックとして2CB編成）
  B：LW or RW に入る（追加ウィング展開）
  C：ポストポジションに入る（ダブルポスト化）
```

| 項目 | 内容 |
|---|---|
| リスク | 相手がボール奪取 → 無人ゴールへロングシュート即失点 |
| 対策 | 最後尾に「GKカバー役」1名を残す変形版もある |
| 最高効率アクション | 突破シュート（Breakthrough Shot）— 研究で有意差が最大（adjusted residuals 2.32~2.87） |
| 使用タイミング | 後半残り5分・3点差以内が最多（Frontiers in Sports, 2024） |
| 心理効果 | 相手DFに「1失点でも絶対ダメ」というプレッシャーをかける |

## Part 5：スローイン / コーナーからのプレー

### P27：クイックスローイン

```
相手がラインを割った → スローインの権利
① サイドラインのプレーヤーが素早くボールを取る
② DFが3m離れていない → すぐ展開してOK（合法）
③ パラレルに走るチームメートへのロングパス or 短パス即攻
```

### P28：スローイン→クロスプレー

```
スローイン位置の近く：
① スロワーがサインを出す（口頭 or 手）
② 2名が9m付近でクロスの動き
③ スロワーが有利な方へパス
④ 受け手が即1on1またはシュートへ
```

---

### P29：コーナー→シンプルスタート

```
エンドラインを守備側が割った → コーナーエリアからのスロー
① スロワーがコーナーへ → CB/バックへ短パス
② そのままセットオフェンス構築
もしくは：
③ ウィング+ポストがクロスの動き → スロワーがベストな方へ
```

### 6-2. タイムアウト中のコーチの流れ

```
タイムアウト60秒のプロトコル：

0〜15秒：選手を落ち着かせる（水分・呼吸）
15〜40秒：コーチが磁気ボード上でプレーを図解
           「このサインプレーを使う：○○（コードワード）」
           起点プレーヤーを指名
           第1選択・第2選択を説明
40〜55秒：DF体制の変更指示（あれば）
55〜60秒：コートへ戻る・全員でのコール
```

### 6-3. タイムアウト後の定番プレー設計（残り30秒・1点差）

```
理想の「ラストプレー」設計例：

コーチ：「次のフリースローは『ポーレン』。
         打つのは RL。第1選択はRLの直接シュート、
         ダメなら PV への縦パス、最終はタイムアウト。」

実行：
  笛 → C が「ポーレン！」と叫ぶ
  LL と RL がクロス → 中央混乱
  C が勢いのある位置へ移動
  RL がフリースローを打つ or PV へ縦
```

### 7-2. フランス女子 — リオ2016の6パターン分析

**大西論文（handball.kikirara.jp）によるリオ五輪ノックアウトラウンド分析：**

| パターン | 内容 |
|---|---|
| パターン1 | LL + RL のクロス（**異利き同士が最も効果的**） |
| パターン2 | C→ポストのクロス + バックコートの動き（2段階起動） |
| パターン3 | LL と RL の中央収束クロス（片方はカットイン、片方はクロス） |
| パターン4 | 逆サイドバックがデコイカット + Cとサイドのクロス |
| パターン5 | **5vs6（1名退場）でも中央双クロス** → パッシブにならない姿勢 |
| パターン6 | DFが読み始めたらパスのタイミング変化で対応 |

> **フリーフロー切り替え：** DFが1on1の機会を与えた場面、またはフリースローのたびに、センター（#29 Pineau）が即興攻撃（フリーフロー）に切り替える権限を持つ

### 7-3. ドイツ — コードワード文化の本家

```
Heiner Brand 体制での具体的なプレー：

「Polen（ポーレン）」：
  目的：中央のカオスを作ること
  手段：バックプレーヤーが勢いを持って良い位置に到達
  LLとRLの空クロス → Cが最良ポジションへ滑り込む

「Jugo Links（ユーゴ・リンクス）」：
  CLLがボールを受けて左へ1on1
  Cが中央へ補填

「Jugo Links Kreuz（クロス版）」：
  同上 + RLが背後からクロス → 中央フリー

コードワード命名方針：
  ・チームで記憶しやすい言葉を選ぶ
  ・意味は不要（Hawaii = 右か左かさえ決まれば十分）
  ・同じ言葉を「フェイク（実行しない）」で呼んでDFを混乱させる
```

### 7-4. スペイン — ローリングオフェンス（Circulación）

```
Carlos Ortega（FCバルセロナ→日本代表も指揮）のシステム：

  LW → LL → C → RL → RW
   ↑                    ↓
   PV ←←←←←←←←←←←←←←

・6人全員が常に動く
・ポストは左右に往復
・バックは常にポジションを入れ替え
・「どの選手が何をするか」ではなく
  「全員が動き続けることでDFが崩れる」哲学
```

| 要件 | 全選手が複数ポジションでプレーできるマルチポジション能力 |
|---|---|
| 強み | DFが担当を決められないまま崩れる |
| 弱み | 高い戦術理解と多ポジション熟練が前提 |

### 7-5. ノルウェー — Triggere（トリガー）システム

```
「Triggerahandling（起動動作）」：
  特定のトリガー動作がサインの代わりになる
  選手Aが決まった場所に動く → 選手B/Cがそれに対応する動きを始める

Sagosen の哲学：
「考えずにプレーするほうが良いプレーができる」
→ 決められた反応（反射的判断）を訓練で刷り込む

Sagosen-Myrhol Taktik（2019年世界選手権 vs ドイツ）：
  Sagosen（CB）が突破起点
  Myrhol（PV）がドイツDFの前衛DF（Freigänger）にスクリーン
  → ドイツの5-1守備の前衛を機能不全にした
```

### 7-6. アイスランド — ウィング直撃型

```
「Spectacular and efficient wings」を中心に据える

プレーの特徴：
  複雑なセットプレーよりも1on1質を重視
  速攻からウィングへのロングパスが基本トリガー
  ポストとウィングのコンビプレーを多用
  複雑なコードワード体系より「誰が勝てるかの直感」優先
```

## Part 9：プレー選択じゃんけん

### どのサインプレーをどのDFに当てるか

| 相手DF | 有効なプレー | 理由 |
|---|---|---|
| **6-0（フラット）** | ケンパトリック、ダイビング、ウェーブ、早打ち | 高さの壁 → 上を越えるか壁が整う前に打つ |
| **5-1** | Pick&Roll、ユーゴ、ポーレン | 前衛の裏or脇を突く |
| **3-2-1** | ドライアー・クロイツ、ダブルスクリーン | 3層の間のギャップをクロスで混乱 |
| **3-3（プレス）** | シュネレ・ミッテ、クイックフリースロー | 整列前に速攻 |
| **マンツーマン** | ローリングオフェンス、スクリーン系全般 | 常に動き続けることでマークを外す |

## Part 10：バックプレーヤーのサインプレー実行チェック

```
フリースローを受け取った直後のルーティン：

① サインを確認（コードワード聞く / 手のサインを見る）
② 自分の役割を確認（トリガー役？受け手？スクリーナー？）
③ DF陣形を0.5秒で確認（前衛の有無、ポスト担当の確認）
④ サイン通りに動く OR 即興（フリーフローへ）を判断
⑤ タイミングが「DFが切り替わる瞬間」に合っているか確認

バックプレーヤーが知っておくべき原則：
  ・スクリーンを使うときは「スクリーナーに近づいてから離れる」
  ・パスを受けるのは「走りながら」→ 勢いがそのままシュート力になる
  ・第1選択が消えたら即フリーフロー判断（止まらない）
  ・クロスプレーの「タイミング」はパスを出す側が決める
```
