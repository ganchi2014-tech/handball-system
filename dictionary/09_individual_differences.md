## 身長

| 特性 | 有効な戦術 | 弱点 |
|---|---|---|
| 高身長（195cm超） | ジャンプシュート・上部コーナー・ロブパス・ブロック圧力 | スピード不足→速攻対応困難 |
| 低身長（170cm以下） | 低弾道シュート・股下・腰重心でドリブル突破・ファウル誘発 | 高身長DFのブロックに弱い |
| 平均身長 | バランス戦術・オールラウンド | 突出した武器がない |

## 利き手

| 組み合わせ | 特徴 | 戦術的優位 |
|---|---|---|
| 左利き×右サイド | 内切りで利き手がゴール方向 | 最も自然なシュート角度 |
| 左利き×バックセンター | DFが逆向きの壁を作る必要がある | 非対称プレッシャー |
| 右利き×左サイド | 内切りで利き手がゴール方向 | 同上 |
| 両利き | 読まれない | 最も対応困難 |

**左右混在の「非対称プレッシャー」**：左利きと右利きが交互に攻撃することで、DFが毎回ブロック方向を変えなければならない

## ジャンプ力

| タイプ | 有効な戦術 |

**最難対応コンビ**：高ジャンプ×左利き（高さと角度が両方非標準）

## 左利き右バックの構造的優位（グローバル最重要分析）

世界トップレベルで「左利き右バック」は最も警戒されるポジション。理由は構造的。

```
通常の右利き右バック：
　シュートの腕 → 右（ゴール中央〜左コーナー方向）
　DFは右から押さえる（標準的な対処）

左利き右バック：
　シュートの腕 → 左（ゴール右コーナー方向 = 逆クロスコーナー）
　DFが標準の守り方をすると「逆コーナー」が完全に空く
　DFが逆から押さえようとすると体勢が不自然になる
```

**左利き右バックへのDFが難しい理由**：
- 標準的な0-6ゾーンは「右利き前提」の配置で組まれていることが多い
- 左コーナー（右利きには難角度）が左利きには自然なシュートコース
- DFが正面から来た場合、利き腕が「DFの守りやすい方向」の逆にある

**戦術的応用**：
| 場面 | 左利き右バックが使うべきアクション |
|---|---|
| DFがクローズ（内側） | 左への外シュート（逆クロス・DFの反対コーナー） |
| DFがワイド（外側） | 内突破からの左ニアシュート |
| DF正面 | 左へのステップフェイント → 右への切り返し（DFの利き手方向が逆） |
| 右バックから左バックへの連続攻撃 | DFがブロック方向を毎回切り替えなければならない → 疲弊 |

## 身長別シュートコース選択

| 身長 | 使いやすいシュートコース | 理由 |
|---|---|---|
| 高身長（190cm+） | 上部コーナー（左上・右上） | ジャンプ最高点からGKを見下ろす角度になる |
| 高身長 | スマッシュシュート（ネット真上から叩きつける） | 高い位置からの弾道がGKに防ぎにくい |
| 平均身長 | 中央コース・ファーコーナー | オールラウンド |
| 低身長（175cm以下） | 股下・低コーナー（左下・右下） | ジャンプ最高点が低い → 低弾道が自然 |
| 低身長 | バウンスシュート（手前でバウンドさせる） | GKの手前を通す低弾道 |

**高身長DFに対する低身長OF**：
```
高身長DFがブロックに飛ぶ
→ 低弾道でDFの手の下を通す（股下・低コーナー）
→ DFが手を下げる
→ ループシュートでDFの頭上を越す
→ DFがまた飛ぶ → 低弾道に戻る
```

## スピードタイプ別1on1戦術

| スピードタイプ | 1on1の仕掛け方 | 使わない方がいい技 |
|---|---|---|
| 最高速度型 | スピードフェイント（速さでDFのリズムを崩す） / 直接突破 | 多段フェイント（スピードを活かせない） |
| 加速型（瞬発） | ジャブステップ → 爆発的加速 / ヘジテーション | 長距離スプリント（持続しない） |
| 持久型 | 後半の疲弊DFを突く / 繰り返しフェイントで消耗させる | 序盤の1on1勝負 |
| 低速型 | フェイントで主導権を取る / 接触でファウルを誘う | スピード勝負（追い抜けない） |

## 自己タイプ診断フレーム

```
Q1: 自分の身長は？
  → 185cm以上 → 上部コーナー・スマッシュシュートを主武器に
  → 175cm以下 → 低弾道・バウンスシュートを主武器に

Q2: 得意な動き出しは？
  → 急加速が得意 → 加速型。ジャブステップからの爆発突破
  → 持久力が高い → 持久型。後半勝負・繰り返しフェイント
  → 一番速い → 最高速度型。スピードフェイント・直接突破

Q3: 利き手は？
  → 左利きでバックライト/センター → 左利き右バックの優位を最大活用
  → 右利きでバックライト → 標準戦術
  → 両利き → 最も対応困難。両手シュートをオプションに

Q4: 接触への強さは？
  → 強い → コンタクト・廻旋フェイント・ファウル誘発を積極利用
  → 弱い → 接触前にフェイントで外す・スピードで離す
```

## じゃんけん（身体特性）

```
高身長ブロックDF
  → 低弾道早打ち（踏み切らない）で股下/低コーナー
  → GKが下に合わせる
  → 高跳躍ジャンプシュートで上部コーナー
  → 高身長DFブロックに戻る

左利きの非対称プレッシャー
  → 右利き前提のゾーンDF（左コーナーが弱い）
  → ハーフタイムで左コーナー担当強化
  → 左コーナーDFを引き付けてセンターパス → 右利きバックライト
  → 両方にマーク強化 → ダブルスクリーン
  → 左利きエースに戻る
```
