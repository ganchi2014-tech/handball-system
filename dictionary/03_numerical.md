# 数的状況・速攻・バックチェック

---

## 数的有利（OF：多い側）

### 基本原則
- 「先に読んだ方が選択肢を持てる」
- 数的有利は**即座に活かす**：時間をかけるとDFが戻ってくる
- 最終的には必ずフリーの選手が出る → その選手を探す

### 2on1（2人対1人）
- DFがどちらかに寄った瞬間に逆へパス
- DFが中間にいる場合：どちらに動いても対応できると思わせて近い方からシュート
- じゃんけん：DFが1人に集中→逆にパス / DFが中間→シュート判断

### 3on2（3人対2人）
- DF2人の外側を使う
- サイドに広がってDFを引き伸ばす → 中央が空く
- 中央が空いたら真ん中の選手がシュート

---

## 数的有利（DF：多い側）

### 5on6（数的不利守備）
- DFラインを引いてコンパクトゾーンを形成
- 外シュートはGKに任せる（リスクを冒さない）
- ボールロストからの速攻に備えて1人後方待機
- 早い攻撃は禁物（カウンターリスク）
- **2分間を0点で凌ぐ**のが目標

---

## 数的同数

### OFの考え方
- 1on1の局面を作ることが目標
- スクリーンや動きでマークを剥がす
- ポストを使ってDFを動かす

### DFの考え方
- 数的同数でも個人の1on1で負けない
- ヘルプの声かけ・連携が重要
- 位置関係を常に把握する

## 速攻

### 速攻の発動条件
- 相手のシュートミス → GKがボールを保持した瞬間
- インターセプト
- 相手ボールのミス

### 速攻の種類
| 種類 | 人数 | 説明 |
|---|---|---|
| 1stウェーブ | 1〜2人 | GKスロー直後の最速カウンター |
| 2ndウェーブ | 3〜4人 | 1stが止められた後に合流する第2波 |
| セカンドチャンス速攻 | 全員 | DFが整い切る前の全員参加型 |

### 速攻の具体的役割分担

**1stウェーブ（GKスロー直後・1〜2人）**

```
GKがボールを保持した瞬間に開始。全員の理解が必要なのは「**速攻判断2秒ルール**」：
2秒以内に「速攻か遅攻（DFが戻る前を狙わず、いったん攻撃を組み立て直してゆっくり攻めること）か」を全員が把握して走るコースを決める。
（※04_cognition.mdの「2秒の壁」は別概念：個人がボールを受けた後の判断時間）

右サイド走者：相手コートの右コーナー方向へ全力スプリント（全力疾走）
左サイド走者：相手コートの左コーナー方向へ全力スプリント
→ GKがどちらかをケアした瞬間にもう一方へパス
→ 2人ともマークされた場合 → 2ndウェーブを待つ
```

**GKスローのパスパターン：**
| パスの種類 | 使う条件 |
|---|---|
| ロングパス（ダイレクト＝中継を挟まず受け手へ直接） | 走者が完全にフリーで距離がある |
| ミドルパス（中継経由） | 完全フリーがいない。走りながら中継を使う |
| バウンスパス（低弾道） | DFが途中でインターセプトを狙うとき |
| 即時ドリブル（GK自身が運ぶ） | 誰もスプリントできていないとき（緊急手段） |

---

**2ndウェーブ（3〜4人・1stが止まった後）**

```
1stウェーブが止められた後に後続が合流する場面

中央走者：ゴール正面のセンターポジションに走り込む
  → 中央から1on1を仕掛けてGKを引き付ける
  → GKが中央を意識したら左右へはたく

右・左サイド走者：6mラインの角（バックコーナー＝ゴールエリアの線の左右の隅）を取る位置へ
  → パスが来たらそのままシュート（角度はあるが距離は近い）

リバウンド役（遅れて来た選手）：ゴール正面のリバウンドポジションへ
  → シュートが外れた場合の詰め
```

**3人速攻（3on2）のじゃんけん：**
```
中央がドライブしてDF2人を引き付ける
  → 右or左にパス → シュート
  → DF2人が両サイドをケア
  → 中央が直接シュート
  → DFが中央をケア
  → 再び外へパス
```

---

### 速攻のじゃんけん
```
速攻（アウトナンバー）
  → バックチェックで数を整える
  → セットDF完成
  → セットOF始動
  → シュートミスまたは得点
  → 相手の速攻または自チームGKスロー
  → 速攻に戻る
```

## バックチェック

### 基本原則
- シュートを打った瞬間から始まる（シュート後に止まらない）
- 最低2名が即時帰陣（全員でなくてよい）
- コースをカットして速攻の成功率を下げる

### バックチェックの判断
- 相手GKがボールを保持：全力で帰陣
- 相手がカウンターで抜け出している：追いかけてコースをカット（ファウル覚悟）
- 相手が速攻を断念した：帰陣しながらセットDFに移行

### よくあるミス
- シュートの結果を見てから動く（→ 遅い）
- 全員で速攻に出て誰も帰陣しない
- カウンター1人目を止めてもカバーがない
