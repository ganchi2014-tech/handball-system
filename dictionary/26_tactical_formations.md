# 26. 戦術フォーメーション一覧

> **情報源：** IHF ルール 2025、EHF タクティカル 用語集、DHB Rahmentrainingskonzeption、ノルウェー代表分析、各国協会指導書、handball-planet.com

## フォーメーション記法の読み方

```
数字表記の原則（DF側）

  ← 9m線方向（高い位置）
  前衛 ー 中衛 ー 後衛
  ← 6m線方向（低い位置）

例：「5-1」= 後衛5枚 + 前衛1枚
例：「3-2-1」= 後衛3 + 中衛2 + 前衛1（3層）

コートの見方（この辞書の全図共通）

          相手ゴール方向
  ┌────────────────────────┐
  │  9m線（フリースロー線）  │
  │                        │
  │  6m線（ゴールエリア線）  │
  │    ┌──────────┐        │
  │    │    GK    │        │
  │    └──────────┘        │
  └────────────────────────┘
          自陣ゴール側
```

## パート 1：DF（ディフェンス）陣形

### DFフォーメーション全体マップ

```
積極性（高）←━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━→ 保守性（高）
  0-6   1-5   2-4   3-3   3-2-1   4-2   5-1   6-0
   │     │     │     │      │      │     │     │
 超攻  攻  バランス  バランス  守寄  守備  守  超守
 撃的 撃的   寄り   寄り  攻撃  守備  備   備
```

### ① 6-0（ゼロ-ゼクス / フラットシックス）

```
          攻撃側
  ──────────────────────── 9m線
  
  ──────────────────────── 6m線
     D₁  D₂  D₃  D₄  D₅  D₆     ← 6名全員が6m線に並ぶ

          ┌──────┐
          │  GK  │
          └──────┘
```

| 項目 | 内容 |
|---|---|
| 別名 | Flache アプヴェーア（守備）（独）/ Flat defense（英）/ ブロック守備（日）|
| 配置 | 6名が6m線沿いに横一列 |
| 特徴 | コンパクトブロック、クロスシュートコース絞り、ポスト入りスペース消去 |
| **強み** | バックライン高さを活かしたシュートブロック、チーム連動が容易 |
| **弱み** | 9m〜10mの遠距離シュートに無防備、DFが引き出されたら即スペース発生 |
| カウンター（OF） | ロングシュート集中、9mフリーポジション確保、ポスト不使用 |
| 採用する状況 | リード守備、フィジカル強い大型DFチーム、疲労蓄積時 |
| 代表的採用国 | ドイツ（DHB）、スウェーデン、スペイン（状況別） |

**6-0の3基本動作（コーチ直伝 ドイツ語専門用語）**

| 動作 | ドイツ語 | 内容 |
|---|---|---|
| シフト | **フェアシーベン（シフト）**（フェルシーベン） | ボール側へラインが一体となって横移動。ボールサイドに密集を作る |
| 飛び出し | **ヘラウスリュッケン（前飛び出し）**（ヘラウスリューケン） | インナーDFがボールを持つバックに前進プレッシャー |
| 受け渡し | **ユーバーゲーベン（引き渡し）**（ユーバーゲーベン） | 飛び出した後、隣のDFにマークを渡してラインを維持 |

**ティーフェンラウム（深部空間）**（ティーフェンラウム）= ゴール前の危険なスペース。6-0の目的はここを最小化すること。

**崩し方（OF視点）詳細**：
- フェアシーベン（シフト）直後 → 逆サイドへ素早い展開でスペースを使う
- ヘラウスリュッケン（前飛び出し）背後 → ピボットを走り込ませる
- ユーバーゲーベン（引き渡し）（受け渡しの一瞬） → クイックパスで瞬間的なマーク外れを突く
- 遠距離から積極的に外シュート → DFラインを引き出す

### ② 5-1（フュンフ-アイン）

```
          攻撃側
  ──────────────────────── 9m線
           D₆               ← 前衛1枚（センター対面に出る）
  
  ──────────────────────── 6m線
    D₁  D₂  D₃  D₄  D₅     ← 後衛5枚

          ┌──────┐
          │  GK  │
          └──────┘
```

| 項目 | 内容 |
|---|---|
| 別名 | フュンフ（独：5）-アインス（独：1）（独）/ Five-One（英） |
| 前衛の役割 | 相手センターバック（CB）へのプレッシャー、パスコース遮断 |
| **強み** | CBのオーガナイズを妨害、速攻抑制（前衛が即プレス） |
| **弱み** | 前衛の裏スペース（前衛両脇）がオープンになりやすい |
| カウンター（OF） | 前衛の脇をバックプレーヤーが突くカットイン、前衛を誘い出してその裏へパス |
| 採用する状況 | 相手CBが強力な場合、速攻を封じたい場面、拮抗した試合 |
| 代表的採用国 | デンマーク、フランス、クロアチア（定番） |

**トップDFの連動（コーチ直伝）**：
- トップDFは単独で動くのではなく、**裏のフルバックDFとの位置関係**を常に把握した上で動く
- **ポストの位置**も連動判断の要素（ポストがどこにいるかでカバーの優先順位が変わる）
- カバー（穴を埋める動き）と牽制（相手を動かす動き）を状況に応じて使い分ける
- 試合の流れによって判断するもの。数式的に「この場面はカバー」ではなく流れを読む

### ④ 3-2-1（ドライ-ツバイ-アイン）

```
          攻撃側
  ──────────────────────── 9m線
              D₆            ← 前衛1枚（最前線）
        D₄       D₅        ← 中衛2枚
  ──────────────────────── 6m線
      D₁    D₂    D₃       ← 後衛3枚

          ┌──────┐
          │  GK  │
          └──────┘
```

| 項目 | 内容 |
|---|---|
| 別名 | ドライ（独：3）-ツヴァイ（独：2）-アインス（独：1）（独）/ Pyramid defense / 3層守備（日） |
| 特徴 | 3層構造。中衛がポストと前衛の穴を埋めるハイブリッド |
| **強み** | 全エリアに対応できる柔軟性、速攻の起点にもなりやすい |
| **弱み** | 各選手の担当範囲が広い、コーチングの複雑さ、連動ミスが即失点につながる |
| カウンター（OF） | 中衛と後衛の間に入り込むポジション取り、クロスプレーで層を混乱させる |
| 採用する状況 | 攻撃的守備をしたい場面、リードされている終盤 |
| 代表的採用国 | フランス（伝統的な派生系）、クロアチア |

```
          攻撃側
  ──────────────────────── 9m線
       D₄   D₅   D₆        ← 前衛3枚（9m線付近）

  ──────────────────────── 6m線
       D₁   D₂   D₃        ← 後衛3枚
```

| 項目 | 内容 |

| カウンター（OF） | ポスト中央で起点作り、速いボール回しで前衛を置き去り |
| 採用する状況 | 大差で追う終盤、相手の遅攻を止めたい時、運動量が勝っている場合 |

### ⑧ ハイブリッド / 個人マークシステム

#### 部分マンツーマン（Gemischte アプヴェーア（守備））
```
          攻撃側
  ──────────────────────── 9m線
              D₆ →（相手エースをマンマーク）

  ──────────────────────── 6m線
     D₁  D₂  D₃  D₄  D₅  ← ゾーン5枚

```

- 相手の特定エース選手（例：強力なCB）に1名がマンマーク
- 残り5名はゾーン維持
- **利点：** エース封じ ＋ ゾーンの安定性
- **欠点：** マンマーカーのフィジカル消耗、ローテーション複雑

#### スイッチングシステム（Wechselsystem）
- 試合中に6-0 → 5-1 → 3-3 を流動的に切り替え
- 欧州トップチームが多用
- 選手全員が複数システムを習熟している必要がある

## パート 2：OF（オフェンス）陣形

### OFフォーメーション記法

```
数字表記（OF側）

  バックライン ー ポスト（前衛）ライン
  例：「3-3」= バック3（LB/CB/RB）+ 前線3（LW/PV/RW）
  例：「2-1-3」= バック2 + ポスト1（前衛） + ウィング2（フォワード）
                （実際には LB/RB + 中央ポスト + 両ウィング = 5人バック1ポスト）
```

---

### ① 3-3（バックスリー / スタンダード）

```
         相手ゴール方向
  ──────────────────────── 9m線
  LW         PV        RW   ← 前線3（左ウィング・ポスト・右ウィング）

  LB         CB        RB   ← バック3（左・センター・右バック）
  ──────────────────────── 

```

| 項目 | 内容 |
|---|---|
| 別名 | Standard formation（英）/ 標準陣形（日）|
| 特徴 | バックプレーヤー3名が弧を描き、ポストが中央インサイドに入る |
| **メリット** | 全方向への展開が可能、シュートバリエーション豊富 |
| **デメリット** | 相手に読まれやすい、バックプレーヤーが揃って高さ不足なら6-0に封殺 |
| バリエーション | 「クローズ」ポスト（6m付近）vs「オープン」ポスト（9m付近） |
| 対DFじゃんけん | 6-0に対しては遠距離シュートが有効。5-1に対してはCBのドライブ |

### ② 2-1-3（ダイアモンド / ウィングオフェンス）

```
         相手ゴール方向
  ──────────────────────── 9m線
    LW         PV      RW   ← 前線3（両ウィング + ポスト）
               CB           ← センター（前に出る）
    LB                 RB   ← バック2（両サイド）
  ──────────────────────── 
```

| 項目 | 内容 |
|---|---|
| 別名 | Diamond formation（英）/ Rhombus / ダイアモンド陣形（日） |
| 特徴 | CBが中央を牽引、両サイドバックとウィングが幅を作る |
| **メリット** | CBによる牽引からの展開が多彩、ウィングを活かしやすい |
| **デメリット** | CBへの依存度が高い、CB封じられると機能しない |
| 対DFじゃんけん | 5-1の前衛を中央から引き出し、左右サイドに展開で有効 |

### ③ ダブルポスト（2-2-2 / ツインポスト）

```
         相手ゴール方向
  ──────────────────────── 9m線
    LW    PV₁  PV₂    RW   ← ポスト2枚が中央で並立
           CB               ← CBが中央で指揮
    LB                 RB   ← 両サイドバック
  ──────────────────────── 
```

| 項目 | 内容 |
|---|---|
| 別名 | Doppelpivot（独）/ Twin pivot（英）/ ダブルポスト（日） |
| 特徴 | ポスト2枚でDF2枚を引きつけ、バックラインにスペース |
| **メリット** | DFの混乱誘発、ポスト同士のコンビプレー（ロールオフ等） |
| **デメリット** | バックラインが1枚減る（CB or バック1名がポスト役）、遠距離シュートが減少 |
| 代表的採用国 | スペイン（Ortega体制）、デンマーク |
| 対DFじゃんけん | 6-0が相手の場合にポスト2枚で内側を崩す |

### ④ ワイドオフェンス（3-3 ワイド）

```
         相手ゴール方向
  ──────────────────────── 9m線
LW                       RW  ← ウィングが最大限に開く
      LB   CB    RB           ← バック3が幅を保つ
                PV             ← ポストが中央に残る
  ──────────────────────── 
```

| 項目 | 内容 |
|---|---|
| 別名 | Wide offense（英）/ ブレイテス System（独）/ ワイド展開（日） |
| 特徴 | 横幅を最大化してDFを引き延ばす |
| **メリット** | DFを横に引き延ばしてバック線スペース拡大、ウィングシュート増加 |
| **デメリット** | ポストへのパスルートが長い、パス強度が必要 |
| 対DFじゃんけん | 3-3守備（密集系）に対して有効、6-0の圧縮DFには逆効果になることも |

## パート 3：特殊状況のフォーメーション

### ① 7on6（セブン-オン-シックス / GK外し）

```
         相手ゴール方向（自チームがGKを上げた状態）
  ──────────────────────── 9m線
LW      LB    CB    RB      RW
              PV                ← ポスト中央
  ←─────────────────────────→ 
  追加フィールドプレーヤー（元GK）が参加 → 7人攻撃

一般的配置オプション：
  [A] CB + 元GK がバック並列（2CB編成） 
  [B] 元GK がLW/RWに入る（7人ウィング展開）
  [C] 元GK がポスト（ダブルポスト化）
```

| 項目 | 内容 |
|---|---|
| 使用場面 | 試合終盤でリードされている場合（残り数分） |
| **最大リスク** | 相手にボール奪われたら無人ゴールへロングシュート即失点 |
| カウンター対策 | 最後方に「GKコバー役」を1名残す変形版もある |
| IHFルール | GKをベンチに下げてフィールドプレーヤー7名起用。合法。 |
| 心理効果 | 相手DFに「絶対に失点できない」プレッシャーを与える |

### ② 6on7（数的不利守備）

```
         攻撃側が7on6仕掛けてくる局面
  ──────────────────────── 9m線
              D₇              ← 7人目の相手選手

  D₁  D₂  D₃  D₄  D₅  D₆    ← 自チーム6名守備

          ┌──────┐
          │（GKなし）       ← 自陣は空ゴール
          └──────┘
```

| 対応戦略 | 内容 |
|---|---|
| **速攻特化** | ボール奪取→即前方への速攻（相手GKは上がっている） |
| **ファウル戦略** | 故意ファウルで時計を使い（イエカード覚悟） |
| **ゾーン維持** | パニックにならず組織的ゾーン守備を維持 |

### ③ 速攻フォーメーション（ファスト（速い） Break Setup）

#### 一次速攻（First Wave）
```
    GK がボールを持った瞬間のポジション
    
    W（即ダッシュ）           W（即ダッシュ）
                                              ← ゴールに向かって走る
    
    [GK]──────→  B₁（受け手）──── → ゴール
    
    ルール：
    ・ウィングは相手ゴール方向に走りながらパスを受ける
    ・GKは速く、正確に前方へ
    ・B₁は相手DF組織が整う前に決定的アクション
```

#### 二次速攻（Second Wave）
```
    一次速攻が間に合わなかった場合：
    
    ─────────────────── 速攻ライン（DF組織化前）
    
    ウィング or バック が深く走り込んでパスを受ける
    → 2-1状況や1-0状況を探す
```

### ④ フリースロー攻撃陣形（9m攻撃）

```
パターンA：センター突破型
  ──────────── 9m線 ────────────
  LW   LB   CB   RB   RW
                PV                 ← ポスト中央待機
  CBがドライブ or LB/RBへ展開

パターンB：サイン「ダブルスクリーン」型
  ──────────── 9m線 ────────────
  LW        CB        RW
       LB [PV] RB              ← LBとRBがポスト両脇でスクリーン
  → バック選手がスクリーン裏から走り込んでシュート
  
パターンC：速攻型（早打ち）
  ──────────── 9m線 ────────────
  W ────── CB ────────── W
  → DF組織が整う前に即シュート（3秒以内）
```

## パート 4：じゃんけん対応マトリクス

### OFフォーメーション × DFフォーメーション

| OF ↓ / DF → | **6-0** | **5-1** | **3-2-1** | **3-3** |
|---|---|---|---|---|
| **3-3 スタンダード** | ロングシュート有効、ポストは困難 | CB突破 or 前衛裏パス | 中衛と後衛の間狙い | 速いパス回しで翻弄 |
| **ダブルポスト** | ポスト2枚で内側崩し★有利 | ポストが前衛引き出す | ポスト2枚で3層崩し | 中央密集＋速い展開 |
| **ワイド** | DFを横に引き延ばす★有利 | ウィングへの裏パス | 後衛3枚の幅を超える | 広い分スペース×（DFが追いつく） |
| **クローズ** | 密集でコンビプレー | CB抜けでポスト直結 | 密集がDFと噛み合わない | スクリーン効かない |
| **7on6** | GKなし相手に6人シュート★超有利 | — | — | — |

> **★凡例：** ★有利＝このOFがこのDFに対して有効な傾向　×＝不利

---

### DFじゃんけんの本質原則

```
6-0の強さ：
  → バックラインを高さでブロック
  → ポストへのパスをDFが潰す
  → 9mロングシュートには無力

5-1の強さ：
  → CBを封じ込め → 展開力を奪う
  → 速攻の起点選手を高い位置でプレス
  → 前衛の裏スペースが穴

3-2-1の強さ：
  → 全方向に層がある
  → 速攻奪取からカウンターが狙いやすい
  → 疲労と連動ミスが欠点

3-3の強さ：
  → 相手のセットOF構築を高い位置で妨害
  → 運動量多い選手で短時間使用
  → 体力消耗極大
```

## パート 5：フォーメーション移行パターン

### DF陣形のダイナミック切り替え

```
ゲーム状況に応じた陣形変化（流れ）

 スコア同点〜リード：     6-0 または 5-1
   ↓（追い上げが必要）
 わずかに追う（-3点以内）：5-1 または 3-2-1
   ↓（終盤追い上げ）
 大差追う（-5点以上）：   3-3 または 2-4
   ↓（残り数分絶体絶命）
 残り2〜3分：             1-5 または 3-3 + 7on6

切り替えのタイミング：
  ・タイムアウト後
  ・GKセーブ直後
  ・速攻で点を取った直後（そのまま高プレスへ）
  ・退場者（イエカード）で数的不利になった後
```

### OF陣形のダイナミック切り替え

```
相手DFを読んでOFを切り替える

 相手が6-0 →「ワイド」または「ロングシュート集中」
   ↓（相手が5-1に変えたら）
 5-1を見たら →「CB突破 or 前衛脇を抜けるカット」
   ↓（相手が3-2-1に変えたら）
 3-2-1を見たら →「ポスト起点のハイロー」
   ↓（終盤追い上げ相手が3-3プレス）
 3-3プレスを見たら →「落ち着いてポスト縦パス」
```
