# じゃんけんマップ — 全カテゴリ横断統合（索引兼マップ）

> 「この世にある全てのハンドボールを私は認めている。相性や向き不向きがあるだけ」

---

## 核心原則

1. **絶対最強の戦術は存在しない** ── 全てにカウンターがある
2. **「先に読んだ方が選択肢を持てる」** ── 認知の速さが結果を決める
3. **「2/3閉じて1/3開ける」** ── GKの法則→DF全体に転用可能
4. **「予測の予測を予測する」** ── 軍拡競争としての戦術
5. **「数的優位は条件付きで有効」** ── 自チームの強みと合致するときのみ
6. **「コーチが言うとバイアスがかかる」** ── 辞書はAIによる全羅列

## 【OF↔DFの基本サイクル】

```
クローズDF（内側に立つ）
  → OFに外パス・外シュートの選択肢 → ワイドOF優位
  → DFがワイドに対応 → ラインが広がる
  → 内側にスペース発生 → クローズOF（内突破）優位
  → クローズDFに戻る
```

```
マンツーマンDF

  → ゾーンDF（スペース消し）
  → 外シュートで引き出す → ゾーン間にスペース → 1on1突破
  → マンツーマンに戻る
```

## 【GKサイクル】（各国理論統合版）

```
先読み型GK（コミット＝先に決め打ちで動くタイプ。リリース＝ボールが手から離れる瞬間の193ms（約0.2秒）前に動く）
  → 早打ちでGKの動作前に放つ
  → コース絞り型GK（待ち：クワイエット Eye＝視線を一点に保って見極める技術。これを延長してリリースの瞬間までボールを見続ける）
  → 引きつけてから角度変化 or 空き2エリアを精度で打つ
  → GKが次第にそのコースを先読みで塞ぐ → 先読み型に戻る

※科学的根拠：GKの反応+動作に500ms必要、シュートフライトは350ms
  → 純粋反応では不可。エリートGKはリリース193ms前から動き始めている
※3方向の法則：常に「左・中・右」の2/3が閉、1/3が開いている

【大型GK（スペイン型 200cm）サイクル】
大型GKが面積でコースを消す
  → 高跳躍ジャンプシュートで上コーナーを狙う
  → Schrittsprungtechnik（シュリットシュプルング技術＝ドイツ式の踏み込みジャンプ。ループシュートなど上のコースを塞ぐGK技術）でGKが上を塞ぐ
  → 低コース（股下・腰）を狙う
  → GKが低姿勢 → 上コーナーが空く → 高跳躍に戻る

【左利き非対称サイクル】
左利きシューター登場 → GKの精度が低下（学習経験の非対称性）
  → 左利き専用練習 → 精度回復
  → 左利きの希少性が消える → 右利き並みの対策完成
```

## 【フェイントサイクル】（全7種＋ダブル原則）

```
DFが接触してくる（ホールディング・プッシング）
  → ①廻旋フェイント（腕を外して差し込む）
  → DFが接触をやめる → 腕が自由 → ③⑤⑥（シュート・ステップ・ボディの各フェイント）が使いやすくなる

DFが腕・手を見ている
  → ③シュートフェイント / ④パスフェイント
  → DFが骨盤を見るようになる
  → ⑤ステップフェイント / ⑥ボディフェイントが効きにくくなる
  → ②ターンフェイント（体全体動かす）or ⑦スピードフェイント（あるいはダブルフェイント原則で7種から2種連続）

GKが先読み型
  → ③シュートフェイント or ③-B バックハンドフェイク
  → GKがコース絞り型に変化
  → 引きつけてから角度変化
```

## 【ドリブル・打点サイクル】

```
スピードドリブルで縦突破
  → DFが体を張って縦を切る
  → 止まってシュートフェイント
  → DFが前に出る（ブロックを狙う）
  → 脇を抜けてドリブル突破

早打ち（上昇中の打点）
  → GKが最高点予測で先に動いている → 動く前に放てる
  → GKが早打ちに慣れる
  → 最高点に戻す → GKが過剰に早く動いて空く
  → GKがコース絞り型に変化 → 角度変化で打つ
```

## 【ポストプレーサイクル】

```
ポストを守るDFがV字位置（ポストとゴールを結ぶ線上に先回りして立ち、パスコースを切る守り方＝先読み）
  → ポストOFが位置を変える
  → DFが追う → 別スペース発生
  → バックプレーヤーが突破
  → DFがバックに集中
  → ポストへの供給ルート開く
  → DFがポストのV字位置に戻る

ポストの4プレー（回転/フェイク/フラッシュ/スクリーン）の連携：
回転 → DFが背後を強くカバー → フェイク（逆方向）→ DFが反応 → 回転に戻る
フラッシュ（止まっていたポストが一瞬で動き出してパスを受けに飛び込む動き） → DFがフラッシュを警戒（背後集中） → スクリーンで離す → ロールアウト（スクリーンのあとゴール方向へ回転しながら離れてパスを受ける動き）
```

## 【7mスローサイクル】

```
シューターが特定コース（例：左コーナー）を狙う
  → GKが先読みで左コーナーに動く
  → 右コーナーへの切り替えシュート（逆を突く）
  → GKが両コーナーを警戒してコース絞り型に変化
  → ロブヴルフ（ループシュート＝GKの頭上をふわりと越すシュート）で上部を突く
  → GKが高く構えてロブヴルフ（ループシュート）対策
  → 低弾道・バウンスシュートで股下コース
  → GKが低く構える → 再び上部コーナーが空く
  → 先読み型GKに戻る

※飛翔時間0.35秒：GKが動き始める前にボールが届く速度で打てば先読み関係なし
```

## このマップの使い方

**試合中に思い出す順序**：
1. 自分の状況を「OFかDFか」「数的状況」「時間・スコア」で位置づける
2. 該当するサイクルを思い出す
3. 「相手は何をしている」を確認
4. その逆を取るアクションを選ぶ

**詳細が必要なときは**：上の【索引】から該当ファイルを開く。
