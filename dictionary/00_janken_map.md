# じゃんけんマップ — 全カテゴリ横断統合

> 「この世にある全てのハンドボールを私は認めている。相性や向き不向きがあるだけ」

---

## 核心原則

- 絶対最強の戦術は存在しない
- 全てにカウンターがある
- 「先に読んだ方が選択肢を持てる」

---

## 【OF↔DFの基本サイクル】

```
クローズDF（内側に立つ）
  → OFに外パス・外シュートの選択肢 → ワイドOF優位
  → DFがワイドに対応 → ラインが広がる
  → 内側にスペース発生 → クローズOF（内突破）優位
  → クローズDFに戻る
```

## 【数的状況サイクル】

```
マンツーマンDF
  → スクリーンでマーク剥がし → フリー確保
  → ゾーンDF（スペース消し）
  → 外シュートで引き出す → ゾーン間にスペース → 1on1突破
  → マンツーマンに戻る
```

## 【GKサイクル】

```
波型GK（コース予測で動く）
  → 早打ちでGKが動く前に打つ
  → スポット型GK（動かずコース絞り）
  → 引きつけて最後に角度で打つ
  → 波型GKに戻る

※常に「左・中・右」の2/3は閉じている。開いている1/3を読む勝負
```

## 【身体特性サイクル】

```
高身長ブロックDF
  → 低弾道早打ち（踏み切らない）で股下/低コーナー
  → GKが下に合わせる
  → 高跳躍ジャンプシュートで上部コーナー
  → 高身長DFのブロックに戻る
```

## 【ゲームクローズサイクル】

```
大量リード → ポゼッション戦術
  → パッシブプレー警告（強制攻撃）
  → 攻撃ミス → カウンター機会
  → 点差縮まる → 負けチームが7on6
  → 空ゴールリスク → DFが引く
  → 7on6でピボットがフリー
  → ゾーン守備でコンパクト化 → 得点率低下 → 時間切れ
```

## 【ファウル損得サイクル】

```
ファウルしてでも止める（確実な失点回避）
  → 累積退場リスク上昇
  → 退場発生 → 数的不利（5on6）
  → コンパクト守備で凌ぐ
  → 退場明けで数的回復 → 通常に戻る
```

## 【DF陣形サイクル】

```
6-0ゾーン（外シュート多発）
  → 1-5（前衛1人でプレッシャー）
  → 前衛を引き付けてピボットパス（5-1の裏）
  → 3-2-1（縦に対応）
  → 縦の連係でスペース作り
  → 6-0に戻る

マンツーマン
  → スクリーンでマーク剥がし
  → スイッチ対応 → ミスマッチ発生
  → ゾーンDF
  → 外シュートで引き出す
  → マンツーマンに戻る
```

## 【ポストプレーサイクル】

```
ポストがV字位置（先読み）
  → ポストOFが位置を変える
  → DFが追う → 別スペース発生
  → バックプレーヤーが突破
  → DFがバックに集中
  → ポストへの供給ルート開く
  → ポストのV字位置に戻る

「そこにいるだけでDFが迷う」= 存在自体が脅威
```

## 【速攻サイクル】

```
速攻（アウトナンバー）
  → バックチェック（数を整える）
  → セットDF完成
  → セットOF始動
  → シュートミス or 得点
  → 相手の速攻（または自チームのGKスロー）
  → 自チームの速攻に戻る
```

## 【スクリーンサイクル】

```
スクリーン利用（カットイン）
  → DFがスイッチ対応
  → ミスマッチ発生（スクリーナーが有利）
  → スクリーナーをポップアウト（外に開く）
  → 外シュート機会発生
  → DFがポップを追う → カットインのフリー復活
  → スクリーン利用に戻る
```

---

## カテゴリ横断じゃんけんマップ（複合）

```
高身長エースの外シュート
  → スクリーンでフリー確保
  → スカウティングで対策：スイッチDF
  → ミスマッチ → スクリーナーが1on1
  → スクリーン＆ロールでフリーに
  → ヘルプアンドリカバーで対応
  → ボールをさらにキックアウト
  → スコアリード → ポゼッション戦術
  → パッシブプレー警告 → 強制攻撃
  → クイックフリースロー発動
  → 最初の局面に戻る

左利きエースの非対称シュート
  → 右利き前提のゾーンDFが弱い（左コーナー）
  → ハーフタイムで左コーナー担当を強化
  → 左コーナーのDFを引き付けてセンターパス → 右利きバックライト
  → スカウティングで右利きにもマーク強化
  → ダブルスクリーンでバックライトをフリーに
  → 左利きエースに戻る
```
