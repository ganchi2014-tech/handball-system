# じゃんけんマップ — 全カテゴリ横断統合（索引兼マップ）

> 「この世にある全てのハンドボールを私は認めている。相性や向き不向きがあるだけ」

---

## 核心原則

1. **絶対最強の戦術は存在しない** ── 全てにカウンターがある
2. **「先に読んだ方が選択肢を持てる」** ── 認知の速さが結果を決める
3. **「2/3閉じて1/3開ける」** ── GKの法則→DF全体に転用可能
4. **「予測の予測を予測する」** ── 軍拡競争としての戦術
5. **「数的優位は条件付きで有効」** ── 自チームの強みと合致するときのみ
6. **「コーチが言うとバイアスがかかる」** ── 辞書はAIによる全羅列

## 【OF↔DFの基本サイクル】

```
クローズDF（内側に立つ）
  → OFに外パス・外シュートの選択肢 → ワイドOF優位
  → DFがワイドに対応 → ラインが広がる
  → 内側にスペース発生 → クローズOF（内突破）優位
  → クローズDFに戻る
```

---

## 【数的状況サイクル】

```
マンツーマンDF
  → スクリーンでマーク剥がし → フリー確保
  → ゾーンDF（スペース消し）
  → 外シュートで引き出す → ゾーン間にスペース → 1on1突破
  → マンツーマンに戻る
```

---

## 【GKサイクル】（名称統一済み）

```
先読み型GK（コース予測で動く）
  → 早打ちでGKが動く前に打つ
  → コース絞り型GK（動かずコース絞り）
  → 引きつけて最後に角度で打つ
  → 先読み型GKに戻る

※常に「左・中・右」の2/3は閉じている。開いている1/3を読む勝負
```

---

## 【フェイントサイクル】（全7種＋ダブル原則）

```
DFが接触してくる（ホールディング・プッシング）
  → ①廻旋フェイント（腕を外して差し込む）
  → DFが接触をやめる → 腕が自由 → ③⑤⑥が使いやすくなる

DFが腕・手を見ている
  → ③シュートフェイント / ④パスフェイント
  → DFが骨盤を見るようになる
  → ⑤ステップフェイント / ⑥ボディフェイントが効きにくくなる
  → ②ターンフェイント（体全体動かす）or ⑦スピードフェイント（あるいはダブルフェイント原則で7種から2種連続）

GKが先読み型
  → ③シュートフェイント or ③-B バックハンドフェイク
  → GKがコース絞り型に変化
  → 引きつけてから角度変化
```

---

## 【ドリブル・打点サイクル】

```
スピードドリブルで縦突破
  → DFが体を張って縦を切る
  → 止まってシュートフェイント
  → DFが前に出る（ブロックを狙う）
  → 脇を抜けてドリブル突破

早打ち（上昇中の打点）
  → GKが最高点予測で先に動いている → 動く前に放てる
  → GKが早打ちに慣れる
  → 最高点に戻す → GKが過剰に早く動いて空く
  → GKがコース絞り型に変化 → 角度変化で打つ
```

## 【身体特性サイクル】

```
高身長ブロックDF
  → 低弾道早打ち（踏み切らない）で股下/低コーナー
  → GKが下に合わせる
  → 高跳躍ジャンプシュートで上部コーナー
  → 高身長DFのブロックに戻る

左利き右バック
  → 右利き前提のDFが弱い（左コーナーが空く）
  → DFが左コーナー担当強化
  → センターパスで右利き選手へ
  → 両方マーク強化 → ダブルスクリーン
  → 左利き右バックに戻る
```

## 【ゲームクローズサイクル】

```
大量リード → ポゼッション戦術
  → パッシブプレー警告（強制攻撃）
  → 攻撃ミス → カウンター機会
  → 点差縮まる → 負けチームが7on6
  → 空ゴールリスク → DFが引く
  → 7on6でピボットがフリー
  → ゾーン守備でコンパクト化 → 得点率低下 → 時間切れ
```

## 【ファウル損得サイクル】

```
ファウルしてでも止める（確実な失点回避）
  → 累積退場リスク上昇
  → 退場発生 → 数的不利（5on6）
  → コンパクト守備で凌ぐ
  → 退場明けで数的回復 → 通常に戻る

優先順位：
①フリースロー → ◎ 積極的に選べる
②7m → ✗ 絶対に避ける
③警告 → △ 修正シグナル
④退場 → ✗✗ いかなる場合も避ける
⑤失格 → ✗✗✗ 絶対にダメ
```

## 【DF陣形サイクル】

```
6-0ゾーン（外シュート多発）
  → 5-1（前衛1人でプレッシャー）
  → 前衛を引き付けてピボットパス（前衛裏）
  → 3-2-1（縦に対応）
  → 縦の連係でスペース作り
  → 6-0に戻る

マンツーマン

  → スイッチ対応 → ミスマッチ発生
  → ゾーンDF
  → 外シュートで引き出す
  → マンツーマンに戻る
```

## 【ポストプレーサイクル】

```
ポストがV字位置（先読み）
  → ポストOFが位置を変える
  → DFが追う → 別スペース発生
  → バックプレーヤーが突破
  → DFがバックに集中
  → ポストへの供給ルート開く
  → ポストのV字位置に戻る

ポストの4プレー（回転/フェイク/フラッシュ/スクリーン）の連携：
回転 → DFが背後を強くカバー → フェイク（逆方向）→ DFが反応 → 回転に戻る
フラッシュ → DFがフラッシュを警戒（背後集中） → スクリーンで離す → ロールアウト
```

## 【速攻サイクル】

```
速攻（アウトナンバー）
  → バックチェック（数を整える）
  → セットDF完成
  → セットOF始動
  → シュートミス or 得点
  → 相手の速攻 or 自チームGKスロー
  → 速攻に戻る

1stウェーブ（GKスロー直後2人）→ 止められたら 2ndウェーブ（3〜4人合流）→ セット移行
```

```
スクリーン利用（カットイン）
  → DFがスイッチ対応 → ミスマッチ発生
  → スクリーナーが有利な1on1
  → スクリーナーをポップアウト（外に開く）
  → 外シュート機会発生
  → DFがポップを追う → カットインのフリー復活
  → スクリーン利用に戻る
```

## 【コンビネーションサイクル】

```
ギブ＆ゴー → DFがスイッチで対応 → ミスマッチ発生
 → ミスマッチを攻める → DFがヘルプ → ヘルプが空けたスペースに3人目

ピック＆ロール → DFがハードブリッジ → ポップアウト
 → DFがポップ追う → ロール復活

ウェーブ → DFが第1波対応 → 第2波 → DFが追いつかない → 第3波で決める
```

## 【OFタイプ × 相手DF対応サイクル】

```
速攻特化型 → セットDFが整うと得点力低下 → ポゼッション切替が必要

ポゼッション型 → パッシブ警告 → 強制攻撃 → ミスからカウンター

バック中心型 → 外シュート警戒 → DFが前に出る → ピボットスペース発生

ピボット中心型 → ピボット2枚マーク → バックがフリー → バック中心型へ切替

ウィング中心型 → サイド警戒 → 中央スペース発生 → 中央突破へ切替
```

詳細マトリクスは `16_of_combinations.md`。

## 【7on6サイクル】

```
7on6発動（負けチーム終盤）
  → 数的有利でピボットへ通る
  → ボールロスト時 → 空ゴール失点リスク
  → DFは「ボールキープ」を最優先（ロスト17.7%が失点に直結）
  → 7on6解除（GK戻し）→ 通常DFへ

フォーメーション選択：
3-3（標準）/ 2-4（ピボット2枚）/ 4-2（外野4人）
→ 相手の対応を見て切替
```

## 【カテゴリ横断じゃんけんマップ（複合）】

```
高身長エースの外シュート
  → スクリーンでフリー確保
  → スカウティングで対策：スイッチDF
  → ミスマッチ → スクリーナーが1on1
  → スクリーン＆ロールでフリーに
  → ヘルプアンドリカバーで対応
  → ボールをさらにキックアウト
  → スコアリード → ポゼッション戦術
  → パッシブプレー警告 → 強制攻撃
  → クイックフリースロー発動
  → 最初の局面に戻る

左利き右バックの非対称シュート
  → 右利き前提のゾーンDFが弱い（左コーナー）
  → ハーフタイムで左コーナー担当を強化
  → 左コーナーDFを引き付けてセンターパス → 右利きバックライト
  → スカウティングで右利きにもマーク強化
  → ダブルスクリーンでバックライトをフリーに
  → 左利き右バックに戻る
```

## このマップの使い方

**試合中に思い出す順序**：
1. 自分の状況を「OFかDFか」「数的状況」「時間・スコア」で位置づける
2. 該当するサイクルを思い出す
3. 「相手は何をしている」を確認
4. その逆を取るアクションを選ぶ

**詳細が必要なときは**：上の【索引】から該当ファイルを開く。
