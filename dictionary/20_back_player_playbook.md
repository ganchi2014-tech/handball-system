# バックプレーヤー判断プレイブック — 状況別「次の一手」連鎖辞書

> **使い方**：試合中・試合直後に「いま何の状況だったか」から引く。
> 各状況には【第1選択】→【DFの対応】→【第2選択（連鎖）】を必ず記載。
> `17_scene_index.md` が**単発引き**なら、このファイルは**判断ツリー＝じゃんけん連鎖**。

> **このファイルの独自性**：他ファイルが「技術別 / カテゴリ別」整理なのに対し、ここは**バックプレーヤーが試合で直面する瞬間別**の整理。同じ技術が複数カテゴリに登場するのは意図的（試合では状況から技術を引くため）。

## 【A】DF個人の位置・姿勢

### A-1. DFがクローズ（内側）

```
【第1選択】 外シュート / サイド展開 / 外パス
【DFの対応】 ワイドに変更（外を切る）
【第2選択】 内突破 or ピボットパス
【DFの対応】 クローズに戻る
【サイクル完成】 → 最初に戻る（じゃんけん）
```

### A-2. DFがワイド（外側）

```
【第1選択】 内突破・ドリブル突破 / ピボットへの内パス
【DFの対応】 クローズに変更（内を切る）
【第2選択】 外シュート or サイド展開
【サイクル完成】 → 最初に戻る
```

### A-3. DFが前傾姿勢

```
【第1選択】 バックステップしてシュート
【DFの対応】 後傾に切り替える
【第2選択】 前への加速でドリブル突破
【補足】 前傾DF = ブロックを狙っている → シュートフェイク有効
```

### A-4. DFが後傾姿勢

```
【第1選択】 前への加速・ドリブル突破
【DFの対応】 前に出る（前傾になる）
【第2選択】 急停止からシュートフェイク → ドライブ
```

### A-5. DFが斜め構え（山側 / 谷側）

```
【第1選択】 山側フェイク → 谷側突破（DFが山側ケアに動く）
【DFの対応】 谷側ケアに変える
【第2選択】 山側突破（DFが体勢を戻しきれない）
【DFの対応】 正面構えに変える
【第3選択】 ステップフェイント or クロスオーバードリブル
詳細：01_basics.md「コンタクト技術（山側/谷側）」
```

### A-6. DFが正面構え（非対称なし）

```
【第1選択】 ステップフェイント / クロスオーバードリブル
【DFの対応】 斜め構えに変える（片側を切る）
【第2選択】 切られた側と逆へドライブ
```

## 【B】DF陣形

### B-1. 0-6（コンパクトゾーン）

```
【第1選択】 外シュートで引き出す（バック陣で連続シュート）
【DFの対応】 5-1へ変更（前衛1人で外シュート封じ）
【第2選択】 前衛を引きつけてピボットパス（前衛の裏スペース）
【DFの対応】 前衛がそれ以上飛び出すのを禁止（ヘラウスリュッケン＝前飛び出しを止める） → 0-6に戻る
【サイクル完成】 → 最初に戻る
```

### B-2. 5-1（前衛1人）

```
【第1選択】 前衛を左右に動かして疲弊させる / クロスプレーで前衛1人を翻弄
【DFの対応】 ピボットマーク強化
【第2選択】 ピボットを前衛の背後（ティーフェンラウム（深部空間））に走らせる
【DFの対応】 0-6に戻る or マンツーマンへ
【鍵】 前衛が「飛び出し中（ヘラウスリュッケン（前飛び出し））」の0.5秒以下が最大の窓
```

### B-3. 2-4（前衛2人）

```
【第1選択】 横の素早い展開で前衛2人の間を使う
【DFの対応】 前衛2人を狭める
【第2選択】 中央でピボットを使う（前衛2人の真ん中にスペース）
【補足】 ポルトガル代表が「中央のティーフェンラウム（深部空間）が広がる」と認識して使う変形（13参照）
```

### B-4. 3-2-1（縦積み上げ）

```
【第1選択】 縦のスペースを使う / ピボットで縦に切り込む
【DFの対応】 縦の連係強化
【第2選択】 横の連係が薄い → 横展開
【弱点】 全員の連動が必要 → 切り替えの瞬間が狙い目
```

### B-5. マンツーマン

```
【第1選択】 スクリーンでマーク剥がし / 1on1勝負
【DFの対応】 スイッチ対応 → ミスマッチ発生
【第2選択】 ミスマッチを攻める / バックドアカット
【DFの対応】 ヘルプ → スペースが空く
【第3選択】 ヘルプが空けたスペースに3人目（3-man play）
```

### B-6. ハイブリッド（5-1↔6-0切り替え型・ドイツ／ハンガリー）

```
【第1選択】 切り替えの瞬間（0.5〜1秒）を狙う
【ポイント】 DFが「形を変えている途中」が最大の窓
【第2選択】 規律的DFには想定外のクイックパス（パターン崩し）
【DFの対応】 形を一つに固定 → 通常陣形の対策を使う
```

### B-7. アグレッシブ6-0（フランス／ポルトガル／FCバルセロナ）

```
【第1選択】 6-0の3原則（圧縮/飛び出し/受け渡し）の完成前を突く
【3原則別の窓】
- フェアシーベン（シフト）（圧縮中）：逆サイドへ即展開
- ヘラウスリュッケン（前飛び出し）（飛び出し中）：背後にピボット走り込み
- ユーバーゲーベン（引き渡し）（受け渡し中）：クイックパスで瞬間マーク外れ
```

### B-8. バルセロナ 型4-2密集ブロック

```
【特性】 前線2人で外シュート封じ + 後衛4人でゴール前を固める
【第1選択】 1on1依存は不利 → 組織連携でImpar（浮き＝DFのマークが外れてフリーになった味方）を作る
【第2選択】 ピボット周辺の パル/インパル（偶数/奇数システム） 連鎖（スペイン式）
【鍵】 GKの先読みを外す早打ち or 引きつけ角度変化（コースを先読みして動くタイプのGK＝ニールセン型への対策）
```

## 【C】GK状況

### C-1. 先読み型GK（コース予測で先に動く）

```
【第1選択】 早打ち（上昇中の打点で放つ）
【GKの対応】 早打ちに慣れて早めに動く
【第2選択】 最高点に戻す（GKが過剰に動いてコースが空く）
【第3選択】 ③-Bバックハンドフェイク（手首を逆方向に返す）
【GKの対応】 動きを遅らせる（コース絞り型へ変化）
【サイクル完成】 → C-2へ移行
```

### C-2. コース絞り型GK（動かず2/3を塞ぐ）

```
【第1選択】 引きつけてから最後に角度変化
【第2選択】 空き2エリア（GKが塞いでいないコース）を正確に狙う
【GKの対応】 次第に先読みで塞ぎ始める（先読み型へ変化）
【サイクル完成】 → C-1へ戻る
【補足】 低めのコース（腰〜膝）が両タイプに有効
```

### C-3. GKタイプ不明（試合序盤）

```
【行動】 最初の3〜5本は「普通に打って」GKの反応を観察
【観察結果】
- 動いた → 先読み型 → C-1へ
- 動かなかった → コース絞り型 → C-2へ
- 場面で変わる → 混合型 → 場面ごとに使い分け
【保険】 序盤は低コース（腰〜膝）を選ぶと外れにくい
```

### C-4. GKが動きを見せた瞬間別

| GKの動き | 即時アクション |
|---|---|
| 左右どちらかに動いた | 反対コースへシュート |
| 前に出た | ループシュート（頭上を越す） |
| 仁王立ち（動かず低重心） | 股下シュート / バウンスシュート |
| 高く構えている | 低コーナー（腰〜膝） |
| 低く構えている | 上部コーナー |

## 【D】ポスト連携

### D-1. ピボットがフリーで6mに入っている

```
【第1選択】 即パス（DFを引きつけずに直送）
【失敗パターン】 引きつけすぎてパスコースを失う
【判断基準】 ピボットがDFの前 = 即パス、DFの裏 = 引きつけてから
```

### D-2. ピボットDFが「自分側」を強く守っている

```
【第1選択】 シュートフェイント → ピボットDFがブロックに動く → ピボットがフリー
【第2選択】 ドライブで自分が突破（ピボットDFが自分側に集中 → ピボットの逆スペースが空く）
```

### D-3. ピボットが V字位置（先読み）

```
【ピボットの動き】 OFが位置を変える → DFが追う → 別スペース発生
【自分のアクション】 そのスペースへドライブ
【鍵】 ピボットとのブリックコンタクト（目合わせ）で開始を合図
```

### D-4. ポスト4プレーの選択フロー

| DFの状態 | ポストが選ぶ動き |
|---|---|
| 背後を強くカバー | 1. 回転プレー（利き手側へ） |
| 正面から来ている | 2. フェイクプレー |
| ボールやバックに集中 | 3. フラッシュ（空きスポットに急加速） |
| マンツーマン | 4. スクリーン＆ロール |

## 【E】数的状況

### E-1. 速攻：1stウェーブ（GKスロー直後）

```
【第1選択】 左右サイドにスプリント開始（GKがボールを持ってから2秒以内）
【GKのパス選択】
- 完全フリー＋距離あり → ロングパス
- 中継経由 → ミドルパス
- DFがインターセプト狙い → バウンスパス
- 誰も走れない → GK自身がドリブル
【止められた場合】 → E-2へ
```

### E-2. 速攻：2ndウェーブ（1stが止まった後）

```
【第1選択】 中央走者がドライブしてDF2人を引きつける
【DFの対応】 中央をケア
【第2選択】 左右へはたく（角度はあるが至近距離）
【DFの対応】 両サイドをケア
【第3選択】 中央が直接シュート
【リバウンド役】 ゴール正面で詰め（シュート外れ用）
```

### E-3. 数的有利（2on1）

```
【DF位置別】
- DFが一方に寄っている → 逆へ即パス
- DFが中間 → 両者フリー → 近い方からシュート判断
```

### E-4. 数的有利（3on2）

```
【第1選択】 両サイドに広がってDF2人を引き伸ばす
【効果】 中央が空く → 真ん中の選手がシュート
【DFの対応】 中央ケアに集中 → サイドが空く
【第2選択】 サイドにパス → サイドからシュート
```

### E-5. 数的不利（5on6）

```
【方針】 「2分間を0点で凌ぐ」
【守備】 コンパクトゾーン・外シュートはGKに任せる
【攻撃】 早い攻撃禁止（カウンターリスク）／1人後方待機
```

### E-6. 7on6（GK外し・自チームが負けてる）

```
【第1選択】 ピボットへの即パスコースを探す
【コースあり】 即パス → ピボットがシュート
【コースなし】 外野でボール回し → DFを横に動かす
【絶対回避】 パスミス（GKを外しているため、奪われると無人ゴールへ決められる。その失点確率は約17.7%）
```

### E-7. 7on6対応（自チームが勝ってる）

```
【攻撃】 ロスト時に空ゴールへロングシュート（インターセプト＝空ゴール）
【守備】 コンパクトゾーンで5on6を守り切る
【絶対禁止】 7mを与える行為
```

## 【F】試合文脈

### F-1. 大量リード（5点以上・残10分以上）

```
【方針】 ポゼッション・主力温存・確実なシュートのみ
【リスク】 パッシブ警告 → 攻撃強制 → ミス → カウンター
【対応】 警告後はパス4回以内にシュートが必要なルールのため、「ピボットへの当てパス」で時間調整しながらシュート（IHF 2022〜）
```

### F-2. 同点・終盤

```
【方針】 先取点で心理的優位
【選択】 ハイテンポ + GKビッグセーブ後の即カウンター
【避ける】 ファウル誘発（退場リスク）
```

### F-3. 2点差ビハインド・残2分

```
【発動】 7on6（GK外し）の定番タイミング
【手順】 フリースロー時 or ゴールスロー後にGK交替
【鍵】 ボールロストを絶対避ける（空ゴール失点で点差拡大）
```

### F-4. ラスト30秒で負けてる

```
【方針】 無理でもシュート → リバウンド狙い
【7on6中】 確実だが得点しにくい無難なシュート（セーフシュート）より、多少無理でも決まれば得点のシュート（リスクシュート）を選ぶ（撃たずに時間切れの方が悪い）
```

### F-5. ラスト30秒で勝ってる

```
【方針】 ファウルさせながらボールキープ
【動き】 コーナーへ移動 / ピボットがボール保持 / 立ち上がりはゆっくり
```

### F-6. 連続失点3点（モメンタム逆転中）

```
【最優先】 即タイムアウト（モメンタムを切る）
【次】 選手交代（空気を変える）
【次】 DF陣形変更（パターンを崩す）
【次】 GKへの声かけ（精神的サポート）
```

### F-7. パッシブ警告が出た

```
【ルール】 4回以内にシュートが必要（IHF 2022〜、旧6回）
【選択】 ピボット当てパスを繰り返して「シュート脅威」を見せ続ける
【NG】 静止したボール回し（さらに警告が早まる）
```

## 【G】DFが見ているもの（フェイント選択）

### G-1. DFが腕・手だけを見ている

```
【第1選択】 ③シュートフェイント / ④パスフェイント（腕の動きで釣る）
【DFの対応】 骨盤を見るようになる
【第2選択】 ②ターンフェイント or ⑦スピードフェイント（体全体で偽る）
```

### G-2. DFが肩・上半身を見ている

```
【第1選択】 ⑥ボディフェイント（肩だけ傾ける）
【DFの対応】 ステップに目移り
【第2選択】 ⑤ステップフェイント
```

### G-3. DFが足の踏み込みを見ている

```
【第1選択】 ⑤ステップフェイント（踏み込みの方向を偽る）
【DFの対応】 骨盤を見るようになる
【第2選択】 ②ターン or ⑦スピードフェイント
```

### G-4. DFが骨盤まで見ている（上級）

```
【第1選択】 ②ターンフェイント（体全体を動かす）
【第2選択】 ⑦スピードフェイント（速さで見る余裕を奪う）
【補足】 これ以上の対応はダブルフェイントの原則（7種から2種連続）
```

### G-5. DFが1回のフェイントは耐える

```
【選択】 ダブルフェイントの原則を発動
【組み合わせ例】
- ④パスフェイント → ③シュートフェイント → 本シュート
- ⑤左ステップ → ⑤右ステップ → 左突破
- ②ターン → ①廻旋 → シュート
```

詳細：02_individual.md「フェイント7種＋ダブル原則」

## 【H】接触対応

### H-1. DFがホールディング（腕を掴む）

```
【第1選択】 ①廻旋フェイント（腕を円運動で外す → 差し込む）
【DFの対応】 接触をやめる
【効果】 ③⑤⑥のフェイントが使いやすくなる（腕が自由になるため）
```

### H-2. DFがプッシング（腕を押す）

```
【第1選択】 ①廻旋フェイント or 体でシールド
【補足】 押されている方向と逆の山側へドライブが最速の抜け方
```

### H-3. DFが背後から密着

```
【第1選択】 ②ターンフェイント / ロールターンドリブル
【ポストの場合】 背中をDFに当てながら一歩前に出る
```

### H-4. DFが横から押してくる

```
【第1選択】 押されている方向（谷側）と逆の山側へドライブ
【理由】 山側突破 = DFの体勢が完全に崩れる方向
```

## 【I】メンタル状態

### I-1. プレッシャーが高い

```
【症状】 視野が狭くなる / 判断が遅れる / 体が硬くなる
【対処】
① 深呼吸4回（深く吐く呼吸でリラックスを司る神経＝副交感神経が働き、心拍が下がって落ち着く）
② 視野を意識的に広げる（周辺視野復活）
③ 単純化：「第一選択を即実行」
④ インドレ・ディアログ（内的対話）：「やれることをやる」
```

### I-2. ミスをした直後（3秒ルール）

```
0-1秒：自分のミスを認識（隠さない）
1-2秒：インドレ・ディアログ（内的対話）「次は○○する」
2-3秒：トリッゲレ（誘発する）発動（深呼吸・ジェスチャー）
3秒以降：完全に切り替えて次のプレーへ
```

### I-3. 接戦終盤で焦り始めている


## 【J】ポジション特化

### J-1. 右バック（自分が右側）

```
【右利き右バック】
- 内切り（左方向へドライブ）が最強オプション → 山側/谷側の選択肢が多い
- 外シュートは半身を開く必要があり角度が狭い → 引きつけ角度シュート向き

【左利き右バック】★構造的優位
- シュートの腕（左）がゴール正面を向く → 角度が広い
- DFの標準守備（右利き前提）に左コーナーが空く
- 第1選択：左への外シュート（逆クロスコーナー）
- 第2選択：内突破 → 左ニアシュート
- 第3選択：DFが左コーナー強化 → 右利き隣バックへ展開
```

### J-2. 左バック（自分が左側）

```
【右利き左バック】★構造的優位（右バックの左利きと同じ理屈）
- シュートの腕（右）がゴール正面を向く → 右コーナーが自然
- 第1選択：右への外シュート

【左利き左バック】
- 内切り（右方向へドライブ）が最強
- 外シュートは半身 → 引きつけ角度向き
```

### J-3. センターバック（自分が中央）

```
【役割】 攻撃の組み立て役＝味方にパスを配り攻撃の起点を作る司令塔（ロシア式 ラズヴォドヤシュチ（露：プレーメーカー） / ドイツ式 シュピールマッハー（ゲームメーカー））
【第1選択】 自分のマークDF（Par＝自分に対応している相手DF）を引きつけてその場に「固定（fijar）」する（スペイン式 パル/インパル（偶数/奇数システム））
【第2選択】 ピボットへの対角パス（2:2状況）
【第3選択】 自分でシュート（GKと1on1の局面が作れる場合）
【鍵】 ボールに触れる回数が最多 → 「テンポを決める」責任
```

### J-4. ピボット（自分が6m）

```
【判断順序】
① DFはどこにいる？
   背後 → 1.回転プレー
   正面 → 2.フェイクプレー
   離れている → そのままシュート

② GKはどこ？
   動いた → 空いたコースへ
   動いてない → 引きつけて角度変化

③ フリーのバックは？
   いる → DFを引きつけてからパスアウト
   いない → シュート判断で解決
```

## このプレイブックの「使い方」

### 試合中（30秒で引く）

1. 「いま何の状況か」を A〜J のカテゴリから1つ選ぶ
2. 該当項目の【第1選択】を実行
3. DFの対応を観察 → 【第2選択】へ

### 試合直後（5分で振り返り）

1. 印象的な場面を3つ思い出す
2. それぞれを A〜J のどれかに分類
3. 自分の選択が【第1選択】通りだったか？
4. 違ったなら：選んだ選択肢の根拠は？／プレイブック通りに行くと結果はどう変わっていたか？

### 試合前（10分で確認）

- 相手の主要DFタイプ・GKタイプ・陣形を予測
- 該当する A〜J を再読
- 「最初の5分で確認すべきこと」を3つ決める
