## 【K】試合直後 5分振り返りチェックリスト

> 試合終了直後に**1回だけ**やる。長くなりすぎない。3部構成（良かった/課題/改善案）。

### Step 1：今日の試合タイプを特定（30秒）

```
□ 速攻が多い試合だった
□ ポゼッション中心の試合だった
□ 接戦終盤だった
□ 大量得点 or 大量失点
□ 自分の出場時間：　　分
```

### Step 2：認知チェック（1分）

```
ボールを受ける前の5ポイント確認はできていたか？
□ ① 自分のDFの位置・状態を見ていた
□ ② ピボットの位置を確認していた
□ ③ GKの位置を確認していた
□ ④ 隣の選手の状況を見ていた
□ ⑤ スペース（Tiefenraum）を見ていた

→ 5つ全部○ → 認知バッチリ
→ 3つ以下 → 次の練習で「受ける前の確認」を意識
```

### Step 3：判断速度チェック（1分）

```
ボールを受けてから2秒以内に判断できたか？
□ 自分が打った/パスを出した全ての場面で2秒以内
□ ほぼ2秒以内（1〜2回オーバー）
□ 何度もオーバー（3回以上）

→ オーバーが多い場合：「考えてから受ける → 受ける → 動く」に切り替える練習
```

## 【L】特定状況の即時対応リファレンス

### 自分がボールを持って固まったとき

```
Q: 何をすればいい？
A: 順番に確認：
  ① シュート打てる距離・角度か → YES：シュートフェイントを入れて打つ or ドライブ
  ② ピボットへのパスコースあるか → YES：即パス
  ③ 隣にはたけるか → YES：はたく
  ④ どれもなし → コントロールドリブルで時間を作る → 再び①へ
```

### DFがめちゃくちゃ詰めてきたとき

```
Q: 何をすればいい？
A:
  ① 接触してきたら廻旋フェイント
  ② シュートフェイントで一旦距離を作る
  ③ ピボットに即逃がす
  ④ ファウルもらえる位置なら無理せずファウル誘発
```

### GKのタイプが分からないとき

```
Q: 試合序盤、どうする？
A:
  最初の3〜5本は「普通に打って」GKの反応を観察
  → 動いた → 先読み型 → 早打ち or バックハンドフェイクへ
  → 動かなかった → コース絞り型 → 引きつけて角度変化へ
  → 状況で変わる → 混合型 → 場面ごとに使い分け
```

### ミスをしてしまった直後

```
Q: 次のプレーへどう切り替える？
A:
  → 3秒以内に切り替える（長く引きずらない）
  → Indre dialog（内的対話）：「次は○○する」と自分に言う
  → Triggere（メンタルトリガー）：握手・深呼吸・特定のジェスチャーで集中再起動
  → 次のプレーで小さな成功を作る（Mestring = 達成感）→ 自信が戻る
詳細は `18_mental.md`
```
