# 32. 試合当日マニュアル（日本高校生向け）

> 「明日大会、何時に起きて何持って行く？アップどうする？」を自分で決められるようになるための辞典。
> **設計哲学**：コーチの指示通りに動くのではなく、**自分で1日を設計する**。

---

## 【A】このファイルの使い方

試合当日は **時間刻みで判断が必要**。本ファイルは：

1. **前日準備**（B章）
2. **当日朝の活性化**（C章）
3. **会場到着後の流れ**（D章）

5. **ハーフタイムの使い方**（F章）
6. **試合終了直後の動き**（G章）
7. **連戦時の調整**（H章）

## 【B】前日準備（試合の前夜）

### 食事（19:00〜20:00）
| 食べる | 避ける |
|---|---|
| 白米・パスタ・うどん（高炭水化物） | 揚げ物・油っこい肉 |
| 鶏むね肉・魚（高タンパク・低脂質） | 生もの（食中毒リスク） |
| バナナ・スポーツドリンク | カフェイン（夜のは睡眠妨害） |
| 味噌汁・スープ（水分補給） | アルコール（高校生は当然NG） |

### 睡眠（22:00〜23:00 就寝）
- **目標 8時間以上**
- スマホ・ゲームは就寝1時間前にOFF
- ブルーライト遮断 = メラトニン分泌促進
- 寝室は暗く、室温18〜22℃
- 寝る前に **明日の試合をポジティブにイメージ**（イメージング）

### 装備チェック（前夜のうちに）
- [ ] ユニフォーム上下（背番号確認）
- [ ] インドアシューズ（紐の状態）
- [ ] 靴下2足
- [ ] タオル2枚
- [ ] スポーツドリンク・水
- [ ] テーピング・サポーター
- [ ] 携帯電話・充電器
- [ ] 着替え（試合後の通学用）

→ 詳細リスト：`31_equipment.md` 【G】試合に持って行くものリスト

### メンタル準備
- 試合プランを **頭の中で3回シミュレーション**
- 「今日の目標」を1つだけ決める（例：「フェイント7種のうち2種を試す」）
- 不安な点を **紙に書き出す → 対策を1つ書く**

## 【C】当日朝の活性化

### 起床時間の決め方
| 試合開始時刻 | 起床時刻 | 朝食時刻 |
|---|---|---|
| 9:00（朝一） | **5:30〜6:00** | 6:30 |
| 11:00 | 7:00 | 7:30 |
| 13:00（昼） | 8:00 | 9:00（軽め） |
| 15:00（午後） | 8:30 | 8:30（普通）+ 12:00（軽食） |
| 17:00（夕方） | 9:00 | 通常朝食 + 13:00昼食 + 15:00補食 |

### 朝食メニュー
**目的：消化が良く、エネルギーに変わる炭水化物中心**

#### 推奨
- おにぎり2〜3個（梅・鮭・昆布）
- バナナ1本
- 味噌汁
- スポーツドリンク 500ml
- ヨーグルト（少量）

#### NGメニュー
- 揚げ物（消化に時間がかかる）
- 牛乳大量（お腹がゆるくなる選手も）
- 食物繊維過多（試合中の便意リスク）
- 辛い物（胃刺激）

### 朝の身体活性化（試合の3〜4時間前）
1. **5分の散歩** — 軽い有酸素で血流UP
2. **動的ストレッチ 5分**（肩回し・股関節・膝）
3. **シャワー** — 38〜40℃で身体を起こす
4. **音楽 or 動画** — 集中力を高める個人ルーチン

### 移動中の過ごし方
- 携帯ゲーム → 集中力消耗、避ける
- 試合の動画レビュー → 効果的（前回試合の良かったプレー）
- イメージング → 自分の成功シーンを頭の中で再生
- 友人と雑談 → 緊張ほぐし、ただし試合の話題は控えめ

## 【D】会場到着後の流れ

### 到着時刻の目安
**試合開始の 90〜120分前** に会場到着

例：13:00試合開始 → **11:00〜11:30到着**

### 到着後のタイムライン
| 時刻（試合開始から逆算） | 行動 |
|---|---|
| **-120分** | 会場到着、受付・控室確認 |
| **-110分** | 着替え、シューズチェック |
| **-100分** | 軽い動的ストレッチ（個人） |
| **-90分** | チーム集合・コーチからのオーダー確認 |
| **-75分** | ウォームアップ開始（チーム全体） |
| **-50分** | アップシュート（GKへのシュート練習） |
| **-30分** | 戦術確認・サインプレー復習 |
| **-15分** | 個人ルーチン・集中タイム |
| **-5分** | 入場準備、整列 |
| **0分** | 試合開始 |

### 着替え時のチェック
- ユニフォーム着用
- アクセサリー全て外す（指輪・ピアス・ネックレス）
- 髪を結ぶ（女子・長髪男子）
- 爪チェック
- テーピング・サポーター装着
- シューズの紐確認

## 【E】試合直前ルーティン

### ウォームアップ構成（試合前60〜75分）
1. **動的ストレッチ 10分** — 関節可動域UP
2. **軽いランニング 5分** — 心拍数UP
3. **パスキャッチ 10分** — ボール慣れ
4. **シュート練習 15分** — 各ポジション基本シュート
5. **GKシュート 10分** — アップシュート
6. **チーム戦術確認 5〜10分** — サインプレー、約束事

### アップシュート（GKへの試合前シュート練習）
**目的**：
- GKの感覚を試合モードに上げる
- シューターの感覚調整
- チーム全体の準備度確認

**やり方**：
1. シューター列を作り、順番にGKへシュート
2. 各ポジション（バック・サイド・ピボット）から最低3本
3. GKは **本気でセーブ**（手抜きNG）
4. 軽いコース・強いコース・フェイントを混ぜる
5. 最後は **「俺は今日いける」感覚** で締める

**注意**：
- 怪我リスクを下げるため **全力すぎない**
- GKを傷つけない（顔面狙いNG）
- 床の松ヤニ確認（滑らないか）

### 試合前15分の個人ルーチン
**目的：自分のメンタルを試合モードに切り替える**

#### 個人ルーチン例
- 音楽を聴く（テンポ高めの曲）
- 深呼吸4-7-8法（4秒吸う・7秒止める・8秒吐く）×3回
- イメージング（自分の成功プレー3つを頭で再生）
- 軽いストレッチ
- チームメイトとアイコンタクト（連携の合図）

### 入場時の心構え
- 笑顔で入場（緊張をほぐす）
- 相手と握手（リスペクト）
- 国歌（試合による）は背筋を伸ばして
- キックオフ直前 = 集中の最高潮を持っていく

## 【F】ハーフタイムの使い方（10〜15分）

### 時間配分
| 時間 | 行動 |
|---|---|
| 0〜2分 | 整列退場、控室・ベンチへ |
| 2〜4分 | **水分・補食**（バナナ・エナジーバー） |
| 4〜8分 | **コーチからの修正指示** |
| 8〜10分 | **チーム内の確認**（次の20分の目標） |
| 10〜12分 | 個人準備（テーピング再装着・トイレ） |
| 12〜15分 | 入場前の集中、再アップ |

### ハーフタイム補食
**目的：後半のエネルギー切れ防止**

#### 推奨
- バナナ1本
- エナジーバー（SOYJOY等）
- スポーツドリンク300〜500ml
- 経口補水液（脱水気味なら）

#### NG
- 大量の食事（消化に時間）
- 冷たすぎる飲み物（胃を冷やす）
- 炭酸飲料（ゲップで気分悪化）

### 前半の振り返り（自分で）
3つだけ自問する：
1. **何が良かった？** — 続けるべきこと
2. **何が悪かった？** — 直すべきこと
3. **後半の目標は？** — 1つに絞る

→ コーチの指示と自分の判断を **照らし合わせる**

### 体温管理
- 涼しい会場 → ジャージ羽織って体温保持
- 暑い会場 → 冷たいタオルで首・脇冷却
- アイスベスト（あれば）で深部体温下げる

## 【G】試合終了直後の動き

### 試合直後の5分
1. 相手と握手（リスペクト）
2. 主審・副審に「ありがとうございました」
3. 自チームベンチに戻る
4. **水分補給**（500ml以上）
5. ベンチで深呼吸

### クールダウン（試合後10〜15分）
**目的：疲労物質の除去・怪我予防**

#### メニュー例
1. **軽いジョギング 5分** — ふくらはぎを動かして血流促進
2. **静的ストレッチ 10分** — 大腿四頭筋・ハムストリング・肩・腰
3. **アイシング** — 痛む箇所、または膝・足首予防的に10〜15分

### 試合後の食事（30〜60分以内）
**ゴールデンタイム**：筋肉修復に最も効果的

#### 推奨
- タンパク質（鶏肉・魚・卵・プロテインシェイク）
- 炭水化物（おにぎり・パン・パスタ）
- 水分（500ml以上）
- ビタミンC（フルーツ・ジュース）

### 試合後の振り返り（自分で）
**試合後30分以内** に書く（記憶が新鮮）

#### メモする3項目
1. 良かったプレー3つ（次回も再現）
2. 悪かったプレー3つ（次回改善）
3. 次の練習で意識すること1つ

→ 詳細フォーマット：`28_video_analysis.md` セルフスカウティング

## 【H】連戦時の調整（試合間1〜2時間）

### 高校大会では **同日に複数試合** が多い

### 試合間60分の使い方
| 時間 | 行動 |
|---|---|
| 0〜10分 | クールダウン |
| 10〜20分 | シャワー・着替え（ユニフォームは試合用キープ） |
| 20〜35分 | 食事（軽め・消化重視） |
| 35〜45分 | 横になって休息（足を上げる） |
| 45〜55分 | 動的ストレッチで再活性化 |
| 55〜60分 | 次試合のメンタル準備 |

### 試合間の食事
| 試合間時間 | 食事内容 |
|---|---|
| 30分以内 | スポーツドリンク + バナナのみ |
| 1時間 | おにぎり1個 + ドリンク + バナナ |
| 2時間 | 軽い定食（うどん・小盛りカレー） |
| 3時間以上 | 通常食（脂質控えめ） |

### 連戦疲労対策
- **アイシング** — 膝・足首を10分ずつ
- **着圧スパッツ** — 試合間も履き続けて血流促進
- **足を高くして横になる** — リンパ流促進
- **マッサージ** — ふくらはぎ・大腿を軽く揉む

### 次試合への切替メンタル
- 前試合の結果は **完全に手放す**（勝敗関わらず）
- 「次は別の試合」「ゼロからスタート」を口に出す
- 過去のミスを引きずらない = 短期記憶リセット

## 【I】大会期間中（連日試合）の生活

### 高校大会は **2〜5日連続** が普通

### 日々のルーティン
| 時間 | 行動 |
|---|---|
| 起床（試合4〜5時間前） | 朝食・装備チェック |
| 試合 | 集中・最大努力 |
| 試合終了後 | クールダウン・補食 |
| 移動・宿泊 | 着替え・休息 |
| 夕食（19:00頃） | 高炭水化物・高タンパク |
| 風呂（20:00頃） | 38〜40℃で疲労回復 |
| ストレッチ（21:00頃） | 全身10〜15分 |
| 翌日準備（21:30頃） | 装備・メンタル準備 |
| 就寝（22:00〜22:30） | 8時間睡眠確保 |

### 大会期間中の栄養
- 普段以上に **タンパク質摂取**（体重×2g/日）
- ビタミンB群（豚肉・卵）でエネルギー代謝促進
- 水分は **1日2リットル以上**
- アルコールNG（高校生だが、コーチ含めて）

### メンタル管理
- 試合のたびに切り替え（前試合を引きずらない）
- 仲間との会話で緊張ほぐす
- 個人時間も確保（音楽・読書・睡眠）
- スマホ過剰使用NG（疲労蓄積）

## 【J】よくある失敗と対策

| 失敗 | 対策 |
|---|---|
| 朝食食べすぎでお腹重い | 試合3時間前までに食べ終わる |
| 試合直前にトイレに行きたくなる | 試合30分前にトイレを済ます |
| ウォームアップで全力を出しすぎる | 試合本番のために70%で抑える |
| ハーフタイムでスマホを見て集中が切れる | スマホはロッカーに置く |
| 試合後の振り返りを忘れる | 試合バッグにメモ帳を常備 |
| 連戦で2試合目に体が動かない | 試合間に動的ストレッチ必須 |
| 大会期間中に体調を崩す | 手洗い・うがい・睡眠優先 |

## 【K】試合当日のメンタル管理

### 緊張の対処
| 状態 | 対処 |
|---|---|
| 緊張で寝られない | 深呼吸 + イメージング、最悪寝なくても試合はできる |
| 試合直前に手が震える | アドレナリンの正常反応。深呼吸で落ち着く |
| ミスを引きずる | 1プレー1リセット、過去は手放す |
| 相手が強そうで怖い | 「相手も自分を恐れている」と認識する |
| 仲間とのコミュニケーションが固い | 笑顔・声出しで雰囲気を作る |

### 試合中の集中切れ対策
- 失点後すぐに **次の1点を取りに行く** マインド
- ベンチに戻った時は **次の作戦を考える** 時間
- タイムアウト中は **聞く** に徹する（自分の意見は控えめ）

## 【L】じゃんけん：試合当日の選択
