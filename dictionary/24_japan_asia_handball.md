## 1. 日本男子代表「彗星JAPAN」概要

### ニックネームの由来
| 項目 | 内容 |
|---|---|
| 男子ニックネーム | **彗星JAPAN（Suisei Japan）** — 2018年6月公募で決定（687名、1,000案） |
| 意味 | チームのスピード・シュートの軌道・「ファンを魅了する」野心を表現 |
| 旧ニックネーム | ムササビJAPAN（モモンガ＝滑空スピード） |
| 女子ニックネーム | **織姫ジャパン（Orihime Japan）** — 七夕伝説の星の姫（天体テーマを統一） |

### 戦術アイデンティティ
- **スピードハンドボール**：速攻中心、コメットの比喩がそのまま戦術思想
- 「ディフェンス一本、守って速攻」——守備を徹底した上での高速カウンター
- 欧州と比較した場合の物理的ハンデを**認知速度・判断速度・移動速度**で補う哲学

## 2. 日本代表の戦術システム

### オフェンス基本構造

#### セットオフェンス陣形
- **1-5 / 3-3ベース**から状況に応じてポストを絡める
- **ダブルポスト**（ポスト2枚）採用時もあり

#### 主要セットプレー
```
ワンクロス・ツーストレート（One Cross, Two Straight）
  → バックプレーヤー2名がクロスプレー1本を起点に同時ストレートダッシュ
  → 片方がDFを引き付け、もう一方がフリーでシュート
  → 日本代表が教科書的に使用する得点パターン
```

#### 速攻タイプ分類（日本特有3分類）
| 名称 | 内容 | 備考 |
|---|---|---|
| **一次速攻**（いちじそくげき） | ボール奪取直後、即座にゴール前の味方へ展開 | 最高効率の得点手段「試合の流れを変える」 |
| **二次速攻**（にじそくげき） | 一次速攻が得点にならなかった後、追い上がる選手へ継続 | 一次が外れても流れを切らない |
| **逆速攻**（ぎゃくそくげき） | 相手の速攻に対するカウンター — 守備的速攻対応 | DFがボールを取りきった後に発動 |

#### カットインプレー（Cut-in プレー）
- DFラインを突破する縦突破
- スピード変化＋ステップワークでDFを置き去りにする

#### ブロックプレー
- スクリーン（ピック）を使ったバックプレーヤーのシュート機会創出
- `dictionary/12_set_plays.md` のスクリーン技術と連携

### 認知・判断の文化的核心

| 日本語概念 | 英語/解説 |
|---|---|
| **観察力と判断力** | Observation & Decision-making — ハンドボールの基本能力として最重視 |
| **瞬間の判断力** | Instantaneous decision-making — 「成功の鍵」とJHAが定義 |
| **スピード変化** | スピード variation — マークを外すための速度変化、単純なスピードではない |
| **自分の前を攻める** | "Attack the space directly in front of you" — パターン実行より決断的動作を優先 |

### 欧州用語の日本語化（指導現場で使用）
| 日本語表記 | 原語 | 意味 |
|---|---|---|
| テンポシュピール | Tempo-Spiel（独） | テンポプレー：守備組織を崩すリズム変化 |
| グループ戦術 | Gruppentaktik（独） | グループ戦術：個人技術と全体システムの中間レベル |
| ランアンドパス | Run & Pass | ノードリブル・パスドリル、日本指導の基礎 |

## 9. フィジカル差と戦術的代償策

### データ比較

| 指標 | アジア選手平均 | 欧州エリート平均 | 差 |
|---|---|---|---|
| 身長 | 約182cm | 約185cm | **−3cm** |
| 体重 | 約81.7kg | 約87kg | **−5.3kg** |

### アジアチームの対応戦略

| 戦略 | 内容 |
|---|---|
| **スピード優先哲学** | 身体的不利を速度で相殺。PMC研究でも速攻効率がチーム差の主因と確認 |
| **技術精度の追求** | 欧州型の力任せではなくフォーム・フェイント・タイミングの精緻化 |
| **欧州監督の招聘** | 日本（Dagur→Ortega→Gerona）、韓国（Freitas→Signell）、女子日本（Soubak）— 戦術体系のショートカット |
| **カタール方式** | 帰化によるフィジカル問題の完全回避（論争的だが最も成功した方法） |
| **ノードリブル文化** | 「ランアンドパス」を基礎から叩き込む→走りながらのパス精度向上 |

> **PMC（PMC3588687）の知見：** 速攻効率は同レベルチーム間での最大の成功要因。これがアジアチームのスピード優先戦略の科学的根拠。

## 12. じゃんけん構造：日本のスタイルに合うDFとは

```
日本のスタイル（速攻重視）
  ↑ 弱い：6-0ゾーン（高さでバック線シュートを封殺）
  ↑ 弱い：身体接触の圧力（欧州型3-2-1で前に出て不意を突く守備）
  ↓ 強い：5-1オープン（広いスペースで速攻活かせる）
  ↓ 強い：2-4（ポスト2枚を準備していない相手）

対戦時の参照
  → 相手が6-0ならば：ポスト起点のハイ-ロー（ポストとバックが高い位置・低い位置で連係する攻め）／速攻ではなくセットOFに切替
  → 相手が積極的プレスなら：そのプレスの裏で速攻の準備
  → 欧州チームには「先に動いてからの判断」vs「考えてから動く」の差が大きい
```
