# ポストプレー（OF・DF）

---

## ポストOFの核心

> 「そこにいるだけでDFが迷う」= 存在自体が脅威

ポストプレーヤーの最大の価値は、6mライン付近に構えることでDFの注意を引き、バックプレーヤーの突破スペースを作ること。

---

## ポストの4プレー（コーチ直伝）

### 1. 回転プレー

**体の動き**：
```
DFに背を向けた状態でボールを受ける
→ 足を軸に（利き手側の方向へ）体を回転
→ 回転しながらゴール方向を向く
→ 回転の遠心力を乗せてシュート
```
- 回転方向：**利き手側**が基本（右利きなら右回転）。逆回転はDFを最も惑わせるが難度が高い
- DFが背後から密着：回転で振り切る
- DFが横から来る：体でシールドしながら逆方向へ回転

**いつ使うか**：DFがポストの背後（ゴール方向）を強くカバーしているとき

---

### 2. フェイクプレー

**体の動き**：
```
パスを受ける → 右方向へのシュートフォーム（腕を引く）
→ DFが右方向にブロックを出す（or 重心が右へ）
→ 逆の左へシュート or 回転シュート
```
- GKとDFを同時に騙せる
- フェイクの「ため」が長いほど効果大。ただし長すぎるとファウルの隙を与える
- 先読み型GKに特に有効。コース絞り型には効きにくい

**いつ使うか**：DFが正面から来ているとき / GKが先読み型のとき

---

### 3. フラッシュプレー

**タイミングと動き**：
```
DFがボールを見た瞬間（0.3〜0.5秒の窓）
→ ポストが6mライン上の「空きスポット」に急加速で飛び込む
→ バックがその動きを見てパス → 受けてシュート
```

**飛び込む「空きスポット」の種類**：
| スポット | 飛び込む方向 | パスの種類 |
|---|---|---|
| ゴール正面（中央） | DFのブロックが薄い中央 | 速いフラットパス |
| 右コーナー（6m角） | DFが左寄りのとき | 低めのバウンスパス |
| 左コーナー（6m角） | DFが右寄りのとき | 低めのバウンスパス |

**バックとの連携サイン**：
- 言語：「来い！」「フラッシュ！」
- 動作：ポストが手を上げた → パスOK
- アイコンタクト：目が合った → パス送る

**いつ使うか**：DFがボールやバックに集中して背後のスペースが空いたとき

---

### 4. スクリーンプレー

**体の動き**：
```
ポストがバックプレーヤーのDFの進路上に立つ
→ バックプレーヤーがポストの陰から抜け出す
→ ポストはその場で静止（合法スクリーンの要件）
→ バックが突破した後、ポストがゴール方向にロールアウト
→ ロールアウト後にパスを受けてシュート（スクリーン＆ロール）
```
- ロールアウトの有無はバックとの事前サインで決める
- スクリーン詳細は `12_set_plays.md` 参照

**いつ使うか**：マンツーマンDF / セットプレー後

---

### ポストがパスを受けてからの判断フレーム

```
ボールを受けた瞬間に確認する順番：

① DFはどこにいるか？
   背後 → 回転プレー
   正面 → フェイクプレー → シュート
   離れている（スペースあり） → そのままシュート

② GKはどこにいるか？
   動いた → 空いたコースへ打つ
   動いていない（コース絞り型） → 引きつけてから角度変化

③ フリーのバックはいるか？
   いる → DFを引きつけてからパスアウト
   いない → シュート判断で解決
```

## ポストのポジション取り（DFとの争い）

```
DFはV字（ポスト〜ゴールを結ぶ線上）を維持しようとする
ポストOFはそのV字の前に入ろうとする

→ 争いに勝つ = ボールを受けやすい位置確保
→ 争いに負けても = DFが動いた分だけバックのスペースが開く
```

**ポジション取りの技術**：
- 腰を落として重心を低く保つ（押し出されにくい）
- DFが前から来た → 体を横向きにしてDFを外に逃がす
- DFが背後から密着 → 背中をDFに当てながら一歩前に出る

**揺さぶりのじゃんけん**：
```
左に動く → DFが追う → 右が空く → 右に急加速してポジションを取る
静止してDFを油断させる → 一気に動き出す（フラッシュプレーへ）
```

## ポストOFのじゃんけん

```
ポストが6mラインに構える
  → DFがポストを警戒してバックに集中できない
  → バックプレーヤーの突破スペース発生
  → DFがバックに集中
  → ポストへのパスコース開く
  → ポスト経由のシュート機会
  → DFがポスト警戒に戻る

回転プレー
  → DFが回転を予測して先に体を張る
  → フェイクプレー（逆方向フェイクで先読みを利用）
  → DFが逆フェイクを読む
  → フラッシュプレー（タイミング変化）
  → DFが回転プレーへの対応に戻る
```

## ポストDFの原則（コーチ直伝）

### V字位置取り
- ポストOFとゴールを結ぶ線上（V字の頂点）に立つ
- ポストOFより**先に**ボールを受ける位置を取る
- OFがフラッシュで飛び込んでも、DFはすでに先回りしている状態

### ポストDFのじゃんけん
```
V字位置取りでパスコースを切る
  → ポストOFが動いてV字の外に出る
  → DFが追う → 別スペースが発生（バックへの影響）
  → バックプレーヤーが突破
  → DFがバックに集中
  → ポストへのパスコース開く
  → V字位置取りに戻る
```

### ポストDFのポイント
- 常にポストOFとゴールの間に体を置く
- ポストOFの動きを先読みして先回り
- バックプレーヤーとの連携：どちらをケアするかの声かけが必須
- 接触プレーで押し込まれないよう重心を低く保つ

## トピックJ：スクリーン（ブロック）の法的限界と審判判定基準

### IHFルールにおける合法スクリーンの要件

IHFルール8:2では、相手選手（ボール非保持者を含む）を妨害・阻止・危険にさらす行為を禁止している。合法スクリーンの核心は「受動性（パッシブ）」にある。

**合法スクリーンの三条件（IHFルール8:2bに基づく）**：
```
① 静止していること（Stationary Position）
   → スクリーンを設定した後、その場に止まっていること
   → 相手の進行方向に向かって動いてはならない

② 接触を起こさないこと（No Contact）
   → 相手に自分から接触しに行かない
   → 相手がぶつかってくることは許容される（相手の過失）

③ 動的妨害をしないこと（No Active Obstruction）
   → 腕を広げて通路を塞ぐ
   → 脚を横に出してブロックする
   → これらは「能動的妨害」としてファウル
```

**IHFルール8:2の正確な位置づけ**：
- ルール8は「許可される・禁止される」身体的プレーを規定
- 8:2aでは「体を使って相手を妨害する」ことは禁止
- ただし「ボールへのチャレンジに向かう途中」の接触は別途評価される
- スクリーンは「ボールへのプレーではなく、選手へのプレー」であるため、常に受動性が求められる

### 「動く」か「止まっているか」：審判が判定する実際の基準

審判判定は「動きの意図と方向」で決まる：

| 状況 | 判定傾向 |
|---|---|
| スクリーナーが静止→DFが当たってきた | 合法（DFの責任） |
| スクリーナーがDFの方向に一歩動いた | 違反（能動的妨害） |
| スクリーナーが腕を横に広げた | 違反（空間の拡大による妨害） |
| スクリーナーが背中でDFを押した | 違反（能動的接触） |
| スクリーナーが位置取り中に動いた | グレーゾーン（速度・意図次第） |

**EHFの解釈傾向** ⚠️確認推奨：
EHFはトップリーグ（CL）において、スクリーンの「位置取り段階の動き」に対して比較的寛容な傾向があると指摘されることがある。ただし「位置取り完了後の静止」は厳格に求める。

**日本（JHA）の判定傾向** ⚠️確認推奨：
JHA公式資料における明確な公開基準は確認できなかった。コーチや審判講習での解釈が実態に近い可能性がある。国内大会では接触の強さと意図に基づいて判定されるという証言があるが、文書化されたガイドラインは未確認。

### 合法スクリーンの限界を最大限活用するための技術ポイント

```
【設定フェーズ】
① DFの予測移動ルートを先読みし、先にそのライン上に立つ
② 位置取りは「一歩早く」＝DFが動く前に静止する
③ 静止完了後は重心を下げ、足幅を広げて安定させる

【設定後フェーズ（静止段階）】
④ 腕は体の前で交差か体側に密着（広げない）
⑤ 視線をボールに向ける（審判への「意図の見せ方」）
⑥ DFが接触してきたら吸収する（倒れない＝安定したスクリーン）

【ロールアウトフェーズ】
⑦ バックプレーヤーが通過した直後にゴール方向へロール
⑧ ロールは「DFとの接触から離れる方向」が合法
⑨ ロール後のポジションでパス要求（スクリーン＆ロール完成）
```

**スクリーンの「幅」を合法的に最大化する方法**：
- 肩幅＋わずかに足を広げた状態が最大合法幅
- 足の向きをスクリーン方向に対して垂直に立つ（体の断面を最大化）
- これにより接触なしで最大の「壁」を作れる

## トピックK：重心移動の物理（小柄ピボットが大柄DFを動かせる理由）

### 重心の高さと安定性の物理：支持基底面（Support Polygon）の概念

**安定性の基本方程式**：
```
安定性 ∝ 支持基底面の広さ × (1 / 重心の高さ)

→ 重心が低いほど安定
→ 足幅が広いほど安定
→ 相手の重心が高ければ、小さな力でも不安定にできる
```

**支持基底面（Support Polygon）とは**：
- 両足が地面に接する点を結んだ多角形の領域
- 重心の垂直投影がこの領域内にある間は安定
- 投影がはみ出た瞬間に倒れる（または踏み出す）

**フットボール・ラグビーとの共通知見**：
フットボール選手はタックル時に腰を落とし、足幅を広げ、重心を前方かつ低く保つ。これにより支持基底面が広がり、相手から「押し出されにくく」なる。同じ原理がピボットのポジション取りに直結する。

### 慣性モーメント（Moment of Inertia）：体が大きいほど方向転換が遅い理由

**慣性モーメントの基本**：
```
慣性モーメント（I）= Σ（質量 × 回転軸からの距離²）

→ 体が大きい（質量が多い・手足が長い）ほど I が大きい
→ I が大きいほど回転を開始・停止するのに多くの力が必要
→ 大柄なDFは方向転換の開始が遅れる
```

**スポーツバイオメカニクスの知見**：
大きな筋肉と長い力腕はトルクを増やすが、重く長い手足は慣性モーメントを増大させる。結果として、大柄なアスリートは絶対的なパワーは大きいが、「向き直り」「急停止からの再加速」は小柄なアスリートより遅い。

**ピボットにとっての意味**：
```
小柄ピボット（I が小さい）
  → 素早い回転・方向転換が可能
  → DFが反応する前に向きを変えられる

大柄DF（I が大きい）
  → 方向転換の慣性が大きい
  → 一度動かしたら止めにくい（慣性の活用）
  → 「ついてきた方向」に誘導してから逆を突ける
```

### てこの原理：低重心からの水平力が大きな相手を動かせる仕組み

**てこの原理（柔道の知見から）**：
柔道における研究が示す基本原理：
- 低い位置から相手を崩す（崩し＝Kuzushi）のが最も効率的
- 相手が押してくる力（例えば7の力）に逆らわず、その向きに合わせて自分も引く力（例えば3の力）を加えると、相手の力と自分の力が同じ向きにそろって合わさり、大きな力（例の場合10の力）として働く
- 相手の重心をその「支持基底面の外」に移動させれば、相手の体重が武器になる

```
【てこの原理のピボット応用】

支点 = 自分の腰（低く安定した位置）
力点 = 相手のDFの体（腰〜肩の高さ）
作用点 = DFの重心

→ 低重心からの水平・斜め方向の押し出しは
  「てこの力点を長くする」ことと同義
→ 自分の体重 + てこの増幅 で実際の体格差を縮小できる
```

### 柔道・ラグビーからの応用知見

**柔道（崩し＝Kuzushi）**：
崩しの基本は「相手の支持基底面の外に重心を移動させること」。直接の力比べではなく、相手の構造的安定性を壊すことで、体格差を超えた投げが可能になる。

**ラグビー（スクラム・コンタクト）**：
ラグビーのコンタクト理論では「低い姿勢からのヒットで高い姿勢の相手を押せる」ことが基本原則。接触の瞬間に低重心を維持した選手が有利になる。

### ピボット実践への落とし込み

```
腰を落とす = 重心を下げる = 安定性増大

具体的な動作原則：
① ボールを持つ前：膝を曲げ、腰の高さをDFの腰より低く保つ
② DFと接触する瞬間：瞬間的にさらに腰を落とし、支持基底面を広げる
③ ポジション争い中：重心を水平方向に移動させてDFの慣性を利用
④ 「揺さぶり」の原理：左→右への素早い体重移動で
   DFの「方向転換の慣性」が追いつかない状態を作る

【身長差への対応】
DFが高い場合、その「重心も高い」ことを利用する：
- 低重心からの水平力は高重心の相手を横に動かしやすい
- DFの腰・骨盤付近を支点として引き出し、支持基底面の外に誘導する
- この原理により「体重60kgのピボット」でも
  「体重80kgのDF」のポジションを動かすことが可能
```

⚠️確認推奨：ここで示したバイオメカニクスの数値（7の力→10の合力など）は柔道の原理説明から引用した概念的な比喩であり、ハンドボール競技の実測データではない。

## トピックL：「選択肢を意図的に減らす」逆説的コミットメント戦術

### ゲーム理論のコミットメント：「自分の手を縛る」ことで相手の読みを固定させる逆説

ゲーム理論における「コミットメント戦略」の核心命題：

```
【逆説的コミットメントの命題】
「選択肢を持つこと」は常に有利ではない
→ 相手が自分の行動を予測できないとき、相手も読みを固定できず
  「最適なリアクション」を取ろうとして迷う
→ しかし「自分の行動を宣言・固定してしまう」ことで
  相手に「その行動前提で動く」ことを強制できる
```

**コミットメント戦略が有効になる条件（ゲーム理論）**：
1. コミットメントが**観察可能（Observable）**：相手に見えること
2. コミットメントが**不可逆（Irreversible）**：相手が「覆せない」と認識すること
3. これにより相手は「そのコミットメントを前提とした最適行動」しか取れなくなる

**ハンドボールへの翻訳**：
```
ピボットが「必ずここにいる」と宣言する
→ DFはそのポジションを守らざるを得ない
→ DFが「このポジションを守る」ことにコミットすると
  バックプレーヤーの突破経路が固定（予測可能）になる
→ バックがDFの動きを読んで裏を取れる
```

### 選択の逆説（Paradox of Choice）：DFに選択肢が多すぎると反応が遅れる

**ヒックの法則（Hick's Law）**：
```
反応時間 = 処理速度 × log₂(選択肢の数 n)

→ 選択肢が2倍になるたびに反応時間が一定量増加する
→ DFが「右にも行けるし左にも行ける」と感じるほど
  初動の判断が遅れる
```

**「選択の逆説」（Barry Schwartz）のスポーツ応用**：
- 選択肢が多い状況では「決定疲労（Decision Fatigue）」が発生する
- DFに「Aとも読めるし、Bとも読める」状況を作るほど、DFの反応は遅くなる
- コーチング現場では「選択肢を減らすことで選手の意思決定を速める」という手法が確立されている

**ピボットへの応用**：
```
DFに対して「2択の脅威」を常に維持する：

ケース1：「シュートを打つか、パスを捌くか」
  → DFが「どちらかを捨てる」まで初動が遅れる

ケース2：「右に回転するか、左に回転するか」
  → DFがどちらかにコミットした瞬間、逆を突く

ケース3：「自分が打つか、パスして外から打たせるか」
  → DFがマークを絞れない状態を作る
```

### フランスのフィクサシオン（Fixation）理論の延長：「そこにいることを宣言する」ピボット

**フランスの Fixation du demi-centre 理論**：
フランスのハンドボール指導体系では「フィクサシオン（固定化）」が重要な概念。バックセンターの動きがDFを「釘付け（固定）」にすることで、周辺のスペースが生まれる。

```
【ピボットのフィクサシオン的コミットメント】

ピボットが6mライン上の特定位置に「確定的に」構える
  → DFは「あのピボットはそこから動かない」と認識
  → DFはそのポジションを常にケアしなければならない
  → ケアするために常にピボット方向に意識の一部を使う
  → バックプレーヤーへの集中度が下がる
  → バックの突破成功率が上昇
```

⚠️確認推奨：「Fixation du demi-centre」はフランスのコーチング文書に確認できる戦術概念だが、それをピボットに直接適用した公式理論として文書化されたものは未確認。

### 予測可能であることを逆用する具体的な戦術設計

**「あえて予測可能にする」ことで相手を縛る戦術**：

```
ケース A：必ず右から入るピボット

前半3〜4回：右から入る動きを見せる
  → DFの脳が「右から来る」パターンを学習する
  → DFが右を優先カバーし始める

重要な局面：逆（左）からのフラッシュ
  → DFの「右を先に守ろう」という先読みが逆効果になる

核心：パターンを「見せる」のも意図的な戦術
```

**「お墨付きのプレー」として機能させる**：
- 同じプレーを繰り返すことで、DFに「このピボットはこれをやる」という確信を持たせる
- その確信を使い、「確信に反するプレー」のタイミングを狙う
- これはランダム性ではなく「意図的なパターン構築→意図的な裏切り」

### 「あえて1択」と「多択の揺さぶり」をどう使い分けるか

```
【局面別の使い分けフレーム】

①試合序盤（情報収集フェーズ）
  → 多択の揺さぶり
  → DFに「このピボットはどう動くかわからない」という不安を植える
  → 同時にDFのリアクションパターンを観察する

②試合中盤（パターン構築フェーズ）
  → あえて1択を数回繰り返す
  → DFに「このプレーをやる」という確信を持たせる

③勝負所（パターン破壊フェーズ）
  → 構築したパターンと逆の行動でDFの先読みを裏切る
  → DFが「1択と思っていた」状態で多択の選択肢を使う

【スコア・時間との連動】
  → 接戦の後半：あえて1択コミットメントで「確実なポジション取り」
  → リードして時間を使うとき：1択に固定してDFを疲弊させる
  → ビハインドで攻めるとき：多択揺さぶりでDFの反応時間を削る
```

**「あえて予測可能」と「意図的なアンプレディクタビリティ」のじゃんけん**：
```
DF：先読みで対応
  → ピボット：あえて予測可能にしてDFの先読みを固定 → 逆を突く
  → DF：先読みをやめてリアクション型に
  → ピボット：多択の揺さぶりで反応時間を遅らせる
  → DF：再び先読みに戻る
```

⚠️確認推奨：ヒックの法則の数値（反応時間の具体的な増分）はボタン押しの実験データに基づいており、実際のスポーツ場面での適用可能性には「事前のトレーニングによるパターン認識」が大きく影響するという研究も存在する。数値を絶対視せず、概念として活用することを推奨。
