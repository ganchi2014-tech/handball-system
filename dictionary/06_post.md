# ポストプレー（OF・DF）

---

## ポストOFの核心

> 「そこにいるだけでDFが迷う」= 存在自体が脅威

ポストプレーヤーの最大の価値は、6mライン付近に構えることでDFの注意を引き、バックプレーヤーの突破スペースを作ること。

---

## ポストの4プレー（コーチ直伝）

### 1. 回転プレー
- DFに背中を向けた状態からパスを受ける
- 体を回転させてゴール方向を向きながらシュート
- DFが背後をカバーしきれない角度への攻撃
- 回転方向：利き手側に回転するのが基本

### 2. フェイクプレー
- パスを受けた瞬間に逆方向へのフェイク動作
- DFが反応して体重移動したところで本来の方向へシュート
- GKも釣ることができる（DFとGKを同時に惑わせる）

### 3. フラッシュプレー
- 6mライン付近に**瞬間的に**飛び込んでパスを受ける
- DFが追い付く前にシュート
- タイミングが命：パスを出す側との連携が必要
- DFが油断した瞬間（ボールを見ている瞬間）に飛び込む

### 4. スクリーンプレー
- ポストがDFに壁を作り、バックプレーヤーの突破を助ける
- ポスト自身はシュートを打たない（壁役に徹する）
- バックプレーヤーが突破した後、ポストがロールアウトしてフリーになる可能性

---

## ポストOFのじゃんけん

```
ポストが6mラインに構える
  → DFがポストを警戒してバックに集中できない
  → バックプレーヤーの突破スペース発生
  → DFがバックに集中
  → ポストへのパスコース開く
  → ポスト経由のシュート機会
  → DFがポスト警戒に戻る

回転プレー
  → DFが回転を予測して先に体を張る
  → フェイクプレー（逆方向フェイクで先読みを利用）
  → DFが逆フェイクを読む
  → フラッシュプレー（タイミング変化）
  → DFが回転プレーへの対応に戻る
```

---

## ポストDFの原則（コーチ直伝）

### V字位置取り
- ポストOFとゴールを結ぶ線上（V字の頂点）に立つ
- ポストOFより**先に**ボールを受ける位置を取る
- OFがフラッシュで飛び込んでも、DFはすでに先回りしている状態

### ポストDFのじゃんけん
```
V字位置取りでパスコースを切る
  → ポストOFが動いてV字の外に出る
  → DFが追う → 別スペースが発生（バックへの影響）
  → バックプレーヤーが突破
  → DFがバックに集中
  → ポストへのパスコース開く
  → V字位置取りに戻る
```

### ポストDFのポイント
- 常にポストOFとゴールの間に体を置く
- ポストOFの動きを先読みして先回り
- バックプレーヤーとの連携：どちらをケアするかの声かけが必須
- 接触プレーで押し込まれないよう重心を低く保つ
