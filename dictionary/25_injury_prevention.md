## 【A】バックプレーヤーの傷害リスク

```
【最重要リスク】肩の過負荷傷害
  肩の過負荷傷害報告者の89%がバックコートプレーヤー
  成人プロ選手で投球肩の過負荷傷害有病率：26〜28%

原因：
  1試合で最大100本以上の投擲動作の繰り返し
  外旋筋力低下（decreased external rotation strength）
  肩甲骨機能異常（Skapuladyskinesie / scapular dyskinesis）
  間接コンタクト傷害（indirect contact）：31.5%

【第2リスク】膝の過負荷傷害（膝蓋腱炎）
  ジャンプ・加速・減速の繰り返しによる腱の過負荷反応
  12〜14歳の男子に多い

【第3リスク】足関節捻挫
  再受傷率：40〜70%（非常に高い）→ 二次予防が重要
```

## 【B】ウォームアッププログラム

### エビデンス

```
15件の研究のメタ解析（PMC 2022）：
  体系的なウォームアッププログラムで傷害発生率36%削減（pooled injury rate ratio = 0.64）

ノルウェー（skadefri.no）のプログラム：
  急性膝・足関節傷害を最大50%削減可能
```

### B-1. DHB公式：M.A.P.S.プログラム（ドイツハンドボール連盟 2024）

**4段階の体系的ウォームアップ。毎練習前15〜20分で実施。**

**M = Mobilisieren（モビライゼーション）**
```
関節・キネティックチェーンを能動的に全可動域で動かす
関節粘性の向上、機械受容器・ゴルジ受容器の刺激
例：股関節・胸椎・肩甲骨のモビリティドリル
```

**A = アクティヴィーレン（活性化）（アクティベーション）**
```
筋の柔軟性・血流・神経支配・代謝の活性化
伸張耐性の向上
例：グルート・ハムストリングス・コアの活性化エクササイズ
```

**P = Potenzieren（ポテンシエーション）**
```
神経筋システムの興奮性向上
最大動作速度でのサイクリック・非サイクリック動作、プライオメトリクス
例：スプリント・ジャンプ・方向転換
```

**S = Spezifizieren（スペシフィケーション）**
```
競技動作への直接的な準備
ハンドボール特異的な動作・戦術的アクション
例：シュート動作・ステップ・ポジション別プレー
```

### B-2. VBG「ハンドボールの4×4（Das 4×4 des ハンドボールs）」

**VBG（ドイツ職業保険組合）開発のハンドボール特化型予防トレーニング。**

```
4つの柱：
1. Stabilisation und Kräftigung（安定化と強化）
2. Mobilisation und ベヴェークリヒカイト（柔軟性）（モビライゼーションと柔軟性）
3. Schulterkontrolle（肩コントロール）
4. 脚軸制御と着地技術（下肢アライメントと着地技術）

実施方法：各領域から3〜4種目を選んでウォームアップに組み込む
収録：100種目以上。器材少なく実施可能。
```

### B-3. ノルウェー「Skadefri Håndball」（怪我のないハンドボール）

```
NHF（ノルウェーハンドボール連盟）が推奨する包括的プログラム
肩・膝・足関節にフォーカス（ハンドボール傷害の50%超がこの3部位）
ウォームアップ15〜20分：ボール使用可、バランスボード活用
週3回以上実施推奨
URL：https://www.skadefri.no/idretter/handball/
```

## 【C】肩傷害予防（バックプレーヤー最重要）

### C-1. OSTRC（オスロスポーツ外傷研究センター）ショルダーコントロールプログラム

**Oslo スポーツ Trauma Research Center（ノルウェー）開発。最大のエビデンスを持つ肩予防プログラム。**

```
エビデンス（RCT・45チーム660名のエリートハンドボール選手）：
  介入群の肩傷害有病率：17%
  対照群：23%
  → 28%削減

実施方法：週3回・10分間・ウォームアップとして組み込む
```

**5つの目的と具体的エクササイズ（チューブ/エラスティックバンド使用）：**

**① 外旋筋力強化（肩関節（グレノヒューメラル） external rotation strength）**

```
エクササイズA（90°外転位）：
  肩を90°外転、肘を90°屈曲した立位
  チューブを外旋方向に引く
  同心性：1秒 / 遠心性（戻り）：2秒
  3セット×10回、セット間休憩30秒

エクササイズB（中間位）：
  肩を体側に保ち、肘90°屈曲
  外旋方向へチューブを引く
  3セット×10回
  ノルウェー語：Skulderrotasjon bakover med strikk
```

**② 肩甲骨筋コントロール（Scapular muscle control）**
```
Yレイズ（Y Raise）：下部僧帽筋・回旋筋腱板・肩甲骨安定筋群
バンド外旋（バンド エクスターナル ローテーション）：低負荷・高反復。毎日実施可
```

**③ 肩関節内旋可動域改善（肩関節（グレノヒューメラル） internal rotation ROM）**
```
後部関節包ストレッチ（cross-body stretch）：30秒×3回
スリーパーストレッチ（横向き内旋ストレッチ）
```

**④ 運動連鎖活性化（Kinetic chain activation）**
```
体幹安定化（Rumpfstabilisation / コア stabilization）
腰椎・骨盤のコントロールが肩への過負荷を軽減
```

**⑤ 胸椎モビリティ（Thoracic mobility / Brysthvirvelmobilitet）**
```
胸椎回旋ストレッチ
フォームローラーを使った胸椎伸展
→ 胸椎の硬さ＝肩の代償運動増加につながる
```

**注意：** 単独のエクセントリック外旋エクササイズ8週間のみでは効果不十分。多要素プログラムが推奨。

### C-2. 肩予防の追加ポイント（VBG Schulterkontrolle）

```
筋不均衡（Muskelimbalanz）の修正：投球側の一側性過負荷由来
肩甲骨スタビライゼーション（Schulterblatt-Stabilisierung）
制御された動作パターン（kontrollierte Bewegungsmuster）
→ 投球負荷への段階的適応
```

## 【D】膝傷害予防

### D-1. ACL（前十字靭帯）傷害予防

```
ACL = 前十字靭帯 / Vorderes クロイツ（クロス）band (VKB) / Fremre korsbånd

リスクプロファイル：
  女性ハンドボール選手（15〜19歳）が特に高リスク
  カットプレー・着地・急停止の際に発生
  ACL再建術後2年以内の再受傷率：20〜30%

ノルウェー女子ハンドボールの実績：
  神経筋トレーニング（NMT）の導入でACL傷害50%削減を達成
```

**ニー（膝） コントロールプログラム（スウェーデン開発・RCT有効性証明）：**

```
6種の主要エクササイズ（各A〜D難易度レベル）：
1. One-legged knee squat（片脚膝スクワット）
2. Pelvic lift（骨盤挙上）
3. Two-legged knee squat（両脚膝スクワット）
4. The bench（ベンチ）
5. The lunge（ランジ）
6. Jumping/landing technique（ジャンプ・着地技術）

プログレッション：週1回ずつ難易度を上げる
頻度：週3回
```

**追加エクササイズ：**
```
ノルディック/Russian ハムストリング Curl（ハムストリングス遠心性強化）
Single-leg squat（シングルレッグスクワット）
Plank variations（プランク）
バランスボード練習
```

### D-2. 膝蓋腱炎（ジャンパー膝）予防

```
Jumper's ニー（膝） / Patellarsehnentendinopathie / Hopperkneet

予防・管理エクササイズ：
  Heavy Slow Resistance（HSR）トレーニング：ゆっくりとした高負荷トレーニング
  Eccentric Decline スクワット（遠心性ディクラインスクワット）：ゴールドスタンダード

プログレッション：
  Leg press → スクワット → Hack squat → Split squat → Seated knee extension

研究（思春期ハンドボール選手でのRCT）：
  機能的高負荷エクサイズで競技シーズン中の腱痛有病率を有意に削減
```

## 【E】足関節捻挫予防

```
ハンドボール最多の急性傷害（22.2%）
再受傷率：40〜70%（非常に高い）→ 一度捻挫したら二次予防が必須
```

### 予防の3本柱

**① バランス・神経筋トレーニング（proprioceptive training）**
```
バランスボード・ウォブルボード使用
片脚立ち・ジャンプ着地訓練
再発リスクを約50%削減
→ 捻挫歴ありの選手に最も有効
```

**② テーピング・ブレーシング（Taping / バンデージ（包帯））**
```
捻挫歴あり選手：テーピング・ブレーシングで再発リスク50〜70%削減
捻挫歴なし選手への予防効果は限定的
テープ vs. ブレース：同等の効果

バディテープ（Buddy Taping / パートner-Taping）：
  非弾性スポーツテープで隣接指同士を固定
  締め付けすぎず、緩くない張力で
  指の捻挫・脱臼予防にも活用
```

**③ 着地技術（Landetechnik / Landing technique）**
```
VBG 4×4の「脚軸制御と着地技術」が直接対応
着地時の膝内側への崩れ（knee valgus）を防ぐ意識
→ 足関節・膝の両方を守る
```

## 【F】成長期（ユース・高校生）の過負荷傷害予防

### F-1. 成長期特有のリスク

**Osgood-Schlatter病（オスグッド・シュラッター病）**
```
脛骨粗面の成長板への反復牽引ストレス
発症年齢：男子10〜15歳・女子8〜13歳
発症率：スポーツ選手の13〜17%
症状：脛骨上部の痛み・腫れ

予防：
  急激なトレーニング量増加を避ける（10%ルール参照）
  大腿四頭筋のストレッチ強化
  症状が出たら負荷を下げる
```

**Sever病（セーバー病）**
```
踵骨骨端炎（成長板）
好発年齢：8〜12歳（女子が早い傾向）
症状：踵の痛み

予防：
  踵への反復衝撃を管理
  下腿三頭筋のストレッチ
  クッション性の高い靴・インソール
```

### F-2. 予防戦略（成長期共通）

```
① 10%ルール：週あたり練習量の増加を10%以内に抑える
② スポーツ多様化：複数スポーツへの参加推奨（特に12歳以下）
   → 特定動作の反復過負荷を防ぐ
③ 休息期の設定：競技シーズン後に積極的休息期（active rest）
④ 栄養管理：カルシウム・ビタミンD不足は骨ストレス傷害リスクを増加
⑤ 痛みのシグナルを無視しない：
   「痛みをこらえてプレー」は悪化・慢性化・ストレス骨折を招く

トレーニング負荷モニタリング（Load Monitoring）：
  RPE（主観的運動強度）で週・月単位の練習量を管理
  推奨目安（21_physical_training参照）：週sRPE 1,000〜1,400 AU以内
```

## 【G】クールダウンとリカバリー

### G-1. クールダウン（練習・試合後5〜15分）

```
静的ストレッチング（static stretching）：
  ※練習前ウォームアップには不適。練習・試合後に実施。
  推奨：各ストレッチ2回×75秒
  対象：大腿四頭筋・ハムストリングス・股関節・肩・胸椎

フォームローリング（Faszienrolle / Foam Rolling）：
  20〜30秒間、硬い部位で一時停止
  筋の硬直軽減、可動域改善
  筋痛の軽減にも有効
```

### G-2. リカバリー手段（エビデンスレベル別）

**高エビデンス：**

```
Cold Water Immersion / Eisbad（氷浴）：
  水温10〜15℃（50〜59°F）
  10〜15分間浸水（3分間の分割でも可）
  効果：筋痛軽減・腫脹抑制

Compression Garments / Kompressionskleidung（コンプレッションウェア）：
  代謝廃棄物の除去促進、腫脹抑制

Foam Rolling（フォームローリング）：
  筋の硬直軽減、可動域改善
```

**その他：**
```
積極的回復（active recovery）：
  軽い有酸素運動（サイクリング・スイミング）
  血流改善・代謝老廃物除去

間欠的空気圧迫（Intermittent Pneumatic Compression）：
  多競技大会での有効性を示すエビデンスあり
```

## 【H】睡眠・栄養・水分補給

### H-1. 睡眠（Schlaf / Søvn / Sleep）

```
推奨睡眠時間（思春期アスリート）：1夜8〜10時間
  ※現実平均：約6.3時間（目標と大きくかけ離れている）

研究エビデンス：
  8時間未満で傷害リスク1.7倍上昇
  平日8時間以上確保した選手：新規傷害のodds ratioが61%低下

睡眠不足の影響：
  コルチゾール増加 → 筋タンパク分解促進
  グリコーゲン補充障害
  炎症性サイトカイン上昇 → 結合組織の脆弱化
  運動性疼痛・疲労の主観的増大
```

### H-2. 栄養（Ernährung / Ernæring / Nutrition）

```
炭水化物（Kohlenhydrate）：
  推奨量：6〜10g/体重kg/日（練習1〜3時間/日の場合）
  最重要エネルギー源。認知・身体パフォーマンスの制限因子

タンパク質（Protein）：
  推奨量：1.2〜2.0g/体重kg/日
  繰り返しの筋損傷回復に必須

脂質：
  総エネルギーの25〜30%以内

タイミング：
  試合・練習後2時間以内：高糖質＋良質タンパク質の食事
  → グリコーゲン回復・筋タンパク合成促進
```

### H-3. 水分補給（Flüssigkeitszufuhr / Hydration）

```
試合後：失った体重の150%相当を2時間以内に補給
  例：体重1kg減少 → 1.5L補給
電解質（特にナトリウム）の補充が重要
試合中：6〜12%糖質電解質飲料が推奨（環境・個人差に応じて調整）
```

## 【I】復帰プロトコル（リターン-to-プレー / Spielwiederaufnahme）

### 一般原則：段階的5ステップ

```
1. 組織治癒期（tissue healing）：
   痛みは軽減するが組織は未修復

2. 運動機能回復：
   患側の筋力は急速に低下（受傷後でも20〜30%の筋力差が残存）

3. スポーツ特異的動作の再習得

4. 全体チーム練習への復帰

5. 競技復帰（return to competition）

重要：疼痛消失 ≠ 組織治癒完了
  性急な復帰が再受傷の最大リスク
```

### 部位別の目安

| 傷害 | 復帰目安 | 注意事項 |
|---|---|---|
| 肩（SLAP修復後） | 12〜16週 | 段階的なスローイングプログラムが必須 |
| ACL再建後 | **平均7.7ヶ月** | 若年選手ほど再受傷リスク高（20〜30%/2年） |
| 足関節捻挫 | 重症度による | 復帰後もブレーシング＋バランストレーニング継続 |
| 膝蓋腱炎 | 症状管理次第 | 痛みがないことを確認しながら段階的に負荷増加 |

## じゃんけん構造（ケガ予防）

```
トレーニング量を増やす
  → 傷害リスクが上がる（特に成長期）
  → 体系的ウォームアップで36〜50%削減できる
  → ウォームアップをしない
  → 傷害リスクが戻る

肩を酷使する（投球反復）
  → 外旋筋力低下・肩甲骨機能異常
  → OSTRC（オスロスポーツ外傷研究センター）プログラムで28%削減できる
  → 睡眠・栄養が不足
  → 組織回復が遅れて傷害悪化

捻挫した
  → 再受傷率40〜70%（高い）
  → バランストレーニング＋ブレーシングで50〜70%削減
  → 急いで復帰する
  → 再受傷する

→ 絶対安全はない。リスク管理の継続が唯一の答え。
```
