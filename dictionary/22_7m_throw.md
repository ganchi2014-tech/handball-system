> 出典：IHF ルール 2025・EHF 用語集 2025・DHB・RFEBM・PMC査読論文（複数）

## 【基本定義】

```
7メートルスロー
  日本語：7メートルスロー
  独：ジーベン（独：7）meter（ジーベンメーター）
  英：7-metre throw / penalty throw
  西：lanzamiento de 7 metros
  諾：strafekast（ストラフカスト）
  丁：straffekast（ストラフカスト）

重要原則（ドイツ語圏の教育で強調）：
「Ein ジーベン（独：7）meter ist keine Bestrafung」
（7メートルスローは罰則ではない）
→ 反則への制裁ではなく「奪われた得点機会の回復手段」として位置付けられる
```

## 【A】IHF規則（第14条 / Rule 14 / Regel 14）

### A-1. 与えられる条件

```
以下のいずれかに該当するとき、審判は7メートルスローを宣告する：

① 明確な得点機会（klare Torchance / clear chance of scoring）が
  相手の反則行為によって妨げられた場合

② ゴールエリア内で守備側が違反行為を行い、明確な得点機会を阻止した場合

③ 守備側以外の人物（交代席・観客など）が故意に妨害した場合

④ GKがゴールエリア外に出て不正に妨害した場合

重要：「明確な得点機会」の解釈はIHFガイドライン補足事項No.6に詳細規定。
  攻撃側がボールと身体を完全にコントロールしている場合、
  違反があっても7メートルにならない場合がある（その後得点できなくても）。
```

### A-2. 実施規則

| 規則 | 内容 |
|---|---|
| 制限時間 | 審判の笛の後、シューターは**3秒以内**に投じる |
| シューターの位置 | 7メートルラインの背後0〜1メートル以内。ライン上は不可 |
| **ジャンプスロー禁止** | **少なくとも一方の足が地面に触れていること**（通常フィールドと大きく異なる重要規則）|
| フェイントの許可 | ボディフェイント（身体動作による欺き）は合法。ただしGKが体勢を取った後に意図的に中断するのは不正 |
| GKの位置 | **4メートルライン（Torwartgrenzlinie）より前に出てはならない**。ボールが離れるまで |
| 他選手の位置 | 9メートルラインの外側、シューターから3メートル以上離れること |

### A-3. 2025年7月施行の規則改正

```
【頭部接触ルール変更（Rule 8:8d）】
  改正前：シューターがGKの頭部に当てた場合 → 退場（rote Karte / 赤カード）
  改正後：GKが意図的にボールの方向へ頭を動かした場合を除き → 2分間退場（Zeitstrafe）に軽減

【数学的決着済みの場合】
  すでに勝敗が数学的に確定している場合、7メートルスローを継続しなくてよい
```

## 【C】シューター技術（Wurfvarianten / Técnica de ランサミエント（シュート））

### C-1. 主要な4シュートタイプ

**① スタンディングスロー（シュタントヴルフ（立ち打ちシュート） / Standing Shot）**
```
両足を地面につけたまま投じる基本形
安定性が高く、コントロール重視
7メートルスローで最も一般的な選択肢
フェイント（フィンテ（フェイント））との組み合わせが有効
```

**② ロブショット（ロブヴルフ（ループシュート） / Lob Shot）**
```
ボールに比較的急な放物線軌道をつける
GKが前傾姿勢・前方に出てきた場合に効果的
スペイン女子代表 Carmen Martín（カルメン・マルティン）が名手
心理戦：GKを前に引き出してからロブを使う高度な判断が必要
```

**③ バウンスショット（アウフゼッツァー（バウンドシュート） / Bounce Shot）**
```
フロアにバウンドさせてからゴールに入れる
GKの手や足の下を通す目的
研究：7mスローの4分の1以上でバウンスシュート or ボディフェイントが使われる
ドイツ・スペインコーチ教材で重要技法として位置付けられる
```

**④ スピンショット（ドレーヴルフ（回転シュート） / Spin Shot / Screwshot）**
```
ボールに横回転（sidespin）をかけてバウンド後に方向を変える高度技法
Uwe Gensheimer（ウーヴェ・ゲンスハイマー）・Rune Dahmke（ルーネ・ダムケ）らが使用
ウィングシュートで多用される技法のペナルティ応用版
```

### C-2. 生体力学的原理

```
運動連鎖（Kinetic Chain）：
  近位部から遠位部への順次動作（proximal-to-distal sequencing）が最大球速を生む

ボール速度：
  7メートルスローの最大球速：約20 m/s（時速72 km）
  ゴールまでの飛行時間：0.35秒以下

速度-精度トレードオフ（スピード-Accuracy Trade-Off）：
  7mではGKの存在が球速と精度のバランスに影響する
  エキスパート選手はこのトレードオフを示さない
  → 初心者はGKがいると速度を上げるが精度が落ちる
  → ジュニア以上では有意差なし（訓練で克服可能）

ジャンプスロー禁止の意味：
  通常の試合では74%のシュートでジャンプスローが使われる
  → 7メートルではこれが禁止されるため、スタンディング技術とフェイント技術が特に重要
```

## 【D】シューター戦略（Shooter Strategy）

### D-1. 2つの基本戦略

**戦略①：GK非依存型（ゴールキーパー-Independent Strategy）**
```
事前に狙いを決め、GKの動きを無視して強さと精度を最大化
高速シュートで角（corner）を正確に狙う
上段コーナー（クロスバー直下）への強烈なシュートは最難防御ゾーン
→ GKタイプが読めない時・精神的余裕がない時に有効
```

**戦略②：GK依存型（ゴールキーパー-Dependent Strategy）**
```
GKの動き出しを読み、その逆を突く
GKが早めに動いた場合 → ロブや逆サイドを狙う
GKが静止している場合 → フェイントで誘ってから投じる
→ 高い観察力と判断力が必要。GKを事前に観察しておくことが前提
```

### D-2. コーナー・高さ選択

```
スペイン研究（Universidad de サラゴサ 2017）の知見：

得点になりやすい条件：
  ・左利き選手による高いアーム位置からのスロー
  ・強くて速い直線軌道またはバウンドシュート
  ・フェイントなし（直接投じる）
  ・GKがシューターから距離を取っている場合
  ・GKが「閉じた姿勢（closed posture）」のとき

左利き優位の研究（PMC 2024）：
  左利きシューターはGKにとって有意に予測しにくい
  GKは左利きへの予測精度が低い
  → 視線の当て方ではなく、左利きの動きパターンの視覚解釈が困難なことが原因
```

### D-3. GKタイプ別の対応

```
早期動作型（アンティツィパツィオン（先読み）型 / アンティシペーター（先読み型））：
  ボール離手前から動く → フェイントに弱い
  → 対応：スタンディングスローで引き付けてから逆サイドへ
         ロブで動いた方向の上を越す
         フェイント後に動き出しを確認してから投げる

反応型（Reaktion型 / Reactor）：
  ボール離手後に方向確認 → 物理的に間に合わない可能性大
  → 対応：フェイントはあまり効かない
         強速シュートで角を狙う
         コース絞りで力押し
```

## 【E】GKの7メートル防御技術

### E-1. 根本的ジレンマ

```
ボール飛行時間：0.35秒以下
人間の反応時間：約0.2〜0.3秒

→ 純粋な「見てから動く」反応では物理的に防御不可能

エリートGKの結論：
  予測的な動作開始（anticipatory movement initiation）が必須
  しかしフェイントに騙されるリスクが生じる
  → このジレンマが7メートルをハンドボール最大の心理戦場面にする
```

### E-2. 基本ポジション（グルントシュテルング（基本姿勢） / Versetzter Ausgangsposition）

```
膝を軽く曲げ、重心は前足部（拇指球）に
両腕を体側から少し離し、手のひらをシューターに向ける
ゴールラインから1〜1.5メートル前に立ち、4メートルラインまでの空間を活用
→ シューターの角度を狭める（Winkelverkleinerung / angle closing）

オフセット基本姿勢（Versetzter Ausgangsposition）：
  体をシューターの利き腕側の角度を優先してカバーする斜め構え
  → 特定コーナーへのシュートを誘発しながら、反対側をカバーする戦術的配置
```

### E-3. タイミング戦略

| タイプ | ドイツ語 | 内容 | リスク |
|---|---|---|---|
| 早期動作型 | アンティツィパツィオン（先読み） | 身体キューを読んでボール離手前に動く | フェイントに弱い |
| 反応型 | Reaktion | ボール離手後に方向確認してから動く | 物理的に間に合わない |
| 混合型 | Kombiniert | キューを読みながら動作開始を微調整 | 高い経験と練習が必要 |

### E-4. 視線戦略（科学的根拠）

```
研究（PMC 2025・エリートGK40名対象）：

男性GKの視線：上腕・前腕・ボールを主に観察
女性GKの視線：胴体・頭部を主に観察
→ 異なる戦略で同程度の防御成功率を達成

クワイエット・アイ（クワイエット Eye）：
  有効な防御動作では、無効な動作に比べてクワイエット・アイ時間が有意に長い
  男性：平均差122.83 / 女性：平均差92.26
  → 一点を静止して見る時間が長いほど防御成功率が高い

エキスパートGKは非専門家より正確に方向予測できる（特に右利きシューターに対して）
左利きへの対応：GKの予測精度が有意に低い（視覚パターン認識の問題）
```

### E-5. 読み取る身体的キュー（Visual Cues）

```
優先順位順：
① 肩と腕の方向（上腕・前腕の向き） ← 最重要
② 体幹の傾き・回転方向
③ 手首の角度とグリップ
④ アプローチの速度とリズム
⑤ 足の向き・踏み出しの角度
⑥ 目線（ただし経験者は意図的に外す）
```

### E-6. GKの心理的プレッシャー戦術

```
合法的な戦術：
  ・ゴールラインからの前進（4メートルラインまで）→ 角度を狭める
  ・「見せかけの傾き（Scheinbewegung）」でシューターの決断を誘発
  ・ポジション変更・大袈裟な動作でシューターの注意を分散
  ・ボール位置を確認するふりなどの遅延戦術（Verzögerungstaktik）

注意：
  GKはシューターの投げる動作を開始した後に不正な動作で妨害してはならない

スポーツ心理学的優位性：
  GKはペナルティで「負け役（expected loser）」として位置付けられる
  → 入れられても「想定通り」。精神的プレッシャーはシューターより軽い
  → この非対称性を認識し、積極的かつ大胆に動くことが奨励される
```

## 【F】心理的側面（Psychologische Aspekte）

### F-1. シューター側の心理管理

```
7メートルスローは技術的条件が固定されながら心理的プレッシャーが最大化される稀有な場面

コントロールすべき4要素：
  注意集中 / 覚醒レベル / 感情反応 / 運動実行

統計的傾向：
  男性選手：ファイナル5分間で成功率が下がる傾向（平均 -1.4pp）
  女性選手：ファイナル5分間で成功率が上がる傾向（+2.7pp）
```

**プレショット・ルーティン（Pre-shot Routine / Vorbereitungsroutine）**
```
一貫したルーティンを持つ選手はプレッシャー下でより安定したパフォーマンスを発揮
例：ボールを3回ドリブル → 深呼吸 → 視線をコーナーへ → 投げる
→ 毎回同じ動作で「集中モード」を条件反射化する
```

**セルフトーク（Self-talk / Selbstgespräch）**
```
インストラクショナル型：「右上角、強く」など技術的な言葉
モチベーショナル型：「できる」「落ち着け」など自信強化
→ 両方が科学的に有効と証明されている
```

**チョーキング vs Yips（違いを知る）**
```
チョーキング（Choking under Pressure）：
  高プレッシャー時の不安による運動パフォーマンス低下
  正しい動作は認識できるが身体が思い通りに動かない
  → 訓練とルーティンで対処可能

Yips（イップス）：
  ストレスで誘発される制御不能な不随意運動
  チョーキングより深刻な状態
  → 専門的なメンタルサポートが必要
```

### F-2. メンタルイメージ（Imagery / Mentale Vorstellung）

```
成功シュートの視覚化が集中力・自信に貢献（IHF/EHFコーチング教材より）
実践方法：
  投げる前に「成功するシーン」を0.5〜1秒間頭に描く
  視覚だけでなく、ボールの感触・音・歓声も含める
  失敗するシーンは描かない
```

## 【G】タクティカルファウル（Taktisches Foul）

### G-1. 戦略的に7mを与えることの判断

```
防御側が意図的に反則を犯して7メートルを相手に与える戦略が存在する

有利になる条件（7mを与える価値がある状況）：
  ・相手が数的優位の速攻に出ており、防がなければ「確実なゴール」になる場合
  ・7メートル成功率（〜75%）が相手の速攻成功率より低い場合
  ・残り時間が少なく、次の攻撃機会を確保したい場合

リスク計算式：
  P(相手速攻成功率) × ゴール価値
  > P(7m成功率≈75%) × 1ゴール + 退場ペナルティの期待コスト
  → この式が成立するときのみタクティカルファウルが合理的

注意点：
  反則の態様によっては退場（rote Karte）が加わり数的不利になる
  7m + 退場 = 二重不利
  残り30秒以内：Rule 8:10dにより故意の妨害は自動的に7m + 退場処分になる場合あり
```

## 【H】トレーニング方法

### H-1. シューターのプレッシャー練習

```
ゲーム状況のシミュレーション：
  試合開始直後・タイムアウト直後・大量ビハインド・残り30秒
  など具体的な状況設定で練習する

連続7mシュート：
  成功・失敗に関わらず次のシュートへの心理的リセットを習慣化
  → 「次の1本だけ」の意識（18_mental.mdのインドレ・ディアログ（内的対話）参照）

プレッシャー環境の設定：
  チームメイトの観衆の前で打つ
  成功/失敗に対するゲームルールの導入（失敗したらダッシュなど）

射法の習得順序：
  1. スタンディングスロー（基本）
  2. バウンスショット
  3. ロブショット
  4. フェイント付きスタンディング
  5. GKタイプ別の対応
```

### H-2. GKの防御練習

```
視覚キュー訓練：
  ビデオ分析でシューターの身体部位への注意集中を訓練
  実地：上腕→前腕→ボールの順に視線を移す練習

クワイエット・アイ訓練（クワイエット Eye トレーニング）：
  一点を静止して見る時間を意識的に長くする練習
  → ホッケーGKで有効性が証明。ハンドボールでも研究中

動作開始タイミング訓練：
  早すぎず（フェイントに弱い）遅すぎない（間に合わない）バランスを訓練
  フェイント付き・なしを混在させた練習が有効

左利きシューターへの対応：
  左利き選手との反復練習で視覚パターン認識を向上させる
```

## じゃんけん構造（7メートル）

```
シューター：GK非依存型（強速コーナー）
  → GK：事前に角度を絞って角を狭める
  → シューター：ロブや逆コーナーを選ぶ

シューター：フェイント型（GK依存型）
  → GK：早期動作をやめて待つ（Reaktion型）
  → シューター：強速シュートで押し切る

GK：積極的前進（角度絞り）
  → シューター：ロブショットが有効
  → GK：前進を控えて待つ
  → シューター：角への強速シュートが有効

→ 7メートルに「絶対に入る」方法も「絶対に防ぐ」方法も存在しない
→ 相手の特性・試合状況・心理状態によって最適解が変わる
```
