## 【グローバルトレンド 2025】

### IHF 2025世界選手権の記録
- **64.0点/試合**（男子史上最高記録）
- 平均攻撃時間：**31.7秒/攻撃**（高速化の証拠）
- デンマークの平均得失点差：**+13.4**（記録的な支配力）

### 7on6（GK外し）戦術の実態データ（2020-2023研究）
- **使用タイミング**：主に試合終盤、小差でビハインドのときに発動
- **空ゴール失点率**：17.7%（= ロングシュートで簡単には入らない）
- **内訳**：シュートなし攻撃での失点40.9% / 不成功シュート後の失点19.7% / 得点シュートの失点5.9%
- **実用示唆**：「ボールを持ち続けさえすれば空ゴール失点リスクは低い」。問題はボールロスト
- 2016年ルール改正で2ピボット配置、5+1など多様な7on6フォーメーションが登場

### じゃんけん（グローバル）
```
ゾーン守備（コンパクト0-6） → 外シュートを打たれ続ける → 1-5に変更
→ 前衛の裏をピボットで使われる → ゾーンに戻る

守備優位（低失点チーム） → 速攻で少ない得点を守る → ポゼッション攻撃に弱い
→ 相手が時間を使うポゼッション → カウンターチャンスが減る → 守備型チームが不利に

7on6（GK外し） → コンパクトゾーンで守られる → ピボット2枚配置でゾーン崩す
→ 空ゴールへのロングシュートカウンター → 7on6を早めに解除 → 得点機会が減る
```

## 【デンマーク】 ★★★ 現在の世界最強

### 基本データ
- 2024年パリ五輪金メダル（決勝39-26でドイツに勝利 = 五輪決勝史上最大差）
- 2025年世界選手権優勝（平均得失点差+13.4 = 記録的）
- 失点平均22点/試合（世界最少水準）
- 直近28試合で26勝

### 監督：Nikolaj Jacobsen
- 哲学：「チームダイナミクスを作り、各選手が自分の役割を理解すること」
- 戦術的にコード化されたシステム（サインプレーが高度に体系化）
- 選手の自由を大切にする（「オフコートで楽しめる空気が頭をクリアにする」）

### 戦術的特徴
**攻撃：**
- **クイックスローオフ**：得点直後に素早く再開。相手守備が整う前に攻撃開始
- 高い得点力（スローオフから即時攻撃で得点）
- 全選手が高スコアに貢献（エース依存ではなく集団得点）

**守備：**
- 22点/試合という記録的な失点の少なさ
- コレクティブDF（集団的組織守備）が核心
- 特定のDF陣形より「全員が役割を果たすシステム」が強み

### じゃんけん（デンマーク対策）
```
クイックスローオフ → 相手が整う前に攻撃 → DF未完成の隙を突く
→ 相手がスローオフを急ぎ始める → DF準備が不十分のまま守備に入る
→ カウンターDFとして数人を残す → 速攻が出にくくなる → セット攻撃中心に戻る
```

## 【フランス】 ★★★ école française（フランス流）

### 基本データ
- 2024年EHF EURO金メダル
- 2025年世界選手権3位（怪我者続出の中）
- Guillaume Gille監督（2024 EURO金後、2026年2月退任 → 後任就任）

### 戦術的特徴
**école française（フランス流）の核心：**
- 戦術的な複合性：攻守の役割を単純化しない
- 各選手が複数の役割を担える多機能性
- 高い個人技＋集団的組織の融合

**攻撃：**
- ダイナミックなピボット活用
- バックプレーヤーが内外を使い分ける
- セットプレーのバリエーションが豊富

**守備：**
- アグレッシブな6-0ゾーン
- 前線プレッシャーとゾーン守備の組み合わせ

### 人材育成
- フランス国内リーグ（Starligue）が高い競争環境を提供
- 若い才能のクラブ育成から代表への流れが確立
- 練習哲学：試合状況に近い条件での練習（エコロジカルアプローチ）

## 【アイスランド】 ★★★ 小国モデルの教科書

### 基本データ
- 人口約37万人で継続的な国際競争力を維持
- 2008年北京五輪銀メダル（歴史的快挙）
- Snorri Steinn Guðjónsson監督（元国際選手、257キャップ）
- 2026年EHF EURO準決勝進出（強豪と互角）

### なぜ小国で強いのか？

**組織的基盤：**
- 人口の**28%**がスポーツ協会会員
- 12歳の子供の**80%以上**が放課後スポーツプログラムに参加
- 子供の**90%以上**がいつかは組織スポーツを経験

**コーチング輸出大国：**
- アイスランド人コーチが欧州各国の代表・クラブを指導
- 2026年EHF EUROの準決勝4チームのうち**3チームがアイスランド人監督**
- 東欧（ポーランド・ロシア）からの外部コーチ招聘 → 技術・戦術水準を上げて独自発展

**戦術的特徴：**
- 規律あるゾーン守備 → 速攻（カウンター）のサイクル
- 大国に5〜10点差をつけられても育成が続く「長期目線」

### 示唆（小規模チームへの応用）
```
人数が少ない ≠ 弱い
組織力×コーチングの質×長期育成 = 大国超え
全員がスポーツ文化の中にいる環境が基盤
```

## 【GKシステムの国際比較】

### 共通の現代トレンド
- ハンドボールの高速化→GKが**至近距離の1on1場面に頻繁に対処**する必要
- 手や腕より**フットワークと立ち位置が優先**（足が合ってから手）
- 各ポジション（左/右/中）ごとの「セットポジション」の反復練習

### 国別のGK育成傾向
| 国 | 傾向 |
|---|---|
| デンマーク・フランス・ドイツ・ルーマニア | 13〜14歳からDF・GKシステムを系統的に指導（比較研究あり） |
| アイスランド | 幼少期からの高い競技参加率が底上げ |
| 全欧州 | コーチ教育の体系化（IHFコーチシンポジウムを活用） |

### 技術的焦点
1. **基本姿勢**：ゴールを守る角度を常に維持
2. **フットワーク**：ポジション移動は足から始まる
3. **ゾーン対応**：左/中/右（2/3閉じ・1/3開け）は国際共通
4. **早打ち対応**：シューターの最高点に先取りする癖を逆用される（コーチ知識と一致）
