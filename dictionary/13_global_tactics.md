## 【グローバルトレンド 2025】

### IHF 2025世界選手権の記録
- **64.0点/試合**（男子史上最高記録）
- 平均攻撃時間：**31.7秒/攻撃**（高速化の証拠）
- デンマークの平均得失点差：**+13.4**（記録的な支配力）
- ポルトガル4位（自国史上最高）、フランス3位

### 7on6（GK外し）戦術の実態データ（2020-2023研究）
- **使用タイミング**：主に試合終盤、小差でビハインドのときに発動
- **空ゴール失点率**：17.7%（= ロングシュートで簡単には入らない）
- **内訳**：シュートなし攻撃での失点40.9% / 不成功シュート後の失点19.7% / 得点シュートの失点5.9%
- **実用示唆**：「ボールを持ち続けさえすれば空ゴール失点リスクは低い」。問題はボールロスト
- 2016年ルール改正後：2ピボット配置、5+1等の多様な7on6フォーメーションが登場

### じゃんけん（グローバル）
```
ゾーン守備（コンパクト0-6） → 外シュートを打たれ続ける → 1-5に変更
→ 前衛の裏をピボットで使われる → ゾーンに戻る

守備優位（低失点チーム） → 速攻で少ない得点を守る → ポゼッション攻撃に弱い
→ 相手が時間を使うポゼッション → カウンターチャンスが減る → 守備型チームが不利に

7on6（GK外し） → コンパクトゾーンで守られる → ピボット2枚配置でゾーン崩す
→ 空ゴールへのロングシュートカウンター → 7on6を早めに解除 → 得点機会が減る
```

## 【デンマーク】 コレクティブDF + クイックスローオフ

### 戦術的特徴
| 局面 | 特徴 |
|---|---|
| OF | **クイックスローオフ**（得点直後の素早い再開）が軸。エース依存ではない集団得点 |
| DF | コレクティブDF（集団的組織守備）。失点22点/試合の記録的少なさ |
| サインプレー | コード化されたサインプレーが体系化されている |
| 強み | 「特定の陣形」より「全員が役割を果たすシステム」 |

### じゃんけん（デンマーク対策）
- クイックスローオフ → 数人をDFに残して速攻カウンターを準備
- 整ったDFを崩す → 外シュート+ピボットの2軸で同時に守れない状況を作る
- コード化システム → 同じ陣形で対応せず、毎回少しずつ違うパターンを見せて読みを外す

## 【フランス】 école française（多機能性 + アグレッシブDF）

### 戦術的特徴
| 局面 | 特徴 |
|---|---|
| OF | ダイナミックなピボット活用、バックプレーヤーが内外を使い分け、セットプレーのバリエーション豊富 |
| DF | アグレッシブな6-0ゾーン、前線プレッシャーとゾーン守備の組み合わせ |
| 多機能性 | 各選手が複数の役割を担う（左利き右バックの伝統育成あり） |

### じゃんけん（フランス対策）
- アグレッシブ6-0 → 6-0の3原則（圧縮/飛び出し/受け渡し）の完成前を突く
- 多機能選手 → マンツーマンで追わず、ゾーンで領域を渡す（個別マークが追い切れない）
- 左利き右バック → 右ブロックDFで対応（シュートコースを左側から塞ぐ）

- フィジカル勝負を避ける → 接触対決ではなく速さと位置取りで勝負

|---|---|

### じゃんけん（スペイン対策）

- 「読み」が強い → シンプルな単発アクション（連続シュートフェイント等）で読みを混乱させる

|---|---|

### じゃんけん（クロアチア対策）

## 【スロベニア】 革新的（5on5戦術）

### 戦術的特徴
| 局面 | 特徴 |
|---|---|
| OF | 高速トランジション、流動的なポジション |
| DF | コンパクト守備 |
| 革新性 | **6on5でもあえて5on5を維持**する判断（Zorman流） |

### Zorman流5on5戦術（重要な戦術原則）
- 通常6on5（ライン選手投入で数的優位）の場面で**あえて5on5を維持**
- 狙い：数的有利の混乱より、速い切り替えと守備のコンパクト性を優先
- 結論：「**数的優位は条件付きでのみ有効**」（自チームの強みと合致するときのみ）

### じゃんけん（スロベニア対策）
- 高速トランジション → バックチェック必須
- 流動的ポジション → マンツーマン気味の対応が無効化される → ゾーンで対応
- 5on5志向 → 数的優位を作っても効果が薄い → 質的優位（個人技差）で勝負

## 【GKシステムの国際比較】

### 共通の現代トレンド
- ハンドボールの高速化 → GKが至近距離の1on1場面に頻繁に対処する必要
- 手や腕より**フットワークと立ち位置が優先**（足が合ってから手を出す）
- 各ポジション（左/右/中）ごとの「セットポジション」の反復練習
- シューターの最高点に先取りする癖が「早打ちの標的」になる（コーチ知識と完全一致）

### GKコーチ養成
- IHFが2段階のGKコーチライセンス制度を整備：
  - Lv.1：ユース年代向けGKコーチ育成
  - Lv.2：シニア（パフォーマンス志向）GKコーチ育成

### 国別育成傾向
| 国 | GK育成の特記事項 |
|---|---|
| デンマーク・フランス・ドイツ・ルーマニア | 13-14歳からDF・GKシステムを体系的に指導（比較研究あり） |
| アイスランド | 全国的な高い競技参加率が底上げ。コーチ育成も輸出水準 |
| ノルウェー | 専門家チーム（フィジカル・タクティクス・メンタル）が年代別にGKも指導 |

## 【辞書への応用：じゃんけんマップ追記】

### 国際水準から逆算した原則
```
「失点22点/試合（デンマーク）」が示すもの
  → 守備の質 × 組織力 = 得点より失点管理が勝敗を決める
  → 日本のバックプレーヤーが学ぶべきは「守備から速攻へのスイッチ精度」

「64点/試合（2025世界選手権平均）」が示すもの
  → 試合は高速化。1秒のズレが致命的
  → 「認知→判断→行動」の速度が全ての前提

「7on6の空ゴール失点17.7%」が示すもの
  → ボールを保持し続けることがリスク管理の本質
  → ロストが最大の敵

「5on5を選んだZorman」が示すもの
  → 数的有利は必ずしも最善手ではない
  → トランジションの速度と組織の安定を優先する選択肢がある

「アイスランドコーチ現象」が示すもの
  → 戦術は選手より「コーチ」を通じて伝播する
  → 一人の優れたコーチが複数の国を強くする連鎖が生まれる
```

## 【デンマーク補足：コードワード戦術（タイムアウト指示の体系化）】

タイムアウトで短いコードワードを使い、全員に攻撃意図を一瞬で共有する仕組み。日本のチームでも応用可能。

### ポジション番号体系
- **1's** = ウィング（両翼）
- **2's** = バック（両サイドバック）
- **3's** = 中央のインナー

### 主要コードワードと動きの内容
| コード | 動きの内容 | 目的 |
|---|---|---|
| **Barca** | センターバックが左へ動く → 右バックにパス → 左バックがセンターバックとクロスして中央を走り抜けてボールを受ける | バックが1on1状況を作る。複数の選択肢を内包 |
| **Billund** | ウィングがボールを持ち、弧を描いて右バックと相手中央DFの間のスペースへ走り込む | ランニングレーン（走るルート）の指定 |
| **det brede punkt** | 選手がベンチ側サイドライン際からスタート、相手バックとウィングの間を攻撃 | DFの横を大きく使って間を割く |

### 応用：自チームでコードワード化する手順
1. 主要パターンに2文字以内の名前を付ける（例：「クロス」「ウィング走り」）
2. タイムアウトと試合中の声でその名前だけを叫ぶ
3. 全員がパターンを把握していることが前提 → 練習で繰り返し定着させる

### じゃんけん（コードワード戦術への対処）
- DF側：ビデオスカウティングでパターンを解析する
- OF側：パターンを2〜3にとどめず、各パターンに「分岐」を作って読まれにくくする

## 【6-0守備の3基本動作（ドイツ語一次情報）】

> 出典：handballtraining.de、Wikibooks Handball Taktik、VIBSS

### ドイツ語技術用語（世界標準）

| 用語（独） | 読み | 意味 |
|---|---|---|
| **Verschieben** | フェルシーベン | ボール側へのライン全体シフト |
| **Herausrücken** | ヘラウスリューケン | インナーDFが前に飛び出すプレッシャー |
| **Übergeben** | ユーバーゲーベン | 飛び出した後の隣へのマーク受け渡し |
| **Tiefenraum** | ティーフェンラウム | ゴール前の危険な深さスペース（これを最小化するのが6-0の目的） |
| **Spielmacher** | シュピールマッハー | ゲームメーカー＝センタープレーヤー（5-1が狙う対象） |

### 6-0の3動作サイクル

```
① Verschieben（シフト）
   ボールが右に渡る → ライン全体が右へシフト
   → ボールサイドに密集 / 逆サイドに空間が生まれる（リスク）

② Herausrücken（飛び出し）
   インナーDFがボールを持つバックに前進プレッシャー
   → シュートコース圧迫 / パスコースを切る
   → 飛び出した背後にTiefenraumが生まれる（最大のリスク）

③ Übergeben（受け渡し）
   飛び出したDFが戻れない場合 → 隣のDFにマークを渡してラインを維持
   → 受け渡し失敗 → 穴が生まれる（攻撃のチャンス）
```

### じゃんけん（6-0の3動作を崩す）

```
Verschieben → 逆サイドへの素早い展開でスペースを使う
Herausrücken → 飛び出した背後にピボットを走り込ませる
Übergeben → 受け渡しの一瞬にクイックパス → 瞬間的マーク外れを突く
```

### 5-1の本質（ドイツ語原典から）
> 「5:1はHerausrückenをシステム化したもの。前衛1人が常に相手のSpielmacher（センター）を制限する。特に相手センターが強力な場合に有効」

## 【スペイン補足：トランジション構造化の原則】

### 1stウェーブ/2ndウェーブの体系化
速攻を「第1波（1〜2人）」と「第2波（3〜4人合流）」に明確に分離し、全選手が役割を把握する。

| 局面 | 1stウェーブ | 2ndウェーブ |
|---|---|---|
| 出発タイミング | ボール奪取/GKキャッチ即時 | 1stウェーブが減速し始めた瞬間 |
| 人数 | 1〜2人 | 3〜4人 |
| 役割 | スピードで先に走る・ロングパス受け | クイック攻撃またはセット移行の準備 |
| 判断基準 | GK外しか低速か（時間圧力下の即決） | 1stが決められなかったら即セットアップ |

### トランジションの判断訓練
- 「プレッシャー下でも正確な選択ができる」よう、時間制限付きの判断練習を体系化
- 相手の選手交代のタイミング・パターンを読んで戦術を調整（後半の疲労交代を利用）

### じゃんけん（トランジション構造化への対処）
- 1stウェーブ → 守→攻の切り替え時、ハーフコート付近にバックチェック要員を残しておく
- 2ndウェーブ → 1stが防げた直後の「2波目」を警戒（攻撃が止まったと油断しない）

## 【IHF分析：2025年世界選手権から読む戦術トレンド】

> 出典：IHF World Handball Magazine Tech（2025年世界選手権分析号）
> 分析担当：Dietrich Späte（IHF CCM委員長）、Jorge Dueñas（元スペイン・ブラジル・スロバキア代表監督）

### ① 近距離突破（Breakthrough）が最高の得点効率
- **ゴール至近距離へのブレークスルー（突破）が一貫して最も高い成功率**を示した
- センタープレーヤーのクローズレンジシュートエリアへの侵入が重要
- 示唆：パスを回すよりもゴールに向かって突破する判断が有効

```
外シュート（ミドル） → 確率は低い
ブレークスルー（近距離突破） → 確率が最も高い ← 積極的に選ぶべき
ペナルティ誘発 → 確率は高いがリスクも高い
```

### ② ライン選手なし攻撃（イタリア）
- **イタリアが2025年世界選手権でライン選手（ピボット）なしのポジショナル攻撃を実施**
- 一般常識（ピボット必須）を覆す予想外の戦術
- 分析：ライン選手がいないことで守備側の混乱を引き出すとともに、バックプレーヤーとウィングの組み合わせが増加
- **逆説的示唆**：ライン選手の「存在しない場合の脅威」が証明された

### ③ 7on6の質的向上
- 7on6戦術は**頻度が下がったが、実行の精度・質が向上**
- 単純な数的優位（7vs6）より、**2ピボット配置（7vs6 with 2 line players）や5+1フォーメーション**など高度な形が浸透
- 空ゴール失点を防ぎながら得点する技術が洗練されてきた

### ④ GKのロングシュート対応改善
- 遠距離シュートに対するGKの反応・成功率が向上傾向
- 示唆：**外シュートだけで勝てる時代は終わりに近い** → 近距離突破・ピボット活用の相対価値が上がる

### ⑤ 守備vs各種ポジショナル攻撃のアダプテーション
- 守備側が各チームのポジショナル攻撃に対して独自の「開口部（タクティカルオープニング）」を見つけ対応
- 攻撃の多様化に守備が追いついてきている → 双方の進化が続く「軍拡競争」

## 【スロベニア補足：5on5戦術と「数的優位の条件付き有効性」】

### 5on5維持の戦術的根拠
**問題提起**：なぜ6on5（数的優位）を使わないのか？
- 6on5（ファウルで得た1人多い状況）→ 整ったセット攻撃 → スロベニアの「速さ」の強みが消える
- **5on5を維持**（あえて数的優位を使わず素早く攻める）→ 相手DFが整う前に得点

> 自チームの強みが「速さ」なら、数的優位より「速さ」を優先する判断。
> 原則：「**数的優位は自チームの強みと合致するときのみ有効**」

### 7on6（GK外し）の実行品質
7on6は決断だけでなく、チームの準備（事前パターン）が必須。準備が不十分なまま発動するとボールロスト→空ゴール失点に直結する。

### じゃんけん（スロベニア戦術への対応）
- 高速トランジション → 得点後に素早く守備陣形に戻る（相手の「速さ」を消す）
- セット攻撃での対決に持ち込む → スロベニアは個人技×組織が融合
- ゴール前での強引な決断を迫る展開に持ち込む（決定力が課題のチームへの対策）

## 【6-0の3原則 — 多言語対照（戦術用語の普遍性）】

**「6-0の3原則は世界共通の発見」**：独語・露語で完全に同じ概念が体系化されている。同じ動作を異なる言語で呼んでいるだけ。

| 動作 | 内容 | 独語 | 露語 |
|---|---|---|---|
| 圧縮 | ボール側にラインが横移動 | **Verschieben** | **Сжатие** (Szhatie) |
| 飛び出し | 前線へ飛び出してプレッシャー | **Herausrücken** | **Выход** (Vykhod) |
| 受け渡し | マークを隣に渡す | **Übergeben** | **Передача** (Peredacha) |

### 3原則を逆手に取る攻め方
| DFの動作中 | OFの攻め方 |
|---|---|
| 圧縮中 | 逆サイドへのクイック展開（圧縮の完成前） |
| 飛び出し中 | 飛び出したDFの背後をピボットで使う |
| 受け渡し中 | 受け渡しの一瞬（0.5秒以下）にクイックパス |

→ どの動作も「完成前」「直後」が攻めどき。完成された状態を正面から崩そうとしない。

## 【左利き右バックのグローバル戦術】

### なぜ右バックは左利きが有利か（物理的理由）
```
右バックは右サイドのスペースからシュート/パスを出す
↓
右利き：体を半身に開いてシュート → 角度が狭い
左利き：体が自然にゴール正面を向く → 角度が広い・パワーが乗る

つまり「右バック × 左利き」は構造的優位
逆に「左バック × 右利き」も同じ理由で構造的優位
```

### 各国の左利き育成傾向
| 国 | 左利き活用度 | 戦術への影響 |
|---|---|---|
| フランス | 高（伝統的に育成） | 右バックの中距離・カットインに左利き定石 |
| セルビア・クロアチア | 高（旧ユーゴ伝統） | Bek（バック）の左利き輩出多数 |
| スペイン | 中（ピボット中心） | 左利きピボットの伝統 |
| デンマーク・ドイツ | 中 | システムの中で活用 |
| 日本 | 低（一般的でない） | **左利き育成の意識的取り組みが課題** |

### 左利き右バックへの対策（DF側）
- **右ブロックDF**：シュートコースを左側（ファーサイド）から塞ぐ
- **山側／谷側の選択**：左利きには通常と逆側（左側）を山にする
- **アングルカット**：左利きが好む「正面シュート」のコースをDFの体で塞ぐ

### 自チームに左利きがいる場合
- 右バック or 左ウィングに配置するのが構造的最適
- 「左利きはサウスポーシュート」と相手が誤認する場合がある → 通常の右肩シュートとは反対の体勢になるためGKの先読みが狂う

## 【新普遍原則3つ（批判的分析＋知的好奇心担当から導出）】

```
【原則13：構造的知識循環】
小規模集団ほど知識が速く深く循環する
→ アイスランド：レイキャビク周辺の密集した小クラブ群が全指導者の「共同知識プール」を形成
→ チーム内：選手全員が戦術を共有することで判断速度が最大化する（Jacobsenのコードワード）
→ 応用：チームが小さい≠弱い。知識の循環速度で補える

【原則14：数的優位の条件付き有効性】
数的有利は「自チームの強みと適合するときのみ」有効
→ Zormanの5on5選択：6on5の「遅い整った攻撃」より5on5の「速いトランジション」を優先
→ 強みを活かせない数的優位は優位ではない
→ 応用：ファウルで得た6on5も、チームの実力・スタイルに合わなければ使わない選択がある

【原則15：予測の予測を予測するスパイラル】
GKの予測精度が上がるほど「予測を外す技術」の価値も上がる
→ エリートGKはボールリリースの193ms「前」から動き始める（研究）
→ 早打ちはその「先取り予測」を逆用する
→ GKが進化するほど、早打ちの価値も比例して上がる（軍拡競争）
→ 応用：相手GKが上手いほど「GKが動く前に打つ」ことの意味が大きくなる
```
