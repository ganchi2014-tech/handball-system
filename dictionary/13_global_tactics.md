## 【グローバルトレンド 2025】

### IHF 2025世界選手権の記録
- **64.0点/試合**（男子史上最高記録）
- 平均攻撃時間：**31.7秒/攻撃**（高速化の証拠）
- デンマークの平均得失点差：**+13.4**（記録的な支配力）
- ポルトガル4位（自国史上最高）、フランス3位

### 7on6（GK外し）戦術の実態データ（2020-2023研究）
- **使用タイミング**：主に試合終盤、小差でビハインドのときに発動
- **空ゴール失点率**：17.7%（= ロングシュートで簡単には入らない）
- **内訳**：シュートなし攻撃での失点40.9% / 不成功シュート後の失点19.7% / 得点シュートの失点5.9%
- **実用示唆**：「ボールを持ち続けさえすれば空ゴール失点リスクは低い」。問題はボールロスト
- 2016年ルール改正後：2ピボット配置、5+1等の多様な7on6フォーメーションが登場

### じゃんけん（グローバル）
```
ゾーン守備（コンパクト0-6） → 外シュートを打たれ続ける → 1-5に変更
→ 前衛の裏をピボットで使われる → ゾーンに戻る

守備優位（低失点チーム） → 速攻で少ない得点を守る → ポゼッション攻撃に弱い
→ 相手が時間を使うポゼッション → カウンターチャンスが減る → 守備型チームが不利に

7on6（GK外し） → コンパクトゾーンで守られる → ピボット2枚配置でゾーン崩す
→ 空ゴールへのロングシュートカウンター → 7on6を早めに解除 → 得点機会が減る
```

## 【デンマーク】 ★★★ 現在の世界最強

### 基本データ
- 2024年パリ五輪金メダル（決勝39-26 vs ドイツ = 五輪決勝史上最大差）
- 2025年世界選手権優勝（平均得失点差+13.4 = 記録的）
- 失点平均**22点/試合**（世界最少水準）
- 直近28試合で26勝

### 監督：Nikolaj Jacobsen
- 「チームダイナミクスを作り、各選手が自分の役割を理解すること」
- 戦術的にコード化されたシステム（サインプレーが高度に体系化）
- 「オフコートでも楽しめる空気が頭をクリアにする」（選手の自由を重視）

### 戦術的特徴
**攻撃：**
- **クイックスローオフ**：得点直後に素早く再開 → 相手守備が整う前に攻撃開始
- エース依存ではなく集団得点（全員が高スコアに貢献）

**守備：**
- 22点/試合という記録的な失点の少なさ（守備の質が強さの核心）
- コレクティブDF（集団的組織守備）が軸
- 特定DF陣形よりも「全員が役割を果たすシステム」

### じゃんけん（デンマーク対策）
```
クイックスローオフ → 相手が整う前に攻撃 → DF未完成の隙を突く
→ 数人をDFに残す（速攻カウンター対策） → 速攻が出にくくなる → セット攻撃中心に戻る
→ 整ったDFでの守備 → デンマークの組織が最大限機能 → 崩すには外シュート+ピボット
```

```

- 「私はパッションを求めていた」（クロアチア就任時のコメント）

### 攻撃的特徴
- 個人技術（1on1）の高さ（ユーゴスラビア以来の伝統）
- 速攻（カウンター）の発動速度
- 「チームを削る（grinding down）」フィジカルなプレースタイル

### Jonas Willeの戦術哲学
- 「守ることより、目の前のチャンスを攻めることに集中する」
- 前任者Christian Berge（8年間）からの移行期 → 戦術的再構築中
- 軸：守備組織からの速攻（カウンター）発動

### ノルウェーのユース育成モデル（研究で証明された体系）
```
13歳以前：専門化を法律で禁止（多種目経験を優先）
13-15歳：地域タレント育成プログラム（月1回、地域横断トーナメント）
15-25歳：NHF（ノルウェーハンドボール連盟）の10年育成プログラム
         → フィジカル・タクティクス・メンタルを専門家チームで継続支援
         → クラブ + スポーツ学校 + 地域活動に同時参加
全期間：脱落=終わりではない。一度外れても再選考のチャンスあり（インクルーシブ方式）
```

**特徴的な原則**：
- 「約1/3のユース選手がタレント育成プログラムに参加」（選抜が緩やか）
- 閾値なしの継続的な選考（厳密なカット制度がない）
- 身体的・精神的な成熟の個人差を考慮した長期目線

## 【アイスランド】 ★★★ 小国最強モデルの教科書

### 基本データ
- 人口約37万人で継続的な国際競争力を維持
- 2008年北京五輪銀メダル（歴史的快挙）
- Snorri Steinn Guðjónsson監督（元国際選手、257キャップ）
- 2026年EHF EURO準決勝進出

### なぜ小国で強いのか？

**組織的基盤：**
- 人口の**28%**がスポーツ協会会員
- 12歳の80%以上が放課後スポーツプログラムに参加
- 子供の90%以上がいつかは組織スポーツを経験

**コーチング輸出大国（最重要ポイント）：**
```
2026年EHF EURO準決勝4チームのうち3チームがアイスランド人監督
代表例：Dagur Sigurdsson（クロアチア）、他複数のアイスランド人コーチが欧州各国で活躍
→ 人口37万人の小国が欧州のコーチング思想を形成している
```

**外部知識の取り込み：**
- 歴史的に東欧（ポーランド・ロシア）から指導者を招聘 → 技術・戦術水準を引き上げ → 独自発展

**戦術的特徴：**
- 規律あるゾーン守備 + 速攻（カウンター）のサイクル
- 防御的構造からのトランジション

### 示唆（小規模チームへの応用）
```
人数が少ない ≠ 弱い
組織力 × コーチングの質 × 長期育成 = 大国超え
全員がスポーツ文化の中にいる環境が最大の基盤
「学んで→自国で昇華→輸出する」コーチングサイクルが正の連鎖を生む
```

**哲学的キーワード**：
- 「youth, speed and positive madness（若さ・スピード・前向きな狂気）」

**攻撃：**

## 【GKシステムの国際比較】

## 【辞書への応用：じゃんけんマップ追記】

### 国際水準から逆算した原則
```
「失点22点/試合（デンマーク）」が示すもの
  → 守備の質 × 組織力 = 得点より失点管理が勝敗を決める
  → 日本のバックプレーヤーが学ぶべきは「守備から速攻へのスイッチ精度」

「64点/試合（2025世界選手権平均）」が示すもの
