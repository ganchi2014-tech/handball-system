## 【グローバルトレンド 2025】

### IHF 2025世界選手権の記録
- **64.0点/試合**（男子史上最高記録）
- 平均攻撃時間：**31.7秒/攻撃**（高速化の証拠）
- デンマークの平均得失点差：**+13.4**（記録的な支配力）
- ポルトガル4位（自国史上最高）、フランス3位

### 7on6（GK外し）戦術の実態データ（2020-2023研究）
- **使用タイミング**：主に試合終盤、小差でビハインドのときに発動
- **空ゴール失点率**：17.7%（= ロングシュートで簡単には入らない）
- **内訳**：シュートなし攻撃での失点40.9% / 不成功シュート後の失点19.7% / 得点シュートの失点5.9%
- **実用示唆**：「ボールを持ち続けさえすれば空ゴール失点リスクは低い」。問題はボールロスト
- 2016年ルール改正後：2ピボット配置、5+1等の多様な7on6フォーメーションが登場

### じゃんけん（グローバル）
```
ゾーン守備（コンパクト0-6） → 外シュートを打たれ続ける → 1-5に変更
→ 前衛の裏をピボットで使われる → ゾーンに戻る

守備優位（低失点チーム） → 速攻で少ない得点を守る → ポゼッション攻撃に弱い
→ 相手が時間を使うポゼッション → カウンターチャンスが減る → 守備型チームが不利に

7on6（GK外し） → コンパクトゾーンで守られる → ピボット2枚配置でゾーン崩す
→ 空ゴールへのロングシュートカウンター → 7on6を早めに解除 → 得点機会が減る
```

## 【デンマーク】 ★★★ 現在の世界最強

### 基本データ
- 2024年パリ五輪金メダル（決勝39-26 vs ドイツ = 五輪決勝史上最大差）
- 2025年世界選手権優勝（平均得失点差+13.4 = 記録的）
- 失点平均**22点/試合**（世界最少水準）
- 直近28試合で26勝

### 監督：Nikolaj Jacobsen
- 「チームダイナミクスを作り、各選手が自分の役割を理解すること」
- 戦術的にコード化されたシステム（サインプレーが高度に体系化）
- 「オフコートでも楽しめる空気が頭をクリアにする」（選手の自由を重視）

### 戦術的特徴
**攻撃：**
- **クイックスローオフ**：得点直後に素早く再開 → 相手守備が整う前に攻撃開始
- エース依存ではなく集団得点（全員が高スコアに貢献）

**守備：**
- 22点/試合という記録的な失点の少なさ（守備の質が強さの核心）
- コレクティブDF（集団的組織守備）が軸
- 特定DF陣形よりも「全員が役割を果たすシステム」

### じゃんけん（デンマーク対策）
```
クイックスローオフ → 相手が整う前に攻撃 → DF未完成の隙を突く
→ 数人をDFに残す（速攻カウンター対策） → 速攻が出にくくなる → セット攻撃中心に戻る
→ 整ったDFでの守備 → デンマークの組織が最大限機能 → 崩すには外シュート+ピボット
```


- 「私はパッションを求めていた」（クロアチア就任時のコメント）

### 攻撃的特徴
- 個人技術（1on1）の高さ（ユーゴスラビア以来の伝統）
- 速攻（カウンター）の発動速度
- 「チームを削る（grinding down）」フィジカルなプレースタイル

### Jonas Willeの戦術哲学
- 「守ることより、目の前のチャンスを攻めることに集中する」
- 前任者Christian Berge（8年間）からの移行期 → 戦術的再構築中
- 軸：守備組織からの速攻（カウンター）発動

### ノルウェーのユース育成モデル（研究で証明された体系）
```
13歳以前：専門化を法律で禁止（多種目経験を優先）
13-15歳：地域タレント育成プログラム（月1回、地域横断トーナメント）
15-25歳：NHF（ノルウェーハンドボール連盟）の10年育成プログラム
         → フィジカル・タクティクス・メンタルを専門家チームで継続支援
         → クラブ + スポーツ学校 + 地域活動に同時参加
全期間：脱落=終わりではない。一度外れても再選考のチャンスあり（インクルーシブ方式）
```

**特徴的な原則**：
- 「約1/3のユース選手がタレント育成プログラムに参加」（選抜が緩やか）
- 閾値なしの継続的な選考（厳密なカット制度がない）
- 身体的・精神的な成熟の個人差を考慮した長期目線

## 【アイスランド】 ★★★ 小国最強モデルの教科書

### 基本データ
- 人口約37万人で継続的な国際競争力を維持
- 2008年北京五輪銀メダル（歴史的快挙）
- Snorri Steinn Guðjónsson監督（元国際選手、257キャップ）
- 2026年EHF EURO準決勝進出

### なぜ小国で強いのか？

**組織的基盤：**
- 人口の**28%**がスポーツ協会会員
- 12歳の80%以上が放課後スポーツプログラムに参加
- 子供の90%以上がいつかは組織スポーツを経験

**コーチング輸出大国（最重要ポイント）：**
```
2026年EHF EURO準決勝4チームのうち3チームがアイスランド人監督
代表例：Dagur Sigurdsson（クロアチア）、他複数のアイスランド人コーチが欧州各国で活躍
→ 人口37万人の小国が欧州のコーチング思想を形成している
```

**外部知識の取り込み：**
- 歴史的に東欧（ポーランド・ロシア）から指導者を招聘 → 技術・戦術水準を引き上げ → 独自発展

**戦術的特徴：**
- 規律あるゾーン守備 + 速攻（カウンター）のサイクル
- 防御的構造からのトランジション

### 示唆（小規模チームへの応用）
```
人数が少ない ≠ 弱い
組織力 × コーチングの質 × 長期育成 = 大国超え
全員がスポーツ文化の中にいる環境が最大の基盤
「学んで→自国で昇華→輸出する」コーチングサイクルが正の連鎖を生む
```

**哲学的キーワード**：
- 「youth, speed and positive madness（若さ・スピード・前向きな狂気）」

**攻撃：**

## 【GKシステムの国際比較】

## 【辞書への応用：じゃんけんマップ追記】

### 国際水準から逆算した原則
```
「失点22点/試合（デンマーク）」が示すもの
  → 守備の質 × 組織力 = 得点より失点管理が勝敗を決める
  → 日本のバックプレーヤーが学ぶべきは「守備から速攻へのスイッチ精度」

「64点/試合（2025世界選手権平均）」が示すもの
  → 試合は高速化。1秒のズレが致命的
  → 「認知→判断→行動」の速度が全ての前提

「7on6の空ゴール失点17.7%」が示すもの

  → ロストが最大の敵

「5on5を選んだZorman」が示すもの
  → 数的有利は必ずしも最善手ではない

「アイスランドコーチ現象」が示すもの
  → 戦術は選手より「コーチ」を通じて伝播する
  → 一人の優れたコーチが複数の国を強くする連鎖が生まれる
```

## 【6-0守備の3基本動作（ドイツ語一次情報）】

> 出典：handballtraining.de、Wikibooks Handball Taktik、VIBSS

### ドイツ語技術用語（世界標準）

| 用語（独） | 読み | 意味 |
|---|---|---|
| **Verschieben** | フェルシーベン | ボール側へのライン全体シフト |
| **Herausrücken** | ヘラウスリューケン | インナーDFが前に飛び出すプレッシャー |
| **Übergeben** | ユーバーゲーベン | 飛び出した後の隣へのマーク受け渡し |
| **Tiefenraum** | ティーフェンラウム | ゴール前の危険な深さスペース（これを最小化するのが6-0の目的） |
| **Spielmacher** | シュピールマッハー | ゲームメーカー＝センタープレーヤー（5-1が狙う対象） |

### 6-0の3動作サイクル

```
① Verschieben（シフト）
   ボールが右に渡る → ライン全体が右へシフト
   → ボールサイドに密集 / 逆サイドに空間が生まれる（リスク）

② Herausrücken（飛び出し）
   インナーDFがボールを持つバックに前進プレッシャー
   → シュートコース圧迫 / パスコースを切る
   → 飛び出した背後にTiefenraumが生まれる（最大のリスク）

③ Übergeben（受け渡し）
   飛び出したDFが戻れない場合 → 隣のDFにマークを渡してラインを維持
   → 受け渡し失敗 → 穴が生まれる（攻撃のチャンス）
```

### じゃんけん（6-0の3動作を崩す）

```
Verschieben → 逆サイドへの素早い展開でスペースを使う
Herausrücken → 飛び出した背後にピボットを走り込ませる
Übergeben → 受け渡しの一瞬にクイックパス → 瞬間的マーク外れを突く
```

### 5-1の本質（ドイツ語原典から）
> 「5:1はHerausrückenをシステム化したもの。前衛1人が常に相手のSpielmacher（センター）を制限する。特に相手センターが強力な場合に有効」

## 【IHF分析：2025年世界選手権から読む戦術トレンド】

> 出典：IHF World Handball Magazine Tech（2025年世界選手権分析号）
> 分析担当：Dietrich Späte（IHF CCM委員長）、Jorge Dueñas（元スペイン・ブラジル・スロバキア代表監督）

### ① 近距離突破（Breakthrough）が最高の得点効率
- **ゴール至近距離へのブレークスルー（突破）が一貫して最も高い成功率**を示した
- センタープレーヤーのクローズレンジシュートエリアへの侵入が重要
- 示唆：パスを回すよりもゴールに向かって突破する判断が有効

```
外シュート（ミドル） → 確率は低い
ブレークスルー（近距離突破） → 確率が最も高い ← 積極的に選ぶべき
ペナルティ誘発 → 確率は高いがリスクも高い
```

### ② ライン選手なし攻撃（イタリア）
- **イタリアが2025年世界選手権でライン選手（ピボット）なしのポジショナル攻撃を実施**

- 分析：ライン選手がいないことで守備側の混乱を引き出すとともに、バックプレーヤーとウィングの組み合わせが増加
- **逆説的示唆**：ライン選手の「存在しない場合の脅威」が証明された

### ③ 7on6の質的向上
- 7on6戦術は**頻度が下がったが、実行の精度・質が向上**
- 単純な数的優位（7vs6）より、**2ピボット配置（7vs6 with 2 line players）や5+1フォーメーション**など高度な形が浸透
- 空ゴール失点を防ぎながら得点する技術が洗練されてきた

### ④ GKのロングシュート対応改善
- 遠距離シュートに対するGKの反応・成功率が向上傾向
- 示唆：**外シュートだけで勝てる時代は終わりに近い** → 近距離突破・ピボット活用の相対価値が上がる

### ⑤ 守備vs各種ポジショナル攻撃のアダプテーション
- 守備側が各チームのポジショナル攻撃に対して独自の「開口部（タクティカルオープニング）」を見つけ対応
- 攻撃の多様化に守備が追いついてきている → 双方の進化が続く「軍拡競争」

## 【スロベニア補足：Zormanの戦術哲学（スロベニア語一次情報）】

> 出典：SZR（スロベニア・ハンドボール連盟）、ekipa.svet24.si、rtvslo.si（国営放送）

### Zormanの言葉（スロベニア語原典）

> **「Obramba je ključna. Maksimalno disciplinirani in agresivni.」**
> （守備が鍵だ。最大限に規律正しく、そして攻撃的に）

> **「Manjka nam konkretnosti in ubijalskega instinkta.」**
> （私たちに足りないのは、具体性とキラーインスティンクト（仕留める本能）だ）

> **「Hitrost prehoda je naša največja prednost.」**
> （トランジション（切り替え）の速さが私たちの最大の強みだ）

### 5on5革新戦術の背景

**問題提起：なぜ6on5（数的優位）を使わないのか？**
- Zormanの答え：「スロベニアの強みは**速いトランジション（高速切り替え）**にある」
- 6on5（ファウルで得た1人多い状況）→ 整ったセット攻撃 → スロベニアの「速さ」の強みが消える
- **5on5（あえて数的優位を使わず素早く攻める）** → 相手DFが整う前に得点

> 原則14「数的優位の条件付き有効性」の具体例

**2024年パリ五輪4位の実証：**
- 高速トランジションで複数の強豪を撃破
- ただし金メダル争いに届かなかった課題：「具体性とキラーインスティンクト」の欠如（Zorman自身の言葉）

### 2026年EURO vs クロアチア（7on6失敗の事例）

- スロベニアが終盤に7on6（GK外し）を選択 → クロアチアが空ゴールへのカウンターで対応
- Zormanのコメント：「7vs6の実行の質が十分でなかった。GK外しは決断だけでなく、チームの準備が必要」
- ⚠️詳細スコアは検索時点で未確認（確認が必要）

### じゃんけん（スロベニア戦術への対応）
```
高速トランジション（5on5） → 守備が整う前に失点リスク
→ 速攻を封じる = 得点後に素早く守備陣形に戻る → 相手の速さが消える
→ セット攻撃での対決 → スロベニアは個人技×組織が融合（Zorman自身バックプレーヤー出身）
→ Zormanの弱点「キラーインスティンクト欠如」を逆用：ゴール前での強引な決断を迫る展開に持ち込む
```

## 【新普遍原則3つ（批判的分析＋知的好奇心担当から導出）】

```
【原則13：構造的知識循環】
小規模集団ほど知識が速く深く循環する
→ アイスランド：レイキャビク周辺の密集した小クラブ群が全指導者の「共同知識プール」を形成
→ チーム内：選手全員が戦術を共有することで判断速度が最大化する（Jacobsenのコードワード）
→ 応用：チームが小さい≠弱い。知識の循環速度で補える

【原則14：数的優位の条件付き有効性】
数的有利は「自チームの強みと適合するときのみ」有効
→ Zormanの5on5選択：6on5の「遅い整った攻撃」より5on5の「速いトランジション」を優先
→ 強みを活かせない数的優位は優位ではない
→ 応用：ファウルで得た6on5も、チームの実力・スタイルに合わなければ使わない選択がある

【原則15：予測の予測を予測するスパイラル】
GKの予測精度が上がるほど「予測を外す技術」の価値も上がる
→ エリートGKはボールリリースの193ms「前」から動き始める（研究）
→ 早打ちはその「先取り予測」を逆用する
→ GKが進化するほど、早打ちの価値も比例して上がる（軍拡競争）
→ 応用：相手GKが上手いほど「GKが動く前に打つ」ことの意味が大きくなる
```
