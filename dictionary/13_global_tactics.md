## 【グローバルトレンド 2025】

### IHF 2025世界選手権の記録
- **64.0点/試合**（男子史上最高記録）
- 平均攻撃時間：**31.7秒/攻撃**（高速化の証拠）
- デンマークの平均得失点差：**+13.4**（記録的な支配力）
- ポルトガル4位（自国史上最高）、フランス3位

### 7on6（GK外し）戦術の実態データ（2020-2023研究）
- **使用タイミング**：主に試合終盤、小差でビハインドのときに発動
- **空ゴール失点率**：17.7%（= ロングシュートで簡単には入らない）
- **内訳**：シュートなし攻撃での失点40.9% / 不成功シュート後の失点19.7% / 得点シュートの失点5.9%
- **実用示唆**：「ボールを持ち続けさえすれば空ゴール失点リスクは低い」。問題はボールロスト
- 2016年ルール改正後：2ピボット配置、5+1等の多様な7on6フォーメーションが登場

### じゃんけん（グローバル）
```
ゾーン守備（コンパクト0-6） → 外シュートを打たれ続ける → 5-1に変更
→ 前衛の裏をピボットで使われる → ゾーンに戻る

守備優位（低失点チーム） → 速攻で少ない得点を守る → ポゼッション攻撃に弱い
→ 相手が時間を使うポゼッション → カウンターチャンスが減る → 守備型チームが不利に

7on6（GK外し） → コンパクトゾーンで守られる → ピボット2枚配置でゾーン崩す
→ 空ゴールへのロングシュートカウンター → 7on6を早めに解除 → 得点機会が減る
```

## 【デンマーク】 コレクティブDF + クイックスローオフ

### 戦術的特徴
| 局面 | 特徴 |
|---|---|
| OF | **クイックスローオフ**（得点直後の素早い再開）が軸。エース依存ではない集団得点 |
| DF | コレクティブDF（集団的組織守備）。失点22点/試合の記録的少なさ |
| サインプレー | コード化されたサインプレーが体系化されている |
| 強み | 「特定の陣形」より「全員が役割を果たすシステム」 |

### 現代の支配（2026年5月時点）
- **世界選手権4連覇**（2019・2021・2023・2025）— ニコライ・ヤコブセン監督による史上初記録（フランス「Les Experts」以来の偉業）
- **欧州選手権2026優勝** — 同時に世界・欧州・五輪のすべてのタイトルを保持（フランス Les Experts に次ぐ史上2チーム目）
- 監督契約は2030年まで延長済
- 中核選手: マティアス・ギゼル（IHF世界最優秀選手 2023/2024/2025 3年連続）、GK エミール・ニールセン（2025世界選手権セーブ率43%・オールスターチーム選出GK。※大会MVPはギゼル）

### じゃんけん（デンマーク対策）
- クイックスローオフ → 数人をDFに残して速攻カウンターを準備
- 整ったDFを崩す → 外シュート+ピボットの2軸で同時に守れない状況を作る
- コード化システム → 同じ陣形で対応せず、毎回少しずつ違うパターンを見せて読みを外す

## 【フランス】 école française（多機能性 + アグレッシブDF）

### 戦術的特徴
| 局面 | 特徴 |
|---|---|
| OF | ダイナミックなピボット活用、バックプレーヤーが内外を使い分け、セットプレーのバリエーション豊富 |
| DF | アグレッシブな6-0ゾーン、前線プレッシャーとゾーン守備の組み合わせ |
| 多機能性 | 各選手が複数の役割を担う（左利き右バックの伝統育成あり） |
| 監督 | **Guillaume Gille**（2020〜）／2026年2月で退任予定 |

### 2024-2025 フランス代表の現状

- **ピボット**：**Ludovic ファブレガス**（ユーロ 2024 最優秀ピボット、主将）／**Karl コナン**（Montpellier→PSG、防御スペシャリスト）
- パリ五輪2024：「**carburation et justesse**（連携と正確性）」を欠き、想定パフォーマンスに届かず
- Gille 自己評価：「**jeu et connexion**（プレーと連携）でポジティブダイナミクスを取り戻す必要」
- 2025 W杯：35人→22人選抜、再起動シーズン
- 「Experts時代」（2008-2017）の集約的才能起用から、**世代分散型**へ移行中

### 「école française」 = 多機能性の具体

```
1人の選手が複数ポジションをこなす：
- バックがピボットに入る瞬間 → 突然の2ピボット
- ウィングが中央に切り込む瞬間 → 突然の3バック
- センターがウィングに動く瞬間 → DFのマーク混乱

→ DFが「誰が何をするか」を読み切れない
→ マンツーマンが効かない（追従が追いつかない）
```

**ファブレガス型ピボット**（参考にすべき動き）：
- パワーで6mラインを守る ＋ ブリックコンタクト（味方バックと目を合わせて「今だ」の合図を送ること）でバックに「今だ」を伝える
- DFが見ていない側に走る「フラッシュタイミング」（DFの視線が外れた一瞬を突いてゴール前へ素早く走り込むこと）が極めて正確

### じゃんけん（フランス対策）
- アグレッシブ6-0 → 6-0の3原則（圧縮/飛び出し/受け渡し）の完成前を突く
- 多機能選手 → マンツーマンで追わず、ゾーンで領域を渡す（個別マークが追い切れない）
- 左利き右バック → 右ブロックDFで対応（シュートコースを左側から塞ぐ）
- ファブレガス型ピボット → ブリックコンタクト（目合わせ） 成立前にバック側にプレッシャーをかけて目を合わせさせない

## 【クラブチームレベル：Paris Saint-Germain】 ★デンマーク式速攻 × 国際的フィジカル

> **メイン：ステファン・マドセン が オールボー から持ち込んだ「速いボール進行 × コンパクト守備」**
> 監督：**ステファン・マドセン**（2025年夏〜、前任 Raul Gutiérrez）
> オールボー時代（2018-2024）：デンマークリーグ4回優勝（2019/2020/2021/2024）、EHF チャンピオンズ リーグ 決勝2回到達

### Madsen の戦術哲学

```
「デンマーク式のスピードと守備」を「国際レベルのフィジカルと戦術」に適応する。
（Madsen本人の言葉に近い：「リスペクトしつつ、自分のやり方で小さな変化を加える」）
```

3つの軸：
1. **Quick ball progression（素早いボール進行）**：GKセーブ即座のカウンター、構造化攻撃（時間をかけて組み立てるセット攻撃）でも展開速度を落とさない
2. **Compact defense（コンパクト守備）**：ゴール前を厚くして遠距離シュートをGKに任せる
3. **構造化攻撃のバリエーション**：同じ陣形でも複数の派生パターン

### 攻撃の特徴と課題（2025-26シーズン現状）

**主力選手**：
- **ピボット**：**Kamil Syprzak**（ポーランド代表、フィジカル型）
- **右ウィング**：**Sebastian Karlsson**（フィニッシュ精度高い、カウンター完結役）
- **バック**：**Yahia Omar**、**Elohim プランディ**（プランディ はバルセロナ戦で14得点・CL個人最高記録）
- **GK**：**Mikkel Løfquist**（試合中の覚醒型・短時間で大量セーブ可能）

**特徴的な戦術**：
- **GKを外して数的優位（7on6 / attaque sans gardien）**：タイムアウト時など計画的に発動
- **両サイド展開（前任からの継承）**：左右バックから外シュート＋ピボット連携

**現在の課題**（vs バルセロナ戦などから判明）：
- 構造化攻撃成功率 **34%**（＝セットで作り込んだ攻撃が決まった割合。vs バルセロナ戦。一方で速攻なども含めた全体シュート効率は 54%）
- **右サイドの予測不能性が不足**：機械的にパターン化され、バルセロナ ニールセン に読まれる
- 個人技依存（dribble et du 1×1）：「ボールなしの走り（courses sans ballons）」が少ない
- 防御の脆さ：226得点を許す（StarLigueリーグ最多被失点／2025-26序盤時点）

### Karl コナン が示すDF技術

PSG守備の核。Montpellier 時代から「science défensive（守備の科学）」と称される：
- 接触ファウルを取られない距離感
- ヘラウスリュッケン（前飛び出し） のタイミングの正確さ
- ヘルプとリカバーを瞬時に切り替え
- → **DFスペシャリスト育成の好例**（14_position_roles_international.md と連動）

### じゃんけん（PSG対策）

- 速いボール進行 → **バックチェック2人即時帰陣**を絶対遵守
- GKなし攻撃 → **ロストしたら空ゴールへ即ロング**で得点機会
- 機械的なセット攻撃 → **同じパターンを2回見たら3回目を読む**（DF対応）
- Karl コナン の守備 → **接触に巻き込まれず、廻旋フェイントで腕を外して差し込む**

### バックプレーヤーへの示唆

- 「Quick ball progression」の意識：自分が打った後・パスした後に**走り続ける**（止まらない）
- 「コンパクト守備」を真似る：自陣で広がりすぎず、ピボット周辺を埋める
- プランディ 型バック：個人で14得点取れる「決定力」と、チームへの組織貢献を両立

モダン × 速さ × 戦術的可変性

3つの軸：
1. アグレッシブで知的な守備（aggressive und intelligente アプヴェーア（守備）arbeit）
  → カウンター発動の「基盤」として位置づけ

### CL 2025決勝：マグデブルク vs フックセ・ベルリン（32:26）の決定要因

「**ハンドボール界最高選手 マティアス・ギゼル を制限した**」という快挙：

| 要因 | 内容 |
|---|---|
| 高い集中的なポジショニングプレー | DFが「予測位置」より「ギゼルの実際の動き」を見る訓練 |
| 優れた守備転換（攻撃から守備への切り替え＝ディフェンシブ・トランジション） | OF→DFの切り替えで一瞬の隙を作らない |
| 決断力のある守備（entschlossene Verteidigung） | 接触をためらわない／フェイントに釣られない |
| GK **Sergey Hernández** の活躍 | ギゼルのコース予測を外し続ける |

→ **「世界最強の個人を組織で止める」モデル**として、バックプレーヤーDF側の学びが大きい

### Wiegert型カウンターの構造

```
強い6-0で外シュートを限定
  → 相手が「決めるか、外すか」の状況に追い込まれる
  → 外した瞬間、GKがロングパス（または速いミドルパス）
  → 全ポジションが走り出している（事前にスプリント開始済み）
  → 至近距離シュート or ピボット経由ですぐ決める
```

「**全ポジションが脅威**」＝ DFが「どこをマークすべきか」決められない。

### バックプレーヤーへの示唆

- DF局面：**「予測位置」ではなく「相手の実際の動き」を見る訓練**（ギゼル制御モデル）
- OF局面：シュート判断と同時にカウンターのスプリント準備（**シュート結果を見る前に走り始める**）
- 守備転換：シュートを打った後、5秒以内に守備位置へ
- 「世界最強でも組織で止められる」事実 → **個人能力より組織連携を信じる**

## 【スペイン】 ポジショナル + ピボット中心

### 戦術的特徴
| 局面 | 特徴 |
|---|---|
| OF | 厳密なポジショニング、空間を作って使う、ピボット経由セットプレー |
| DF | 組織的ゾーン守備 |
| 哲学 | 「一手先を読む」ポジショナルプレー、パル/インパル（偶数/奇数システム）理論によるピボット浮き |
| 代表監督 | Jordi Ribera（2025 W杯時点）／2025年から世代交代サイクル本格化 |

### パル/インパル（偶数/奇数システム） 理論（核心戦術原則）

スペイン式戦術の中核。バックプレーヤーが**常に確認すべき2つの状況**：

```
Par（パル＝偶数）  = 自分の直接的なDFマーク相手
Impar（インパル＝奇数）= 直接DFの両側に立つDF選手（自分にとっては「ズレた」DF）

【連続突破（penetraciones sucesivas）の論理】
① OFが自分のParを「固定（fijar）」する（ドライブ・脅威）
② 隣のDF（Impar）が寄ってカバーしようとする
③ 反対側にImpar（浮き）が生まれる → そこが空く
④ そのフリー選手を使う
⑤ DFが対応 → さらに別のImparが生まれる … 連続的に発生
```

→ 「**1人を固定して、隣を浮かせる**」を連鎖させるのがスペイン式。

### 2:2（バック×ピボット）の永続的探求

スペイン代表・バルセロナ・ASOBALリーグで共通して見られる**最重要パターン**：

```
バックプレーヤー（前線）＋ ピボット の2人で「ダイヤモンド状況」を作る

① セントラル/バックがDFを「固定（fijar al defensor par）」
② ピボットがImpar（浮きDF）の側のスペースへ走り込む（Freilaufen）
③ バックがピボットへ対角パス → ピボットがシュート or 倒れ込み

または：
① バックがシュートフェイント → DFを引きつける
② ピボットDFがヘルプに来る → ピボットが浮く
③ 逆サイドのバックがフリーになる（連鎖突破）
```

詳細は `14_position_roles_international.md` の「センター×ピボット連携（最重要）」参照。

### 2025 W杯時点のスペイン代表（Ribera体制）

- 世代交代サイクル本格化：2004-05世代（Cikuša兄弟、Victor Romero、Ferran Castillo、Ian Barrufet）、2002-03世代（Javi ロドリゲス、Jan Gurri、Gorka Nieto、Carlos Álvarez）が主軸へ
- グループF（2025 W杯）：チリ、**日本**、スウェーデン → 日本にとってスペインは直接の対戦相手
- バルセロナ系のパル/インパル（偶数/奇数システム）理論をベースに、若手の機動力を加味

### じゃんけん（スペイン対策）
- ポジショナル攻撃 → 想定外の位置取り（オフボール動き直し）でDFラインを崩す
- ピボット経由 → ピボットへのパスコース封じが基本
- パル/インパル（偶数/奇数システム）連鎖 → **「最初の固定」段階で全力プレッシャーをかけて連鎖を止める**
- 2:2状況 → ピボットDFに「ヘルプに行かない」訓練が必要（ヘルプ＝Imparが浮く）
- 「読み」が強い → シンプルな単発アクション（連続シュートフェイント等）で読みを混乱させる

## 【クラブチームレベル：FCバルセロナ】 ★スペイン式の頂点

> **メイン：スペイン代表のパル/インパル（偶数/奇数システム）理論を最も精度高く実装するクラブ**
> 監督：**アントニオ・カルロス・オルテガ**（2021〜、2027年まで契約）
> 2024年 EHF チャンピオンズ リーグ 12回目優勝。ASOBALリーグ連覇中。

### 守備の特徴

**基本陣形：6-0をベースに、状況で 4-2 の密集変形へ**

```
通常時：6-0（カタルーニャ式「堅固な防御ブロック（solid bloc défensif catalan）」）
PSG戦などビッグマッチで：4-2に密集変形
  → 前線2人で外シュートを封じ、後衛4人でゴール前を固める
  → 相手の構造化攻撃を「ブロックに何度もぶつける」
```

**GK中心の構造（最重要）**：
- **エミール ニールセン**（デンマーク）：75セーブで35%セーブ率／PSG戦で19シュート中9セーブ（前半）／**カウンター攻撃のブロック能力**が特徴
- **Viktor Hallgrimsson**（アイスランド、2025夏Plockから加入）：ニールセンと補完デュオ／両者Nantes時代の経歴
- **意味**：DFは「外シュートのコースを限定する」だけでよく、最終ラインはGKに任せる構造 → 6-0の3原則を最も効率化した形

### 攻撃の特徴

**両サイドにペア配置（戦術的柔軟性の核心）**：

| サイド | ペア | 役割 |
|---|---|---|
| **左バック** | Timothey N'Guessan（仏）／Jonathan Carlsbogard（瑞） | 経験あるベテラン×予測不能性。29ゴール水準 |
| **右バック** | **Dika Mem**（仏）／Đorđe Cikuša（西、19歳） | パワー＋ビジョン＋チームワーク。Memが主力 |
| **センター** | Luis フラーデ／**Ludovic ファブレガス**（仏） | フラーデがビルドアップ。ファブレガスがヴェスプレームから復帰しピボット兼任 |
| **右ウィング** | **アレイクス・ゴメス**（今季40得点トップスコアラー）／Daniel Fernandez | Gomezは マグデブルク 戦2試合で17得点 |
| **左ウィング** | Ian Barrufet（20得点） | 若手台頭 |

**特徴的なプレースタイル**：
1. **両サイドからの展開（attaque large）**：ワイドアタックでDFラインを引き伸ばす → 中央が空く
2. **素早いカウンター（engagements rapides）**：GKセーブ即時の速攻
3. **遠距離シュート**（"ça allume de loin"）：N'Guessan、Memのミドルレンジ
4. **2:2の永続的探求**：センター/バック×ファブレガス（ピボット）の パル/インパル（偶数/奇数システム） 連鎖
5. **中盤からのビルドアップ**：Luis フラーデが「ゲームを組み立てる役」

### 試合結果（実例）：vs PSG（2025-26 EHF CL）

```
前半：ニールセンが圧倒（19シュート中9セーブ）+ Memスコア=バルセロナ優勢
後半：PSG Løfquist覚醒で7-0ラン → 反撃される
最終：38-33でバルセロナ勝利（or 27-30で勝利／試合により）

決定要因：
- 困難な局面でも「耐える」攻撃継続性
- ニールセンのカウンターブロック
- Ortegaの「ペース調整」判断（第2半で意図的にスローダウン）
```

### じゃんけん（FCバルセロナ対策）

- 4-2密集ブロック → **狭いスペースでの組織連携が必要**。1on1依存は不利。
- ニールセン+Hallgrimsson → **遠距離より、引きつけて角度変化＋至近距離シュート**で対応
- 両サイドペア → **片サイドへの集中守備は不可**。バランス維持必須
- 2:2連鎖 → **ピボット周辺のヘルプを禁止**（ヘルプ＝Imparが浮く）
- 遠距離シュート → **早めの前線プレッシャー**で打たせない

### バックプレーヤーへの示唆

- 「2人ペア配置」を学ぶ：自分と隣のバックで**役割分担**（ベテラン×新人、突破×シュート、左×右）
- **フラーデ型ビルドアップ**：センターは「ボールを進める」より「組み立てる」役割
- **Mem型バック**：パワーだけでなく「ビジョン（パスの選択肢を見る）」を持つ
- カウンターを徹底的に走る（GKがフリーになる場面で5秒以内にスプリント開始）

### ゾルマン流5on5戦術（重要な戦術原則）
- 通常6on5（ライン選手＝ピボット〈ポスト〉を1人追加投入して数的優位を作る）の場面で**あえて5on5を維持**

- 結論：「**数的優位は条件付きでのみ有効**」（自チームの強みと合致するときのみ）

- 高速トランジション → バックチェック必須

## 【デンマーク補足：コードワード戦術（タイムアウト指示の体系化）】

タイムアウトで短いコードワードを使い、全員に攻撃意図を一瞬で共有する仕組み。日本のチームでも応用可能。

### ポジション番号体系
- **1's** = ウィング（両翼）
- **2's** = バック（両サイドバック）
- **3's** = 中央のインナー

### 主要コードワードと動きの内容
| コード | 動きの内容 | 目的 |
|---|---|---|
| **バルサ** | センターバックが左へ動く → 右バックにパス → 左バックがセンターバックとクロスして中央を走り抜けてボールを受ける | バックが1on1状況を作る。複数の選択肢を内包 |
| **ビルン** | ウィングがボールを持ち、弧を描いて右バックと相手中央DFの間のスペースへ走り込む | ランニングレーン（走るルート）の指定 |
| **det brede punkt** | 選手がベンチ側サイドライン際からスタート、相手バックとウィングの間を攻撃 | DFの横を大きく使って間を割く |

### 応用：自チームでコードワード化する手順
1. 主要パターンに2文字以内の名前を付ける（例：「クロス」「ウィング走り」）
2. タイムアウトと試合中の声でその名前だけを叫ぶ
3. 全員がパターンを把握していることが前提 → 練習で繰り返し定着させる

### じゃんけん（コードワード戦術への対処）
- DF側：ビデオスカウティングでパターンを解析する
- OF側：パターンを2〜3にとどめず、各パターンに「分岐」を作って読まれにくくする

## 【6-0守備の3基本動作（ドイツ語一次情報）】

> 出典：handballtraining.de、Wikibooks ハンドボール Taktik、VIBSS

### ドイツ語技術用語（世界標準）

| 用語（独） | 読み | 意味 |
|---|---|---|
| **フェアシーベン（シフト）** | フェルシーベン | ボール側へのライン全体シフト |
| **ヘラウスリュッケン（前飛び出し）** | ヘラウスリューケン | インナーDFが前に飛び出すプレッシャー |
| **ユーバーゲーベン（引き渡し）** | ユーバーゲーベン | 飛び出した後の隣へのマーク受け渡し |
| **ティーフェンラウム（深部空間）** | ティーフェンラウム | ゴール前の危険な深さスペース（これを最小化するのが6-0の目的） |
| **シュピールマッハー（ゲームメーカー）** | シュピールマッハー | ゲームメーカー＝センタープレーヤー（5-1が狙う対象） |

### 6-0の3動作サイクル

```
① フェアシーベン（シフト）
   ボールが右に渡る → ライン全体が右へシフト
   → ボールサイドに密集 / 逆サイドに空間が生まれる（リスク）

② ヘラウスリュッケン（前飛び出し）
   インナーDFがボールを持つバックに前進プレッシャー
   → シュートコース圧迫 / パスコースを切る
   → 飛び出した背後にティーフェンラウム（深部空間）が生まれる（最大のリスク）

③ ユーバーゲーベン（引き渡し）
   飛び出したDFが戻れない場合 → 隣のDFにマークを渡してラインを維持
   → 受け渡し失敗 → 穴が生まれる（攻撃のチャンス）
```

### じゃんけん（6-0の3動作を崩す）

```
フェアシーベン（シフト） → 逆サイドへの素早い展開でスペースを使う
ヘラウスリュッケン（前飛び出し） → 飛び出した背後にピボットを走り込ませる
ユーバーゲーベン（引き渡し） → 受け渡しの一瞬にクイックパス → 瞬間的マーク外れを突く
```

### 5-1の本質（ドイツ語原典から）
> 「5:1はヘラウスリュッケン（前飛び出し）をシステム化したもの。前衛1人が常に相手のシュピールマッハー（ゲームメーカー）（センター）を制限する。特に相手センターが強力な場合に有効」

## 【スペイン補足：トランジション構造化の原則】

### 1stウェーブ/2ndウェーブの体系化
速攻を「第1波（1〜2人）」と「第2波（3〜4人合流）」に明確に分離し、全選手が役割を把握する。

| 局面 | 1stウェーブ | 2ndウェーブ |
|---|---|---|
| 出発タイミング | ボール奪取/GKキャッチ即時 | 1stウェーブが減速し始めた瞬間 |
| 人数 | 1〜2人 | 3〜4人 |
| 役割 | スピードで先に走る・ロングパス受け | クイック攻撃またはセット移行の準備 |
| 判断基準 | このまま速攻を仕掛けるか、無理せず攻めを遅らせるかを時間圧力の中で即決する | 1stが決められなかったら即セットアップ |

### トランジションの判断訓練
- 「プレッシャー下でも正確な選択ができる」よう、時間制限付きの判断練習を体系化
- 相手の選手交代のタイミング・パターンを読んで戦術を調整（後半の疲労交代を利用）

### じゃんけん（トランジション構造化への対処）
- 1stウェーブ → 守→攻の切り替え時、ハーフコート付近にバックチェック要員を残しておく
- 2ndウェーブ → 1stが防げた直後の「2波目」を警戒（攻撃が止まったと油断しない）

## 【IHF分析：2025年世界選手権から読む戦術トレンド】

> 出典：IHF ワールド ハンドボール Magazine Tech（2025年世界選手権分析号）
> 分析担当：Dietrich Späte（IHF CCM委員長）、Jorge Dueñas（元スペイン・ブラジル・スロバキア代表監督）

### ① 近距離突破（Breakthrough）が最高の得点効率
- **ゴール至近距離へのブレークスルー（突破）が一貫して最も高い成功率**を示した
- センタープレーヤーのクローズレンジシュートエリアへの侵入が重要
- 示唆：パスを回すよりもゴールに向かって突破する判断が有効

```
外シュート（ミドル） → 確率は低い
ブレークスルー（近距離突破） → 確率が最も高い ← 積極的に選ぶべき
ペナルティ誘発 → 確率は高いがリスクも高い
```

### ② ライン選手なし攻撃（イタリア）
- **イタリアが2025年世界選手権でライン選手（ピボット）なしのポジショナル攻撃を実施**

## 【スロベニア補足：5on5戦術と「数的優位の条件付き有効性」】

### 5on5維持の戦術的根拠
**問題提起**：なぜ6on5（数的優位）を使わないのか？
- 6on5（ファウルで得た1人多い状況）→ 整ったセット攻撃 → スロベニアの「速さ」の強みが消える
- **5on5を維持**（あえて数的優位を使わず素早く攻める）→ 相手DFが整う前に得点

> 自チームの強みが「速さ」なら、数的優位より「速さ」を優先する判断。
> 原則：「**数的優位は自チームの強みと合致するときのみ有効**」

### 7on6（GK外し）の実行品質
7on6は決断だけでなく、チームの準備（事前パターン）が必須。準備が不十分なまま発動するとボールロスト→空ゴール失点に直結する。

### じゃんけん（スロベニア戦術への対応）
- 高速トランジション → 得点後に素早く守備陣形に戻る（相手の「速さ」を消す）
- セット攻撃での対決に持ち込む → スロベニアは個人技×組織が融合
- ゴール前での強引な決断を迫る展開に持ち込む（決定力が課題のチームへの対策）

## 【左利き右バックのグローバル戦術】

> **メイン辞書：`09_individual_differences.md`「左利き右バックの構造的優位」**
> このセクションは「グローバル戦術視点」の補足（各国の育成傾向・DF対策）。
> 個人差・戦術的応用の詳細は09を参照。

### なぜ右バックは左利きが有利か（物理的理由）
```
右バックは右サイドのスペースからシュート/パスを出す
↓
右利き：体を半身に開いてシュート → 角度が狭い
左利き：体が自然にゴール正面を向く → 角度が広い・パワーが乗る

つまり「右バック × 左利き」は構造的優位
逆に「左バック × 右利き」も同じ理由で構造的優位
```

### 各国の左利き育成傾向
| 国 | 左利き活用度 | 戦術への影響 |
|---|---|---|
| フランス | 高（伝統的に育成） | 右バックの中距離・カットインに左利き定石 |
| セルビア・クロアチア | 高（旧ユーゴ伝統） | ベック（バック）の左利き輩出多数 |
| スペイン | 中（ピボット中心） | 左利きピボットの伝統 |
| デンマーク・ドイツ | 中 | システムの中で活用 |
| 日本 | 低（一般的でない） | **左利き育成の意識的取り組みが課題** |

### 左利き右バックへの対策（DF側）
- **右ブロックDF**：シュートコースを左側（ファーサイド）から塞ぐ
- **山側／谷側の選択**：左利きには通常と逆側（左側）を山にする
- **アングルカット**：左利きが好む「正面シュート」のコースをDFの体で塞ぐ

### 自チームに左利きがいる場合
- 右バック or 左ウィングに配置するのが構造的最適
- 「左利きはサウスポーシュート」と相手が誤認する場合がある → 通常の右肩シュートとは反対の体勢になるためGKの先読みが狂う

## 【新普遍原則3つ（批判的分析＋知的好奇心担当から導出）】

```
【原則13：構造的知識循環】
小規模集団ほど知識が速く深く循環する
→ アイスランド：レイキャビク周辺の密集した小クラブ群が全指導者の「共同知識プール」を形成
→ チーム内：選手全員が戦術を共有することで判断速度が最大化する（Jacobsenのコードワード）
→ 応用：チームが小さい≠弱い。知識の循環速度で補える

【原則14：数的優位の条件付き有効性】
数的有利は「自チームの強みと適合するときのみ」有効
→ ゾルマンの5on5選択：6on5の「遅い整った攻撃」より5on5の「速いトランジション」を優先
→ 強みを活かせない数的優位は優位ではない
→ 応用：ファウルで得た6on5も、チームの実力・スタイルに合わなければ使わない選択がある

【原則15：予測の予測を予測するスパイラル】
GKの予測精度が上がるほど「予測を外す技術」の価値も上がる
→ エリートGKはボールリリースの193ms「前」から動き始める（研究）
→ 早打ちはその「先取り予測」を逆用する
→ GKが進化するほど、早打ちの価値も比例して上がる（軍拡競争）
→ 応用：相手GKが上手いほど「GKが動く前に打つ」ことの意味が大きくなる
```
