# 練習メニュー辞書

> 各技術・戦術を**どう練習すれば身につくか**。個人・ペア・チームの3階層構成。

---

## 【A】個人練習（自主練・1人で可能）

### A-1. シュート練習

**早打ち練習**
```
目的：上昇中の打点で正確に打てるようになる
内容：
  ① ゴール前7m地点に立つ
  ② 助走なしの1ステップ踏み切りでシュート
  ③ ジャンプの最高点を「待たない」意識
  ④ コース別に各10本（左上・右上・中央・左下・右下）
セット数：5コース × 10本 × 3セット = 150本
注意：パワーは出にくい → コース精度を優先
```

**最高点シュート練習**
```
目的：最も威力・精度の出る打点を体に覚えさせる
内容：
  ① 助走3歩 → 2ステップ踏み切り → 最高点で打つ
  ② ボールリリースの瞬間に「最も高い位置」を意識
  ③ 1本ごとに最高点と打点を確認
セット数：30本 × 3セット
```

**バックハンドフェイク練習（上級）**
```
目的：先読み型GK対策の最重要技術
内容：
  ① 通常のシュートフォームで構える
  ② ボールを離す直前に手首を逆方向に返す
  ③ 最初は壁打ちで手首の感覚を作る
  ④ 慣れたらゴールへ
  ⑤ コーチに「フォーム → 実際のコース」が分かるか聞く
注意：手首の柔軟性が必要。怪我に注意して徐々に
```

### A-2. フェイント練習

**フェイント7種＋ダブル原則ローテーション**
```
目的：7種のフェイントを試合で使えるレパートリーにする
内容：
  ① 仮想DF（コーン）の前で順番に練習
     1.廻旋（腕を空中で大円運動）
     2.ターン（軸足回転）
     3.シュートフェイント（フォーム作って打たない）
     4.パスフェイント（腕振って出さない）
     5.ステップフェイント（踏み込み変える）
     6.ボディフェイント（肩傾け）
     7.スピードフェイント（急停止→加速）
     ＋ ダブル原則：任意の2種を連続でかける（例：⑤→③、②→①など）
セット数：7種 × 10回 × 3セット = 210回（＋ ダブル原則10回）
```

**鏡前フェイント練習**
```
目的：自分のフェイントの「動きの大きさ」を客観視
内容：
  ① 鏡の前でフェイントを行う
  ② DFから見えている動きが何かを確認
  ③ 大きすぎる動き = 予備動作 → 小さく洗練
ポイント：エリート選手は「予備動作が消えている」
```

### A-3. ドリブル練習

**ドリブル6種ローテーション**
```
目的：状況別ドリブルの使い分け
内容：コーンを5本並べてジグザグドリブル
  1. スピードドリブル：全速力
  2. コントロールドリブル：体の横で守りながら
  3. 低位置ドリブル：腰の高さ
  4. クロスオーバー：体の前で左右切替
  5. バックビハインド：体の後ろで切替
  6. ロールターン：1回転して逆方向へ
セット数：6種 × 5本 × 3セット
```

---

### A-4. 踏み切り・ジャンプ練習

**4種踏み切り**
```
1. 2ステップ踏み切り：右→左（または左→右）標準
2. ユーロステップ：1歩目を一方向、2歩目を逆方向に大きく
3. 1ステップ踏み切り：最後の1歩だけで打つ（早打ち用）
4. ノーステップ：立ったまま打つ（外角フリー用）

各踏み切りからシュートを各10本 × 3セット
```

---

### A-5. パス練習（壁打ち）

**スロー4種（ノルウェー式）**
```
1. Piskekast（むち打ちスロー）：腕のしなりで最速
2. Sirkelkast（サークルスロー）：腕を円を描くように
3. Støtkast（プッシュスロー）：押し出すような短距離速射
4. Skjult kast（隠しスロー）：相手に見えにくい角度から

壁の特定の場所（5cm×5cmのマーク）に各種10球当てる
セット数：4種 × 10本 × 5セット
```

## 【B】ペア練習（2人で行う）

### B-1. 1on1

**勝ち位置取り練習**
```
目的：DFより先に有利な位置を取る
内容：
  ① OFがコーン地点からスタート
  ② DFは6mライン上の自由な位置
  ③ OFは「ボールを受ける前」にDFより先にどこかへ動く
  ④ DFは追って遅らせる
  ⑤ コーチがパスを出す → OFが受けてシュート
判定：OFが勝ち位置を取れたか／DFが先回りできたか
セット数：10回 × 攻守交代 × 3セット
```

**フェイント1on1（フィードバック付き）**
```
目的：フェイントが本当にDFを動かしているか確認
内容：
  ① OFが任意のフェイントを1種選んで仕掛ける
  ② DFは反応する（足を動かす）
  ③ ペアでフィードバック：
     - DFは「何を見て反応したか」を伝える
     - OFは「狙った動きが出せたか」を確認
セット数：10回 × 攻守交代 × 各フェイント種
```

---

### B-2. 2on2

**ピック＆ロール練習**
```
目的：スクリーン→ロールの連携を体に覚えさせる
内容：
  ① OF2人（ピボット役・バック役）
  ② DF2人
  ③ ピボットがバックのDFの前にスクリーンセット
  ④ バックがスクリーン経由で抜ける
  ⑤ ピボットがロールアウト
  ⑥ DFはスイッチ・ファイトオーバー・ゴーアンダーを試す
判定：OFがどの局面で得点機会を作れたか
セット数：DF対応3種 × 10回
```

**ギブ&ゴー練習**
```
目的：パス後の走り込みのタイミング
内容：
  ① Aがボールを持つ → Bにパス
  ② Aがそのままゴール方向に走り込む（マーカーが目を切った瞬間）
  ③ Bが即リターンパス
  ④ Aがシュート
判定：パスからシュートまで何秒か（理想2秒以内）
```

---

### B-3. パスキャッチ精度

**動きながらキャッチ**
```
目的：止まらず受けて即動作
内容：
  ① 2人がコート横一列に並ぶ
  ② 走りながらパス交換
  ③ 1度も止まらない・1度も後ろを向かない
  ④ 受けた瞬間に次のパス相手を見つけている
セット数：往復5回 × 3セット
```

## 【C】チーム練習（4人以上）

### C-1. 速攻練習

**1stウェーブ役割練習（3人）**
```
目的：GKスローからの2人速攻＋自分の位置取り
内容：
  ① GK + 右走者 + 左走者
  ② GKボール → 同時に左右走者がスプリント
  ③ GKがフリー走者を見つけてパス
  ④ 走者がシュートまで持っていく
判定：GKスローからシュートまでの時間（理想5秒以内）
```

**2ndウェーブ合流練習（5〜6人）**
```
目的：1stが止まった後の3〜4人合流
内容：
  ① 1stウェーブ2人 + 2ndウェーブ3人
  ② 1stが止められた想定
  ③ 中央走者・左右走者・リバウンド役の役割確認
  ④ 中央がDFを引き付けて左右へはたく
判定：DFバランス維持しながら役割完遂できたか
```

---

### C-2. セットOF練習

**3-3フォーメーション**
```
内容：6人のOFが3-3で並ぶ（ピボット入れて）
  → 外野3人 + ウィング2人 + ピボット1人
  → ボール回しでDFを動かす
  → クローズ/ワイドを確認 → 適切なアクション
セット：5分間連続でシュートまで持っていく回数を計測
```

**2-1-3フォーメーション**
```
内容：ピボットを9mライン付近に引き出す
  → ピボットが引き出したスペースにバックが切り込む
  → DFが対応できるか試す
```

### C-3. DF練習

**0-6基本動作（Verschieben/Herausrücken/Übergeben）**
```
目的：6-0の3原則を体に覚えさせる
内容：
  ① ボール側へVerschieben（横シフト）
  ② インナーDFがHerausrücken（飛び出し）
  ③ 飛び出し後Übergeben（隣にマーク受け渡し）
  ④ コーチが「右へボール」「左へボール」「外シュート」のコールでDFが動く
セット数：10分連続
```

**マンツーマン→スイッチ練習**
```
目的：スクリーンへの対応
内容：
  ① 2人ずつペアでマンツーマン
  ② OFがスクリーンを使う
  ③ DFは「ファイトオーバー」「ゴーアンダー」「スイッチ」を試す
  ④ 結果を3つ比較
```

---

### C-4. 連携練習

**Blickkontakt訓練（バック×ピボット）**
```
目的：パスする側と受ける側の「目合わせ」
内容：
  ① バックとピボットがアイコンタクト
  ② 目が合った瞬間にピボットがフラッシュ
  ③ バックが即パス
  ④ ピボットがシュート
判定：アイコンタクトからシュートまで何秒か（理想1.5秒）
```

**コードワード練習（デンマーク式）**
```
目的：短い言葉で全員がパターン理解
内容：
  ① 主要パターンに2文字以内の名前を付ける（例：「クロス」「ピボ」「外」）
  ② コーチがコールワード → 全員が動く
  ③ 5パターンを覚える
セット数：5パターン × 各10回
```

## 【D】メンタル練習（普段から）

### Visualiseringルーティン

```
時間：寝る前 5分
内容：
  ① 翌日の練習・試合を思い浮かべる
  ② 自分が成功するシーンを具体的に描く
  ③ 視覚 + 聴覚 + 触覚で再現
  ④ ネガティブシーンは描かない
```

### Triggere作成

```
時間：練習で試す
内容：
  ① 自分だけの「集中スイッチ」を決める
     例：ボールを2回手で叩く・深呼吸3回・特定の言葉「集中」
  ② 練習中毎回同じ動作を繰り返す
  ③ 1ヶ月続けると条件反射化
```

### 3秒切り替え練習

```
時間：1on1練習中
内容：
  ① ミスをしたら3秒以内に「次は○○する」と言う
  ② 4秒以上引きずったら次のプレーを始められない
  ③ コーチが時間を計測してフィードバック
```

## 【F】練習で意識する原則

```
①「受けてから考える」をやめる
   → 「考えてから受ける」に切り替える
   → 詳細：04_cognition.md, 17_scene_index.md

②「同じ練習を毎日」より「変奏のある練習」
   → 同じシュートを100本より、コース・打点・状況を変えて練習

③「成功率の管理」
   → 各練習の成功率を記録（例：早打ち成功率70%）
   → 数字で自分の成長を可視化

④「実戦に近い文脈」（エコロジカルアプローチ）
   → DFがいない練習は試合に直結しない
   → 必ず仮想DF（コーン・人）を入れる

⑤「自分の弱点 → 練習計画」
   → 自己診断（18_mental.md）で弱点を特定
   → 弱点を埋める練習を週2回入れる
```
