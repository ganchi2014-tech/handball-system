## 【ドイツ DHB】 判断明確化 + バック-ピボット連携語彙

### AIMポジション（狙い姿勢）（全動作可能受球姿勢）

**アレス・イスト・メーグリッヒ姿勢（全可能性姿勢）**（アレス-イスト-メーグリッヒ-ポジション）＝ 「すべてが可能な姿勢」。

```
DHBが定義するバックプレーヤーの理想的なボール受球姿勢：

① 両膝を軽く曲げ、重心は均等（前後左右いずれにも即動けるニュートラル）
② 上体は正面向き（DF・GK・ピボットを同時に視野に入れる）
③ 利き手はシュート体勢に入れる位置、非利き手はパス方向を示さない
④ 足の向きでシュート方向を読まれないよう45°以内の角度制限

目的：ボールを持った瞬間に「シュート/ドライブ/パス」の3択が全て残っている。
    どれか1択が消えた状態でボールを受けると、DFはもう読んでいる。
```

**なぜ重要か（じゃんけん視点）**：
- AIMポジション（狙い姿勢） の逆 = DF・GKに「読まれている」ポジション
- 上体が傾いている、利き手が先に動く、足がシュート方向を向く → これを見てDFは先読みする
- **バックプレーヤーの習慣チェック**：ボールを受けるとき、自分はAIMポジション（狙い姿勢）か？（→ 映像で確認 `28_video_analysis.md`）

### 4択判断ツリー（DHB ハンドボール Praxis 15）

DHBの指導書「ハンドボール Praxis 15」が定義する、バックプレーヤーが**ボールを持った瞬間に行う4択判断**。

> ⚠️ **重要：4択は「順番に考える」ものではない**
> 受球前の認知（`04_cognition.md`の3段構造）で**すでに答えを出しておく**。
> 下図はあくまで「4択の概念整理」。実際は全択を並列で評価し、ボールを持った瞬間には行動が決まっている状態が理想。

```
【4択の概念マップ】（受球前に並列評価する）

  ┌─────────────────────────────────┐
  │ [択1] 即シュート                  │
  │  スペースあり + GKが動く前に放てる  │
  ├─────────────────────────────────┤
  │ [択2] ドライブ                    │
  │  DFが近い + 1v1で勝てる角度がある  │
  ├─────────────────────────────────┤
  │ [択3] ピボットパス                 │
  │  DFが遠い + ポストへのコースが空    │
  ├─────────────────────────────────┤
  │ [択4] 横展開（リセット）            │
  │  択1〜3が全て潰されている          │
  │  → DFが動いたら択1〜3へ戻る       │
  └─────────────────────────────────┘
```

**使い方のポイント**：
- AIMポジション（狙い姿勢） はこの「並列評価を最後まで保つ姿勢」として機能する
- 択4（横展開）は「逃げ」ではなく「じゃんけんをリセットして次の択を作る」判断
- 「受けてから考えた」試合は択4が多い。「受ける前に考えた」試合は択1〜3が多い

### バック-ピボット連携3概念（DHB標準語彙）

| 概念 | ドイツ語 | カタカナ | 意味 |
|---|---|---|---|
| シフト | フェアラーゲン（シフト） | フェアラゲルン | バックが動いてDFをずらす。ピボットのためのスペース創出 |
| スクリーン | シュペレン（妨害） | シュペレン | ピボットがバックのDFに対してスクリーンをかける |
| リリース | アプゼッツェン（離れる動作） | アプゼッツェン | スクリーン後にバックがDFから切り離れる動き |

```
連携の流れ：

  LB がフェアラゲルン（右へシフト）
      ↓
  DFが引きずられる
      ↓
  ピボットがシュペレン（LBのDFへスクリーン）
      ↓
  LBがアプゼッツェン（スクリーン裏に抜ける）
      ↓
  LBフリー → シュート or ピボットへリターンパス
```

**じゃんけん循環**：
- DFがシュペレンを嫌って早めにスイッチ → アプゼッツェンでピボットがフリー → ピボットシュート
- DFがスイッチしない → バック（LB）がアプゼッツェンでフリー → バックシュート
- どちらかが必ずフリーになる → ピック&ロールと同構造

## 【フランス FFHB】 フィクサシオン・デュ・ドゥミ・サントル

### Fixation du ドゥミ（仏：半）-Centre（CBバイナリ判断ドリル）

**Fixation du ドゥミ（仏：半）-Centre**（フィクサシオン・デュ・ドゥミ・サントル）＝ 「センターバックによる固定」。

> **概念の位置づけ**：「CBがDFのインターバル前に立つ」という考え方は多くの国が実践している普遍的な原則。フランスが独自なのは、**この動作を独立したドリルとして体系化・命名した点**（entrainement-handball.frに練習メニューとして掲載確認済み）。スペインでは「フィハシオン（Fijación）」と呼ぶ（確認済み）。ドイツ語圏では「**Binden / Gegenspieler binden**（バインデン＝相手を縛る）」と表現し、「Fixierung」はDHB公式術語として確認されていない。

**コンセプト**：
CB（センターバック＝中央のバックプレーヤー）が、隣り合う2人のDFの間（インターバル）の「正面」に立ち続けることで、DFに「左を切るか・右を切るか」の2択を強制する。

```
DF配置（例：5-1の後衛2枚）：

       [D_L]         [D_R]
              ↑
           ここにCBが立つ（インターバルの正面）

DFがとれる選択肢はバイナリになる：
  A：D_L（左側のDF）が前に出てCBを止める → D_R（右側のDF）の隣にスペース → RB（右バック）/LW（左ウィング）へパス
  B：D_R（右側のDF）が前に出てCBを止める → D_L（左側のDF）の隣にスペース → LB（左バック）/RW（右ウィング）へパス
  C：両者がプレッシャーしない → CBがシュート
```

**なぜこれが強力か**：
- DFが「どちらかが動かなければならない」状況を作り出す
- CBが**動かないこと**でDFを動かす（自分は動かないのに相手を動かす、という受け身に見えて実は効いている動き＝消極的能動性）
- 「そこにいるだけでDFが迷う」（`CLAUDE.md` 普遍原則5の例証）

**バックプレーヤーへの示唆（CBでない場合）**：
- LB/RBは「CBがインターバルに立っている」を確認してから動く
- CBが固定している間に、LB or RBはスペースを探してカットの準備
- 全員が動こうとするとCBの固定効果が消える

## 【デンマーク】 序盤20分コーディング + デュアルバック集中

### ギセル式「序盤20分コーディング」

EHFトップレベルのハンドボールで広く実践される**試合序盤の情報収集フレームワーク**。

> ⚠️ **帰属注記**：「ギセル式」「ランデル体系化」という特定人物への帰属は文献的根拠が確認されていない。「試合中に相手守備が何を優先しているかを読み解く」という発想はマティアス・ギセルがEHFインタビューで語っており（eurohandball.com）、デンマーク代表が高水準で実践していることは確かだが、特定人物・デンマーク固有の手法ではなく、ナクスポートやダートフィッシュ等の映像分析ツールを活用するエリートチーム全般の慣行である。

```
序盤20分でコーディングする4項目：

① DFポジショニング傾向
   → 自分の担当DFは「高い位置に出やすいか / 引いているか」
   → ボールサイドに寄りすぎる癖があるか

② GKの先読みパターン（→ 05_gk_df.md のGKタイプ判定と連動）
   → 3〜5本シュートで反応パターンを確認
   → 「何を見てから動くか」を観察

③ DF陣形の切り替えタイミング
   → 何点差になると守備陣形が変わるか（セーフティーDF＝リードを守るための安全重視の守備に切り替えるポイント）
   → タイムアウト後に変わるか

④ ピボット対応DFの強度
   → ポストへのパスコースを積極的に消しているか
   → ポスト離れ気味に守るか
```

**コーディング後の活用**：
- 前半終了時点で「相手チームのじゃんけん法則」が1つ以上見える
- 後半最初の攻撃でそこを突く（相手が修正する前の「旬」）
- ハーフタイム分析（→ `28_video_analysis.md`）と連動させると精度が上がる

### デュアルバック集中モデル（50.2%データ）

IHF統計（2025年世界選手権）でデンマークの得点50.2%が**LB（左バック）/RB（右バック）の2名から**発生。

```
デュアルバック集中の構造：

  CBが攻撃を組み立てる（オーガナイズ＝味方へのパス配給とスペース管理）
      ↓
  LB / RB のどちらかに「解放する」判断
      ↓
  解放されたバックが「即シュートゾーン」に入る

  相手DFにとって：
  ・CBを警戒するとLB/RBが空く
  ・LB/RBを警戒するとCBが空く
  → じゃんけん循環（CBが管理、LB/RBが消費）
```

**バックプレーヤーへの示唆**：
- CBポジション：自分がシュートを打つよりも、LB/RBを「解放する」判断を優先する場面がある
- LB/RBポジション：CBが保持している時間は「シュートゾーンへの入り込み」に使う
- デュアルバックモデルでは**2枚目のバックがどれだけ積極的に動けるか**が鍵

## 【アイスランド】 マグヌッソン式 1v1特化訓練法

### 80-100回1v1 / セッション法

アイスランドのSCマクデブルク所属右バック、**オーマル・インギ・マグヌッソン**選手がEHF公式メディアのインタビュー（*"Magnusson's game: トレーニング one-on-one situations 100 times per session"*, eurohandball.com）で語った少年時代（Selfossクラブ）の練習回顧。特定コーチが確立した手法というよりも、**アイスランドの地方クラブ育成文化全体に根付いている哲学**。

```
訓練の原則：

  1セッション（90分）の構成：
  ・ウォームアップ：15分
  ・1v1ドリル集中：60分以上（80〜100回の反復）
  ・チーム練習：残り時間

  1v1ドリルの種類（すべてを繰り返す）：
  ① ステップフェイント（右/左 交互）
  ② ボディフェイク（上体/骨盤）
  ③ スピード変化（減速→加速）
  ④ ダブルフェイント
  ⑤ 接触を伴う1v1（DFが手を出す状況）
```

**なぜ80-100回が基準か**：
- 技術が**反射レベル（意識せずに出る）**になるためには数千回の反復が必要
- 1セッション80-100回 × 週4〜5回 × 数年 = 自動化のしきい値を超える
- チーム戦術はその後に乗せるもの（「土台なき家」を建てない）

**アイスランドが世界で通用する理由の核心**：
- 平均身長（短め）のハンデを個人技術の極限化で補う
- タクティクスを知っていても、1v1で負けると戦術は無意味
- 「技術が先、戦術は後」というフィロソフィー

**バックプレーヤーへの示唆**：
- 週あたりの1v1反復回数を数える
- 試合に出た全フェイントを記録して「成功率の低いフェイント」を次セッションで多く練習
- 「苦手なフェイントを避ける」のではなく「苦手を繰り返す量で埋める」

> ⚠️ **レベル別注記**：80-100回はSelfoss育成（将来のEHFエリート向け）での実践値。年齢・練習環境が異なる場合の具体的推奨回数について科学的推奨値は現時点の文献では明確でない。重要な原則は「回数より毎回フルコミット（本番と同じ強度）」であり、疲労で精度が落ちる手前までが有効な反復。

## 【クロアチア】 ジャンプシュート精度論 + フィート哲学

> **2つの理論の関係**：「ステップ数が少ない＝ジャンプシュートのエリート」と「立ちシュートが基本」は矛盾しない。
> ジャンプシュートを2〜3歩で打てるようになって初めて、「跳ぶ/跳ばない」の選択肢が生まれる。
> 最終形：**「2歩ジャンプ」と「立ちシュート」を同じ体勢から打ち分けられる状態** → DFもGKも読めない。

### ジャンプシュートバイオメカニクス：踏み切りステップ数理論

クロアチア代表の分析（EHF ユーロ 2022分析、Matkovićら）から導出された**踏み切りステップ数とエリートマーカーの相関**。

```
踏み切りステップ数と技術レベルの関係：

  ステップ多い（4〜5歩） → 助走距離を要する
    = まだシュートフォームが身体に馴染んでいない段階
    = DFが「助走開始を見て予測できる」

  ステップ少ない（2〜3歩） → 短い助走から最高点
    = フォームが自動化（意識せずに筋収縮できる）
    = DFが「どこから打つか読めない」
    → これがクロアチアのエリートバックの判定基準
```

**測定方法（自己分析）**：
- 自分のシュート映像を見て「踏み切り開始からジャンプまでのステップ数」を数える
- 同じシュートコースで3〜5本確認して平均を出す
- 疲労時・プレッシャー下で何ステップになるかも確認（ここが本当の技術レベル）

**なぜステップ数が減るか**：
- 体幹が安定し、踏み切り1〜2歩でも最高打点に到達できる
- 「助走から打点」ではなく「重心コントロールから打点」に変わる
- → 練習方法：立ち位置から1歩または2歩でのジャンプシュートを繰り返す

### 「フィート・オン・ザ・グラウンド」哲学

**"Keep Feet on the Ground" from 9m**（EHF ユーロ 2022 クロアチア分析）。

コンセプト：9m前後からのシュートは「必ずジャンプしなければいけない」わけではない。**立った状態からの直接シュートが基本**。

```
立ちシュート（スタントシュート）の優位性：

① DFが「跳ぶタイミング」で読んでいるところを裏切れる
   → DFとGKは「バックは跳ぶ」という経験値で待っている
   → 跳ばずに打つと読みのタイミングがずれる

② 打点の変化幅が大きい
   → 跳んでから打つ場合：最高点しか打点がない
   → 立ちシュート：どの高さからでも打てる（最高点・肩口・腰高）

③ 速い（フライトタイム＝ボールが手を離れてゴールに届くまでの飛行時間が短い）
   → 助走+ジャンプの時間ゼロ
   → GKの先読みが間に合わない
```

**じゃんけん（ジャンプ vs 立ちシュート）**：
- DFがジャンプシュートに慣れて「跳んだら体を張る」 → 立ちシュートで裏をとる
- DFが「立ちシュートを警戒して前傾」 → ジャンプシュートで頭上を抜く
- これを試合内で1〜2回切り替えるだけで相手の読みを壊せる

## 【スペイン RFEBM】 スキャン-識別-コミット + 統一語彙

### 「スキャン-識別-コミット」3ステップ

スポーツ認知科学における知覚-判断-実行モデル（RPD: Recognition-Primed Decision、Perception-Action Coupling等）のハンドボール実践応用版。スペインのエリートバックプレーヤーが体現している判断サイクル。

> ⚠️ **帰属注記**：「ドゥシェバエフ式」という帰属はEHF等の一次ソースで確認されていない。Dani Dujshebaev個人が「スキャン→識別→コミット」を体系化・言語化したという文献はなく、一般的な認知科学フレームワークをハンドボールに当てはめた概念として理解する。スペインの高水準バックプレーヤーがこのサイクルを高速で実行していることは映像分析から確認されている。

```
3ステップ判断ループ：

  [SCAN スキャン] ← ボールを受ける前0.5〜1秒
    ↓
    ・DFの数・位置・重心方向を素早く把握
    ・GKの構えを確認
    ・ピボットがどこにいるか

  [IDENTIFY 識別] ← ボールが手に来る瞬間
    ↓
    ・最大脅威はどのDFか
    ・最大チャンスはどのコースか
    ・「やっていい」か「やってはいけない」かのバイナリ判断

  [COMMIT コミット] ← ボールを受けた後即座
    ↓
    ・迷わない。「IDENTIFY」で出た答えを全力で実行
    ・中途半端は全て失敗（DFもGKも両対応できる）
```

**なぜ「コミット」が最重要か**：
- スキャンと識別が完璧でも、実行で迷うと全て無駄になる
- 迷い = DFに「もう一回見る時間」を与えること
- エリートバックプレーヤーの特徴：受球から行動開始まで**0.3秒以内**（→ 04_cognition.md）

**練習での習慣化**：
- 1v1練習中、「受ける前に声に出してスキャン結果を言う」（「右が空いている！」）
- コミットの確認：行動を起こした後「なぜそうしたか」を説明できるか
- できない = スキャンが機能していない（感覚だけで動いている）

### スペイン RFEBM「統一ハンドボール語彙」

**"Unified ハンドボール Mind"（統一ハンドボール思考）**：スペイン連盟（RFEBM）が全国の指導者・選手に導入している**標準化された戦術語彙体系**。

| スペイン語 | カタカナ | 意味 |
|---|---|---|
| **クルセ（西：クロス）** | クルセ | バック-バック間のクロス（クロスプレー全般） |
| **Cortina** | コルティーナ | スクリーン（DFの視野を遮る動作全般） |
| **ブロケオ（ブロック）** | ブロケオ | ブロック（身体で道を作る静的スクリーン） |

## 【GK逆工学】 シューターがGKのキュー読みを逆用する

### GKキュー階層（PMC10591308）をシューターが逆から読む

GK研究論文（PMC10591308）が特定した**GKがシュート方向を予測するために読むキューの優先順位**。
シューター（バックプレーヤー）は「GKが何を見ているか」を知ることで、見せ方をコントロールできる。

```
GKがシュート前に読むキュー（優先順位順）：

  1位：上体回転（体幹）    ← GKが最も重視するキュー
  2位：手首の向き          ← リリース直前の方向決定キュー
  3位：助走経路           ← どの角度からアプローチするか
  4位：ブロックコンステレーション ← チーム全体の配置
```

**シューターが各キューを「偽る」方法**：

| GKが読むキュー | フェイクの方法 | 技術名 |
|---|---|---|
| **①上体回転** | 肩・胸を右に向けて実際は左へ打つ | ボディフェイク（上体フェイント）|
| **②手首** | 腕の振りは右方向、手首の返しで左コース | リストシュート |
| **③助走経路** | 右方向に走り込んで左ポストへ斜めシュート | クロスシュート |
| **④コンステレーション** | CBがフィクサシオン中、LBが逆サイドから無警戒ゾーンへ | ルーティング（陣形で空間を作る） |

**優先度の高いキューほど「偽る効果が大きい」**：
- ①上体回転を偽れると、GKは「行先が逆」と判断してしまう（最大効果）
- ②手首は難易度が高い（シュート精度が下がりやすい）
- ③助走経路はウィングが最も使う（バックはコースが限定されるため補助的）
- ④陣形全体でフェイクするのが最も持続可能（個人の体力に依存しない）

**GKの先読みを逆用する「ダブルブラフ」**：
```
通常：GKが上体フェイクを学習 → GK側は「上体フェイクの逆に動く」に変化
      ↓
  「上体フェイクをするとGKは逆に動く」を知っているシューター
      ↓
  上体フェイクなし（真正面）で本当に打つ → GKが「フェイク」と判断して反対に動く
      ↓
  これが「予測の予測を予測するスパイラル」（→ CLAUDE.md 普遍原則9）
```

## 【物理学】ジャンプシュートの滞空時間と入射角の優位性

> **このセクションの位置づけ**：クロアチアセクションで扱った「ステップ数理論」と「フィート哲学」が**何のため**にあるかを物理的に説明する補足。
> 「なぜ高い打点から打つと有利なのか」「なぜ滞空中にシュートするとGKが困るのか」を力学・幾何学から裏付ける。

---

### 1. 滞空時間の実測データ

バイオメカニクス研究（ResearchGate / International Journal of Sports Physiology and Performance）から確認されているジャンプシュートの滞空時間。

```
ジャンプシュートの滞空時間：

  平均滞空時間（バックプレーヤー）：約 0.47秒（±0.10秒）
    出典：複数のバイオメカニクス研究の中央値
    この間に重心の垂直変位は約 0.28m（±0.10m）

  ウィングプレーヤー研究（IJSPP 2017）での比較：
  ・通常条件：  566 ± 30.5 ms（約0.57秒）
  ・加重ベスト着用：635 ± 31 ms（約0.64秒）
  ・踏み台あり：615 ± 32.9 ms（約0.62秒）

  参考：立ちシュートのフライトタイム（ボールが6m地点で0.24〜0.27秒）
  ※05_gk_df.md「6mシュートのフライトタイム約240ms」と整合
```

**滞空時間が長いほどGKのブロックタイミングが難しくなる理由**：

```
GKの対応タイミングのジレンマ：

  滞空中の選手に対してGKが「いつ反応するか」

  ① 踏み切り時点で反応 → 滞空中に軌道を変えられる（フェイク効果）
  ② リリース直前に反応 → 反応時間が約193ms（05_gk_df.md掲載値）なので
                          滞空0.47秒 - リリース後のフライト時間 ≒ ギリギリ
  ③ 最高点でリリース  → GKが地面に足をついている状態から上部コーナーに
                          間に合わない可能性が高い

  結論：滞空時間0.47秒はGKが「早読みすればフェイク被害」
        「遅読みすれば間に合わない」のちょうど境界に位置する
```

### 2. 打点の高さがGKカバー面積に与える影響

**打点とゴール幾何学の関係**：

シュートの軌道は「打点の高さ」と「ゴールからの距離」と「入射角」の3変数で決まる。

```
打点が高いほど入射角が急になる（幾何学的必然）：

  打点が低い（例：肩口・1.5m）からの9mシュート：
    → ゴール上部への軌道が水平に近い（浅い角度）
    → GKが前進すると射線をほぼカバーできる

  打点が高い（例：最高点・2.2〜2.5m程度）からの9mシュート（ゴールから9mのライン付近＝バックが打つ距離からのシュート）：
    → ゴール上部への軌道が急角度（斜め上から下へ）
    → GKが前進しても「上から差し込まれる」エリアが残る

  ⚠️確認推奨：打点高さ×距離×入射角の定量的数値は
    一次文献（Journal of Applied Biomechanics 2016等）で個別確認が必要。
    上記は幾何学的定性論として確認されているが、
    具体的な角度変化率（°/cmの打点上昇）は文献値が特定できていない。
```

**GK前進のトレードオフ**：

```
GKが前進したとき（05_gk_df.md「前進30〜40%アングル縮小」と連動）：

  メリット：
  ・シュートコーンを縮小（横方向のコース）
  ・ボールまでの距離が縮まり低いシュートへの反応余裕が増える

  デメリット（高打点シュートに限定）：
  ・前進するほど「頭上の角度」が広がる
  ・急角度の落下ボールに対してグローブが届かない
  ・前進後に後退する時間がない → 頭上コーナーが完全に無防備に

→ これが「ジャンプシュート + 打点最大化」の組み合わせが
   GKにとって最も対処が難しい理由
```

### 3. 上から打つ優位性の物理的説明

**バウンドシュートとの比較**：

```
バウンドシュートの跳ね上がり位置は打点の高さで変わる

  低打点（立ちシュート・肩口）：
    → バウンドゾーンは6m付近（GKが対応しやすい中間距離）
    → GKはバウンドの位置をある程度予測できる

  高打点（ジャンプシュート最高点）：
    → 同じ打ち出し角でも軌道が急になるため
      バウンドゾーンが4〜5m付近に前進する可能性
    → GKが「バウンドシュートの来る位置」として記憶している
      位置と異なるエリアに落下する

  ⚠️確認推奨：高打点バウンドシュートのバウンドゾーン変化については
    定量的な研究データが現時点で特定できていない。
    定性的に「打点が高いほど急角度でバウンド位置が変わる」は
    物理的に正しいが、実測値は要確認。
```

**GK視点から見た「上から来るボール」の反応難度**：

```
人間の視覚・反応系の非対称性（一般運動科学知見）：

  水平方向の動き：
  ・左右に動くボールは眼球追跡（Smooth Pursuit）で追いやすい
  ・足の動作も左右切り替えが比較的速い

  垂直下方向の動き（急角度シュート）：
  ・上から下へ落下するボール（急角度で上から差し込まれるシュート）は軌道予測が水平より難しい
  ・GKがジャンプして高さを合わせるには踏み切りタイミングが必要
    → 「高い打点から打たれた後にGKがジャンプする」のは
      物理的に間に合わない場合がある（反応時間193msの壁）

  ⚠️確認推奨：「垂直方向vs水平方向の反応時間差」について
    ハンドボールGK特化の定量研究は確認できていない。
    一般的な運動科学の知見として記載。
```

### 5. 実践への落とし込み

**打点を高くするための身体的条件と訓練**：

```
打点高さを決める3要素：

  要素1：身長（体の長さ）
    → 変えられない。しかし「最大リーチ」は腕の使い方で変わる

  要素2：跳躍高（ジャンプの高さ）
    → 訓練可能。脚力・踏み切り力の向上で改善
    → 21_physical_training.md「プライオメトリクス」が直接対応

  要素3：リリース位置（最高点でリリースできているか）
    → 技術的課題。最高点の「0.1〜0.2秒の窓」でリリースする習慣
    → 「跳んでから打つ」ではなく「最高点で打てるよう跳ぶ」という逆転発想
```

**踏み切りタイミングとシュートリリースの最適化**：

バイオメカニクス研究（MDPI Applied Sciences 2023「Effects of One-Step and Three-Step Run-Up on Kinematic Parameters」）では、踏み出しステップ数がジャンプシュートのキネマティクスに影響することが確認されている。

```
踏み切りタイミング最適化の考え方：

  ジャンプ → 最高点 → リリース の一連を「一つの運動」として習慣化

  訓練チェックリスト：
  ① 最高点でリリースしているか？（映像で確認 → 28_video_analysis.md）
     ・リリースが上昇中 → 打点が最大化されていない
     ・リリースが下降中 → 跳ぶ意味が半減

  ② 踏み切り〜リリースのタイミングが一定か？
     ・疲労時に「早めにリリース」する癖がついていないか

  ③ 打点の高さを「自己測定」できるか？
     ・壁に向かって跳び、最高点でボールをタッチした跡の高さを記録
     ・週単位で同じ地点から比較（フィジカル向上の可視化）

  ⚠️確認推奨：「最高点リリースと打点最大化」が得点確率に
    直接寄与することを示す定量データ（例：打点1cm上昇で
    ゴール上部コーナー到達率○%増）は一次研究での確認が必要。
    上記は物理学的根拠のある定性的推奨として扱う。
```

**「跳ぶ/跳ばない」判断との統合**（クロアチアセクション「じゃんけん」の完成形）：

```
バックプレーヤーの最終到達形：

  高打点ジャンプシュート → GKは前進して角度を縮小しようとする
      ↓
  DFがジャンプシュートを警戒して「跳ぶのを待つ」
      ↓
  立ちシュート（フィート哲学）でDFのタイミングを外す
      ↓
  GKが「前進していいか／待つべきか」の判断を迷う
      ↓
  この迷いを作り続けることがバックプレーヤーとしての最大の武器

→ 物理的な打点の高さ（ジャンプシュート）と
  タイミングの裏切り（立ちシュート）の両方を
  「同じ体勢から打ち分けられる」ことが、
  クロアチア哲学の最終ゴール
```
