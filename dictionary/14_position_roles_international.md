### バックプレーヤーの3択判断（ドイツ語：Entscheidungstraining）

ドイツの**Entscheidungstraining（決断トレーニング）**が示す核心：

```
バックプレーヤーがボールを持った瞬間の判断

選択①：Durchbruch（ドゥルヒブルッフ）= 突破してシュート
選択②：クロイツ（クロス）en（クロイツェン）= クロスして味方にスペース提供
選択③：Weiterspiel（ヴァイタースピール）= ボールを継続してパス

この判断を「一瞬で」行う訓練がEntscheidungstraining
判断スピードが攻撃の質を決める
```

**ノルウェー語（Valgskolen）の基本選択状況：**
```
バックプレーヤーがワイドポジションにいる場合の典型的選択：
① ウィングへのパス（外展開）
② ブレークスルー（内突破）
この2択の判断訓練が個人戦術の核心
```

### スペイン語の「奇数-偶数」理論（パル/インパル（偶数/奇数システム））

スペイン語学術論文から抽出した重要戦術概念：

```
「Penetración（突破）」の論理：

Par（偶数）= マークされているDF選手
Impar（奇数）= フリーになれる選手

OFが1人のDF選手に正面から仕掛けて、そのDFをその場に引きつけて動けなくする（固定＝fijar）
  ↓
隣のDFが寄ってカバーしようとする
  ↓
反対側にImpar（浮き）が生まれる
  ↓
そのフリーの選手を使う

これを「penetraciones sucesivas（連続突破）」と呼ぶ
```

**示唆**：個人の1on1より「DFを動かして次の選手をフリーにする」連鎖が本質。

### センター×ピボットの連携（最重要）

**スペイン語学術論文から：**

```
「2x2（セントラル＋ピボット）」= スペインが最重要視する「状況ダイヤモンド」

①セントラル（センター）が自分の正面のDFを引きつけて動けなくする（Fijar al defensor par＝マーク役のDFを固定する）
②センターがDFを固定したことで空いたスペースへ、ピボットが走り込む
③セントラルがピボットへの対角パスを選択

または：
①セントラルがシュートフェイクでDFを引きつける
②ピボットに対するDFが助けに来る
③逆サイドのバックがフリーになる
```

**ドイツ語原典（Zusammenspiel Rückraum und Kreisläufer）：**
> 「ブリックコンタクト（目合わせ）（ブリックコンタクト＝目合わせ）とタイミングをとったFreilaufen（スペース走り込み）が連携の核心」

### ピボットの本質（多言語から統合）

**ドイツ語**：「DFよりも先に、良い場所へ良いタイミングで入る（時間と位置の両方で先手を取る＝時間空間的優位）ことが目標（ブリックコンタクト（目合わせ）必須）」

### ピボットの技術体系（ドイツ語）

| 技術 | ドイツ語 | 内容 |
|---|---|---|
| スペース走り込み | **Freilaufen**（フライラウフェン） | DFより先に有利な位置を取るための素早い走り込み。ブリックコンタクト（目合わせ）で始動 |
| スクリーン設置 | **シュペレン（妨害）**（シュペレン） | DFの進路を体で塞ぐ。静止が必要 |
| ロールアウト | **アプゼッツェン（離れる動作）**（アプゼッツェン） | DFがスクリーンを回避しようとした瞬間、生じたスペースへ走り込む |
| 展開 | **フェアラーゲン（シフト）**（フェアラーゲルン） | ボールを一方から他方へ展開（ピボットが起点になる場合も） |
| 目合わせ | **ブリックコンタクト（目合わせ）**（ブリックコンタクト） | バックプレーヤーとの視線確認。連携の開始信号 |

### シュペレン（妨害）→アプゼッツェン（離れる動作）のじゃんけん

```
シュペレン（妨害）（スクリーン設置）
  → DFがスクリーンの前を通る（ファイトオーバー）
    → ピボットはそのまま壁を維持 → バックが抜ける
  → DFがスクリーンの後ろを通る（ゴーアンダー）
    → ピボットがアプゼッツェン（離れる動作）（ロールアウト）でスペースへ → パスを受ける
  → DFがスイッチ
    → ミスマッチ発生 → ピボットが有利な1on1
```

### スペインのピボット5スキル（学術論文）

1. **Ganar la posición**（位置を取る） = Freilaufen
2. **Bloquear al defensor**（DFをブロック） = シュペレン（妨害）
3. **Realizar fintas**（フェイクをかける） = シュートフェイク・方向転換
4. **Lanzar al gol**（ゴールへシュート） = 回転シュート・フォールシュート
5. **Desmarcarse**（マークを外す） = アプゼッツェン（離れる動作）

## 【ポジション間コンビネーションのじゃんけん統合】

### バック×ピボットじゃんけん

```
バックが1on1で突破しようとする
  → DFが縦を切る（クロイツ（クロス）en封じ）
  → バックがピボットへのパスを選択（アプゼッツェン（離れる動作）誘発）
  → ピボットがフリーで受ける
  → DFがピボットに2枚寄る
  → バックがフリーになる（Durchbruch）
  → 最初に戻る
```

### センター×バック×ピボットの三角じゃんけん（スペイン式）

```
センターが固定（fijar）→ バックのDFが動く → バックの突破路が開く
バックが突破（Durchbruch）→ ピボットのDFが助けに来る → ピボットがフリー
ピボットがフリー → センターのDFが戻る → センターの前にスペース
```

### フランス式Décalageのじゃんけん

```
全OFが内側に寄る（Décalage開始）
  → DF側も順次カバーで内側に寄る
  → 逆ウィング（最も外）に広大なスペース発生
  → ウィングが受ける → シュートまたはDFを引いてカットイン
  → DFが戻る → 内側にスペース再生
```

## 【トピックG：ウィングシュートの入射角物理と得点率の矛盾】

### 幾何学的優位性とは何か

ウィング（コーナー）からのシュートは「横入り」と呼ばれる。通常、シューターは9〜12mの距離からゴールに対して非常に鋭角な入射角で打つ。

**幾何学的な見かけ上の優位性：**
```
ウィングの「横入り」シュートの角度構造：

ゴール幅3m × ゴール高さ2m
ウィングからの射角：約10〜20度（ゴールに対してほぼ平行に近い）

→ GKが正面に立つと、ファーポスト（遠い側）が「大きく開いて見える」
→ 射角（シューターから見たゴールを狙える角度の幅）を最大化するように踏み込めば打てるスペースが増える
→ 「入りやすそう」に見える
```

GKの視点：「ニアポスト（近い柱）に自分が立てばファーポストが丸見えになる」

シューターの視点：「角度を広げてから打てば大きなスペースが見える」

この「相互に有利に見える」構造が矛盾の出発点になる。

---

### EHF統計でのポジション別シュート決定率

EHF（ヨーロッパハンドボール連盟）のデータおよび複数の学術研究から確認できた数値：

| ポジション | シュート決定率の目安 | 出典 |
|---|---|---|
| ライン（ピボット・6m） | 約75〜85% | EHFデータ分析・学術論文 |
| バックプレーヤー（9m）| 約40〜55% | EHFデータ分析 |
| **ウィング（コーナー付近）** | **約40〜55%** | EHFデータ分析 |
| 速攻（1on0） | 約70〜85% | 推計 |

**重要な発見（EHF Euro 2016-2024分析）：**
- 男子：ウィングからの得点は2018〜2020年にピークに達し、その後2024年に向けて減少傾向
- 女子：ウィング得点は2022年頃（1試合7点超）にピーク→2024年に減少
- 全体トレンド：6mシュート・突破シュートの割合が増加し、ウィングからの距離シュートは相対的に減少

⚠️確認推奨：具体的なポジション別決定率の完全データはEHF公式統計ページ（ehfeuro.eurohandball.com）で要確認。上記は複数研究の近似値。

### 「入りやすいはずなのに決定率が低い」矛盾の原因分析

#### 原因①：GKのニアポスト寄せ戦略

```
GKの最適戦略（角度管理）：

ウィングが構えた瞬間 → GKはニアポストに密着またはやや前に出る
  ↓
ニアポストを「体で消す」ことでファーポストのみが残る
  ↓
シューターは狭い「ファーポスト1点」を狙うしかない
  ↓
GKはそのコースへ体の一部を向けて待つだけでよい

→ 前に踏み出すことでさらに射角を縮小できる
（前進1歩で、シューターから見えるゴールの開いて見える幅が体感でおよそ20〜30%＝5分の1から3分の1ほど縮まるとされる）
```

出典：vanjaradic.fi「Saves of Wing Shots」/ Handball Evolution「Saving Shots from the Wing」

#### 原因②：シュート距離の問題

- ウィングからのシュートは一般に9〜12mの距離になる（9mラインよりさらに外側）
- バックプレーヤーの9mシュートと同等またはそれ以上の距離
- 「斜めから近い距離」というイメージがあるが、実際は遠い
- ボールの飛行時間が長く、GKの反応時間に余裕が生まれる

#### 原因③：DFのブロック・プレッシャー

- ウィング選手がボールを受けた段階で、すでにDF選手がプレッシャーをかけていることが多い
- ジャンプシュートへの踏み込み角度が限定される（コートライン・ゴールラインが制約）
- 踏み込みすぎるとコート外に出てしまうリスク
- DF選手が外側から追ってくると腕の振り角度が制限される

#### 原因④：シューターの腕の角度制限

```
ウィングシュートの腕動作の制約：

通常のバックプレーヤー：体の正面で大きく振れる
ウィングシュート：体が横向きになり、腕の振りが「押し込む」方向に変わる

→ シュートスピードが低下しやすい
→ ねじり（体幹回転）が使いにくい
→ 「修正ショット（modified slow shot）」と分類される研究がある

⚠️確認推奨：「modified slow shot」の割合データはResearchGate上の学術論文（2022年前後）で確認推奨
```

### 各国のウィング指導での「入射角最大化」アプローチ

**GK視点からの逆算（主要トレーニング思想）：**

1. **踏み込み（飛距離）の最大化**：コーナーから可能な限り踏み込んでゴールに対する角度を広げる
2. **ジャンプの軌道調整**：ゴールライン方向（並行）に飛ぶと角度維持 → ゴール方向（内向き）に飛ぶと角度拡大
3. **シュートタイミングの多様化**：GKが「待ち」の体制に入る前に打つ（早打ち・遅打ち）

**クロアチア語原典の哲学**：「クリロは最速の選手。狭い角度から跳躍してシュートするのが主な仕事」→ 角度よりも速さ・跳躍による予測困難化を優先

**フランスのアプローチ** ⚠️確認推奨：
- アイリエール（ウィング）はDFを引き伸ばす役割として位置づけられることが多く、シュート得点率よりも「スペース創出者」としての役割が重視されるケースがある

**解釈のまとめ：**
```
「入射角を最大化すれば入る」は半分正しい
 → ただしGKも角度に合わせて前進・寄せを調整する
 → シューターとGKの「角度取り合い」になる
 → 角度優位はGKに対する優位ではなく「最小化した不利」にすぎない
 → 決定率が低い本質：距離・腕角度・GK前進の3重の壁
```

## 【トピックH：速攻中の認知負荷管理（振り返りタイミング）】

### 二重課題理論（Dual-Task Theory）：走る＋情報収集

速攻中のウィング選手は「全力走行」と「状況把握」を同時に行わなければならない。これは認知科学の**Dual-Task（二重課題）**に該当する。

```
二重課題の構造（速攻中）：

課題①（自動的）：走る（足の運び・バランス・スプリント維持）
課題②（注意要）：情報収集（味方位置・GK状況・DFの戻り・パスタイミング）

→ 両課題が同時に「注意資源」を奪い合う
→ どちらかに意識を向けると、もう一方のパフォーマンスが低下する
→ 「走りながら振り返る」は認知負荷が非常に高い行為
```

**スポーツ認知科学の知見：**
- 視覚探索行動（Visual Exploratory Behavior）は、認知プロセスを支える「頭・目の能動的な動き」として定義される
- スキャニング頻度と質が高いほど、より多くの情報を予測に使える
- ただし、スキャニング中は一時的に前方への注意が切れる

---

### 振り返るのに最適なタイミング

走行中のフェーズを以下のように分解すると、認知コストが低いタイミングが存在する：

```
速攻の走行フェーズと「振り返り」の最適タイミング：

フェーズ①：スタートダッシュ（0〜5m）
  → 加速中。最も注意資源が走りに取られる
  → 振り返りコスト：高（バランス・加速が崩れるリスク）

フェーズ②：トップスピード（5〜15m）
  → 動作が自動化されやすい（熟練者）
  → 振り返りコスト：中〜低（走行リズムが安定）
  → 「空中視線」が使いやすいタイミング

フェーズ③：ボール受取直前（15m〜）
  → ボールへの集中が必要
  → 振り返りコスト：再び高（受球準備が最優先）
  → この段階での「首振り」は最小化が望ましい
```

**実用的示唆：** トップスピード維持中（フェーズ②）が情報収集の「コスト最低タイミング」。熟練ウィング選手はこのフェーズで1〜2回の視覚スキャンを行うと考えられる。

⚠️確認推奨：ウィング走行中の具体的な視線計測データは現時点での検索では学術文献を未確認。上記はDual-Task理論と一般的スポーツ認知研究からの推論。

### 視覚的走査パターン：エリート選手の視線行動

ハンドボール選手の視覚研究（主にGKを対象にした研究から派生的に示唆）：

| 要素 | エリート選手 | 非エリート選手 |
|---|---|---|
| 視線の動き | より目標指向的 | 散漫 |
| スキャニング頻度 | 高い（が必要なタイミングに集中） | 不規則 |
| 視線固定（Gaze Fixation） | 関連情報に長く留まる | 移動しすぎる |
| 予測との連携 | 動き前にスキャン完了 | 動きながらスキャン |

**重要な原則（認知スポーツ科学）：**
> 「より高い頻度・質のスキャニングは、より多くの情報を先取りし、競争上の優位をもたらす」
（出典：Eye–Head Coordination During Rapid Gaze Shifts in Soccer Scanning, biorxiv 2025）

ハンドボール選手とノンアスリートの比較研究では：
- 追跡精度に差はないが、**選手の眼球運動はより目標指向的**（Styrkowiec et al., 2024）

### 「首振りの最小化」：見ないことで速さを生む技術

逆説的に、**見る回数を減らすことでパフォーマンスが向上する**場合がある。

```
首振りの最小化の論理：

首を振る = 頭の向きが変わる = 空気抵抗・バランス変化 = スプリント速度が落ちる
首を振る = 前方視野を失う = コート端のラインが見えなくなる = 踏み出し位置が不正確になる

→ エリート選手は「事前に状況を読みきってから走り出す」技術を持つ
→ スタート前に状況を収集し、走行中の振り返り回数を最小化
→ 「走りながら見る」より「見てから走る」
```

**知覚行動結合（Perception-Action Coupling）の原理：**
実際に体を動かしながら見る場合（coupled＝動作と知覚が連動した状態）の方が、立ち止まって見るだけの場合（uncoupled＝連動していない状態）より、その後の反応の正確さが高い。つまり、「走りながら見る」行為は情報の質が落ちるリスクがある。

## 【トピックI：ウィングの前からのプレスがDFラインに与える連鎖効果】

### ウィングのハイプレスが成立する条件

ウィング選手が相手ボールに対して前から圧力をかける「ハイプレス」は、一定条件が揃ったときのみ効果的に機能する。

```
ハイプレス成立の必要条件：

条件①：DFラインとの距離感の確保
  → ウィングが前に出るとき、後方のDFラインとの距離が広がる
  → この「ギャップ」にパスが通ると即ピンチ
  → したがって、DFライン全体がウィングの前進と連動して高い位置を取る必要がある

条件②：相手ボール保持者への意図的プレッシャー
  → 相手が自陣を向いている（バックパス受けた直後）などの
    「プレスの効くタイミング（Trigger Press）」に限定する
  → ランダムなプレスは消耗するだけ

条件③：体力・ポジション戻り能力
  → プレスに失敗したとき、素早く後退できる脚力が必要
  → ウィングは「最速の選手」でなければハイプレスは維持できない
```

**5-1守備とウィングプレスの関係：**
5-1守備の「1（前の選手）」はバックプレーヤーへのプレッシャーを担う。これをウィング側にも拡張する概念がウィングのハイプレス的役割に相当する。

---

### プレスの連鎖効果：ウィングが前に出るとDFライン全体が前進できる

```
連鎖効果の構造：

ウィングが前に出てプレス
  ↓
相手ウィングへのパスコースが消える（or遅延する）
  ↓
相手はボールを中央に戻さざるを得ない
  ↓
DFラインのセンター付近もプレスをかけやすくなる
  ↓
DFラインがゴールから3〜4m前に押し上げられる
  ↓
DFが前に出ることで相手バックプレーヤーはより遠くから打たざるを得なくなり、シュート距離が9m→12〜13mに伸びる
  ↓
シュート決定率が低下する
```

**コンパクト守備との違い：**

### トレードオフ：前に出すぎるとカウンターの長いパスコースができる

ハイプレスの最大リスクは**カウンター（逆速攻）**である。

```
ハイプレス失敗時のリスクシナリオ：

ウィングが前に出てプレス → 失敗（かわされる or パス通る）
  ↓
ウィングが前残り or 戻り遅延
  ↓
相手GKor残ったDFが長いパスを打てるスペースができる
  ↓
相手ウィングが無人のコートを走り込む
  ↓
3on2 or 2on1の数的優位カウンター成立
```

**「オーバーコミット」の問題：**
- 中央に寄りすぎるとウィングが露出する
- 前に出すぎるとカウンターのロングパスコースができる
- 「飛び込みすぎ（out of position）」が最も避けるべき状態

### フランス・スペインでの守備的ウィングの役割定義

⚠️確認推奨：以下は確認できた一般情報および推論を含む。

**フランス（確認情報）：**
- フランス語原典より：「現代ハンドボールではウィングはDFを引き伸ばす者の役割」
  → OFでの役割が前提だが、DFでも「引き伸ばし」の逆、つまり「コンパクト化に貢献する者」として位置づけられる可能性がある
- フランス・デンマーク・スペインはトランジション支配に優れることが確認済み
  → これは守備的ウィングの速攻戻り能力が高いことを間接的に示す

**スペイン（確認情報）：**
- スペイン語でウィングは「Extremo（エクストレモ）」
- 「extremo」のDFでの具体的役割定義は、「5-1守備では後方サポートとして6mライン付近での位置取り」が一般的に知られる
- ⚠️確認推奨：スペイン式守備システムでのウィング役割詳細は、スペインハンドボール連盟（RFEBM）の指導者資料で要確認

### 「コンパクト守備 vs ハイプレス」の選択基準

```
選択フローチャート：

現在の状況評価：
  自チームのウィングは「最速クラス」か？
    Yes → ハイプレス可能性あり
    No  → コンパクト守備を基本にする

相手の攻撃スタイルは？
  バックプレーヤー中心（9m多投） → ハイプレスで距離を伸ばす効果大
  速攻・ウィング活用が多い      → ハイプレスのリスク大。コンパクト優先

スコア状況は？
  リード中（守り切り）     → コンパクト守備・省エネ
  ビハインド（逆転必要）   → ハイプレスで相手ターンオーバーを誘う
  残り時間少ない           → ハイプレス積極的採用

チームの体力状況は？
  終盤・疲弊              → ハイプレス維持困難 → コンパクトに戻す
```

**普遍原則との接続：**
- じゃんけん構造：コンパクト守備とハイプレスはじゃんけんの関係にある
- コンパクト → 相手に9mシュートの時間を与えるが、カウンターには強い
- ハイプレス → 9mシュートを防ぎやすいが、カウンターに弱い
- どちらが正しいかは「相手の得意な攻撃」と「自チームの強み」で決まる（原則8と一致）
