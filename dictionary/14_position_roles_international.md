### バックプレーヤーの3択判断（ドイツ語：Entscheidungstraining）

ドイツの**Entscheidungstraining（決断トレーニング）**が示す核心：

```
バックプレーヤーがボールを持った瞬間の判断

選択①：Durchbruch（ドゥルヒブルッフ）= 突破してシュート
選択②：クロイツ（クロス）en（クロイツェン）= クロスして味方にスペース提供
選択③：Weiterspiel（ヴァイタースピール）= ボールを継続してパス

この判断を「一瞬で」行う訓練がEntscheidungstraining
判断スピードが攻撃の質を決める
```

**ノルウェー語（Valgskolen）の基本選択状況：**
```
バックプレーヤーがワイドポジションにいる場合の典型的選択：
① ウィングへのパス（外展開）
② ブレークスルー（内突破）
この2択の判断訓練が個人戦術の核心
```

### スペイン語の「奇数-偶数」理論（パル/インパル（偶数/奇数システム））

スペイン語学術論文から抽出した重要戦術概念：

```
「Penetración（突破）」の論理：

Par（偶数）= マークされているDF選手
Impar（奇数）= フリーになれる選手

OFが1人のDFを固定（fijar）する
  ↓
隣のDFが寄ってカバーしようとする
  ↓
反対側にImpar（浮き）が生まれる
  ↓
そのフリーの選手を使う

これを「penetraciones sucesivas（連続突破）」と呼ぶ
```

**示唆**：個人の1on1より「DFを動かして次の選手をフリーにする」連鎖が本質。

### センター×ピボットの連携（最重要）

**スペイン語学術論文から：**

```
「2x2（セントラル＋ピボット）」= スペインが最重要視する「状況ダイヤモンド」

①セントラルが自分のDFを固定（Fijar al defensor par）
②ピボットが広がったスペースへ走り込む
③セントラルがピボットへの対角パスを選択

または：
①セントラルがシュートフェイクでDFを引きつける
②ピボットに対するDFが助けに来る
③逆サイドのバックがフリーになる
```

**ドイツ語原典（Zusammenspiel Rückraum und Kreisläufer）：**
> 「ブリックコンタクト（目合わせ）（ブリックコンタクト＝目合わせ）とタイミングをとったFreilaufen（スペース走り込み）が連携の核心」

### ピボットの本質（多言語から統合）

**ドイツ語**：「時間空間的優位をDFより先に確保することが目標（ブリックコンタクト（目合わせ）必須）」

**スペイン語**：「唯一相手守備ゾーン内に入るポジション。ゴールに背を向けDFの間に入る」

**ポルトガル語**：「チームのアンカー。体を壁として使う。ピボットの目的：**DFが集団プレーできない位置を取ること**」

**クロアチア語（コーチ知識と一致）**：「そこにいるだけでDFが迷う = 存在自体が脅威」

---

### ピボットの技術体系（ドイツ語）

| 技術 | ドイツ語 | 内容 |
|---|---|---|
| スペース走り込み | **Freilaufen**（フライラウフェン） | DFより先に有利な位置を取るための素早い走り込み。ブリックコンタクト（目合わせ）で始動 |
| スクリーン設置 | **シュペレン（妨害）**（シュペレン） | DFの進路を体で塞ぐ。静止が必要 |
| ロールアウト | **アプゼッツェン（離れる動作）**（アプゼッツェン） | DFがスクリーンを回避しようとした瞬間、生じたスペースへ走り込む |
| 展開 | **フェアラーゲン（シフト）**（フェアラーゲルン） | ボールを一方から他方へ展開（ピボットが起点になる場合も） |
| 目合わせ | **ブリックコンタクト（目合わせ）**（ブリックコンタクト） | バックプレーヤーとの視線確認。連携の開始信号 |

### シュペレン（妨害）→アプゼッツェン（離れる動作）のじゃんけん

```
シュペレン（妨害）（スクリーン設置）
  → DFがスクリーンの前を通る（ファイトオーバー）
    → ピボットはそのまま壁を維持 → バックが抜ける
  → DFがスクリーンの後ろを通る（ゴーアンダー）
    → ピボットがアプゼッツェン（離れる動作）（ロールアウト）でスペースへ → パスを受ける
  → DFがスイッチ
    → ミスマッチ発生 → ピボットが有利な1on1
```

### スペインのピボット5スキル（学術論文）

1. **Ganar la posición**（位置を取る） = Freilaufen
2. **Bloquear al defensor**（DFをブロック） = シュペレン（妨害）
3. **Realizar fintas**（フェイクをかける） = シュートフェイク・方向転換
4. **Lanzar al gol**（ゴールへシュート） = 回転シュート・フォールシュート
5. **Desmarcarse**（マークを外す） = アプゼッツェン（離れる動作）

## 【ポジション間コンビネーションのじゃんけん統合】

### バック×ピボットじゃんけん

```
バックが1on1で突破しようとする
  → DFが縦を切る（クロイツ（クロス）en封じ）
  → バックがピボットへのパスを選択（アプゼッツェン（離れる動作）誘発）
  → ピボットがフリーで受ける
  → DFがピボットに2枚寄る
  → バックがフリーになる（Durchbruch）
  → 最初に戻る
```

### センター×バック×ピボットの三角じゃんけん（スペイン式）

```
センターが固定（fijar）→ バックのDFが動く → バックの突破路が開く
バックが突破（Durchbruch）→ ピボットのDFが助けに来る → ピボットがフリー
ピボットがフリー → センターのDFが戻る → センターの前にスペース
```

### フランス式Décalageのじゃんけん

```
全OFが内側に寄る（Décalage開始）
  → DF側も順次カバーで内側に寄る
  → 逆ウィング（最も外）に広大なスペース発生
  → ウィングが受ける → シュートまたはDFを引いてカットイン
  → DFが戻る → 内側にスペース再生
```
