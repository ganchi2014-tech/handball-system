# 認知・判断・行動・声掛け

---

## 認知→判断→行動の3段構造

全てのプレーはこの3段階で構成される。

```
① 認知：何を見て、何を把握するか
② 判断：どの選択肢を選ぶか
③ 行動：実際に動く
```

### 速度の重要性
- 3段階のどこかで遅れると、相手の先手を取られる
- 認知を速くする → 情報収集の精度（視野の広さ）
- 判断を速くする → 事前の約束事（オートマティズム）
- 行動を速くする → 技術の自動化（考えなくてもできる）

---

## 認知の要素

### 何を認知するか
| 認知対象 | 具体的な情報 |
|---|---|
| DFの位置 | クローズかワイドか、距離感 |
| DFの重心 | 前傾か後傾か、どちらに動こうとしているか |
| DFの目線 | ボールを見ているかOFを見ているか |
| GKの位置 | 左/中/右のどこが開いているか |
| ピボットの位置 | パスコースがあるか |
| スペース | どこが空いているか |
| 味方の位置 | 誰がフリーか |

### 視野の広げ方
- 目線を上げる（ボールだけを見ない）
- 頭の動きで周辺視野を活用する
- ボールを持つ前から情報を収集しておく

---

## 判断の要素

### 判断の速度を上げる方法
1. **オートマティズム**：事前に決めた約束事。DFがクローズなら外、ワイドなら内など
2. **経験の蓄積**：同じ状況を繰り返し経験して直感的に判断できるようにする
3. **優先順位の明確化**：第一選択・第二選択・第三選択を事前に決める

### 判断の優先順位の例（バックプレーヤー）
1. ピボットがフリーなら → ピボットへパス（最優先）
2. 走り込む味方がフリーなら → そちらへパス
3. GKとの1on1になれば → 自分でシュート
4. それ以外 → ボールをキープして次を探す

## ボールを受ける前の認知ルーティン（5ポイント確認）

> 「受けてから考える」は遅い。**受ける前に決まっている**状態を作る。

```
ボールが隣の選手に渡った瞬間から確認を始める：

① 自分のDFの位置・状態
   → クローズ？ワイド？距離感は？接触してくるか？

② ピボットの位置
   → 6m上でポジションを取れているか？パスコースはあるか？

③ GKの位置
   → 左/中/右のどこが空いているか？（2/3が塞がれている法則）

④ 隣の選手の状況
   → 誰かフリーになりそうか？スクリーンを使えるか？

⑤ スペース
   → ヘラウスリュッケン（前飛び出し）（飛び出し）で空いた背後はあるか？
```

**確認の速さが判断の速さになる**：
- 全部確認する必要はない。①と②だけで判断できることも多い
- 試合に慣れると①が「自動認識」になる → ②以降に意識を使える

## プレッシャー下での判断フロー

DFが詰めてきたとき（時間がなくなるとき）の判断フロー：

```
DFが急に詰めてきた
│
├─ シュートが打てる距離・角度か？
│   YES → シュートフェイントを入れてシュート or ドライブ
│   NO  ↓
│
├─ ピボットへのパスコースはあるか？
│   YES → ピボットへ即パス（DFが詰めた背後が空いているはず）
│   NO  ↓
│
├─ 隣へのパスコースはあるか？
│   YES → 素早くはたく
│   NO  ↓
│
└─ ドリブルで時間を作る（コントロールドリブル）→ 再び①へ
```

**よくある認知・判断ミス（5パターン）**：

| ミス | 原因 | 修正 |
|---|---|---|
| ボールだけを見ている | 視野が狭い | 受ける前から周りを見る習慣 |
| 受けてから考える | 事前確認ゼロ | 隣がボールを持った瞬間から確認開始 |
| 第一選択に固執する | オートマティズムが硬直 | 第一選択が使えない場合の第二選択を決めておく |
| DFの反応を待たない | フェイントを使わずに仕掛ける | 相手が動いた後に動く（先に動くのはDF） |
| 時間をかけすぎる | 良い選択肢を探し続ける | 「2秒以内に最善を選ぶ」より「1秒以内にそこそこ良い選択をする」の方が有効 |

## 「2秒の壁」（個人判断の時間基準）

ボールを受けてから2秒以内に判断できているかを試合後に振り返る基準。
（※03_numerical.md・16_of_combinations.mdの「速攻判断2秒ルール」は別概念：チーム全員が「速攻か遅攻か」を把握する時間）

```
0〜0.5秒：受ける → 第一確認（DFの位置）
0.5〜1秒：フェイント or パス or シュートの選択
1〜1.5秒：実行開始
1.5〜2秒：実行完了（シュート or パス or ドライブ）

2秒を超える → DFが体勢を整えてしまう → 仕掛けの効果が半減
```

## エコロジカルアプローチ

> 練習は試合に近い状況（アフォーダンス）で行う

### 原則
- 試合で起こる状況を練習で再現する
- DFがいない状態での練習は試合に直結しない
- 状況判断ごと練習する（シュート練習ではなくシュート判断練習）

### アフォーダンスの活用
- 環境が「何ができるか」を教えてくれる
- クローズDFを見たら自然に外が見える → クローズを認識したら外へ
- ワイドDFを見たら自然に内が見える → ワイドを認識したら内へ

## 声掛け

### 声掛けの種類と目的
| 種類 | 内容 | タイミング |
|---|---|---|
| 状況報告 | 「右！」「後ろ！」「フリー！」 | 味方が見えていない情報を伝える |
| 指示 | 「ここへ！」「パス！」「打て！」 | 次の行動を指定する |
| 確認 | 「俺マーク！」「スイッチ！」 | DFの分担を確認する |
| 励まし | 「ナイス！」「大丈夫！」 | モチベーション維持 |

### 声掛けのじゃんけん
```
声を出す選手（情報共有）
  → チーム全体の認知速度UP → 判断速度UP → 組織プレーが強くなる
  → 声を出さない選手（個人判断）
  → 個人の判断は速いが連携ミスが増える
  → 声を出す選手が正確に伝えられないときは情報ノイズになる
  → 声の質と量のバランスが重要
```

### バックプレーヤーが出すべき声の優先順位
1. DFの位置報告（「クローズ！」「ワイド！」）
2. フリーの選手への呼びかけ（「来い！」）
3. DFのマーク確認（「俺が見る！」）
4. GKへの指示（「右！」「低い！」）
