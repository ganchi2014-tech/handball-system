# 認知・判断・行動・声掛け

---

## 認知→判断→行動の3段構造

全てのプレーはこの3段階で構成される。

```
① 認知：何を見て、何を把握するか
② 判断：どの選択肢を選ぶか
③ 行動：実際に動く
```

### 速度の重要性
- 3段階のどこかで遅れると、相手の先手を取られる
- 認知を速くする → 情報収集の精度（視野の広さ）
- 判断を速くする → 事前の約束事（オートマティズム）
- 行動を速くする → 技術の自動化（考えなくてもできる）

---

## 認知の要素

### 何を認知するか
| 認知対象 | 具体的な情報 |
|---|---|
| DFの位置 | クローズかワイドか、距離感 |
| DFの重心 | 前傾か後傾か、どちらに動こうとしているか |
| DFの目線 | ボールを見ているかOFを見ているか |
| GKの位置 | 左/中/右のどこが開いているか |
| ピボットの位置 | パスコースがあるか |
| スペース | どこが空いているか |
| 味方の位置 | 誰がフリーか |

### 視野の広げ方
- 目線を上げる（ボールだけを見ない）
- 頭の動きで周辺視野を活用する
- ボールを持つ前から情報を収集しておく

---

## 判断の要素

### 判断の速度を上げる方法
1. **オートマティズム**：事前に決めた約束事。DFがクローズなら外、ワイドなら内など
2. **経験の蓄積**：同じ状況を繰り返し経験して直感的に判断できるようにする
3. **優先順位の明確化**：第一選択・第二選択・第三選択を事前に決める

### 判断の優先順位の例（バックプレーヤー）
1. ピボットがフリーなら → ピボットへパス（最優先）
2. 走り込む味方がフリーなら → そちらへパス
3. GKとの1on1になれば → 自分でシュート
4. それ以外 → ボールをキープして次を探す

## ボールを受ける前の認知ルーティン（5ポイント確認）

> 「受けてから考える」は遅い。**受ける前に決まっている**状態を作る。

```
ボールが隣の選手に渡った瞬間から確認を始める：

① 自分のDFの位置・状態
   → クローズ？ワイド？距離感は？接触してくるか？

② ピボットの位置
   → 6m上でポジションを取れているか？パスコースはあるか？

③ GKの位置
   → 左/中/右のどこが空いているか？（2/3が塞がれている法則）

④ 隣の選手の状況
   → 誰かフリーになりそうか？スクリーンを使えるか？

⑤ スペース
   → ヘラウスリュッケン（前飛び出し）で空いた背後はあるか？
```

**確認の速さが判断の速さになる**：
- 全部確認する必要はない。①と②だけで判断できることも多い
- 試合に慣れると①が「自動認識」になる → ②以降に意識を使える

## プレッシャー下での判断フロー

DFが詰めてきたとき（時間がなくなるとき）の判断フロー：

```
DFが急に詰めてきた
│
├─ シュートが打てる距離・角度か？
│   YES → シュートフェイントを入れてシュート or ドライブ
│   NO  ↓
│
├─ ピボットへのパスコースはあるか？
│   YES → ピボットへ即パス（DFが詰めた背後が空いているはず）
│   NO  ↓
│
├─ 隣へのパスコースはあるか？
│   YES → 素早くはたく
│   NO  ↓
│
└─ ドリブルで時間を作る（コントロールドリブル）→ 再び①へ
```

**よくある認知・判断ミス（5パターン）**：

| ミス | 原因 | 修正 |
|---|---|---|
| ボールだけを見ている | 視野が狭い | 受ける前から周りを見る習慣 |
| 受けてから考える | 事前確認ゼロ | 隣がボールを持った瞬間から確認開始 |
| 第一選択に固執する | オートマティズムが硬直 | 第一選択が使えない場合の第二選択を決めておく |
| DFの反応を待たない | フェイントを使わずに仕掛ける | 相手が動いた後に動く（先に動くのはDF） |
| 時間をかけすぎる | 良い選択肢を探し続ける | 「2秒以内に最善を選ぶ」より「1秒以内にそこそこ良い選択をする」の方が有効 |

## 「2秒の壁」（個人判断の時間基準）

ボールを受けてから2秒以内に判断できているかを試合後に振り返る基準。
（※03_numerical.md・16_of_combinations.mdの「速攻判断2秒ルール」は別概念：チーム全員が「速攻か遅攻か」を把握する時間）

```
0〜0.5秒：受ける → 第一確認（DFの位置）
0.5〜1秒：フェイント or パス or シュートの選択
1〜1.5秒：実行開始
1.5〜2秒：実行完了（シュート or パス or ドライブ）

2秒を超える → DFが体勢を整えてしまう → 仕掛けの効果が半減
```

## エコロジカルアプローチ

> 練習は試合に近い状況（アフォーダンス）で行う

### 原則
- 試合で起こる状況を練習で再現する
- DFがいない状態での練習は試合に直結しない
- 状況判断ごと練習する（シュート練習ではなくシュート判断練習）

### アフォーダンスの活用
- 環境が「何ができるか」を教えてくれる
- クローズDFを見たら自然に外が見える → クローズを認識したら外へ
- ワイドDFを見たら自然に内が見える → ワイドを認識したら内へ

## 声掛け

### 声掛けの種類と目的
| 種類 | 内容 | タイミング |
|---|---|---|
| 状況報告 | 「右！」「後ろ！」「フリー！」 | 味方が見えていない情報を伝える |
| 指示 | 「ここへ！」「パス！」「打て！」 | 次の行動を指定する |
| 確認 | 「俺マーク！」「スイッチ！」 | DFの分担を確認する |
| 励まし | 「ナイス！」「大丈夫！」 | モチベーション維持 |

### 声掛けのじゃんけん
```
声を出す選手（情報共有）
  → チーム全体の認知速度UP → 判断速度UP → 組織プレーが強くなる
  → 声を出さない選手（個人判断）
  → 個人の判断は速いが連携ミスが増える
  → 声を出す選手が正確に伝えられないときは情報ノイズになる
  → 声の質と量のバランスが重要
```

### バックプレーヤーが出すべき声の優先順位
1. DFの位置報告（「クローズ！」「ワイド！」）
2. フリーの選手への呼びかけ（「来い！」）
3. DFのマーク確認（「俺が見る！」）
4. GKへの指示（「右！」「低い！」）

## フィールドプレーヤーの視野の広さ研究

> **前提**：GKのQuiet Eye研究（→ 05_gk_df.md）は「的を狙う最終固視」の最適化だが、フィールドプレーヤーの視野研究は「動く複数の対象を並行して処理する」点で根本的に目的が異なる。

### エリート選手のEye-tracking研究で分かっていること

ハンドボールのフィールドプレーヤーを直接対象としたEye-tracking研究は2026年時点でも少なく、サッカー・バスケットボールからの転用知見が主体となっている。確認できた研究の概要：

**ハンドボール選手への直接研究**
- ストロボスコープトレーニング研究（エリートハンドボール選手22名対象、6週間介入）：訓練群は脳が光に反応するまでの時間（P100潜時＝視覚情報を処理する速さの指標）が短縮し、特に「中心窩外視野（視野の中心から外れた周辺部分の見え方）」での初期の視覚処理が改善した（PMC10875695）。これは周辺視野の神経処理速度が訓練可能であることを示す。
- GKのEye-tracking研究：男女GKの間で固視部位・固視回数に有意差あり（→ 05_gk_df.md 参照）。フィールドプレーヤーを対象とした同等の研究は現時点では未確認。⚠️確認推奨

**サッカー・バスケへの転用知見（チームスポーツ共通として適用可）**
- エリート選手は非エリートより「関連情報へのサッカード（視線移動）」が多く「不要な固視時間」が短い
- スキャニング行動（頭の向きを変えて視野を確認する動作）はパスの前後でエリートほど頻度が高い
- バックプレーヤーはウィングより「画像能力（Imagery ability）」スコアが有意に高い（ポジション特性としての認知負荷の高さを示す）
- メンタルファティーグ（SNS使用・Stroop課題後）はビジュアル反応時間を遅延させるが、視覚探索行動パターン自体は比較的安定する（Sports 2025研究）

**現時点での定量目安（確認された数値）⚠️確認推奨**
- サッカーの実験データ：エリートの1回の固視時間は非エリートより短い傾向（固視時間が短い＝より頻繁に視線を動かして広域情報を取得している）
- 「ハンドボールバックプレーヤーの固視時間○ms」という直接測定値は査読論文では現時点で未確認

### GKのQuiet Eye研究とフィールドプレーヤーの違い

| 比較軸 | GK（Quiet Eye）| フィールドプレーヤー |
|---|---|---|
| 視線の目的 | 特定目標への最終固視（的を絞る）| 複数対象の同時並行処理 |
| 最適化方向 | 固視を「長く・安定させる」 | 固視を「短く・多頻度に」する |
| 課題構造 | クローズドスキル（シュートへの反応）| オープンスキル（流動的判断） |
| 訓練の焦点 | Quiet Eye期間の延長 | 視野スキャン頻度・周辺視野の活用 |
| 該当研究 | GKペナルティ研究（05_gk_df.md）| フィールドスキャニング研究（限定的）|

### 周辺視野（Peripheral Vision）の訓練可能性

**研究が示すコンセンサス（2026年時点）**：

1. **訓練可能性：確立**（スポーツ科学のコンセンサス）
   - スポーツ視覚訓練は周辺視野の反応時間を有意に改善する（介入群と対照群で有意差を確認）
   - ハンドボール・バスケットボール・バレーボール選手を対象にFitLightを用いた研究（PMC10294859）で、周辺視野の手動反応時間の改善を確認
   - 6週間のストロボスコープ訓練で中心窩外視野の神経処理速度が改善（P100潜時短縮）

2. **先天性 vs 後天性の現時点でのコンセンサス**：
   - 「遺伝的ベースラインは存在する」：一部の選手は生まれつきより鋭い周辺視野を持つ（遺伝的差異は実在）
   - 「ただし遺伝的基盤は上限ではなく基線」：訓練によって有意な改善が可能であることが複数研究で確認
   - 「アスリートは非アスリートより認識野が広い」：これは訓練効果か選択効果かは確定していない ⚠️確認推奨
   - **結論**：「視野の広さは後天的に訓練可能」が現在の実践的コンセンサス。先天的優位は存在するが、訓練効果がそれを補完・拡張できる範囲内にある。

3. **改善の時間軸**：
   - 短期（2〜4週）：一定の知覚改善が観察されるケースあり
   - 長期（3〜6ヶ月）：神経適応として定着するとされるが、個人差大
   - ストロボスコープ研究（6週間）：介入直後に改善、4週後のフォローアップでは「短期条件での改善の方が長期より顕著」

### デンマーク・ノルウェーのスポーツ科学での知見

- GKのQuiet Eye・Gaze研究（ノルウェー、Landin他）：→ 05_gk_df.md に詳述
- フィールドプレーヤー特化の視野研究としてデンマーク・ノルウェーの機関が出した論文は現時点では未確認 ⚠️確認推奨

### 実践への示唆（研究ベース）

```
「視野を広げる」の神経科学的内訳：
  → 固視時間を短く保つ（1か所に長く注視しない）

## 1v1で「止まる」がDFのリアクションタイムを突く理由

> 関連：02_individual.md「スピードフェイント」と本セクションの違いは最後の比較表を参照。

### 動く刺激 vs 静止刺激への反応時間：神経科学的根拠

人間の視覚神経系は「動いているもの」に対して自動的に高い注意資源を割り当てる。これは動体検出が生存に直結するため進化的に優先されている機構（マグノセルラー系：動き・空間周波数処理）。

**重要な逆説**：

| 状況 | DFの処理 | 結果 |
|---|---|---|
| OFが動き続けている | 動体として処理、次の動きを予測し続ける | DFは常に「準備状態」を維持できる |
| OFが突然止まる | 動体検出シグナルが消える、予測モデルが失立する | DFは次の行動準備の根拠を失う |
| OFが止まった後に再加速 | 新たな動体として再検出、しかし反応には遅延が発生 | DFの出足が遅れる |

- 止まった後の再加速に対するDFの反応時間は「最初から動いている刺激への反応」より遅延する：脳が動体→静止→動体という状態変化を処理する分のコストが加算される
- テーブルテニス（卓球）の研究では、速く動く対象に対する反応時間が258ミリ秒（ms＝1000分の1秒）から232ミリ秒へと速くなったが、これは対象がすでに動いていることが前提。突然の停止はこの準備状態を破壊する。

### アフォーダンス理論：「次の動き」の情報が消えるとDFの行動準備が阻害される

エコロジカルアプローチにおけるアフォーダンス（J.J.Gibson）の観点：

```
アフォーダンス（行動の機会）は連続的に生成・消滅する：
  → OFが右に動く → DFに「右スペースを閉じる」アフォーダンスが生まれる
  → OFが止まる  → そのアフォーダンスが消滅する
  → DFは準備した動作の「根拠」を失い、行動が中断・保留される
  → OFが再加速（逆方向へ）→ 新たなアフォーダンスが生まれるが、DFには再処理の遅延がある
```

**研究が示すメカニズム（バスケットボールの「Pump Fake」研究から転用）**：
- 「point of no return（PNR）」：一度開始された防御反応は特定の時点を超えると中止できない（PLOS ONE 2025研究、PMC12614604）
- ボールを胸の高さまで上げるだけでDFが（打たれると思い込んで飛ぶなど）引っかかった反応を開始する確率が50%に達し、頭上まで上げるとDFはその反応をほぼ止められない
- NBAプロ選手でもPump Fakeに約73%の確率で引っかかる
- 急停止も同様に「一度開始されたDFの前進・接近動作」の中断をPNRを利用して誘発できる ⚠️確認推奨（ハンドボール直接研究ではなくバスケ研究からの類推）

### 「急停止→再加速」の効果メカニズムと02_individual.mdのスピードフェイントとの違い

**02_individual.mdのスピードフェイント（参照）**：
- 「ドリブルのリズム変化でDFを崩す」：速度の連続的な変調
- DFは「速度が変わった」という情報は受け取るが、動体としての知覚は継続している
- 効果の中心：DFの加速・減速へのリアクション遅延

**本セクションの「急停止→再加速」**：

| 比較軸 | スピードフェイント（02） | 急停止→再加速 |
|---|---|---|
| DFの知覚状態 | 動体として処理継続 | 静止→動体への「状態変化検出」が必要 |
| アフォーダンス | 修正されるが消えない | いったん消滅→再生成 |
| 神経処理コスト | 速度変化の補正 | 状態変化の再検出＋新方向の処理 |
| 有効な距離帯 | ドリブル中・接触前 | 接触寸前・DFが前に出てきた瞬間 |
| 効果のピーク | DFが追いきれなくなるとき | DFが前に出た足をついた瞬間 |
| 前提条件 | ドリブルスキル | 「止まる」決断と姿勢制御 |

**急停止の神経科学的根拠補足**：
- DFが接近中に突然止まると、DFは「慣性で前に出てしまう」物理的問題に加え、「反応を止めるための抑制信号（Stop Signal）」を出す必要がある
- Stop Signal Reaction Time（SSRT）：進行中の動作を停止するための処理時間は、単純反応時間より長い（通常200ms以上）
- つまり「止まる」はDFに「2つの処理」を同時に要求する：①接近動作の停止、②次の行動の再計画

### 実践への統合（まとめ）

```
「止まる」の使い方：
  1. DFが接近してきた → まず止まる（アフォーダンスを消す）
