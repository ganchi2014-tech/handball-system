# メンタルマネジメント・自己診断

> 試合中の不安・緊張・ミス後の立て直し。ノルウェー語のメンタル5概念を核に構成。

---

## メンタルの5要素（ノルウェー語 NHF原典）

| ノルウェー語 | 日本語 | 内容 |
|---|---|---|
| **Mestring** | 習得・達成感 | 「できた」という実感がモチベーションの源泉 |
| **Selvtillit** | 自信 | DFに仕掛ける・GKが飛び込む判断力の基盤 |
| **Visualisering** | イメージトレーニング | プレー前に成功シーンを頭の中で描く |
| **Indre dialog** | 内的対話（セルフトーク） | 「できる」「次はこうする」の自己言語化 |
| **Triggere** | メンタルトリガー | 集中を高める特定の合図・ルーティン |

---

## 試合前のメンタル準備

### Visualisering（イメージトレーニング）

```
試合の前夜・当日朝・ウォーミングアップ中に行う

① 自分が成功するシーンを具体的に描く
   例：「右バックから廻旋フェイント → 内突破 → ループシュートが決まる」
② 視覚だけでなく、聴覚（声援・ボールの音）・触覚（ボールの感触）も含める
③ 失敗するシーンは描かない（脳は区別しない）
```

**研究的根拠**：エリート選手の93%がイメージトレーニングを習慣化（複数研究）。
ただし**具体性が低いイメージ**（「頑張る」「勝つ」）は効果薄い。**動作・感覚レベルの具体性**が必要。

---

### Triggere（メンタルトリガー）の作り方

```
試合前・プレー前に「集中スイッチを入れる」自分だけの合図

例：
- ボールを持つ前に右手でボールを2回叩く
- フリースロー前に深呼吸を3回
- タイムアウト後に味方とハイタッチ
- ベンチに座るときに右足から座る

ポイント：同じ動作を毎回繰り返す
→ パブロフ的条件反射で「この動作 = 集中モード」が脳に刷り込まれる
```

**注意**：迷信的になりすぎない（「やらないと負ける」と思うとプレッシャーが増える）。

## 試合中のメンタル管理

### ミス直後の立て直し（3秒ルール）

```
ミスした瞬間：
0〜1秒：自分のミスを認識する（隠さない・否定しない）
1〜2秒：Indre dialog（内的対話）で「次は○○する」と言う
       NG：「あー」「最悪」（過去を引きずる）
       OK：「次はピボットを見る」（未来に向ける）
2〜3秒：Triggere発動（深呼吸・小さなジェスチャー）→ 集中再起動
3秒以降：次のプレーへ完全に切り替え
```

**長く引きずるミス vs すぐに切り替えるミス**：
- 引きずる選手：1つのミスから3〜5プレー連続でミスが続く（メンタル連鎖）
- 切り替える選手：次のプレーで小さな成功 → Mestring（達成感）→ Selvtillit回復

### プレッシャー下の集中

**プレッシャーが高い状況の典型**：
- 残り2分・1点差
- 7m投擲
- タイムアウト後の1本目
- 5連続失点で味方が崩れている
- 大事な試合の開始時（自分への期待）

**プレッシャー下の認知低下**：
- 視野が狭くなる（中央集中視野になる）
- 判断時間が長くなる（情報処理が遅れる）
- 体が硬くなる（フェイントが効かない）

**対処法**：
```
① 呼吸を深くする（深呼吸4回）→ 副交感神経が働き心拍数下がる
② 視野を意識的に広げる（周辺視野を取り戻す）
③ 単純化する：「3つの選択肢から選ぶ」ではなく「最初の選択肢を実行」
④ Indre dialogを使う：「やれることをやる」「次の1プレーだけ」
```

---

### 接戦終盤の心理

**「終盤崩れ型」選手の特徴**：
- 残り時間ばかり気にする
- スコアボードを何度も見る
- 過去のプレーを後悔する
- 「今打たないと」という焦り

**「終盤強い」選手の特徴**：
- 「次の1プレー」だけに集中
- 残り時間を見ない（味方やコーチに任せる）
- ミスをすぐ手放す
- 「自分のシュート」を信じる（Selvtillit）

## メンタル状態の自己診断

### 試合中の自分の状態をチェック

```
Q1: 今、視野は広いか？
  Yes → 集中できている
  No  → プレッシャーで狭くなっている → 深呼吸・視野広げる

Q2: 判断は速くなっているか？
  Yes → リズムに乗れている
  No  → 考えすぎ → 第一選択を即実行する意識

Q3: ミスを引きずっているか？
  Yes → 3秒ルールを思い出す・Triggere発動
  No  → 集中継続

Q4: 体は硬いか？
  Yes → ジャンプ・スキップで体ほぐす
  No  → 良好

Q5: 自信はあるか？
  Yes → 仕掛ける勇気あり
  No  → Mestring（小さな成功）を作る：簡単なパスから始める
```

## 自分のレベル診断（技術・戦術・メンタル3軸）

### 技術レベル診断

```
【シュート】
□ レベル1: ジャンプシュートが安定して打てる
□ レベル2: 早打ち・最高点・下降中の打点を使い分けられる
□ レベル3: バックハンドフェイク・回転シュートが試合で使える
□ レベル4: GKタイプを見抜いて最適な打点・コースを選べる

【フェイント】
□ レベル1: 1〜2種類のフェイントを試合で使える
□ レベル2: 3〜5種類のフェイントを場面で使い分けられる
□ レベル3: ダブルフェイント原則（2種連続）・廻旋フェイントを実戦で使える
□ レベル4: DFの「何を見ているか」を読んで対応フェイントを選べる

【パス・ドリブル】
□ レベル1: 両手キャッチが安定。ワンハンドキャッチも可
□ レベル2: バウンスパス・ロブパス・ノールックパスが使える
□ レベル3: ドリブル中のフェイント（ジャブ・ヘジテーション等）が使える
□ レベル4: 受ける前の勝ち位置取りが意識せずできる
```

### 戦術レベル診断

```
【DF読み】
□ レベル1: クローズ/ワイドの違いがわかる
□ レベル2: 0-6/5-1/マンツーマンを見分けられる
□ レベル3: Verschieben/Herausrücken/Übergebenの動きを読める
□ レベル4: DFタイプから最適なOFアクションを選べる

【GK読み】
□ レベル1: GKが動くか動かないかを観察できる
□ レベル2: 先読み型/コース絞り型を見分けられる
□ レベル3: GKタイプから打点・コースを選べる
□ レベル4: 試合中にGKタイプの変化を読み取って対応できる

【連携】
□ レベル1: ピボットへのパスが出せる
□ レベル2: ギブ&ゴー・ピック&ロールが使える
□ レベル3: クロス・3人ウェーブ等のコンビネーションが使える
□ レベル4: チームのOFタイプを理解して自分の役割を果たせる
```

### メンタルレベル診断

```
【プレッシャー耐性】
□ レベル1: 普通の場面で集中できる
□ レベル2: ミス後3秒で切り替えられる
□ レベル3: 接戦終盤でも視野が狭くならない
□ レベル4: プレッシャーをむしろ集中の源にできる

【ルーティン】
□ レベル1: Triggereを持っている（特定の合図）
□ レベル2: Visualiseringを試合前に行う
□ レベル3: Indre dialogでセルフトークを使える
□ レベル4: Mestring（達成感）→ Selvtillit（自信）の循環を意識して作れる
```

## メンタルじゃんけん

```
プレッシャー高い
  → 視野狭くなる・判断遅れる
  → 深呼吸・第一選択即実行で対応
  → 成功 → Mestring → Selvtillit回復
  → 余裕が出る → プレッシャー耐性アップ

ミスする
  → 3秒以内に切り替え
  → Indre dialog「次は○○」
  → 次のプレーで小さな成功
  → Mestring回復
  → ミス連鎖を断ち切れる

接戦終盤
  → 残り時間気になる
  → 「次の1プレーだけ」に意識を絞る
  → ミスしても引きずらない
  → 終盤強い選手になる
```
