# コンビネーション・OFタイプ・攻撃トリガー辞書

---

## 1. 攻撃のきっかけ（アタックトリガー）

攻撃は「なんとなく始まる」ものではない。**必ずきっかけ（トリガー）がある**。トリガーを把握することで、次のアクションを先読みできる。

### 速攻トリガー（ファスト（速い） Break Triggers）

| トリガー | 説明 | 理想の第1アクション |
|---|---|---|
| GKセーブ直後 | GKがボールを持った瞬間 | 前線2人がすぐにスプリント |
| 相手シュートのゴールアウト | GKがスローに移行する時間がある | 速攻か遅攻かをGKと合図 |
| インターセプト（カット） | 相手パスをカット | カットした選手がそのまま速攻ボールホルダーに |
| 相手のパスアウト | 相手がラインを越えてボールを出した | スローワーが即パスで速攻開始 |
| 7mスロー後の即時移行 | 7mが決まった/外れた直後 | GKが素早くスロー、前線はすでに走り出している |
| 相手のパッシブプレー（攻めが遅すぎる反則）でボール没収 | 相手が攻めきれずボールを奪われた | 先手で前線に送り込む |

**速攻判断2秒ルール**：
```
GKがボールを持ってから2秒以内に「速攻か遅攻か」を全員が把握する必要がある
 → 2秒を超えるとDFが整い始める → 速攻のメリットが薄れる
```
（※04_cognition.mdの「2秒の壁」は別概念：個人がボールを受けてからの判断時間）

---

### セットOF移行トリガー（Set Offense Triggers）

| トリガー | 次のアクション |
|---|---|
| 速攻が止まった（DFが数人戻った） | ボールを回しながらセットOF構築 |
| ファウルをもらってフリースロー | フリースローポジションからセットプレー起点 |
| タイムアウト明け | 事前に決めたサインプレーを実行 |
| 相手交代・サブ（選手交代）直後 | 新しく入ったDFがまだ味方や状況を把握できていないうちに攻める |
| DFが疲弊しているハーフ後半 | 意図的にポゼッション維持でさらに体力を削る |

---

### インプレートリガー（プレー中のきっかけ）

これが最も重要。**試合中にリアルタイムで「今だ」と判断するきっかけ**。

| トリガー | 意味 | 有効なアクション |
|---|---|---|
| クローズDFを確認 | 相手DFが内側（ゴール寄り）に立っている | 外展開・外シュート・サイドパス |
| ワイドDFを確認 | 相手DFが外側（サイドライン寄り）に立っている | 内突破・ドリブル突破・ピボットパス |
| ピボットがポジション取りに成功 | ピボットが6mライン上でDFの前に入った | 即ピボットパス |
| DFのシフト直後 | フェアシーベン（シフト）（横移動）が完了した瞬間 | 逆サイドへ素早い展開 |
| ヘラウスリュッケン（前飛び出し）の背後にスペース | DFが前に出た | ピボットをその背後に走らせる |
| GKが重心を移動した | GKが左右どちらかに動いた | 反対コースへのシュート |
| ウィングが1on1で有利 | ウィングのDFが疲れている・ズレている | ウィングへのパス・クロスアタック |
| 相手DF同士のコミュニケーションが乱れた | DF陣がバタバタしている（交代・混乱） | 素早いテンポで攻める |

## 2. 自チームOFタイプ分類

**「自分たちは何で点を取るチームか」を知ることが戦術選択の出発点。**

### OFタイプ5分類

| タイプ | 特徴 | 強み | 弱み | 主な国・学派 |
|---|---|---|---|---|
| **速攻特化型** | 速攻で点を稼ぐ。セット攻撃は最小限 | DFが整う前に決める | DFが整うと得点力が落ちる | デンマーク・ノルウェー |
| **ポゼッション型** | ゆっくりコースを作ってから打つ | 確率の高いシュートを選択 | テンポが落ちるとパッシブ警告 | スペイン・ドイツ（一部） |
| **バック中心型** | バックプレーヤーの外シュート・突破が軸 | 外からのシュートパワー | ピボットが孤立しやすい | フランス・ロシア/ソビエト学派 |
| **ピボット中心型** | ピボットへのパスを経由して崩す | ゴール正面の密集を突ける | ピボット優秀者がいないと機能しない | クロアチア・スロベニア |
| **ウィング中心型** | ウィングの1on1とコーナーシュートが主得点源 | サイドの得点力が高い | DF2人を内側に引き付けないとウィングは潰される | アイスランド（サイドを活かす型） |

---

### 自チームOFタイプ × 相手DFタイプ 対応マトリクス

**行 = 自チームのOFタイプ ／ 列 = 相手のDF陣形**

| 自チームOF ↓ / 相手DF → | **0-6** | **5-1** | **2-4** | **3-2-1** | **マンツーマン** |
|---|---|---|---|---|---|
| **速攻特化型** | 積極的外シュートでラインを引き出す | 前衛1人を左右に動かして疲弊させる | 横の素早い展開でDF間のズレを使う | DFラインが薄い → カウンター狙い | スペースカット・1on1勝負 |
| **ポゼッション型** | 焦らず外→サイド→ピボットと回してコースを作る | ピボット経由でセンターポジションへ侵入 | サイド展開してウィングのコーナーシュート | ポゼッションで相手を引き付けてからピボット | スクリーン多用・ミスマッチ作り |
| **バック中心型** | 外シュートを積極的に打って0-6ラインを引き出す | クロスプレーで前衛1人を動かす | バックの突破力でDFを個人勝負 | バックが縦の分断スペースに切り込む | バック同士の1on1勝負 |
| **ピボット中心型** | ピボットへパス → DFの崩れを見て2次攻撃 | ピボットが前衛1人を引き付け → 背後のスペースへ | ピボットがDF2人の間に割り込む | ピボットが縦スペースに走り込む | ピボットがスクリーナーを兼任 |
| **ウィング中心型** | サイドへの展開を素早くしてウィングのコーナーを狙う | 前衛が内側を塞ぐ → 逆にサイドが空く | ウィングへの早いパスでコーナー攻略 | 縦のスペースを活かしてウィングを走らせる | ウィングが1on1で相手を剥がす |

> **じゃんけん原則**：マトリクスは「この組み合わせなら必ず勝てる」ではない。相手が対応してくれば対応の対応が必要になる。

## 3. コンビネーションプレー（2人）

### ギブ＆ゴー（ギブ & Go）

```
A → B へパス
A がそのままゴール方向に走り込む（「ゴー」）
B → A へリターンパス
A がシュート or ドリブル突破
```
- **きっかけ**：Aのマーカーが一瞬パスボールを追った瞬間
- **有効DFタイプ**：マンツーマン・5-1（マーカーが一瞬遅れる）
- **失敗パターン**：AがBにパスした後、走り込まずに止まる → せっかくのDFの遅れを無駄にする

---

### ピック＆ロール（Pick & Roll）

```
ピボット（スクリーナー）がバックのDFの前に立つ（ピック）
バックがピボットの陰から抜け出す
ピボットがゴール方向に転身（ロール）してパスを受ける
```
- **きっかけ**：ピボットがスクリーンセットを完了した瞬間
- **有効DFタイプ**：マンツーマン・0-6（ゾーンにはスイッチされにくい）
- **DFの対策**：スイッチ → ミスマッチが生まれる → ミスマッチを活用するのがOFの狙い

---

### クロス（Cross / クロイツ（クロス）ung / クリジャニェ（クロス））

```
AとBが逆方向に走り込む（X字の動き）
どちらにパスが行くか → DFのマーク交換を強制する
交差後に有利なポジションの選手がシュート
```
- **きっかけ**：2人のDFが互いに「誰が追うか」を判断している一瞬
- **有効DFタイプ**：マンツーマン・5-1（交差でマークが乱れる）
- **クロアチア語**：Križanje（クリジャニェ）
- **ドイツ語**：Kreuzung（クロイツング）

---

### バックドアカット（Back Door Cut）

```
AがDFの前でボールを受けようとするふりをする
DFがボールへの注意を向けた瞬間
AがDFの背後（ゴール方向）に急加速して抜ける
Bがロブパスまたはバウンドパスで送る
```
- **きっかけ**：DFがボールを見てAへの注意が緩んだ瞬間
- **有効DFタイプ**：マンツーマン（バックドアはゾーンには効きにくい）
- **補足**：DFが詰めてきた（クローズ）ときこそ使えるカウンターアクション

---

### ハンドオフ（Hand Off / アプガーベ（パス放出））

```
AがDFを引き付けながらBとすれ違う
すれ違う瞬間にAがBにボールを「渡す」（投げるのではなく手渡し）
BがそのままDFを置き去りにして突破
```
- **きっかけ**：AとBが接近して動くときのDF同士の「誰がどっちを追う？」の混乱
- **有効DFタイプ**：マンツーマン・狭い場面での局地戦
- **ドイツ語**：アプガーベ（パス放出）

---

### ブロック＆カット（ブロック & Cut）

```
Aがスクリーナーとして前に立つ（ブロック）
BがAのブロックの陰からカットイン
Aは静止したままスクリーンを完成させる
```
- スクリーン技術の詳細は `dictionary/12_set_plays.md` 参照
- **ここで補足**：スクリーン後のロールがないため「A」は突破の選択肢がない
  → A自身のシュートを狙わない場面で使う（A＝献身的なスクリーナー）

## 4. コンビネーションプレー（3人）

### ウェーブ（Wave Attack）

```
3人が波状に順番に仕掛ける（A→B→C の順）
最初の仕掛けでDFを引き付け
次の選手が空いたスペースに走り込む
```

| ウェーブ段階 | 動き |
|---|---|
| 1st wave | AがDFに対してドライブを仕掛けDFを引きつける |
| 2nd wave | BがAのドライブで空いたスペースにパスを受けに動く |
| 3rd wave | CはBのアクションに対応したDFの背後をカット |

- **スペイン語**：Primera/Segunda/Tercera ola（波）
- **有効DFタイプ**：0-6（シフトが遅れるとどんどん波に対応できなくなる）

---

### トライアングル（Triangle Offense / ドライ（独：3）eck）

```
3人が三角形を形成してポジションを維持
ボール保持者が頂点 → 残り2人が底辺の左右
パスが通ったら全員がポジションをシフト
```
- **目的**：DFが集中しにくい三角形の「死角」を常に作る
- **ドイツ語**：Dreieck（ドライエック）
- **有効DFタイプ**：ゾーンDF全般（特に0-6の穴を探す）

---

### ダブルスクリーン（ダブルスクリーン / ドッペルブロカーデ（ダブルブロック））

```
2人が並んで壁を作る
エース（3人目）がその壁の陰からカットインまたはポップアウト
```
- **使用タイミング**：フリースロー後のセットプレー・試合終盤の1点勝負
- **DFの対策**：スイッチ → ミスマッチが生まれる（それでもOFが有利になるのが理想）
- **弱点**：3人全員の動きが合わないとエースのフリーが一瞬しか続かない

---

### 3人クロスプレー（Triple Cross / Trojno クリジャニェ（クロス））

```
3人が連続してX字に動き、DFのマーク整理を不可能にする
1回目のクロスでDFが混乱 → 2回目のクロスで完全に乱す
```
- **クロアチア語**：Trojno križanje（トロイノ・クリジャニェ）
- **難度**：3人の動線と優先順位の事前共有が必須
- **有効DFタイプ**：マンツーマン（ゾーンDFには効きにくい）

## 7. じゃんけん（コンビネーション視点）

```
ギブ＆ゴー → DFがスイッチで対応 → ミスマッチ発生
 → ミスマッチのあるプレーヤーを攻める → DFがヘルプに来る → ヘルプが空けたスペースにスリーマンプレー

クロスプレー → DFがスイッチで対応 → どちらにパスが行くか分散 → 早いテンポで2回連続クロス → DFが混乱

ピック＆ロール → DFがハードブリッジ（スクリーナーを押して妨害） → ポップアウトでシュートチャンス
 → DFがポップアウトに対応 → ロールに戻る → DFがロールを追う → ポップアウトに戻る

ウェーブ → DFが第1波に対応 → 第2波 → DFが追いつかない → 第3波で決める
 → DFが素早くシフト → ウェーブを逆手に取ったカウンター（DFが前に出た背後を突く）
```

## 8. OFタイプ自己診断チェックリスト

**「自分のチームはどのタイプか」を確認する質問：**

```
Q1: チームの主な得点源はどれか？
  → 速攻が多い           → 速攻特化型
  → バックの外シュートが多い → バック中心型
  → ピボット経由が多い    → ピボット中心型
  → ウィングのコーナーが多い → ウィング中心型
  → ゆっくり崩してから打つ → ポゼッション型

Q2: 相手DFが整った後（セット状態）でも得点できているか？
  → できている  → ポゼッション型またはピボット中心型
  → 苦しくなる → 速攻特化型（速攻が止まると弱い）

Q3: DFがタイトに来るとどうなるか？
  → 外シュートで対応できる → バック中心型
  → 突破で対応できる      → バック中心型 or 速攻特化型
  → ピボットに逃がせる    → ピボット中心型

Q4: 個人の強みは何か？
  → バックの個人突破・シュート → バック中心型
  → ピボットの技術・フィジカル → ピボット中心型
  → ウィングのスピード・1on1 → ウィング中心型
```

> このOFタイプは固定ではなく、**試合の文脈・相手の対応によって切り替える**。
> 「速攻が止まれば即座にポゼッション型に切り替える」など、複数タイプを持つチームが強い。
